# 13-12. キク目2（アザミ亜科・キク亜科1）

## はじめに

本章では、キク科のアザミ亜科（管状花のみ）とキク亜科の一部（アキノキリンソウ連など）を解説します。

---

## 1. アザミ亜科 Carduoideae

### 特徴

| 特徴 | 内容 |
|------|------|
| 小花 | **管状花のみ**で構成 |
| 総苞片 | しばしば刺状 |
| 代表的な属 | アザミ属、ゴボウ属、ヤグルマギク属 |

### 日本産主要種（アザミ属 Cirsium）

アザミ属は日本で約150種と非常に多く、地域固有種が多い。

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ノアザミ](https://ja.wikipedia.org/wiki/ノアザミ) | *Cirsium japonicum* | 本州～九州 | **初夏に開花**（他のアザミは秋） |
| [ノハラアザミ](https://ja.wikipedia.org/wiki/ノハラアザミ) | *Cirsium oligophyllum* | 本州（中部以北） | 秋咲き、総苞片が開出 |
| [タイアザミ](https://ja.wikipedia.org/wiki/タイアザミ) | *Cirsium nipponicum* | 本州～九州 | 秋咲き、総苞片がやや反曲 |
| [フジアザミ](https://ja.wikipedia.org/wiki/フジアザミ) | *Cirsium purpuratum* | 本州（関東～中部） | 大型、頭花が下垂 |
| [キセルアザミ](https://ja.wikipedia.org/wiki/キセルアザミ) | *Cirsium sieboldii* | 本州～九州 | 湿地性、頭花が下垂 |
| [サワアザミ](https://ja.wikipedia.org/wiki/サワアザミ) | *Cirsium yezoense* | 北海道～本州 | 湿地性 |
| [オニアザミ](https://ja.wikipedia.org/wiki/オニアザミ) | *Cirsium borealinipponense* | 本州（日本海側） | 大型、刺が鋭い |
| [ダキバヒメアザミ](https://ja.wikipedia.org/wiki/ダキバヒメアザミ) | *Cirsium amplexifolium* | 本州 | 葉が茎を抱く |
| [アメリカオニアザミ](https://ja.wikipedia.org/wiki/アメリカオニアザミ) | *Cirsium vulgare* | 帰化 | 外来種、刺が強烈、特定外来生物 |

> 📝 **アザミ属の多様性**：日本固有種が多く、地域変異が大きい。同定は困難なことが多い。

### 日本産主要種（その他のアザミ亜科）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ゴボウ](https://ja.wikipedia.org/wiki/ゴボウ) | *Arctium lappa* | 栽培 | 根菜、総苞片が鉤状 |
| [ヤマゴボウ](https://ja.wikipedia.org/wiki/ヤマゴボウ) | - | - | ※漬物の「ヤマゴボウ」はモリアザミの根 |
| [モリアザミ](https://ja.wikipedia.org/wiki/モリアザミ) | *Cirsium dipsacolepis* | 本州～九州 | 根を「ヤマゴボウ」として食用 |
| [ヒレアザミ](https://ja.wikipedia.org/wiki/ヒレアザミ) | *Carduus crispus* | 帰化 | 茎に翼（ヒレ） |
| [タムラソウ](https://ja.wikipedia.org/wiki/タムラソウ) | *Serratula coronata* subsp. *insularis* | 北海道～九州 | アザミに似るが刺なし |
| [キツネアザミ](https://ja.wikipedia.org/wiki/キツネアザミ) | *Hemistepta lyrata* | 本州～沖縄 | 春咲き、葉が羽状に深裂 |
| [ヤグルマギク](https://ja.wikipedia.org/wiki/ヤグルマギク) | *Centaurea cyanus* | 帰化・栽培 | 青色の観賞用 |
| [オケラ](https://ja.wikipedia.org/wiki/オケラ) | *Atractylodes ovata* | 本州～九州 | 薬用、正月の屠蘇散 |
| [アーティチョーク](https://ja.wikipedia.org/wiki/アーティチョーク) | *Cynara scolymus* | 栽培 | 野菜（花蕾） |

> 📝 **タムラソウとアザミの違い**：タムラソウは刺がなく、アザミに似るが別属

## 2. キク亜科 Asteroideae（一部）

### 特徴

| 特徴 | 内容 |
|------|------|
| 小花 | 周辺に**舌状花** ＋ 中央に**管状花** |
| 代表的な属 | キク属、ヨモギ属、ヒマワリ属など |

## 確認問題

### 問1
アザミ亜科の小花の構成を述べよ。

<details>
<summary>解答を見る</summary>

**管状花のみ**で構成される

</details>

### 問2
日本のアザミ属で初夏（春～初夏）に開花する種は何か。

<details>
<summary>解答を見る</summary>

**ノアザミ**（他のアザミ属は主に秋に開花）

</details>

### 問3
春の七草の「ゴギョウ」は何科の何という植物か。

<details>
<summary>解答を見る</summary>

**キク科のハハコグサ**（*Pseudognaphalium affine*）

</details>

### 問4
特定外来生物に指定されているキク科植物を2つ挙げよ。

<details>
<summary>解答を見る</summary>

- **オオハンゴンソウ**
- **ブタクサ**
- **オオブタクサ**
- **アメリカオニアザミ**

などから2つ

</details>
