# 15-6. 水生シダ類

[← 目次に戻る](../NEW_00_目次.md) | [← 前：薄嚢シダ類3](./15-5_薄嚢シダ類3.md)

## はじめに

本章では、水生・湿生環境に適応した特殊なシダ類を解説します。これらは他のシダとは大きく異なる形態をもち、一見シダには見えないものも多いです。

---

## 水生シダ類の概要

### 特徴

| 項目 | 内容 |
|------|------|
| 生育環境 | 水中または水辺 |
| 形態 | 一般的なシダと大きく異なる |
| 胞子 | 多くが異型胞子性 |
| 分類 | 複数の系統に分かれる |

---

## 1. サンショウモ科 Salviniaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 2属 |
| 日本産種数 | 3種 |
| 世界の種数 | 約20種 |
| 特徴 | 浮遊性、異型胞子性 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 水面に浮遊 |
| 根 | なし、または水中葉が変形 |
| 葉 | 小型、浮葉 |
| 胞子 | 大胞子と小胞子（異型胞子性） |

### 日本産主要種

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [サンショウモ](https://ja.wikipedia.org/wiki/サンショウモ) | *Salvinia natans* | 本州～九州 | 浮遊性、葉が楕円形、絶滅危惧II類 |
| [オオサンショウモ](https://ja.wikipedia.org/wiki/オオサンショウモ) | *Salvinia molesta* | 帰化 | 大型、侵略的外来種 |
| [アカウキクサ](https://ja.wikipedia.org/wiki/アカウキクサ) | *Azolla imbricata* | 本州～九州 | 秋に赤くなる、窒素固定 |
| [オオアカウキクサ](https://ja.wikipedia.org/wiki/オオアカウキクサ) | *Azolla japonica* | 本州～九州 | 在来、絶滅危惧II類 |
| [アメリカオオアカウキクサ](https://ja.wikipedia.org/wiki/アメリカオオアカウキクサ) | *Azolla cristata* | 帰化 | 特定外来生物 |

### サンショウモの特徴

| 項目 | 内容 |
|------|------|
| 生活形 | 浮遊性の1年草 |
| 葉 | 3輪生（浮葉2枚+水中葉1枚） |
| 水中葉 | 根のように細裂、根の役割 |
| 名の由来 | 葉がサンショウ（山椒）に似る |

> 📝 **サンショウモの減少**：水田や池沼の減少により、在来のサンショウモは絶滅危惧種

### アカウキクサの特徴

| 項目 | 内容 |
|------|------|
| 生活形 | 浮遊性 |
| 大きさ | 数mm～1cm程度 |
| 共生 | [アナベナ](https://ja.wikipedia.org/wiki/ネンジュモ属)（窒素固定藍藻）と共生 |
| 農業利用 | 水田の緑肥、雑草抑制 |
| 色 | 秋に赤変（アントシアニン） |

> 📝 **窒素固定**：アカウキクサ属は藍藻と共生し窒素固定を行うため、稲作の緑肥として利用される

## 2. デンジソウ科 Marsileaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 2属 |
| 日本産種数 | 3種 |
| 世界の種数 | 約70種 |
| 特徴 | 四葉のクローバー状、異型胞子性 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 根茎 | 細く匍匐 |
| 葉 | 4小葉が掌状につく、または線形 |
| 胞子嚢果 | 堅い殻に包まれる |
| 生育地 | 水田、池沼の浅水域 |

### 日本産主要種

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [デンジソウ](https://ja.wikipedia.org/wiki/デンジソウ) | *Marsilea quadrifolia* | 本州～九州 | 四つ葉、水田・池沼、絶滅危惧II類 |
| [ナンゴクデンジソウ](https://ja.wikipedia.org/wiki/ナンゴクデンジソウ) | *Marsilea crenata* | 九州～沖縄 | 小葉の縁が波状 |
| [ミズニラモドキ](https://ja.wikipedia.org/wiki/ミズニラモドキ) | *Pilularia globulifera* | 本州 | 葉が線形、絶滅危惧IA類 |

### デンジソウの特徴

| 項目 | 内容 |
|------|------|
| 和名 | 「田字草」＝田の字に似た4小葉 |
| 生育環境 | 水田、池沼、休耕田 |
| 減少原因 | 除草剤、圃場整備 |
| 保全状況 | 絶滅危惧II類 |

> ⚠️ **水田雑草の減少**：デンジソウをはじめ、かつて水田で普通に見られたシダや水草が激減している

## 3. ミズワラビ科 Parkeriaceae（またはホウライシダ科に含める）

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 1属 |
| 日本産種数 | 1種 |
| 特徴 | 水生、1年草 |

### 日本産主要種

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ミズワラビ](https://ja.wikipedia.org/wiki/ミズワラビ) | *Ceratopteris thalictroides* | 本州～沖縄 | 水田、1年草、二形性 |

### ミズワラビの特徴

| 項目 | 内容 |
|------|------|
| 生活形 | 1年草の水生シダ |
| 栄養葉 | 軟質、広い |
| 胞子葉 | 細裂、縁が巻いてソーラスを覆う |
| 生育地 | 水田、湿地 |

---

## 水生シダの系統

### 分類上の位置

| 科名 | 系統 |
|------|------|
| サンショウモ科 | 薄嚢シダ類 |
| デンジソウ科 | 薄嚢シダ類 |
| ミズワラビ科 | 薄嚢シダ類（イノモトソウ科に近縁） |
| ミズニラ科 | 小葉植物（リコプシダ綱） |

> 📝 水生への適応は独立に複数回進化している

---

## 外来水生シダの問題

### 侵略的外来種

| 種名 | 原産地 | 問題点 |
|------|--------|--------|
| オオサンショウモ | 南米 | 水面を覆い在来種を圧迫 |
| アメリカオオアカウキクサ | 北米 | 在来のアカウキクサと競合 |

### オオサンショウモの特徴

| 項目 | 内容 |
|------|------|
| 繁殖 | 栄養繁殖のみ（胞子形成不能） |
| 成長 | 非常に速い |
| 被害 | 水路閉塞、在来種駆逐 |
| 対策 | 機械的除去、生物的防除 |

## 確認問題

### 問1
サンショウモ科とデンジソウ科に共通する胞子の特徴を述べよ。

<details>
<summary>解答を見る</summary>

**異型胞子性**（大胞子と小胞子を別々に作る）

</details>

### 問2
アカウキクサ属が共生している生物と、その生態学的意義を述べよ。

<details>
<summary>解答を見る</summary>

- **共生生物**：アナベナ（窒素固定藍藻）
- **意義**：窒素固定を行い、水田の緑肥として利用できる

</details>

### 問3
デンジソウの和名の由来を述べよ。

<details>
<summary>解答を見る</summary>

**「田字草」**＝4枚の小葉が「田」の字のように並ぶことから

</details>

### 問4
侵略的外来種として問題となっている水生シダを1種挙げよ。

<details>
<summary>解答を見る</summary>

**オオサンショウモ**または**アメリカオオアカウキクサ**

</details>
