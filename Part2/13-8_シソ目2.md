## はじめに

本章では、シソ目のうちオオバコ科、ハマウツボ科、ゴマノハグサ科など、APG体系で大きく再編成された科群を解説します。

## 1. オオバコ科 Plantaginaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約25属 |
| 日本産種数 | 約70種 |
| 世界の属数 | 約90属 |
| 世界の種数 | 約1,900種 |

> 📝 **分類の変更**：APG体系で旧ゴマノハグサ科から多くの属が移動し、大幅に拡大された

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 草本（まれに木本） |
| 花 | 合弁、4～5数性 |
| 花冠 | 仮面状、筒状、2唇形など多様 |
| 果実 | 蒴果が多い |

### モクセイ科 Oleaceae

| 項目 | 内容 |
|------|------|
| 日本産属数 | 5属 |
| 日本産種数 | 約20種 |

#### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 木本 |
| 葉 | 対生、単葉または複葉 |
| 花 | 合弁・放射相称・**4裂（二数性）** |
| **雄しべ** | **2本** ← 試験頻出！ |
| 子房 | 上位・2室 |
| 果実 | 核果・翼果・蒴果・液果 |

!!! warning "モクセイ科の識別ポイント：雄蕊2本（2024年度問47で出題）"
    「花は放射相称で二数性、萼と花冠は4裂し、雄蕊は2本である科」→ **モクセイ科**（2024年度問47 正解1）
    - **雄蕊2本**はモクセイ科の最重要識別点
    - 花冠4裂・合弁・放射相称
    - 他の科は一般に4〜5本が多い
    - また「唇形花冠ではない科」としても出題（2022年度問36）→モクセイ科の花冠は**放射相称の4裂**（唇形でない）

## 確認問題

### 問1
APG体系で旧ゴマノハグサ科から拡大したオオバコ科に移動した代表的な属を2つ挙げよ。

<details>
<summary>解答を見る</summary>

- **クワガタソウ属（Veronica）**
- **キンギョソウ属（Antirrhinum）**
- **ウンラン属（Linaria）**
- **ジギタリス属（Digitalis）**

などから2つ

</details>

### 問2
ハマウツボ科に含まれる寄生植物の生活形を2種類述べよ。

<details>
<summary>解答を見る</summary>

- **半寄生植物**（葉緑素あり、例：シオガマギク、ママコナ）
- **全寄生植物**（葉緑素なし、例：ハマウツボ、ナンバンギセル）

</details>

### 問3
モクセイ科の雄しべの数はいくつか。

<details>
<summary>解答を見る</summary>

**2本**

</details>

### 問4
タヌキモ科が属する生態的グループは何か。

<details>
<summary>解答を見る</summary>

**食虫植物**

</details>
