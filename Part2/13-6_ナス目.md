## はじめに

[ナス目](https://ja.wikipedia.org/wiki/ナス目)（Solanales）は、シソ類に属する目で、ナス科とヒルガオ科を含みます。トマト、ジャガイモ、サツマイモなど重要な食用植物を多く含む一方、有毒植物も多い目です。

---

## ナス目の概要

| 項目 | 内容 |
|------|------|
| 日本産科数 | 3科 |
| 世界の科数 | 5科 |
| 世界の種数 | 約4,000種 |
| 主な特徴 | 合弁花、アルカロイドを含むものが多い |

## 1. ナス科 Solanaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約15属 |
| 日本産種数 | 約40種 |
| 世界の属数 | 約100属 |
| 世界の種数 | 約2,500種 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 草本（まれに木本） |
| 葉 | 互生、単葉、托葉なし |
| 花 | 合弁、放射相称、5数性 |
| 花冠 | 車形、漏斗形、筒形など |
| 雄しべ | 5、花筒につく |
| 子房 | 上位、2室（まれに多室） |
| 果実 | 液果または蒴果 |
| 化学成分 | [アルカロイド](https://ja.wikipedia.org/wiki/アルカロイド)を含むものが多い |

### 日本産主要種（野生種）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [イヌホオズキ](https://ja.wikipedia.org/wiki/イヌホオズキ) | *Solanum nigrum* | 全国 | 白花、黒い実、有毒 |
| [アメリカイヌホオズキ](https://ja.wikipedia.org/wiki/アメリカイヌホオズキ) | *Solanum ptycanthum* | 帰化 | イヌホオズキに似る |
| [ヒヨドリジョウゴ](https://ja.wikipedia.org/wiki/ヒヨドリジョウゴ) | *Solanum lyratum* | 北海道～九州 | つる性、赤い実 |
| [ヤマホロシ](https://ja.wikipedia.org/wiki/ヤマホロシ) | *Solanum japonense* | 北海道～九州 | つる性、紫～白花 |
| [ワルナスビ](https://ja.wikipedia.org/wiki/ワルナスビ) | *Solanum carolinense* | 帰化 | 鋭い棘、繁殖力強い |
| [クコ](https://ja.wikipedia.org/wiki/クコ) | *Lycium chinense* | 本州～沖縄 | 薬用、果実・葉を食用 |
| [ホオズキ](https://ja.wikipedia.org/wiki/ホオズキ) | *Alkekengi officinarum* var. *franchetii* | 本州～九州 | 赤い袋状の萼 |

| 和名 | 学名 | 有毒成分 | 症状 |
|------|------|----------|------|
| [チョウセンアサガオ](https://ja.wikipedia.org/wiki/チョウセンアサガオ) | *Datura metel* | スコポラミン、アトロピン | 幻覚、瞳孔散大 |
| [ヨウシュチョウセンアサガオ](https://ja.wikipedia.org/wiki/ヨウシュチョウセンアサガオ) | *Datura stramonium* | スコポラミン | 同上 |

> - ジャガイモの芽や緑色部分には[ソラニン](https://ja.wikipedia.org/wiki/ソラニン)が含まれる

## 2. ヒルガオ科 Convolvulaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約10属 |
| 日本産種数 | 約30種 |
| 世界の属数 | 約60属 |
| 世界の種数 | 約1,900種 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | つる性草本（まれに木本、寄生植物） |
| 乳液 | 白い乳液を含むものが多い |
| 葉 | 互生、単葉 |
| 花 | 合弁、**漏斗形**、5数性 |
| 花冠 | つぼみで回旋状にねじれる |
| 雄しべ | 5、花筒につく |
| 子房 | 上位、2室 |
| 果実 | 蒴果 |

### 日本産主要種

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ヒルガオ](https://ja.wikipedia.org/wiki/ヒルガオ) | *Calystegia japonica* | 全国 | ピンク花、昼に開花 |
| [コヒルガオ](https://ja.wikipedia.org/wiki/コヒルガオ) | *Calystegia hederacea* | 全国 | ヒルガオより小型、葉の基部が角張る |
| [ハマヒルガオ](https://ja.wikipedia.org/wiki/ハマヒルガオ) | *Calystegia soldanella* | 全国 | 海岸、腎臓形の葉 |
| [セイヨウヒルガオ](https://ja.wikipedia.org/wiki/セイヨウヒルガオ) | *Convolvulus arvensis* | 帰化 | 小型、白～ピンク |
| [アサガオ](https://ja.wikipedia.org/wiki/アサガオ) | *Ipomoea nil* | 栽培 | 夏の風物詩、多彩な色 |
| [マルバアサガオ](https://ja.wikipedia.org/wiki/マルバアサガオ) | *Ipomoea purpurea* | 帰化 | 丸い葉、青～紫 |
| [ホシアサガオ](https://ja.wikipedia.org/wiki/ホシアサガオ) | *Ipomoea triloba* | 帰化 | 小型、淡紅色 |
| [マメアサガオ](https://ja.wikipedia.org/wiki/マメアサガオ) | *Ipomoea lacunosa* | 帰化 | 小型、白～淡紅色 |
| [サツマイモ](https://ja.wikipedia.org/wiki/サツマイモ) | *Ipomoea batatas* | 栽培 | 塊根が食用 |
| [クウシンサイ](https://ja.wikipedia.org/wiki/クウシンサイ) | *Ipomoea aquatica* | 栽培 | 空心菜、茎が中空 |
| [ネナシカズラ](https://ja.wikipedia.org/wiki/ネナシカズラ) | *Cuscuta japonica* | 北海道～九州 | 寄生植物、黄色い茎 |
| [アメリカネナシカズラ](https://ja.wikipedia.org/wiki/アメリカネナシカズラ) | *Cuscuta pentagona* | 帰化 | 寄生植物 |
| [グンバイヒルガオ](https://ja.wikipedia.org/wiki/グンバイヒルガオ) | *Ipomoea pes-caprae* | 九州～沖縄 | 海岸性、軍配型の葉 |

### ヒルガオとコヒルガオの違い

| 特徴 | ヒルガオ | コヒルガオ |
|------|----------|------------|
| 花の大きさ | 大型（5～6cm） | 小型（3～4cm） |
| 葉の基部 | 丸く張り出す | 角張る |
| 花柄 | 上部に翼なし | 上部に翼あり |

### ネナシカズラ属（寄生植物）

[ネナシカズラ属](https://ja.wikipedia.org/wiki/ネナシカズラ属)は葉緑素を持たない寄生植物です。

| 特徴 | 内容 |
|------|------|
| 生活形 | 寄生植物（全寄生） |
| 色 | 黄色～橙色（葉緑素なし） |
| 葉 | 退化して鱗片状 |
| 吸器 | 寄主の茎に食い込む |

## ナス科の主要属まとめ

### ナス属 Solanum

| 特徴 | 内容 |
|------|------|
| 花冠 | 車形、裂片が反り返る |
| 葯 | 直立して密着、頂孔開裂 |
| 代表種 | ナス、トマト、ジャガイモ |

### トウガラシ属 Capsicum

| 特徴 | 内容 |
|------|------|
| 花冠 | 車形、白色 |
| 果実 | 多室の液果、辛味成分（カプサイシン） |
| 代表種 | トウガラシ、ピーマン |

### チョウセンアサガオ属 Datura

| 特徴 | 内容 |
|------|------|
| 花冠 | 漏斗形、大型 |
| 果実 | 棘のある蒴果 |
| 毒性 | アルカロイド（スコポラミン等）含有 |

## 確認問題

### 問1
ナス科の花冠の特徴を述べよ。

<details>
<summary>解答を見る</summary>

- **合弁花**
- **5数性**
- **車形、漏斗形、筒形など多様**

</details>

### 問2
ジャガイモの芽に含まれる有毒成分は何か。

<details>
<summary>解答を見る</summary>

**ソラニン**（ステロイドアルカロイド配糖体）

</details>

### 問3
ヒルガオ科の花冠の特徴を述べよ。

<details>
<summary>解答を見る</summary>

- **合弁花**
- **漏斗形**
- **つぼみで回旋状にねじれる**

</details>

### 問4
ネナシカズラ属の生活形は何か。

<details>
<summary>解答を見る</summary>

**全寄生植物**（葉緑素を持たず、他の植物に寄生）

</details>
