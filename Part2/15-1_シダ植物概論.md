## はじめに

[シダ植物](https://ja.wikipedia.org/wiki/シダ植物)（羊歯植物、Pteridophytes）は、維管束をもつが種子を作らない植物群です。日本には約700種が自生し、世界的にも多様性が高い地域です。

---

## シダ植物の概要

### 基本情報

| 項目 | 内容 |
|------|------|
| 世界の種数 | 約12,000種 |
| 日本産種数 | 約700種 |
| 日本産科数 | 約30科 |
| 代表的な群 | 薄嚢シダ類（ウラボシ目など） |

### シダ植物の特徴

| 特徴 | 内容 |
|------|------|
| 維管束 | あり（木部・篩部） |
| 種子 | なし |
| 胞子 | 胞子で繁殖 |
| 世代交代 | 胞子体（2n）と配偶体（n）の明瞭な交代 |
| 受精 | 水を媒介とする（鞭毛をもつ精子） |

---

## シダ植物の形態

### 葉（羽片）の構造

| 部位 | 説明 |
|------|------|
| 葉身 | 展開した葉の部分 |
| 葉柄 | 根茎と葉身をつなぐ部分 |
| 中軸 | 葉身内の主軸 |
| 羽片 | 葉身が分岐した部分 |
| 裂片 | 羽片がさらに分岐した部分 |

### 葉の分裂

| 型 | 説明 | 例 |
|----|------|-----|
| 単葉 | 分裂しない | ヒトツバ、マメヅタ |
| 1回羽状 | 羽片に分かれる | イノデ、オシダ |
| 2回羽状 | 羽片がさらに分裂 | ワラビ |
| 3回羽状 | さらに細かく分裂 | シシガシラ |

### 葉の形態（新芽）

| 特徴 | 内容 |
|------|------|
| ぜんまい巻き | 新芽が渦巻き状に巻く（巻葉） |
| こごみ | 若い葉が巻いた状態（山菜として利用） |

### 胞子嚢群（ソーラス）

| 部位 | 説明 |
|------|------|
| 胞子嚢 | 胞子を作る袋状の構造 |
| 胞子嚢群（ソーラス） | 胞子嚢が集まったもの |
| 包膜 | ソーラスを覆う膜状の構造 |

### ソーラスの位置と形状

| 位置 | 例 |
|------|-----|
| 葉裏全面 | ヒトツバ |
| 羽片の縁 | イノモトソウ |
| 中肋沿い | シシガシラ |
| 脈上に散在 | オシダ |

| 形状 | 包膜の形 | 例 |
|------|----------|-----|
| 円形 | 円形～腎形 | オシダ、イノデ |
| 線形 | 線形 | シシガシラ |
| 馬蹄形 | 馬蹄形 | イヌワラビ |
| なし | 包膜なし | ウラボシ科 |

## シダ植物の生活環

### 世代交代

```
胞子体（2n）
  ↓ 減数分裂
胞子（n）
  ↓ 発芽
前葉体（配偶体）（n）
  ↓ 造卵器・造精器形成
卵（n）＋精子（n）
  ↓ 受精
受精卵（2n）
  ↓ 発生
胞子体（2n）
```

### 特徴

| 世代 | 染色体 | 大きさ | 特徴 |
|------|--------|--------|------|
| 胞子体 | 2n | 大型（私たちが見るシダ） | 独立栄養 |
| 配偶体（前葉体） | n | 小型（数mm） | 心臓形、造卵器・造精器をもつ |

> 📝 **配偶体**：シダの配偶体（前葉体）は小さく、湿った場所で独立生活する

## シダ植物の分類

### PPG I 分類体系

[PPG I](https://ja.wikipedia.org/wiki/PPG_I)（Pteridophyte Phylogeny Group、2016年）は、シダ植物の最新の分類体系です。

### 主要な綱と目

| 綱 | 主な目 | 代表的な科 |
|----|--------|------------|
| [リコプシダ綱](https://ja.wikipedia.org/wiki/リコプシダ綱)（小葉植物） | ヒカゲノカズラ目、イワヒバ目、ミズニラ目 | ヒカゲノカズラ科、イワヒバ科 |
| [トクサ綱](https://ja.wikipedia.org/wiki/トクサ綱) | トクサ目 | トクサ科 |
| [ハナヤスリ綱](https://ja.wikipedia.org/wiki/ハナヤスリ綱) | ハナヤスリ目 | ハナヤスリ科、マツバラン科 |
| [リュウビンタイ綱](https://ja.wikipedia.org/wiki/リュウビンタイ綱) | リュウビンタイ目 | リュウビンタイ科 |
| [ウラボシ綱](https://ja.wikipedia.org/wiki/ウラボシ綱)（薄嚢シダ類） | ウラボシ目など多数 | オシダ科、メシダ科、ウラボシ科など |

### 日本産シダ植物の主要科

| 科名 | 日本産種数 | 代表属 |
|------|------------|--------|
| [オシダ科](https://ja.wikipedia.org/wiki/オシダ科) | 約100種 | オシダ属、イノデ属 |
| [メシダ科](https://ja.wikipedia.org/wiki/メシダ科) | 約80種 | イヌワラビ属、シケシダ属 |
| [ウラボシ科](https://ja.wikipedia.org/wiki/ウラボシ科) | 約50種 | ノキシノブ属、マメヅタ属 |
| [イノモトソウ科](https://ja.wikipedia.org/wiki/イノモトソウ科) | 約40種 | イノモトソウ属、シシガシラ属 |
| [チャセンシダ科](https://ja.wikipedia.org/wiki/チャセンシダ科) | 約30種 | チャセンシダ属 |
| [コバノイシカグマ科](https://ja.wikipedia.org/wiki/コバノイシカグマ科) | 約20種 | コバノイシカグマ属、ワラビ属 |

## 確認問題

### 問1
シダ植物の繁殖方法を述べよ。

<details>
<summary>解答を見る</summary>

**胞子で繁殖する**（種子を作らない）

</details>

### 問2
シダ植物の配偶体の別名と特徴を述べよ。

<details>
<summary>解答を見る</summary>

- **別名**：前葉体
- **特徴**：小型（数mm）、心臓形、造卵器・造精器をもつ

</details>

### 問3
胞子嚢群（ソーラス）とは何か説明せよ。

<details>
<summary>解答を見る</summary>

**胞子嚢が集まったもの**。葉の裏面などに形成され、多くの種で包膜に覆われる。

</details>

### 問4
シダ植物の最新分類体系の名称は何か。

<details>
<summary>解答を見る</summary>

**PPG I**（Pteridophyte Phylogeny Group、2016年）

</details>

## 暗記用まとめ

### シダ植物の特徴

| 特徴 | 内容 |
|------|------|
| 維管束 | あり |
| 種子 | なし |
| 胞子 | あり |
| 精子 | 鞭毛あり（水が必要） |

### 世代交代

| 世代 | 特徴 |
|------|------|
| 胞子体（2n） | 大型、私たちが見るシダ |
| 配偶体（n） | 小型、前葉体 |

### 主要な綱

| 綱名 | 特徴 | 代表 |
|------|------|------|
| リコプシダ綱 | 小葉植物 | ヒカゲノカズラ |
| トクサ綱 | 節のある茎 | トクサ |
| ウラボシ綱 | 薄嚢シダ類（最大） | オシダ、ワラビ |
