## はじめに

[菌類](https://ja.wikipedia.org/wiki/菌類)（Fungi）は、植物でも動物でもない独立した界を構成する生物群です。光合成を行わず、外部から有機物を吸収して生活します。日本には約15,000種以上が知られています。

---

## 菌類の概要

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産種数 | 約15,000種以上 |
| 世界の種数 | 約10万種（推定150万種） |
| 分類 | 真核生物・菌界 |
| 栄養様式 | 従属栄養（吸収栄養） |

### 菌類の特徴

| 特徴 | 内容 |
|------|------|
| 細胞壁 | **キチン**を含む |
| 光合成 | 行わない |
| 栄養 | 有機物を分解・吸収 |
| 体制 | 菌糸（大部分） |
| 貯蔵物質 | グリコーゲン |

> 📝 **菌類と植物の違い**：植物はセルロースの細胞壁でデンプンを貯蔵、菌類はキチンの細胞壁でグリコーゲンを貯蔵

---

## ==真核生物のスーパーグループと菌類の位置（超頻出）==

### ==菌類の所属（Q9対応）==

菌類は真核生物の**オピストコンタ（Opisthokonta）**に所属します。

| スーパーグループ | 含まれる主要な生物群 | 特徴 |
|------------------|----------------------|------|
| ==**オピストコンタ**== | ==**菌類**==、==**動物**== | ==**後方鞭毛型**==（鞭毛が細胞の後ろ側） |
| アーケプラスチダ | 陸上植物、緑藻、紅藻 | 一次共生で葉緑体獲得 |
| SAR | 褐藻、珪藻、渦鞭毛藻 | 二次共生で葉緑体獲得 |
| エクスカバータ | ユーグレナ、トリパノソーマ | 摂食溝を持つ |
| アメーボゾア | アメーバ、変形菌 | 偽足運動 |

### ==オピストコンタの主要特徴（試験重要）==

| 特徴 | 説明 |
|------|------|
| ==**鞭毛の位置**== | ==**後方に1本の鞭毛**==（遊走細胞を持つ場合） |
| **構成グループ** | **菌類**と**動物**（および関連原生生物） |
| **共通祖先** | 約10億年前に他の真核生物から分岐 |
| **系統的関係** | 菌類と動物は姉妹群 |

> 📝 **試験ポイント（Q9）**: 「菌類が所属するスーパーグループは？」→ **オピストコンタ（Opisthokonta）**
>
> 🔴 **重要**: 菌類は**植物とは別のスーパーグループ**に属する！
> - 植物は「アーケプラスチダ」
> - 菌類は「オピストコンタ」（動物と同じグループ）

### オピストコンタ内の分類

```
オピストコンタ（Opisthokonta）
├── ホロゾア（Holozoa）
│   ├── 動物（Animalia）
│   └── 襟鞭毛虫など
└── ヒュロマイコータ（Holomycota）
    ├── 菌類（Fungi）
    └── 関連原生生物
```

## 菌類の生活様式

### 栄養様式

| 様式 | 内容 | 例 |
|------|------|-----|
| 腐生菌 | 死んだ有機物を分解 | 多くのキノコ |
| 寄生菌 | 生きた生物から栄養 | うどんこ病菌 |
| 共生菌 | 他の生物と共生 | 菌根菌 |

### 菌根（きんこん）

| 類型 | 特徴 | 例 |
|------|------|-----|
| [外生菌根](https://ja.wikipedia.org/wiki/外生菌根) | 根の外側を覆う | マツタケ、トリュフ |

| [ラン菌根](https://ja.wikipedia.org/wiki/ラン菌根) | ラン科植物と共生 | ナラタケ属など |

> 📝 **菌根共生**：陸上植物の約80%が菌根菌と共生。菌は無機塩類を植物に供給し、植物は光合成産物を菌に供給

### [ドクツルタケ](https://ja.wikipedia.org/wiki/ドクツルタケ) *Amanita virosa*

#### 基本情報

| 項目 | 内容 |
|------|------|
| 分類 | 担子菌類（テングタケ科テングタケ属） |
| 生活型 | 外生菌根菌 |
| 発生時期 | 夏〜秋 |
| 発生場所 | 広葉樹・針葉樹の混交林の地上 |
| 食毒 | **猛毒**（死亡例多数：英名 "Destroying Angel"「殺しの天使」） |

#### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 傘 | 直径5〜15cm、初め卵形・円錐形→成熟すると中高の平形。**純白**、乾くと絹光沢。縁に条線なし |
| ひだ | **白色**、密、離生 |
| 柄 | 長さ8〜25cm、白色、上部に**白色膜質のツバ**。ツバより下はしばしば繊維状のささくれに覆われる |
| つぼ | 根元に**大型の白色袋状つぼ**（外皮膜の残存物）が明瞭 |
| 胞子紋 | 白色 |

#### 毒成分と症状

| 毒成分 | 種類 | 作用機序 | 症状 |
|--------|------|----------|------|
| α-アマニチン | アマトキシン類（環状8アミノ酸ペプチド） | **RNAポリメラーゼII阻害**→タンパク質合成停止 | 肝・腎細胞壊死→多臓器不全 |
| β-アマニチン | 同上 | 同上 | 同上 |
| ファロイジン | ファロトキシン類 | 肝細胞毒性（経口吸収は限定的） | 肝障害 |

**致死量**：α-アマニチンのヒト経口致死量は体重1kgあたり約0.1mg。成熟した子実体1本にα-アマニチンが約10〜12mg含まれるため、**成人でも1本の摂取で死亡する可能性がある**。

**症状の経過（遅延型）**：
1. 摂食後6〜24時間：激しい下痢・嘔吐・腹痛（コレラ様症状）
2. その後：一時的に症状が消える「**偽回復期**」
3. 24〜72時間後：肝・腎機能障害、黄疸、多臓器不全→死亡

> ⚠️ **重要**：潜伏期間が長く、症状出現時にはすでに重篤な臓器障害が進行していることが多い。

#### なぜ重要か（試験重要度 ★★★★★）

- 日本のキノコ中毒死亡例で最も多い原因種の一つ
- アマトキシンの作用機序（RNAポリメラーゼII阻害）が生化学的に頻出
- 遅延型症状パターン（偽回復期）が試験に出る
- 純白で美しい外見が誤食を招く危険性

#### 試験頻出問題

> **問：ドクツルタケの主要毒成分の作用機序を答えよ。**
> → **答え：アマトキシン類（α-アマニチン等）がRNAポリメラーゼIIを阻害し、細胞のタンパク質合成を停止させる。特に肝臓・腎臓の細胞が障害される。**

> **問：ドクツルタケによる中毒症状の特徴を述べよ。**
> → **答え：摂食後6〜24時間の長い潜伏期間の後、消化器症状が現れ、一時的に回復に見える「偽回復期」を経て、24〜72時間後に多臓器不全を起こす遅延型中毒。**

#### 混同しやすい種との違い

| 特徴 | シロハツ（ベニタケ科・食用） | ドクツルタケ（テングタケ科・猛毒） |
|------|-----------------------------|------------------------------------|
| つぼ | **なし** | **大型袋状つぼあり** |
| ツバ | なし | **あり** |
| ひだ | 密、白→淡黄 | 密、純白・離生 |
| 肉 | 辛み（辛いものあり） | 無臭 |

### [ベニテングタケ](https://ja.wikipedia.org/wiki/ベニテングタケ) *Amanita muscaria*

#### 基本情報

| 項目 | 内容 |
|------|------|
| 分類 | 担子菌類（テングタケ科テングタケ属） |
| 生活型 | 外生菌根菌（針葉樹・広葉樹と共生） |
| 発生時期 | 夏〜秋 |
| 発生場所 | 針葉樹・広葉樹林の地上 |
| 食毒 | **有毒**（幻覚・神経症状；死亡例は少ない） |

#### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 傘 | 直径6〜15cm（最大20cm）。**鮮赤色〜橙黄色**、表面に**白いイボ状の破片**（外皮膜の残存物）が散在。縁に条線あり。湿時に粘性 |
| ひだ | 白色、密、離生 |
| 柄 | 長さ10〜24cm、白色、上部に**白色膜質ツバ**。基部が膨らみ**環状のつぼ破片**がある |
| 肉 | 白色（傘の表皮下は淡黄色） |
| 胞子紋 | 白色 |

#### 毒成分と症状

| 毒成分 | 作用機序 | 症状 |
|--------|----------|------|
| **イボテン酸**（ibotenic acid） | グルタミン酸受容体アゴニスト（神経毒）→体内でムシモールに変換 | 興奮・錯乱 |
| **ムシモール**（muscimol） | **GABA受容体アゴニスト**（神経抑制） | 幻覚・鎮静・運動失調 |
| ムスカリン（微量） | コリン作動性受容体刺激 | 流涎・縮瞳（微量なので主症状でない） |

> 📝 **注意**：「ムスカリン」はベニテングタケの学名 *muscaria* に由来するが、本種の**主要毒成分はムスカリンではなくイボテン酸・ムシモール**。

#### 民間伝承・文化的背景

- **シベリアのシャーマン**（カムチャッカ半島周辺の民族等）が宗教儀式・変性意識状態へ利用してきた歴史がある
- **フィンランド・スウェーデン北部のサーミ人**のシャーマンにも類似した使用例
- ファンタジー・童話で描かれる「典型的な毒キノコ」のイメージの原型

#### なぜ重要か（試験重要度 ★★★★）

- 毒成分（イボテン酸・ムシモール）の名前と作用機序が頻出
- ムスカリンとの混同に注意（ムスカリンは学名に由来するが主成分ではない）
- 担子菌類の典型的な形態（ツバ・つぼ・外皮膜のイボ）の代表例

#### 試験頻出問題

> **問：ベニテングタケの主要毒成分を2つ答えよ。またその作用を述べよ。**
> → **答え：イボテン酸とムシモール。イボテン酸は体内でムシモールに変換される。ムシモールはGABA受容体に作用する神経毒で、幻覚・鎮静・運動失調を引き起こす。**

### [マツタケ](https://ja.wikipedia.org/wiki/マツタケ) *Tricholoma matsutake*

#### 基本情報

| 項目 | 内容 |
|------|------|
| 分類 | 担子菌類（キシメジ科キシメジ属） |
| 生活型 | **外生菌根菌**（主にアカマツと共生） |
| 発生時期 | 秋（9〜11月） |
| 発生場所 | アカマツ林・混交林の地上（貧栄養・酸性土壌の「シロ」と呼ばれる環状帯） |
| 食毒 | 食用（日本最高級食用キノコの一つ） |

#### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 傘 | 直径5〜20cm、**赤褐色〜黄褐色**。幼菌は半球状→成長すると平形 |
| ひだ | 白色、密、離生 |
| 柄 | 長さ10〜20cm、太さ1.5〜3cm、繊維質。上部に**綿毛状のツバ**（部分ベール残存）。下部には鱗状繊維 |
| 香り | **シナモン様の独特の強い香り**（最重要識別特徴）|

#### 菌根共生の詳細

- 菌糸がアカマツの細根の表面を覆い（外生菌根）、根の細胞間隙（Hartig net）に侵入
- 菌糸が土壌中に広がり、水分・無機塩類（リンなど）を効率的に吸収して樹木に供給
- 樹木は光合成産物（糖）を菌に供給
- **「シロ」**：マツタケの菌糸が集まった環状の帯（ring状）。毎年少しずつ外側に広がる

#### 人工栽培が困難な理由

| 理由 | 詳細 |
|------|------|
| **絶対的菌根共生** | 生きたアカマツの根からの有機物供給が必須。原木・菌床での栽培不可 |
| 子実体形成の誘因不明 | 子実体を形成する具体的な条件・トリガーが未解明 |
| 菌糸の競争力の弱さ | 栄養豊富な土壌では他の微生物に競争で負ける |
| 貧栄養土壌の必要性 | かつて里山の落葉かき集め（燃料利用）で維持された貧栄養環境が失われた |
| 宿主樹木の成熟必要 | 樹齢20年以上のアカマツが必要 |

#### なぜ重要か（試験重要度 ★★★★）

- 外生菌根共生の代表例として最重要
- 人工栽培不可能の理由が試験に出る
- アカマツ林の減少・里山荒廃によるマツタケ減少問題

#### 試験頻出問題

> **問：マツタケが人工栽培困難な最大の理由を述べよ。**
> → **答え：マツタケは絶対的な外生菌根菌であり、生きたアカマツの根から有機物供給を受けなければ成長できないため。原木や菌床では代替できず、また子実体形成の条件も未解明。**

| 分生子柄頭部 | 先端に放射状の**ステリグマ（フィアライド）**が形成され、そこから分生子（胞子）が鎖状に生じる |
| コロニー | 黄緑色〜黄褐色（ニホンコウジカビは黄緑色） |

#### 和食文化との関係（国菌としての重要性）

| 製品 | コウジカビの役割 |
|------|-----------------|
| **味噌** | 米・麦・大豆に繁殖し、デンプン→糖・タンパク質→アミノ酸に分解 |
| **醤油** | 大豆・小麦に繁殖し、風味・旨味を生成 |
| **日本酒** | 蒸米に繁殖して糖化（米のデンプン→糖）→酵母がアルコール発酵 |
| **みりん・酢** | 同様の糖化・発酵プロセス |

> 📝 **麹（こうじ）**：コウジカビを繁殖させた穀物（米麹・麦麹・豆麹）のこと。和食の発酵食品の基礎となる。

#### なぜ重要か（試験重要度 ★★★）

- **子嚢菌類であること**（糸状菌であるが分類は子嚢菌門）
- 日本の「国菌」として文化的・産業的重要性が高い
- 分生子（無性胞子）による増殖の仕組み

## 担子菌 vs 子嚢菌 詳細比較表

| 特徴 | 担子菌門（Basidiomycota） | 子嚢菌門（Ascomycota） |
|------|--------------------------|------------------------|
| **胞子形成器官** | **担子器**（basidium） | **子嚢**（ascus） |
| **胞子の位置** | 担子器の**先端・外側**（外生） | 子嚢の**内側**（内生）、通常8個 |
| **代表的な子実体形態** | キノコ形（傘・ひだ・柄） | 多様：杯状・嚢状・地下生・単細胞など |
| **代表的な種** | シイタケ、マツタケ、ドクツルタケ、ベニテングタケ | **アミガサタケ、トリュフ、酵母、コウジカビ** |
| **推定種数** | 約30,000種 | 約65,000種（最大の菌類門） |
| **菌糸の特徴** | クランプ連結（ドルミポア）あり | 隔壁あり（単純隔壁が多い） |
| **主な生活型** | 腐生・外生菌根菌・寄生 | 腐生・寄生・内生菌根 |

> ⚠️ **試験必出**：アミガサタケ・トリュフ・酵母・コウジカビはすべて**子嚢菌類**！キノコ形でも子嚢菌である場合がある。

## 章末確認問題（4択形式）

### 問A
アミガサタケ（*Morchella esculenta*）が属する菌類の門として正しいものはどれか。

（A）担子菌門（Basidiomycota）
（B）接合菌門（Zygomycota）
（C）**子嚢菌門（Ascomycota）**
（D）ツボカビ門（Chytridiomycota）

<details>
<summary>解答を見る</summary>

**（C）子嚢菌門（Ascomycota）**

アミガサタケはキノコ状の外見だが、胞子は子嚢内に形成されるため子嚢菌類に分類される。「キノコ形＝担子菌」という思い込みに注意。トリュフも同様。

</details>

---

### 問B
ドクツルタケ（*Amanita virosa*）の主要毒成分による作用機序として正しいものはどれか。

（A）アセチルコリン受容体の阻害
（B）GABA受容体のアゴニスト作用
（C）**RNAポリメラーゼIIの阻害によるタンパク質合成停止**
（D）ナトリウムチャネルの遮断

<details>
<summary>解答を見る</summary>

**（C）RNAポリメラーゼIIの阻害によるタンパク質合成停止**

主要毒成分はアマトキシン類（α-アマニチン等）。これがRNAポリメラーゼIIを阻害し、mRNA合成が停止することで細胞が死ぬ。特に肝臓・腎臓が障害される。

</details>

---

### 問C
エノキタケ（*Flammulina velutipes*）の栽培株と野生株の形態の違いとして正しいものはどれか。

（A）栽培株は野生株より傘が大きく赤褐色になる
（B）**栽培株は野生株より白く細長くなる（光遮断・高CO₂環境のため）**
（C）野生株は栽培株より白く細長い
（D）野生株と栽培株では形態に差異はない

<details>
<summary>解答を見る</summary>

**（B）栽培株は野生株より白く細長くなる（光遮断・高CO₂環境のため）**

栽培株は暗所・高CO₂・低温環境で育てるため色素が形成されず白くなり、細長く伸びる。野生株は黄褐色〜茶褐色で柄の根元が黒いビロード状。

</details>

---

### 問D
マツタケが人工栽培困難な主な理由として正しいものはどれか。

（A）マツタケは光合成を行うため特殊な光環境が必要だから
（B）マツタケの胞子が非常に小さく採取が困難だから
（C）**マツタケは生きたアカマツとの外生菌根共生なしには成長できないから**
（D）マツタケは増殖速度が遅すぎて商業栽培に不向きだから

<details>
<summary>解答を見る</summary>

**（C）マツタケは生きたアカマツとの外生菌根共生なしには成長できないから**

マツタケは絶対的外生菌根菌であり、アカマツ等の生きた根からの有機物供給が必須。原木栽培・菌床栽培は不可能。

</details>

---

### 問E
担子菌類と子嚢菌類の胞子形成の違いとして正しいものはどれか。

（A）担子菌類は子嚢内に胞子を形成し、子嚢菌類は担子器の先端に形成する
（B）**担子菌類は担子器の先端（外側）に胞子を形成し、子嚢菌類は子嚢の内部に胞子を形成する**
（C）両者とも子嚢内に胞子を形成する
（D）担子菌類は接合胞子を形成し、子嚢菌類は担子胞子を形成する

<details>
<summary>解答を見る</summary>

**（B）担子菌類は担子器の先端（外側）に胞子を形成し、子嚢菌類は子嚢の内部に胞子を形成する**

胞子形成器官と位置関係が逆なのがポイント。担子菌＝外側、子嚢菌＝内側（袋の中）。

</details>

---

### 問F
以下のキノコのうち、子嚢菌類（Ascomycota）に属するものはどれか。

（A）マツタケ
（B）ドクツルタケ
（C）マイタケ
（D）**トリュフ**

<details>
<summary>解答を見る</summary>

**（D）トリュフ**

トリュフ（*Tuber* 属）は子嚢菌類で地下生の外生菌根菌。（A）〜（C）はすべて担子菌類。子嚢菌類のキノコ形としてはアミガサタケとトリュフが試験頻出。

</details>

### 問G
ベニテングタケ（*Amanita muscaria*）の主要毒成分の組み合わせとして正しいものはどれか。

（A）アマトキシン・ファロイジン
（B）ムスカリン・アマトキシン
（C）**イボテン酸・ムシモール**
（D）シロシビン・シロシン

<details>
<summary>解答を見る</summary>

**（C）イボテン酸・ムシモール**

ベニテングタケの主要毒成分はイボテン酸とムシモール。学名 *muscaria* から「ムスカリン」が主成分と誤解しやすいが、ムスカリンは微量しか含まれない。

</details>

### テングタケ属（*Amanita*）の識別キーワード

3. **白いひだ**・白い胞子紋
4. 多くが**外生菌根菌**

> ⚠️ **テングタケ属は猛毒種が多い**：つぼ・ツバを持つキノコには特に注意。専門家以外は採取・食用しないこと。
