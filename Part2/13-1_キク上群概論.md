## はじめに

[キク上群](https://ja.wikipedia.org/wiki/キク上群)（Asterids / Superasterids）は、真正双子葉類の最大の系統群で、被子植物全体の約1/3を占めます。キク科、シソ科、ナス科など、身近で経済的にも重要な植物を多数含みます。

---

## キク上群の概要

### 基本情報

| 項目 | 内容 |
|------|------|
| 含まれる目数 | 約17目 |
| 含まれる科数 | 約125科 |
| 含まれる種数 | 約80,000種 |
| 被子植物に占める割合 | 約1/3 |

### 形態的特徴

| 特徴 | 内容 |
|------|------|
| 花冠 | **合弁花**が基本（花弁が合着） |
| 雄しべ | 花弁と同数または少ない |
| 子房 | 多くは2心皮 |
| 化学成分 | [イリドイド](https://ja.wikipedia.org/wiki/イリドイド)を含むものが多い |

> 📝 **合弁花類**：かつての分類では「合弁花類」としてまとめられていた群の大部分がキク上群に含まれる

## キク上群の主要な特徴

### 1. 合弁花

キク上群の花は基本的に**合弁花**（花弁が合着）です。

| 花冠の型 | 特徴 | 代表的な科 |
|----------|------|------------|
| 筒状花冠 | 花弁が筒状に合着 | キク科 |
| 唇形花冠 | 上下2唇に分かれる | シソ科 |
| 漏斗形花冠 | 漏斗状 | ヒルガオ科 |
| 車形花冠 | 平開、裂片が放射状 | ナス科 |
| 鐘形花冠 | 釣鐘状 | キキョウ科 |

### 2. イリドイド

多くのキク上群植物は[イリドイド](https://ja.wikipedia.org/wiki/イリドイド)（テルペノイド化合物）を含みます。

| 特徴 | 内容 |
|------|------|
| 化学構造 | モノテルペノイド配糖体 |
| 生態的機能 | 草食動物への防御物質 |
| 薬理作用 | 抗炎症、鎮痛などの薬効 |

### 3. 複雑な花序

| 花序型 | 特徴 | 代表的な科 |
|--------|------|------------|
| [頭状花序](https://ja.wikipedia.org/wiki/頭状花序) | 多数の小花が密集 | キク科 |
| [散形花序](https://ja.wikipedia.org/wiki/散形花序) | 花柄が1点から放射状 | セリ科 |
| [集散花序](https://ja.wikipedia.org/wiki/集散花序) | さそり状など | ムラサキ科 |

### APG体系での大きな変更点

| 変更内容 | 旧分類 | APG分類 |
|----------|--------|---------|
| ゴマノハグサ科の解体 | 1科 | 複数科に分割 |
| スイカズラ科の再編 | 複数科 | 旧オミナエシ科・旧マツムシソウ科→スイカズラ科に統合、ガマズミ属・ニワトコ属→レンプクソウ科に移動 |
| アジサイ科の独立 | ユキノシタ科の一部 | ミズキ目・アジサイ科 |
| ツバキ科の移動 | ツバキ目 | ツツジ目 |

## キク上群の進化と多様化

### 進化史

| 時代 | 出来事 |
|------|--------|
| 約1億年前 | キク上群の分化開始 |
| 約7000万年前 | 主要系統の分化 |
| 約4000万年前 | キク科の出現 |
| 約2000万年前 | キク科の急速な多様化 |

### 多様化の要因

1. **合弁花の進化**
   - 特定の送粉者との共進化
   - 効率的な送粉システム

2. **化学的防御**
   - イリドイド等の二次代謝産物
   - 草食動物への適応

3. **頭状花序の発達**（キク科）
   - 送粉効率の向上
   - 種子散布の多様化
