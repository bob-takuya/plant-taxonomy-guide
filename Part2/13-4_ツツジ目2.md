# 13-4. ツツジ目2（ツバキ科・サクラソウ科・カキノキ科）

[← 目次に戻る](../NEW_00_目次.md) | [← 前：ツツジ目1](./13-3_ツツジ目1.md)

---

## はじめに

本章では、ツツジ目のうちツバキ科、サクラソウ科、カキノキ科などを解説します。APG体系での分類変更により、旧ヤブコウジ科がサクラソウ科に統合されています。

---

## 1. ツバキ科 Theaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 4属 |
| 日本産種数 | 約15種 |
| 世界の属数 | 約9属 |
| 世界の種数 | 約200種 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 常緑または落葉の木本 |
| 葉 | 互生、単葉、革質、鋸歯縁 |
| 花 | 両性、花弁5、基部で合着～離生 |
| 雄しべ | 多数、花弁基部に合着 |
| 子房 | 上位、3～5室 |
| 果実 | 蒴果 |

### 日本産主要種（ツバキ属 Camellia）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ヤブツバキ](https://ja.wikipedia.org/wiki/ヤブツバキ) | *Camellia japonica* | 本州～沖縄 | 日本を代表する常緑樹、赤花 |
| [ユキツバキ](https://ja.wikipedia.org/wiki/ユキツバキ) | *Camellia rusticana* | 本州（日本海側） | 多雪地適応、枝がしなやか |
| [サザンカ](https://ja.wikipedia.org/wiki/サザンカ) | *Camellia sasanqua* | 本州～沖縄 | 秋～冬に開花、花弁が散る |
| [チャノキ](https://ja.wikipedia.org/wiki/チャノキ) | *Camellia sinensis* | 栽培 | 茶の原料 |

### ヤブツバキとサザンカの違い

| 特徴 | ヤブツバキ | サザンカ |
|------|------------|----------|
| 開花時期 | 冬～春（12～4月） | 秋～冬（10～12月） |
| 花の散り方 | 花ごと落ちる | 花弁が1枚ずつ散る |
| 子房 | 無毛 | 有毛 |
| 葉柄 | 無毛 | 有毛 |

### 日本産主要種（その他の属）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ヒサカキ](https://ja.wikipedia.org/wiki/ヒサカキ) | *Eurya japonica* | 本州～沖縄 | 常緑低木、神事に使用 |
| [ハマヒサカキ](https://ja.wikipedia.org/wiki/ハマヒサカキ) | *Eurya emarginata* | 本州～沖縄 | 海岸性、葉先が凹む |
| [サカキ](https://ja.wikipedia.org/wiki/サカキ) | *Cleyera japonica* | 本州～沖縄 | 神事に使用、全縁 |
| [モッコク](https://ja.wikipedia.org/wiki/モッコク) | *Ternstroemia gymnanthera* | 本州～沖縄 | 庭木、「木斛」 |
| [ナツツバキ](https://ja.wikipedia.org/wiki/ナツツバキ) | *Stewartia pseudocamellia* | 本州～九州 | 落葉、夏に白花、沙羅双樹 |
| [ヒメシャラ](https://ja.wikipedia.org/wiki/ヒメシャラ) | *Stewartia monadelpha* | 本州～九州 | ナツツバキより小型、赤褐色の樹皮 |

> 📝 **サカキとヒサカキ**：
> - **サカキ**は本来の神事用植物。暖地に分布。
> - **ヒサカキ**は寒冷地でサカキの代用にされる。「非榊」の意。

## 2. サクラソウ科 Primulaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約15属 |
| 日本産種数 | 約50種 |
| 世界の属数 | 約55属 |
| 世界の種数 | 約2,600種 |

> 📝 **分類の変更**：APG体系で旧ヤブコウジ科がサクラソウ科に統合された

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 草本（まれに低木） |
| 葉 | 互生または対生、単葉 |
| 花 | 両性、合弁、5数性 |
| 雄しべ | 5、花弁と対生（花弁の切れ込みと同じ位置） |
| 子房 | 上位、1室 |
| 胎座 | 特立中央胎座 |
| 果実 | 蒴果 |

> 📝 **識別ポイント**：雄しべが花弁と対生（花冠裂片と同位置）なのが特徴

### 日本産主要種（サクラソウ属 Primula）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [サクラソウ](https://ja.wikipedia.org/wiki/サクラソウ) | *Primula sieboldii* | 本州～九州 | 湿地性、絶滅危惧II類 |
| [クリンソウ](https://ja.wikipedia.org/wiki/クリンソウ) | *Primula japonica* | 北海道～四国 | 段状に輪生花、湿地 |
| [ユキワリソウ](https://ja.wikipedia.org/wiki/ユキワリソウ) | *Primula modesta* | 北海道～本州 | 高山、ピンク花 |
| [ハクサンコザクラ](https://ja.wikipedia.org/wiki/ハクサンコザクラ) | *Primula cuneifolia* var. *hakusanensis* | 本州（中部以北） | 高山の湿地 |
| [エゾコザクラ](https://ja.wikipedia.org/wiki/エゾコザクラ) | *Primula cuneifolia* | 北海道 | 高山 |
| [プリムラ](https://ja.wikipedia.org/wiki/プリムラ) | *Primula* spp. | 栽培 | 園芸品種多数 |

### 日本産主要種（その他の属）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [オカトラノオ](https://ja.wikipedia.org/wiki/オカトラノオ) | *Lysimachia clethroides* | 北海道～九州 | 白い総状花序、先端が曲がる |
| [ヌマトラノオ](https://ja.wikipedia.org/wiki/ヌマトラノオ) | *Lysimachia fortunei* | 本州～九州 | 湿地性、花序直立 |
| [コナスビ](https://ja.wikipedia.org/wiki/コナスビ) | *Lysimachia japonica* | 全国 | 黄花、小型の匍匐性 |
| [クサレダマ](https://ja.wikipedia.org/wiki/クサレダマ) | *Lysimachia vulgaris* var. *davurica* | 北海道～九州 | 湿地性、黄花 |
| [ツマトリソウ](https://ja.wikipedia.org/wiki/ツマトリソウ) | *Trientalis europaea* | 北海道～本州 | 高山、白い7弁花 |

### 旧ヤブコウジ科からの統合種

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ヤブコウジ](https://ja.wikipedia.org/wiki/ヤブコウジ) | *Ardisia japonica* | 北海道～九州 | 常緑小低木、赤い実、十両 |
| [マンリョウ](https://ja.wikipedia.org/wiki/マンリョウ) | *Ardisia crenata* | 本州～沖縄 | 常緑低木、赤い実、万両 |
| [カラタチバナ](https://ja.wikipedia.org/wiki/カラタチバナ) | *Ardisia crispa* | 本州～九州 | 百両 |
| [センリョウ](https://ja.wikipedia.org/wiki/センリョウ) | *Sarcandra glabra* | 本州～沖縄 | **センリョウ科**（別科） |

> 📝 **縁起植物の科**：
> - 万両・百両・十両 → サクラソウ科（旧ヤブコウジ科）
> - 千両 → センリョウ科（別系統）
> - 一両（アリドオシ）→ アカネ科

## 3. カキノキ科 Ebenaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 1属 |
| 日本産種数 | 約5種 |
| 世界の種数 | 約500種 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 落葉または常緑の木本 |
| 葉 | 互生、単葉、全縁 |
| 花 | 単性（雌雄異株または同株）、合弁、3～7数性 |
| 果実 | [液果](https://ja.wikipedia.org/wiki/液果)、萼が宿存して大きくなる |
| 材 | 黒檀（エボニー）として有名 |

### 日本産主要種

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [カキノキ](https://ja.wikipedia.org/wiki/カキノキ) | *Diospyros kaki* | 栽培 | 果樹、甘柿と渋柿 |
| [マメガキ](https://ja.wikipedia.org/wiki/マメガキ) | *Diospyros lotus* | 本州～九州 | 小型の果実、台木に使用 |
| [リュウキュウガキ](https://ja.wikipedia.org/wiki/リュウキュウガキ) | *Diospyros maritima* | 九州～沖縄 | 海岸性 |
| [コクタン](https://ja.wikipedia.org/wiki/コクタン) | *Diospyros ebenum* | 栽培 | 黒檀、高級材 |

> 📝 **柿の渋み**：[タンニン](https://ja.wikipedia.org/wiki/タンニン)が原因。渋柿は乾燥や炭酸ガス処理で渋抜きする。

## 確認問題

### 問1
ヤブツバキとサザンカの花の散り方の違いを述べよ。

<details>
<summary>解答を見る</summary>

- **ヤブツバキ**：花ごと（まるごと）落ちる
- **サザンカ**：花弁が1枚ずつ散る

</details>

### 問2
APG体系でサクラソウ科に統合された旧科は何か。

<details>
<summary>解答を見る</summary>

**旧ヤブコウジ科**

</details>

### 問3
サクラソウ科の雄しべと花弁の位置関係の特徴を述べよ。

<details>
<summary>解答を見る</summary>

**雄しべが花弁と対生**（花冠裂片と同じ位置にある）

</details>

### 問4
縁起植物「万両・百両・十両」の科名と、「千両」の科名を述べよ。

<details>
<summary>解答を見る</summary>

- **万両・百両・十両**：サクラソウ科（旧ヤブコウジ科）
- **千両**：センリョウ科

</details>
