# 13-5. リンドウ目

## はじめに

[リンドウ目](https://ja.wikipedia.org/wiki/リンドウ目)（Gentianales）は、シソ類に属する目で、リンドウ科、キョウチクトウ科、アカネ科など薬用植物や観賞植物を多く含みます。

---

## リンドウ目の概要

| 項目 | 内容 |
|------|------|
| 日本産科数 | 5科 |
| 世界の科数 | 5科 |
| 世界の種数 | 約17,000種 |
| 主な特徴 | 合弁花、対生葉が多い、アルカロイドを含むものが多い |

---

## 1. リンドウ科 Gentianaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約10属 |
| 日本産種数 | 約50種 |
| 世界の属数 | 約90属 |
| 世界の種数 | 約1,700種 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 草本（1年草または多年草） |
| 葉 | 対生、全縁、托葉なし |
| 花 | 合弁、4～5数性 |
| 花冠 | 鐘形、漏斗形、高杯形 |
| 雄しべ | 花弁と同数、花筒につく |
| 子房 | 上位、1室 |
| 果実 | 蒴果 |
| 化学成分 | 苦味配糖体（[セコイリドイド](https://ja.wikipedia.org/wiki/イリドイド)）を含む |

### 日本産主要種（リンドウ属 Gentiana）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [リンドウ](https://ja.wikipedia.org/wiki/リンドウ) | *Gentiana scabra* var. *buergeri* | 本州～九州 | 秋の代表花、青紫色 |
| [エゾリンドウ](https://ja.wikipedia.org/wiki/エゾリンドウ) | *Gentiana triflora* var. *japonica* | 北海道～本州 | 湿地性、切り花用 |
| [オヤマリンドウ](https://ja.wikipedia.org/wiki/オヤマリンドウ) | *Gentiana makinoi* | 本州（中部以北） | 高山の湿地 |
| [タテヤマリンドウ](https://ja.wikipedia.org/wiki/タテヤマリンドウ) | *Gentiana thunbergii* var. *minor* | 本州（中部以北） | 高山、小型 |
| [ハルリンドウ](https://ja.wikipedia.org/wiki/ハルリンドウ) | *Gentiana thunbergii* | 本州～九州 | 春に開花、湿地 |
| [コケリンドウ](https://ja.wikipedia.org/wiki/コケリンドウ) | *Gentiana squarrosa* | 本州～九州 | 小型、草地 |
| [フデリンドウ](https://ja.wikipedia.org/wiki/フデリンドウ) | *Gentiana zollingeri* | 北海道～九州 | 春、小型、1年草 |
| [ミヤマリンドウ](https://ja.wikipedia.org/wiki/ミヤマリンドウ) | *Gentiana nipponica* | 北海道～本州 | 高山 |

### 日本産主要種（その他の属）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [センブリ](https://ja.wikipedia.org/wiki/センブリ) | *Swertia japonica* | 北海道～九州 | 薬用（苦味健胃薬）、「千振」 |
| [ムラサキセンブリ](https://ja.wikipedia.org/wiki/ムラサキセンブリ) | *Swertia pseudochinensis* | 本州～九州 | 紫色の花、絶滅危惧II類 |
| [アケボノソウ](https://ja.wikipedia.org/wiki/アケボノソウ) | *Swertia bimaculata* | 北海道～九州 | 白花に暗紫色の斑点 |
| [イヌセンブリ](https://ja.wikipedia.org/wiki/イヌセンブリ) | *Swertia diluta* var. *tosaensis* | 本州～九州 | 湿地性 |

> 📝 **センブリの名の由来**：千回振り出しても（煎じても）まだ苦いことから「千振」

## 2. キョウチクトウ科 Apocynaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約15属 |
| 日本産種数 | 約40種 |
| 世界の属数 | 約400属 |
| 世界の種数 | 約5,000種 |

> 📝 **分類の変更**：APG体系で旧ガガイモ科がキョウチクトウ科に統合された

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 草本、低木、つる性など多様 |
| 乳液 | 白い乳液を含む（多くは有毒） |
| 葉 | 対生または輪生、全縁 |
| 花 | 合弁、5数性、回旋状 |
| 副花冠 | しばしばある |
| 果実 | 袋果が対になる、種子に毛 |

### 日本産主要種（キョウチクトウ亜科）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [キョウチクトウ](https://ja.wikipedia.org/wiki/キョウチクトウ) | *Nerium oleander* | 栽培 | 街路樹、**猛毒** |
| [テイカカズラ](https://ja.wikipedia.org/wiki/テイカカズラ) | *Trachelospermum asiaticum* | 本州～九州 | つる性、芳香の白花 |
| [チョウジソウ](https://ja.wikipedia.org/wiki/チョウジソウ) | *Amsonia elliptica* | 本州～九州 | 青紫色、湿地性、絶滅危惧II類 |
| [ツルニチニチソウ](https://ja.wikipedia.org/wiki/ツルニチニチソウ) | *Vinca major* | 帰化 | 青紫色の花、グランドカバー |
| [ニチニチソウ](https://ja.wikipedia.org/wiki/ニチニチソウ) | *Catharanthus roseus* | 栽培 | 抗がん剤原料（ビンクリスチン） |
| [プルメリア](https://ja.wikipedia.org/wiki/プルメリア) | *Plumeria* spp. | 栽培 | 熱帯の観賞用 |

> ⚠️ **キョウチクトウの毒性**：全草に強心配糖体（オレアンドリンなど）を含む。燃やした煙も有毒。

### 日本産主要種（旧ガガイモ科）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ガガイモ](https://ja.wikipedia.org/wiki/ガガイモ) | *Metaplexis japonica* | 北海道～九州 | つる性、白～淡紫色 |
| [オオカモメヅル](https://ja.wikipedia.org/wiki/オオカモメヅル) | *Cynanchum aristolochioides* | 本州～九州 | 暗紫色の小花 |
| [コバノカモメヅル](https://ja.wikipedia.org/wiki/コバノカモメヅル) | *Cynanchum sublanceolatum* | 本州～九州 | 暗紫色 |
| [スズサイコ](https://ja.wikipedia.org/wiki/スズサイコ) | *Cynanchum paniculatum* | 本州～九州 | 黄緑色、草地 |
| [フナバラソウ](https://ja.wikipedia.org/wiki/フナバラソウ) | *Cynanchum atratum* | 北海道～九州 | 暗紫色、草地 |
| [イケマ](https://ja.wikipedia.org/wiki/イケマ) | *Cynanchum caudatum* | 北海道～九州 | 白花、つる性 |
| [キジョラン](https://ja.wikipedia.org/wiki/キジョラン) | *Marsdenia tomentosa* | 本州～沖縄 | アサギマダラの食草 |

## 3. アカネ科 Rubiaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約25属 |
| 日本産種数 | 約70種 |
| 世界の属数 | 約600属 |
| 世界の種数 | 約13,000種 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 草本または木本 |
| 葉 | **対生で托葉あり**（または輪生状） |
| 托葉 | 葉間托葉、または托葉が葉状で輪生に見える |
| 花 | 合弁、4～5数性 |
| 子房 | 下位 |
| 果実 | 核果、液果、蒴果など |

> 📝 **識別ポイント**：対生葉＋托葉（または托葉状の葉による輪生）＋下位子房

### 日本産主要種（木本）

|------|------|------|------|
| [クチナシ](https://ja.wikipedia.org/wiki/クチナシ) | *Gardenia jasminoides* | 本州～沖縄 | 白い芳香花、黄色染料（果実） |
| [コクチナシ](https://ja.wikipedia.org/wiki/コクチナシ) | *Gardenia jasminoides* f. *radicans* | 栽培 | 矮性品種 |
| [ハクチョウゲ](https://ja.wikipedia.org/wiki/ハクチョウゲ) | *Serissa japonica* | 本州～沖縄 | 小型の常緑低木、生垣 |
| [アリドオシ](https://ja.wikipedia.org/wiki/アリドオシ) | *Damnacanthus indicus* | 本州～沖縄 | 鋭い棘、「一両」 |
| [ルリミノキ](https://ja.wikipedia.org/wiki/ルリミノキ) | *Lasianthus japonicus* | 本州～沖縄 | 青い果実 |
| [コーヒーノキ](https://ja.wikipedia.org/wiki/コーヒーノキ) | *Coffea arabica* | 栽培 | コーヒー原料 |

### 日本産主要種（草本）

| [アカネ](https://ja.wikipedia.org/wiki/アカネ) | *Rubia argyi* | 本州～九州 | つる性、茜染めの原料 |
| [ヤエムグラ](https://ja.wikipedia.org/wiki/ヤエムグラ) | *Galium spurium* var. *echinospermon* | 全国 | 茎葉に逆棘、果実にも棘 |

| [フタバムグラ](https://ja.wikipedia.org/wiki/フタバムグラ) | *Hedyotis diffusa* | 本州～沖縄 | 小型、白花 |

## 5. ゲルセミウム科 Gelsemiaceae

### 概要

旧マチン科から分離された小さな科。

| 和名 | 学名 | 特徴 |
|------|------|------|
| [カロライナジャスミン](https://ja.wikipedia.org/wiki/カロライナジャスミン) | *Gelsemium sempervirens* | 栽培、**有毒** |

## 確認問題

### 問1
リンドウ科の薬用として知られる苦味成分を何というか。

<details>
<summary>解答を見る</summary>

**セコイリドイド配糖体**（苦味配糖体）

</details>

### 問2
APG体系でキョウチクトウ科に統合された旧科は何か。

<details>
<summary>解答を見る</summary>

**旧ガガイモ科**

</details>

### 問3
アカネ科の葉の特徴を2つ述べよ。

<details>
<summary>解答を見る</summary>

- **対生**（または輪生状）
- **托葉がある**（葉間托葉、または托葉が葉状で輪生に見える）

</details>

### 問4
アカネ科で重要な経済植物は何か。

<details>
<summary>解答を見る</summary>

**コーヒーノキ**（コーヒー原料）
その他：アカネ（染料）、クチナシ（染料）

</details>

## 暗記用まとめ

### リンドウ目の科

| 科名 | 分類変更 | 代表種 |
|------|----------|--------|
| リンドウ科 | - | リンドウ、センブリ |
| キョウチクトウ科 | 旧ガガイモ科含む | キョウチクトウ、テイカカズラ |
| アカネ科 | - | アカネ、クチナシ |
| マチン科 | - | マチン |

### 有毒植物

| 植物名 | 有毒成分 |
|--------|----------|
| キョウチクトウ | 強心配糖体（オレアンドリン） |
| マチン | ストリキニーネ |

### アカネ科の識別

```
対生葉 ＋ 托葉 ＋ 下位子房 → アカネ科
```

### 経済植物

| 植物名 | 科名 | 用途 |
|--------|------|------|
| センブリ | リンドウ科 | 苦味健胃薬 |

| コーヒーノキ | アカネ科 | 飲料 |
| アカネ | アカネ科 | 染料 |
| クチナシ | アカネ科 | 染料 |
