## はじめに

本章では、コケ植物のうち[苔類](https://ja.wikipedia.org/wiki/苔類)（Marchantiophyta）と[ツノゴケ類](https://ja.wikipedia.org/wiki/ツノゴケ類)（Anthocerotophyta）を解説します。苔類は蘚類に次いで多様なグループで、ツノゴケ類は最も小さなグループです。

---

## 苔類 Marchantiophyta

### 概要

| 項目 | 内容 |
|------|------|
| 日本産種数 | 約700種 |
| 世界の種数 | 約7,000種 |
| 体制 | 葉状体または茎葉体 |
| 特徴 | 油体をもつ |

### 苔類の特徴

| 特徴 | 内容 |
|------|------|
| 油体 | 細胞内に油滴を含む構造（テルペノイド） |
| 蒴 | 蓋がない、4裂または不規則に裂開 |
| 弾糸 | 胞子散布を助けるらせん状の糸 |
| 体制 | 葉状体（タイ類）または茎葉体（ウロコゴケ類） |

### 苔類の2つのタイプ

| タイプ | 特徴 | 代表 |
|--------|------|------|
| 葉状体（タイ類） | 扁平な葉状 | ゼニゴケ、ジャゴケ |
| 茎葉体（ウロコゴケ類） | 茎と葉が分化 | ムチゴケ、ヤスデゴケ |

### 基本情報

| 項目 | 内容 |
|------|------|

| 日本産種数 | 約10種 |
| 体制 | 葉状体 |
| 特徴 | 杯状体で無性繁殖 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 葉状体 | 二叉分岐、背腹性 |
| 気室 | 内部に気室、表面に気孔 |
| 杯状体 | 無性芽（ジェンマ）を入れる |
| 雌器托・雄器托 | 傘状の生殖器官 |

### 日本産主要種

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ゼニゴケ](https://ja.wikipedia.org/wiki/ゼニゴケ) | *Marchantia polymorpha* | 全国 | 庭の雑草、杯状体あり |
| [フタバネゼニゴケ](https://ja.wikipedia.org/wiki/フタバネゼニゴケ) | *Marchantia paleacea* | 本州～沖縄 | 2列の腹鱗片 |

### ゼニゴケの特徴

| 項目 | 内容 |
|------|------|
| 分布 | 全国、湿った日陰 |
| 繁殖 | 杯状体による無性繁殖が盛ん |
| 雌雄性 | 雌雄異株 |
| 雌器托 | 傘状、放射状の腕 |
| 雄器托 | 円盤状 |

> 📝 **杯状体**：葉状体上のカップ状構造。中に無性芽（ジェンマ）が入り、雨滴で飛散して繁殖

### [ゼニゴケ](https://ja.wikipedia.org/wiki/ゼニゴケ) *Marchantia polymorpha*（詳細解説）

#### 形態的特徴

| 部位/特徴 | 内容 |
|-----------|------|
| 生活形 | 葉状体型苔類（葉状体は茎・葉が未分化） |
| 大きさ | 葉状体の長さ3〜10cm、幅1〜2cm |
| 成長 | 頂端（二叉分岐の先端）から分裂成長を繰り返す |
| 葉状体表面 | 六角形の区画（気室の上方）に**気孔**、表面に鱗状模様 |
| 内部構造 | **気室**（光合成細胞を含む空腔）と**気孔**（外界に通じる） |
| 腹面 | 赤色の**腹鱗片**と褐色の**仮根**（固着用） |
| 雌雄性 | **雌雄異株** |
| 分布 | 全国（北海道〜九州）、湿った日陰の庭・路傍・畦 |

#### 生殖器官の詳細（試験最頻出！）

| 器官 | 形状 | 詳細 |
|------|------|------|
| **雄器托** | **円盤状**（波打つ縁） | 雄株の茎頂端に形成。表面に造精器が埋まっている。雨水で精子が流れ出し水路を使って移動 |
| **雌器托** | **深裂した傘状**（「破れ傘」状） | 雌株の茎頂端に形成。5〜10本程度の腕状に分かれ、裂片の裏側に造卵器→胞子体がつく |
| **杯状体** | カップ状（葉状体表面に散在） | 中に**無性芽（ジェンマ）**が入る。雨粒が落ちると無性芽が飛散して無性繁殖 |
| **無性芽** | レンズ状・多細胞 | クローン体。遺伝的に親と同一の個体になる |

> 📝 **試験の核心**：雄器托＝**円盤状**（平べったい丸）、雌器托＝**深裂した傘状**（「破れ傘」）。この形状の違いを必ず覚える！

#### ゼニゴケの生活環（配偶体世代が主役）

```
配偶体（葉状体）→ 造精器（精子）・造卵器（卵）→ 受精 → 胞子体（寄生）→ 胞子 → 原糸体 → 配偶体
また：葉状体 → 杯状体 → 無性芽 → 新しい配偶体（無性繁殖）
```

#### なぜ重要か（試験重要度 ★★★★★）

- **苔類の代表種**として最も試験に出る種
- 雄器托・雌器托・杯状体・無性芽の4つの構造がセットで出題される
- **雌雄異株**であることも重要（他の苔類では雌雄同株もある）
- モデル植物として2017年にゲノム全解読が完了（生物学研究で重要）
- 庭の北側など湿った場所に生える「雑草」として身近な存在

#### 試験頻出問題

> **問：ゼニゴケの雄器托と雌器托の形状の違いを答えよ。**
> → **答え：** 雄器托は**円盤状**（平たい波打つ円盤形）。雌器托は**深裂した傘状**（「破れ傘」状で5〜10本の腕状に分かれる）。雄株・雌株それぞれの茎の頂端に生じる。

> **問：ゼニゴケの無性繁殖を担う構造を2つ答え、それぞれの役割を述べよ。**
> → **答え：** ①**杯状体**：葉状体表面のカップ状構造。②**無性芽（ジェンマ）**：杯状体の中に形成されるレンズ状の多細胞体。雨粒が杯状体に落ちると無性芽が飛散し、新しい個体（クローン）になる。

> **問：ゼニゴケの葉状体の内部に気室が発達する。この気室に関係する表面構造を答えよ。**
> → **答え：** **気孔**（気室孔）。葉状体の背面表皮に散在し、気室と外界をつなぐ。気室内の光合成細胞（葉緑体をもつ細胞）とガス交換を行う。

#### ゼニゴケとジャゴケの比較

| 比較項目 | ゼニゴケ | ジャゴケ |
|---------|---------|---------|
| 学名 | *Marchantia polymorpha* | *Conocephalum conicum* |
| 葉状体表面 | 鱗状模様、六角形区画 | **蛇皮状**の独特な模様 |
| 杯状体 | **あり**（無性芽を含む） | なし |
| 芳香 | 弱い | **強い芳香**（テルペノイド） |
| 生育環境 | 湿った日陰、庭・路傍 | 渓流沿いの岩上・湿崖 |
| 生殖器官の形 | 雌器托が深裂した傘状 | 尖った傘状の雌器托 |

## ツノゴケ類 Anthocerotophyta

### 概要

| 項目 | 内容 |
|------|------|
| 日本産種数 | 約20種 |
| 世界の種数 | 約300種 |
| 体制 | 葉状体 |
| 特徴 | 角状の胞子体 |

### ツノゴケ類の特徴

| 特徴 | 内容 |
|------|------|
| 配偶体 | 薄い葉状体、ロゼット状 |
| 胞子体 | **角状**（他のコケと大きく異なる） |
| 成長 | 胞子体は基部から継続成長 |
| 葉緑体 | 1細胞に1個（他のコケは多数） |
| ピレノイド | 葉緑体内にピレノイドをもつ |

> 📝 **ピレノイド**：藻類に見られるCO₂濃縮構造。陸上植物ではツノゴケ類のみにある

### ツノゴケ類の詳細解説

#### ツノゴケ類の重要特徴まとめ（試験頻出）

| 特徴 | 内容 | 重要度 |
|------|------|--------|
| **葉緑体の数** | 1細胞に**1個のみ**（蘚類・苔類は多数） | ★★★★★ |
| **ピレノイド** | 葉緑体内に存在（CO₂濃縮機構）、陸上植物唯一 | ★★★★★ |
| **胞子体の形** | **角状（ツノ状）**、他のコケとまったく異なる | ★★★★ |
| **胞子体の成長** | 基部の分裂組織から**継続的に上方へ成長**し続ける | ★★★★ |
| 配偶体 | 薄い葉状体、ロゼット状またはリボン状 | ★★★ |
| シアノバクテリア | 一部の種は葉状体内にシアノバクテリアと共生（窒素固定） | ★★ |
| 系統 | 蘚類・苔類とともにコケ植物を構成。分子系統では**維管束植物に最も近縁**とされることが多い | ★★★ |

#### なぜ重要か（試験重要度 ★★★★）

- **1細胞1葉緑体**はコケ植物中ツノゴケ類のみ（緑藻などの藻類と共通する祖先的特徴）
- **ピレノイド**は陸上植物では唯一ツノゴケ類にある（藻類からの進化の痕跡）
- これらの特徴が「ツノゴケ類は維管束植物に近縁」という系統仮説を支持する
- 胞子体が**基部から成長し続ける**点は他のコケ植物（胞子体が成長を止める）と根本的に異なる
- 日本産は6属17種（世界で約150種）と小さなグループ

#### 試験頻出問題（ツノゴケ類）

> **問：ツノゴケ類の葉緑体の特徴を、蘚類・苔類と比較して答えよ。**
> → **答え：** ツノゴケ類は1細胞に**1個の葉緑体**しかなく、葉緑体の中に**ピレノイド**（CO₂濃縮構造）を持つ。蘚類・苔類では1細胞に**多数の葉緑体**があり、ピレノイドを持たない。

> **問：ツノゴケ類の胞子体（蒴）の形態と成長の特徴を答えよ。**
> → **答え：** 胞子体は**角状（ツノ状）**で、他のコケ植物と異なり**基部の分裂組織から継続的に上方へ成長し続ける**。先端から順に成熟して胞子が放出される。

> **問：コケ植物3群（蘚類・苔類・ツノゴケ類）のうち、葉緑体を細胞当たり1個しか持たないのはどれか。**
> → **答え：** **ツノゴケ類**。さらにピレノイドも持ち、これは陸上植物の中で唯一の特徴。

## 確認問題

### 問1（4択）
苔類の細胞内に特有の構造はどれか。

**a) 薄板（lamellae）　b) ヒアロサイト　c) 油体（oil body）　d) ピレノイド**

<details>
<summary>解答を見る</summary>

**c) 油体（oil body）**

テルペノイドを含む液滴状の構造で、苔類のほぼすべての種の細胞内に存在する。蘚類・ツノゴケ類にはない。

</details>

### 問2（4択）
ゼニゴケの雌器托の形状として正しいものはどれか。

**a) 円盤状　b) 深裂した傘状（破れ傘状）　c) 球状　d) 角状**

<details>
<summary>解答を見る</summary>

**b) 深裂した傘状（破れ傘状）**

雌器托は5〜10本の腕状に深く分かれた傘状。雄器托が円盤状であることと対比して覚える。

</details>

### 問3（4択）
ゼニゴケに関する記述で誤っているものはどれか。

**a) 葉状体表面に杯状体があり無性芽が入る　b) 雌雄同株である　c) 雌器托は破れ傘状　d) 内部に気室が発達する**

<details>
<summary>解答を見る</summary>

**b) 雌雄同株である（誤り）**

ゼニゴケは**雌雄異株**。雄株に雄器托（円盤状）、雌株に雌器托（傘状）が形成される。

</details>

### 問4
ツノゴケ類の胞子体の形態的特徴と成長の特徴を述べよ。

<details>
<summary>解答を見る</summary>

- 形態：**角状（ツノ状）**
- 成長：**基部の分裂組織から継続的に成長し続ける**（他のコケ植物の胞子体は成長が止まる）

</details>

### 問5（4択）
コケ植物3群の比較で、ツノゴケ類のみに当てはまる特徴はどれか。

**a) 茎葉体の形態　b) 油体を持つ　c) 1細胞に葉緑体1個＋ピレノイド　d) 弾糸で胞子を散布する**

<details>
<summary>解答を見る</summary>

**c) 1細胞に葉緑体1個＋ピレノイド**

葉緑体が1細胞1個のみで、ピレノイドを持つのはツノゴケ類のみ（陸上植物中唯一）。

</details>

### 問6
ジャゴケ（*Conocephalum conicum*）の識別に最も有効な特徴を2つ答えよ。

<details>
<summary>解答を見る</summary>

1. 葉状体表面の**蛇皮状の模様**（六角形の気室が蛇のうろこ状に見える）
2. 独特の**強い芳香**（テルペノイド成分）

</details>

### 苔類の主要科（識別ポイント付き）

| 科名 | 体制 | 最重要識別特徴 | 代表種 |
|------|------|--------------|--------|
| ゼニゴケ科 | 葉状体 | **杯状体**（無性芽入り）、雌器托＝破れ傘状・雄器托＝円盤状 | ゼニゴケ |
| ジャゴケ科 | 葉状体 | **蛇皮状模様**＋強い芳香 | ジャゴケ |
| ウキゴケ科 | 葉状体 | **水面浮遊** | ウキゴケ（リシア） |
| ムチゴケ科 | 茎葉体 | **3裂葉**、林床性 | ムチゴケ |
| ヤスデゴケ科 | 茎葉体 | **兜状の腹片**、着生性 | ヤスデゴケ |
| クサリゴケ科 | 茎葉体 | 苔類最大の科、小型 | クサリゴケ |

### コケ植物3群の比較（最頻出！）

| 特徴 | 蘚類 | 苔類 | ツノゴケ類 |
|------|------|------|------------|
| 体制 | **茎葉体**のみ | 葉状体 or 茎葉体 | **葉状体**のみ |
| 油体 | なし | **あり**（テルペノイド） | なし |
| 胞子体形状 | 球形～楕円形（棒状） | 球形（4裂開） | **角状（ツノ状）** |
| 胞子体の成長 | 決まったら停止 | 決まったら停止 | **基部から成長し続ける** |
| 弾糸 | なし | **あり**（胞子散布） | なし |
| ピレノイド | なし | なし | **あり** |
| 葉緑体/細胞 | 多数 | 多数 | **1個** |

### ゼニゴケの生殖器官 識別表（最頻出！）

| 器官 | 形状 | 形成される株 |
|------|------|------------|
| **雄器托** | **円盤状**（平たい波打つ円盤） | 雄株 |
| **雌器托** | **深裂した傘状**（「破れ傘」、5〜10本の腕） | 雌株 |
| **杯状体** | カップ状（中に無性芽） | 雌雄両方 |

### 苔類の2タイプ

| タイプ | 体制 | 代表科 | 識別ポイント |
|--------|------|--------|------------|
| タイ類（葉状苔類） | 葉状体 | ゼニゴケ科・ジャゴケ科・ウキゴケ科 | 気室・気孔が発達 |
| ウロコゴケ類（茎葉苔類） | 茎葉体 | ムチゴケ科・ヤスデゴケ科・クサリゴケ科 | 茎・葉・腹片が分化 |

### ツノゴケ類 vs 他のコケの決定的な違い（暗記）

> ツノゴケ類 = 「**角（ツノ）が生える・葉緑体1個・ピレノイドあり**」
> - 胞子体が角状で基部から成長し続ける
> - 1細胞に葉緑体1個（陸上植物中唯一）
> - ピレノイド（CO₂濃縮構造）を葉緑体内に持つ（陸上植物中唯一）
