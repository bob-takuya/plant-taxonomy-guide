## はじめに

[ムクロジ目](https://ja.wikipedia.org/wiki/ムクロジ目)（Sapindales）は、アオイ類に属する目で、ムクロジ科（旧カエデ科含む）、ミカン科、ウルシ科、ニガキ科など経済的に重要な植物を多く含みます。

---

## ムクロジ目の概要

| 項目 | 内容 |
|------|------|
| 日本産科数 | 5科 |
| 世界の科数 | 9科 |
| 世界の種数 | 約6,000種 |
| 主な特徴 | 木本が多い、しばしば有毒または芳香物質を含む |

> 📝 **識別ポイント**：カエデ属は「対生」「翼果」が重要な識別ポイント。似た葉のフウ（フウ科）は互生。

### カエデ属のAPG体系での統合（試験重要度 ★★★★）

> **問：APG体系でカエデ科（Aceraceae）はどの科に統合されたか？**
> → **ムクロジ科（Sapindaceae）**。旧カエデ科・旧トチノキ科がムクロジ科に統合された。
> 覚え方：「**カエデもトチもムクロジ科**（APGでは3科が1科に）」

### イロハモミジ vs オオモミジ（試験重要度 ★★★）

| 特徴 | イロハモミジ（*Acer palmatum*） | オオモミジ（*A. amoenum* var. *matsumurae*） |
|------|-------------------------------|----------------------------------------------|
| 葉の裂片数 | **5〜7裂**（通常7） | **7〜9裂** |
| 裂片の切れ込み | 深い | やや浅い（裂片幅が広め） |
| 葉縁 | 重鋸歯 | 重鋸歯 |
| 葉の大きさ | 3〜6cm | **5〜10cm**（大型） |
| 分布 | 本州〜九州（最普通） | 北海道〜九州 |
| 別名 | タカオカエデ | - |

> **問：イロハモミジとオオモミジを葉で識別するポイントを述べよ。**
> → **葉の大きさと裂片数**。イロハモミジは小さく5〜7裂、オオモミジはより大きく7〜9裂で裂片幅が広い。

#### カエデ属の例外種

| 和名 | 例外的特徴 |
|------|------------|
| メグスリノキ | 3出複葉（通常は単葉） |
| チドリノキ | 単葉だが裂けない、平行脈 |
| ネグンドカエデ | 羽状複葉 |

#### 概要

| 項目 | 内容 |
|------|------|
| 日本産種数 | 1種 |
| 特徴 | 対生の掌状複葉 |

#### 日本産種

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [トチノキ](https://ja.wikipedia.org/wiki/トチノキ) | *Aesculus turbinata* | 北海道～九州 | 掌状複葉（5～7小葉）、円錐花序、種子は栃餅に |
| [セイヨウトチノキ](https://ja.wikipedia.org/wiki/セイヨウトチノキ) | *Aesculus hippocastanum* | 栽培 | マロニエ、街路樹 |

> 📝 **トチノキ属の特徴**：掌状複葉で対生という独特の特徴

### トチノキの詳細（試験重要度 ★★★）

| 部位/特徴 | 内容 |
|-----------|------|
| 葉 | **掌状複葉（5〜7小葉）で対生**（ムクロジ科の中では独特） |
| 花 | 大型の円錐花序、白〜淡紅色 |
| 果実 | **大型の蒴果**（直径3〜5cm）、1〜2個の大きな種子 |
| 種子の利用 | アク（サポニン）が強く、**水さらし（あく抜き）**してから**栃餅**などに加工 |
| 分布 | 北海道〜九州（渓流沿い、深山） |

> **問：トチノキの種子は生食できない。その理由と食べるための処理を述べよ。**
> → **サポニンなどのアク（有毒成分）**を含むため生食不可。**水にさらしてアク抜き**した後、トチ餅・トチ飯などに加工して食用にする。

### ムクロジの詳細（試験重要度 ★★★）

### [ムクロジ](https://ja.wikipedia.org/wiki/ムクロジ) *Sapindus mukorossi*

| 部位/特徴 | 内容 |
|-----------|------|
| 葉 | 羽状複葉（偶数または奇数）、互生 |
| 果実 | 液果状（核果に似る）、半透明の黄褐色の果皮 |
| 果皮の成分 | **サポニン**を豊富に含む → 泡立つ → **石鹸代わり**に使用 |
| 種子の利用 | **羽根突きの羽根の重り**（黒い種子）に使用 |
| 語源 | 「無患子」＝子が患わないという縁起木 |
| 分布 | 本州（関東以西）〜沖縄 |

> **問：ムクロジの果皮に含まれ、洗浄効果を発揮する成分は何か？**
> → **サポニン**。界面活性剤として作用し、水と脂を乳化させる。石鹸代わりに古くから使用。

### サンショウの詳細（試験重要度 ★★★★）

### [サンショウ](https://ja.wikipedia.org/wiki/サンショウ) *Zanthoxylum piperitum*

| 部位/特徴 | 内容 |
|-----------|------|
| 葉 | **奇数羽状複葉**、小葉の基部に翼状突起、油点あり（芳香） |
| 棘 | 葉枕に対生する棘（一部の節で互生） |
| 花 | **雌雄異株**（重要識別点）、小型黄緑色花 |
| 果実 | 赤い蒴果、内部に黒い種子 |
| 利用 | **葉**（木の芽）、**花**（花山椒）、**若い果実**（粉山椒）、**樹皮**すべてが香辛料 |
| 成分 | **サンショオール**（辛味）・ゲラニオール（芳香）・ リモネン |
| 別名 | ハジカミ |

> **問：サンショウ（*Zanthoxylum piperitum*）が雌雄異株であることを踏まえ、果実（山椒の実）をつける株の性別を答えよ。**
> → **雌株**のみに果実（赤い蒴果）がつく。雌雄異株であるため、果実採取目的の栽培には雌株が必要。

### カラスザンショウの詳細（試験重要度 ★★）

### [カラスザンショウ](https://ja.wikipedia.org/wiki/カラスザンショウ) *Zanthoxylum ailanthoides*

| 部位/特徴 | 内容 |
|-----------|------|
| 幹・枝 | **鋭い棘が幹から突き出る**（識別容易）、扁平な棘が多い |
| 葉 | 大型の奇数羽状複葉（長さ1m近い）、葉痕が馬蹄形 |
| 生態 | **先駆樹種**（崩壊地・伐採跡地に一番乗り） |
| 分布 | 本州〜沖縄 |
| 香り | 強い独特の芳香 |

### ミカン科の油点（試験重要度 ★★★）

> **問：ミカン科の葉に見られる「油点（ゆてん）」とは何か？野外でどう確認するか？**
> → 葉の表皮細胞が変形した**分泌腔**（油分を蓄える）。葉を光に透かすと**透明な小点**として見える。葉を揉むと**芳香**が強く放出される。ミカン科識別の重要手がかり。

| [ニガキ](https://ja.wikipedia.org/wiki/ニガキ) | *Picrasma quassioides* | 北海道～九州 | 樹皮・葉が苦い、薬用 |

> 📝 **シンジュ（ニワウルシ）の注意点**：ウルシ科ではなくニガキ科。名前に「ウルシ」とあるがかぶれない。

## 確認問題

### 問1
APG体系でムクロジ科に統合された旧科を2つ挙げよ。

<details>
<summary>解答を見る</summary>

- **旧カエデ科**
- **旧トチノキ科**

</details>

### 問2
カエデ属の葉の特徴を述べよ。

<details>
<summary>解答を見る</summary>

- **対生**
- **単葉で掌状に裂ける**（ただし例外あり：メグスリノキは3出複葉、チドリノキは裂けない）

</details>

### 問3
ミカン科の葉の特徴を述べよ。

<details>
<summary>解答を見る</summary>

- **互生**
- **複葉が多い**
- **油点がある**（芳香性）

</details>

### 問4
ウルシ科の植物でかぶれの原因となる物質は何か。

<details>
<summary>解答を見る</summary>

**ウルシオール**

</details>

### 問5
コクサギ型葉序とは何か説明せよ。

<details>
<summary>解答を見る</summary>

互生であるが、2枚ずつ同じ側につく特殊な葉序。コクサギ（ミカン科）、サルスベリ（ミソハギ科）で見られる。

</details>

### 問6
サンショウ（*Zanthoxylum piperitum*）の花の性について述べよ。また果実の辛味成分の名前を答えよ。

<details>
<summary>解答を見る</summary>

- 花の性：**雌雄異株**（雌花序と雄花序が別々の株につく）
- 辛味成分：**サンショオール**（sanshoól / hydroxy-α-sanshool）

</details>

### 問7
次のうちムクロジ科（APG体系）に属する植物はどれか（複数選択可）。
① カエデ　② トチノキ　③ ムクロジ　④ センダン　⑤ レイシ（ライチ）

<details>
<summary>解答を見る</summary>

**①②③⑤（カエデ、トチノキ、ムクロジ、レイシ）**

センダンはセンダン科（Meliaceae）。APG体系でカエデ科・旧トチノキ科が統合されたムクロジ科には、カエデ属・トチノキ属・ムクロジ属・レイシ属・フウセンカズラ属などが含まれる。

</details>
