## マメ目 Fabales

[マメ目](https://ja.wikipedia.org/wiki/マメ目)は、マメ類（Fabids）に属する目で、[マメ科](https://ja.wikipedia.org/wiki/マメ科)を含む重要な目です。

| 項目 | 内容 |
|------|------|
| 日本産科数 | 2科 |
| 日本産種数 | 約160種 |
| 世界の科数 | 4科 |

## [マメ科](https://ja.wikipedia.org/wiki/マメ科) Fabaceae（Leguminosae）

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約60属 |
| 日本産種数 | 約150種 |
| 世界の属数/種数 | 約770属/約20,000種 |
| 分布 | 世界中 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 葉 | 互生、複葉（三出・羽状）、托葉あり |
| 花 | 左右相称（蝶形花）、放射相称も |
| 萼 | 合着、5裂 |
| 花弁 | 5枚（旗弁1+翼弁2+竜骨弁2） |
| 雄しべ | 10本（9+1または全部合着） |
| 果実 | **豆果**（莢） |
| 根粒 | **[根粒菌](https://ja.wikipedia.org/wiki/根粒菌)**と共生、窒素固定 |

### 蝶形花の構造

```
蝶形花の構成
├── 旗弁（きべん）：1枚、上側の大きな花弁
├── 翼弁（よくべん）：2枚、両側の花弁
└── 竜骨弁（りゅうこつべん）：2枚、下側で合着、雄しべ・雌しべを包む
```

### 亜科の分類

| 亜科 | 花の特徴 | 代表属 |
|------|----------|--------|
| マメ亜科 | 蝶形花 | ダイズ、フジ、クローバー |
| ジャケツイバラ亜科 | やや左右相称 | ジャケツイバラ、サイカチ |
| ネムノキ亜科 | 放射相称、雄しべ目立つ | ネムノキ、アカシア |

## 試験頻出ポイント（重要まとめ）

### マメ科3亜科の花形比較表

| 亜科 | 花の対称性 | 花弁の構造 | 雄しべ | 代表種 | 試験重要度 |
|------|------------|------------|--------|--------|------------|
| **マメ亜科** Faboideae | **左右相称** | 蝶形花（旗弁1+翼弁2+竜骨弁2） | 10本（9+1） | フジ、ダイズ、クズ | ★★★★★ |
| **ジャケツイバラ亜科** Caesalpinioideae | **やや左右相称** | 5弁花（上弁小・下弁特化しない） | 10本（離生） | ジャケツイバラ、サイカチ | ★★★ |
| **ネムノキ亜科** Mimosoideae | **放射相称** | 小型・合弁または離弁、花弁目立たず | **多数で目立つ**（雄しべが花の主役） | ネムノキ、アカシア | ★★★★ |

> **問：マメ科3亜科のうち、花が放射相称で雄しべが多数あって目立つのはどの亜科か？**
> → **ネムノキ亜科（Mimosoideae）**。ネムノキの淡紅色の糸状の花はすべて雄しべ。

### フジとヤマフジの巻き方向（試験最頻出）

> **問：フジ（*Wisteria floribunda*）とヤマフジ（*W. brachybotrys*）の最も重要な識別点は何か？**
> → **つるの巻き方向**。フジは**右巻き**（上から見て時計回り）、ヤマフジは**左巻き**（反時計回り）。
> 覚え方：「**ヤ（矢）印→左向き**＝ヤマフジは左巻き」

> **問：フジとヤマフジで花序の長さを比較せよ。**
> → フジの花序は**長い（30〜90cm）**、ヤマフジは**短い（10〜20cm）**。フジは基部から先端へ順に開花する。

## クズ詳細（試験重要度 ★★★★）

### [クズ](https://ja.wikipedia.org/wiki/クズ) *Pueraria montana* var. *lobata*

#### 形態的特徴

| 部位/特徴 | 内容 |
|-----------|------|
| 葉 | 三出複葉、小葉は大型（葉長10〜15cm）、裏面に毛 |
| 花 | 赤紫色の蝶形花、総状花序 |
| 果実 | 豆果（莢）、硬毛が密生 |
| 根 | 肥大した根に**葛粉（クズデンプン）**を蓄える |
| 繁殖力 | 非常に強く、つるが他の植物を覆う |

#### 試験頻出問題

> **問：クズの根から作られる食品・生薬を答えよ。**
> → 根のデンプンから**葛粉（くずこ）**を製造。生薬名は**葛根（かっこん）**で、**葛根湯**の主成分。

> **問：クズが「侵略的外来種」として問題になっている国はどこか？**
> → **アメリカ合衆国**（南東部）。成長速度が速く「Kudzu」として悪名高い侵略的外来植物。日本では秋の七草。

## 根粒菌との共生（試験重要度 ★★★★）

| 項目 | 内容 |
|------|------|
| 共生する細菌 | **根粒菌**（Rhizobium 属などリゾビウム科の菌） |
| 共生の仕組み | 根粒菌が根毛から侵入→根粒（こんりゅう）を形成→窒素固定 |
| 固定する物質 | 大気中の窒素（N₂）→アンモニア（NH₃）→植物が利用できる形に |
| 植物側の利益 | 窒素化合物の供給（肥料の節約） |
| 細菌側の利益 | 植物から光合成産物（糖類）の供給 |
| 特殊タンパク質 | **レグヘモグロビン**が根粒内の酸素濃度を調節し、ニトロゲナーゼを保護 |

> **問：マメ科植物の根に形成される瘤状の構造を何というか？また、そこに共生する細菌は何か？**
> → **根粒**（こんりゅう）。共生する細菌は**根粒菌**（Rhizobium など）。大気中のN₂を固定してアンモニアを供給する。

> **問：ハンノキ属（カバノキ科）も窒素固定を行うが、根粒菌は根粒菌（Rhizobium）ではない。何が共生するか？**
> → **フランキア**（*Frankia*）という放線菌が共生する。

## ネムノキの就眠運動

### [ネムノキ](https://ja.wikipedia.org/wiki/ネムノキ) *Albizia julibrissin*（試験重要度 ★★★）

| 部位/特徴 | 内容 |
|-----------|------|
| 葉 | 2回羽状複葉、小葉は多数 |
| 就眠運動 | 夜間・暗くなると小葉を閉じる（睡眠運動） |
| 花 | 淡紅色の糸状の雄しべが多数目立つ（ネムノキ亜科の特徴） |
| 果実 | 扁平な豆果 |
| 分布 | 本州〜沖縄、暖温帯の林縁 |

> **問：ネムノキが夜に葉を閉じる現象を何というか？**
> → **就眠運動**（睡眠運動）。膨圧変化により葉枕（ようちん）が動く。「眠りの木」が名前の由来。

## サイカチ（ジャケツイバラ亜科）の特徴

### [サイカチ](https://ja.wikipedia.org/wiki/サイカチ) *Gleditsia japonica*（試験重要度 ★★★）

| 部位/特徴 | 内容 |
|-----------|------|
| 棘 | **幹・枝に巨大な棘**（枝分かれした複棘）が発達。茎変形。 |
| 葉 | 1〜2回羽状複葉 |
| 果実 | **扁平な大型莢果**（長さ20〜30cm）、サポニンを含む |

### 問7
クズ（Pueraria montana var. lobata）が「秋の七草」である一方、どの国で侵略的外来種として問題になっているか。

### 問8
次のうち、翌年結実（花が咲いた翌年の秋に果実が熟す）するコナラ属の種はどれか。
① コナラ　② ミズナラ　③ アラカシ　④ クヌギ

<details>
<summary>解答を見る</summary>

**③ アラカシ**

アカガシ節の常緑種（アラカシ、シラカシ、アカガシ、ウラジロガシなど）は花から果実まで約2年かかる（翌年結実）。コナラ、ミズナラ、クヌギなどコナラ節の落葉種は当年結実。

</details>
