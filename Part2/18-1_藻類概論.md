## はじめに

[藻類](https://ja.wikipedia.org/wiki/藻類)（Algae）は、光合成を行う生物のうち、陸上植物以外のものの総称です。系統的には多様な群を含み、「藻類」は分類群ではなく便宜的なグループです。

---

## 藻類の概要

### 基本情報

| 項目 | 内容 |
|------|------|
| 種数 | 数万種（推定） |
| 生育環境 | 海水、淡水、湿地、土壌 |
| 体制 | 単細胞～多細胞 |
| 分類 | 複数の界・門にまたがる |

### 藻類の定義

| 特徴 | 内容 |
|------|------|
| 光合成 | 酸素発生型光合成を行う |
| 体制 | 維管束をもたない |
| 生殖 | 胚を形成しない |
| 生態 | 主に水中生活 |

> 📝 **藻類は多系統群**：藻類は進化的に多様な起源をもち、緑藻、紅藻、褐藻などは互いに近縁ではない

---

## 藻類の系統分類

### 主要な藻類のグループ

| 門（または綱） | 和名 | 色素体の起源 |
|----------------|------|--------------|
| Cyanobacteria | [藍藻](https://ja.wikipedia.org/wiki/藍藻) | 原核生物 |
| Chlorophyta | [緑藻](https://ja.wikipedia.org/wiki/緑藻) | 一次共生 |
| Rhodophyta | [紅藻](https://ja.wikipedia.org/wiki/紅藻) | 一次共生 |
| Ochrophyta | [黄色藻](https://ja.wikipedia.org/wiki/不等毛藻)（褐藻など） | 二次共生 |
| Dinophyta | [渦鞭毛藻](https://ja.wikipedia.org/wiki/渦鞭毛藻) | 二次共生 |
| Euglenophyta | [ユーグレナ藻](https://ja.wikipedia.org/wiki/ユーグレナ藻) | 二次共生 |

### 色素体の進化

```
一次共生（藍藻の取り込み）
├── 紅藻
├── 緑藻 → 陸上植物
└── 灰色藻

二次共生（紅藻または緑藻の取り込み）
├── 褐藻、珪藻など（紅藻由来）
├── 渦鞭毛藻（複数の起源）
└── ユーグレナ藻（緑藻由来）
```

## 1. 藍藻（シアノバクテリア）Cyanobacteria

### 基本情報

| 項目 | 内容 |
|------|------|
| 細胞構造 | **原核生物** |
| 光合成色素 | クロロフィルa、フィコシアニン |
| 色 | 青緑色 |
| 生態 | 水中、土壌、岩上、共生 |

### 形態的特徴

| 特徴 | 内容 |
|------|------|
| 核 | 核膜なし（原核） |
| 葉緑体 | なし（細胞質にチラコイド） |
| 窒素固定 | 一部の種が可能（ヘテロシスト） |

### 日本産主要種

| 和名 | 学名 | 特徴 |
|------|------|------|
| [アオコ](https://ja.wikipedia.org/wiki/アオコ) | *Microcystis* spp. | 水の華、毒素産生 |
| [ネンジュモ](https://ja.wikipedia.org/wiki/ネンジュモ) | *Nostoc* spp. | 数珠状、窒素固定 |
| [イシクラゲ](https://ja.wikipedia.org/wiki/イシクラゲ) | *Nostoc commune* | 陸生、食用 |
| [スイゼンジノリ](https://ja.wikipedia.org/wiki/スイゼンジノリ) | *Aphanothece sacrum* | 淡水産、食用、絶滅危惧 |
| [アナベナ](https://ja.wikipedia.org/wiki/アナベナ) | *Anabaena* spp. | 窒素固定、アカウキクサと共生 |
| [スピルリナ](https://ja.wikipedia.org/wiki/スピルリナ) | *Spirulina* spp. | らせん状、健康食品 |

### アオコの問題

| 項目 | 内容 |
|------|------|
| 原因 | 富栄養化 |
| 毒素 | ミクロシスチン（肝臓毒） |
| 被害 | 家畜の中毒死、飲料水汚染 |
| 対策 | 栄養塩の削減 |

> ⚠️ **アオコの毒性**：ミクロシスチンは肝臓毒で、WHOにより飲料水中の基準値が設定されている

## 【重要】クロロフィル・色素比較表（試験最頻出）

| グループ | クロロフィル | 主な補助色素 | 貯蔵物質 | 細胞壁主成分 |
|----------|-------------|-------------|---------|------------|
| 藍藻（シアノバクテリア） | **クロロフィルa のみ** | フィコシアニン、フィコエリスリン | グリコーゲン | ペプチドグリカン |
| 緑藻 | **クロロフィルa + b** | カロテン、キサントフィル | デンプン | セルロース |
| 紅藻 | **クロロフィルa のみ** | フィコエリスリン（赤色）、フィコシアニン | フロリドース（紅藻デンプン） | セルロース |
| 褐藻 | **クロロフィルa + c** | フコキサンチン（褐色） | マンニトール、ラミナリン | アルギン酸 |
| 珪藻 | **クロロフィルa + c** | フコキサンチン（褐色） | 油（脂質） | 珪酸（SiO₂） |
| 渦鞭毛藻 | **クロロフィルa + c** | ペリジニン（特有） | デンプン、油 | セルロース板（鎧板） |
| ユーグレナ藻 | **クロロフィルa + b** | カロテノイド | パラミロン | なし（ペリプラスト） |

> ⚠️ **試験必出ポイント：** 「緑藻 = a+b（陸上植物と同じ）」「褐藻・珪藻 = a+c」「紅藻 = a のみ（+フィコエリスリン）」を確実に覚える。

## 【重要】藻類の生活環タイプ比較表

| タイプ | 説明 | 代表種 |
|--------|------|--------|
| **同形世代交代** | 胞子体と配偶体が外見上同じ形 | アオサ（*Ulva*）、アオノリ |
| **異形世代交代（胞子体優占）** | 胞子体が大型、配偶体が小型微細 | コンブ、ワカメ、アラメ、カジメ |
| **異形世代交代（配偶体優占）** | 配偶体が主体、胞子体が微細な糸状体 | アサクサノリ（*Pyropia*） |
| **減数分裂直後（胞子体なし）** | ほぼ配偶体のみ | 接合藻類（アオミドロなど）|

```
コンブ・ワカメの生活環（異形世代交代）：
　大型胞子体（2n）
　　↓ 減数分裂 → 遊走子（n）放出
　微細糸状配偶体（n）
　　↓ 卵・精子形成 → 受精
　幼胞子体 → 大型胞子体（2n）

アサクサノリの生活環（異形世代交代）：
　葉状体（配偶体、n）← 食用の「海苔」
　　↓ 精子・卵 → 受精 → 接合胞子
　糸状体（コンコセリス、胞子体、2n）← 貝殻内に潜入
　　↓ 殻胞子形成・放出
　葉状体（n）← 秋に発芽

アオサの生活環（同形世代交代）：
　胞子体（2n） ←→ 配偶体（n） ← 外見上同じ形
```

## 各主要種 詳細解説

### [コンブ](https://ja.wikipedia.org/wiki/コンブ)（マコンブ） *Saccharina japonica*

#### 基本情報

| 項目 | 内容 |
|------|------|
| 分類 | 褐藻綱 コンブ目 コンブ科 |
| 色素 | クロロフィルa + c、フコキサンチン |
| 生育環境 | 冷水域の岩礁（水深5〜20m） |
| 分布 | 北海道〜東北（太平洋・日本海側） |
| 体長 | 2〜8m（国内最大級の海藻） |

#### 形態的特徴

- **藻体の構造**：付着器（岩に固着）→ 茎状部（柄）→ 葉状部（光合成部位）
- **葉状部**：幅20〜30cm、縁辺が波打ち、茶褐色（加熱すると緑色に変化）
- **中帯部**：葉状部中央の肉厚な部分（栄養分が多く蓄積）
- **メカブなし**：ワカメと異なり、胞子葉は葉状部表面（子嚢斑）に形成

#### 生活環（異形世代交代）

```
【胞子体】大型（食用部分、2n）
　↓ 秋〜冬に子嚢斑形成
　↓ 遊走子（2本鞭毛、n）放出
【配偶体】糸状・微細（顕微鏡サイズ、n）
　↓ 雌雄配偶体が精子・卵を形成
　↓ 受精
【幼胞子体】（2n）→ 急成長
【胞子体】（食用部分）
```

- **胞子体が主体**（食べているのは胞子体）
- **配偶体は微細な糸状体**（肉眼では見えない）

#### 化学成分と利用

| 成分 | 役割・用途 |
|------|----------|
| **グルタミン酸（うまみ成分）** | 昆布だし（旨味の主成分） |
| **アルギン酸** | 細胞壁多糖→増粘剤・安定剤として食品・医薬品に利用 |
| **マンニトール** | 糖アルコール（炭素貯蔵・浸透圧調節）→干し昆布の白い粉の正体 |
| **ラミナリン** | 貯蔵多糖 |
| **ヨード（ヨウ素）** | 海藻中最大の含有量、甲状腺ホルモン原料 |

#### なぜ重要か / 利用・生態的意義

- 日本の食文化の基礎「だし」の原料（昆布だし）
- 北海道が主産地（利尻昆布・羅臼昆布・日高昆布・真昆布など産地で品種が異なる）
- アルギン酸は食品添加物・医療材料（人工透析膜など）に重要
- コンブ林（ケルプ林）は多くの魚介類の産卵・育成場所として機能

#### 試験頻出問題

> **問：コンブの干し昆布の表面に出る白い粉の正体は何か？**
> → **答え：マンニトール（糖アルコール）**。コンブが浸透圧調節に使うが、乾燥で析出する。

> **問：コンブの生活環における「食べている部分」はどの世代か？**
> → **答え：胞子体（2n）**。コンブの胞子体が大型で食用となる。配偶体は糸状微細で肉眼不可。

### 珪藻 Bacillariophyceae

#### 基本情報

| 項目 | 内容 |
|------|------|
| 分類 | ストラメノパイル（不等毛植物）珪藻綱 |
| 色素 | クロロフィルa + c、フコキサンチン |
| 貯蔵物質 | 油（脂質）・クリソラミナリン |
| 生育環境 | 海水・淡水のプランクトン、底生 |
| 体制 | 単細胞（まれに群体） |

#### 形態的特徴（最重要）

- **珪藻殻（frustule）**：珪酸（SiO₂・nH₂O）からなるガラス質の細胞壁
  - **上殻（epivalve）** + **下殻（hypovalve）** = 弁当箱のような構造
  - 表面に種固有の精巧な微細構造（胞紋）→ 分類の重要形質
- **細胞分裂**：分裂するたびに細胞が小さくなる（上殻側が大きく、新しい下殻を形成するため）
- 鞭毛はない（遊走子形成時を除く）

#### 光合成と生態

- フコキサンチンにより褐色〜黄褐色を呈する
- 緑色光を効率的に吸収（海中での光合成に適応）
- 地球の光合成の約**25%**を担う（最重要な一次生産者）
- 珪藻土（化石化した殻が堆積）→ ダイナマイトの吸着剤（ノーベル発明）、ろ過材、断熱材

#### なぜ重要か / 利用・生態的意義

- **ダイナマイト（1867年）**：ノーベルがニトログリセリンを珪藻土に染み込ませて安定化
- **ろ過材**：ビール・ワインなどの工業ろ過
- **断熱材・建材**：多孔質構造による優れた断熱性
- **放散虫との違い**：放散虫は二酸化ケイ素の殻を持つが、原生動物（アメーバの仲間）で光合成しない

#### 試験頻出問題

> **問：珪藻の細胞壁の主成分と、その構造の名称は？**
> → **答え：珪酸（SiO₂）からなるガラス質の「珪藻殻（frustule）」。上殻（epivalve）と下殻（hypovalve）が組み合わさる。**

> **問：珪藻の光合成色素のうち、褐色の原因となる補助色素は何か？**
> → **答え：フコキサンチン（褐藻と同じ）。クロロフィルa + c とフコキサンチンを持つ。**

> **問：珪藻土を最初に工業利用したのは誰か、何に使ったか？**
> → **答え：アルフレッド・ノーベル。ニトログリセリンを珪藻土に染み込ませてダイナマイトを発明（1867年）。**

#### 基本情報

| 項目 | 内容 |
|------|------|
| 分類 | 緑藻植物門 接合藻綱（Zygnematophyceae） |
| 色素 | クロロフィルa + b |
| 生育環境 | 淡水（池・水田・流水） |

> **問：アオミドロの生活環で二倍体（2n）の段階はどれか？**
> → **答え：接合子（zygospore）のみ**。通常は単相（n）で、接合後に2n接合子→減数分裂で元に戻る。

| 褐虫藻（共生） | サンゴ・シャコガイに共生（*Symbiodinium*属）→サンゴ白化と関連 |

#### 赤潮との関係

- **カレニア・ミキモトイ（*Karenia mikimotoi*）**：日本沿岸の主要赤潮原因種、魚類大量死
- 赤潮は渦鞭毛藻だけでなく珪藻・珪藻などでも起きるが、毒素産生は渦鞭毛藻が多い

#### 試験頻出問題

> **問：渦鞭毛藻の2本の鞭毛はそれぞれどのような役割を担うか？**
> → **答え：横鞭毛（横溝沿い、回転）と縦鞭毛（縦溝から後方、推進）。二種の鞭毛で複雑な運動が可能。**

> **問：渦鞭毛藻がサンゴと共生するときの呼び名は？**
> → **答え：褐虫藻（ゾオキサンテラ、Symbiodinium）**。サンゴが白化すると褐虫藻が失われる。

## 【強化版】章末確認問題（4択形式）

### 問1（クロロフィル）
藻類グループと光合成色素の組み合わせとして正しいのはどれか。

A. 緑藻：クロロフィルa + c  
B. 褐藻：クロロフィルa + b  
C. 紅藻：クロロフィルa + フィコエリスリン  
D. 珪藻：クロロフィルa + b  

<details>
<summary>解答を見る</summary>

**C. 紅藻：クロロフィルa + フィコエリスリン**

緑藻=a+b、褐藻=a+c+フコキサンチン、珪藻=a+c+フコキサンチン、紅藻=aのみ（+フィコエリスリン）

</details>

### 問2（コンブの生活環）
コンブの生活環について正しい記述はどれか。

A. 食用とするコンブの藻体は配偶体（n）である  
B. コンブは同形世代交代を行う  
C. コンブの配偶体は肉眼で見える大型の糸状体である  
D. 食用とするコンブの藻体は胞子体（2n）であり、配偶体は微細な糸状体である  

<details>
<summary>解答を見る</summary>

**D. 食用とするコンブの藻体は胞子体（2n）であり、配偶体は微細な糸状体である**

コンブは異形世代交代（胞子体優占型）。大型の胞子体が食用部分。配偶体は顕微鏡サイズの糸状体。

</details>

### 問3（紅藻の特徴）
紅藻に関する特徴として誤っているのはどれか。

A. 鞭毛を完全に欠く  
B. フィコエリスリン（フィコエリスリン）を含む  
C. クロロフィルbを含む  
D. 不動精子によって受精する  

<details>
<summary>解答を見る</summary>

**C. クロロフィルbを含む**

紅藻はクロロフィルaのみ（bは持たない）。b を持つのは緑藻・陸上植物・ユーグレナ藻。

</details>

### 問4（珪藻の細胞壁）
珪藻の細胞壁（珪藻殻）に関する記述として正しいのはどれか。

A. 主成分はセルロースである  
B. 上殻と下殻に分かれ、主成分は珪酸（SiO₂）である  
C. アルギン酸が主成分で柔軟である  
D. ペプチドグリカンを含む  

<details>
<summary>解答を見る</summary>

**B. 上殻と下殻に分かれ、主成分は珪酸（SiO₂）である**

珪藻殻（frustule）は上殻（epivalve）と下殻（hypovalve）からなるガラス質（SiO₂）の構造。

</details>

### 問5（赤潮・アオコ）
赤潮とアオコについて正しい記述はどれか。

A. アオコは渦鞭毛藻が主に引き起こし、ミクロシスチンを産生する  
B. 赤潮の主な原因藻類は藍藻である  
C. アオコは富栄養化した湖沼で藍藻が異常増殖して起きる  
D. 珪藻は赤潮を引き起こすが毒素を産生しない  

<details>
<summary>解答を見る</summary>

**C. アオコは富栄養化した湖沼で藍藻が異常増殖して起きる**

アオコ=藍藻（*Microcystis*等）、ミクロシスチン（肝毒）産生。赤潮は渦鞭毛藻などが主因。

</details>

### 問6（アサクサノリの生活環）
アサクサノリ（*Pyropia tenera*）の生活環について正しいのはどれか。

A. 食用部分（葉状体）は胞子体（2n）である  
B. 糸状体（コンコセリス）は配偶体（n）であり、夏を海中で過ごす  
C. 葉状体（配偶体、n）から受精後に糸状体（胞子体、2n）が生じる  
D. コンブと同様に胞子体が主体の生活環をもつ  

<details>
<summary>解答を見る</summary>

**C. 葉状体（配偶体、n）から受精後に糸状体（胞子体、2n）が生じる**

アサクサノリ：葉状体＝配偶体（n、食用）→受精→糸状体（コンコセリス）＝胞子体（2n）→殻胞子→葉状体。コンブとは主体が逆。

</details>

### 問7（シャジクモ）
陸上植物に系統的に最も近縁な藻類のグループはどれか。

A. 紅藻  
B. 褐藻  
C. 車軸藻類（シャジクモ類）  
D. 渦鞭毛藻  

<details>
<summary>解答を見る</summary>

**C. 車軸藻類（シャジクモ類）**

シャジクモ類はストレプト植物（Streptophyta）に属し、陸上植物と同じクレードを形成する最近縁の藻類。フラグモプラスト型細胞分裂など、陸上植物と共有する特徴が多い。

</details>
