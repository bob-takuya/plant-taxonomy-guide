# 12-5. その他のマメ類（ウリ目・ニシキギ目・カタバミ目・キントラノオ目）

[← 目次に戻る](../NEW_00_目次.md) | [← 前：ブナ目](./12-4_ブナ目.md)

## はじめに

マメ類（Fabids / Eurosids I）には、マメ目・バラ目・ブナ目以外にも多くの目が含まれます。本章では[ウリ目](https://ja.wikipedia.org/wiki/ウリ目)、[ニシキギ目](https://ja.wikipedia.org/wiki/ニシキギ目)、[カタバミ目](https://ja.wikipedia.org/wiki/カタバミ目)、[キントラノオ目](https://ja.wikipedia.org/wiki/キントラノオ目)などを解説します。

## 1. ウリ目 Cucurbitales

### 概要

| 項目 | 内容 |
|------|------|
| 日本産科数 | 4科 |
| 世界の科数 | 8科 |
| 主な特徴 | 巻きひげをもつつる性植物が多い |

### 1-1. ウリ科 Cucurbitaceae

#### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約12属 |
| 日本産種数 | 約20種 |

| 部位 | 特徴 |

#### 基本情報

|------|------|
| 日本産属数 | 7属 |
| 日本産種数 | 約30種 |

| [ツルウメモドキ](https://ja.wikipedia.org/wiki/ツルウメモドキ) | *Celastrus orbiculatus* | 北海道～九州 | つる性、黄色い果皮と赤い仮種皮 |
| [ウメモドキ](https://ja.wikipedia.org/wiki/ウメモドキ) | *Ilex serrata*（モチノキ科） | - | **注意：モチノキ科**（名前は似るが別科） |
| [ツルマサキ](https://ja.wikipedia.org/wiki/ツルマサキ) | *Euonymus fortunei* | 北海道～九州 | つる性、常緑、付着根で登攀 |

| 特徴 | ニシキギ | コマユミ |

| 分類 | 基本種 | 品種 |
| 別名 | カミソリノキ | - |

## 4. キントラノオ目 Malpighiales

### 概要

キントラノオ目は被子植物で最大級の目の一つで、非常に多様な科を含みます。

| 項目 | 内容 |
|------|------|
| 日本産科数 | 9科 |
| 世界の科数 | 36科 |
| 世界の種数 | 約16,000種 |

### 4-1. トウダイグサ科 Euphorbiaceae

#### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約15属 |
| 日本産種数 | 約50種 |
| 世界の属数 | 約300属 |
| 世界の種数 | 約6,000種 |

#### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 草本または木本 |
| 乳液 | 多くの種で白い乳液を含む |
| 葉 | 互生または対生、托葉あり |
| 花 | 単性花、退化傾向（花被がない種も） |
| 果実 | [蒴果](https://ja.wikipedia.org/wiki/蒴果)（3分果）、弾ける |

#### 日本産主要種

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ナツトウダイ](https://ja.wikipedia.org/wiki/ナツトウダイ) | *Euphorbia sieboldiana* | 北海道～九州 | 春に咲く（名前に反し）、杯状花序 |
| [タカトウダイ](https://ja.wikipedia.org/wiki/タカトウダイ) | *Euphorbia lasiocaula* | 本州～九州 | 草丈1m近くなる大型種 |
| [トウダイグサ](https://ja.wikipedia.org/wiki/トウダイグサ) | *Euphorbia helioscopia* | 全国 | 帰化種、路傍に普通 |
| [ノウルシ](https://ja.wikipedia.org/wiki/ノウルシ) | *Euphorbia adenochlora* | 本州～九州 | 湿地性、黄色い苞葉、準絶滅危惧 |
| [エノキグサ](https://ja.wikipedia.org/wiki/エノキグサ) | *Acalypha australis* | 全国 | 畑の雑草、苞が特徴的 |
| [オオニシキソウ](https://ja.wikipedia.org/wiki/オオニシキソウ) | *Euphorbia nutans* | 帰化 | 茎が立ち上がる |
| [コニシキソウ](https://ja.wikipedia.org/wiki/コニシキソウ) | *Euphorbia supina* | 帰化 | 匍匐性、葉に斑紋あり |
| [ヤマアイ](https://ja.wikipedia.org/wiki/ヤマアイ) | *Mercurialis leiocarpa* | 本州～九州 | 古代の染料植物、対生葉 |
| [アカメガシワ](https://ja.wikipedia.org/wiki/アカメガシワ) | *Mallotus japonicus* | 本州～沖縄 | 先駆樹種、新芽が赤い |
| [シラキ](https://ja.wikipedia.org/wiki/シラキ) | *Neoshirakia japonica* | 本州～九州 | 紅葉が美しい、乳液少ない |
| [ナンキンハゼ](https://ja.wikipedia.org/wiki/ナンキンハゼ) | *Triadica sebifera* | 帰化（中国原産） | 街路樹、白い種子から蝋 |

> ⚠️ **注意**：トウダイグサ科の乳液は有毒。皮膚につくとかぶれることがある。

#### 杯状花序（トウダイグサ属）

[トウダイグサ属](https://ja.wikipedia.org/wiki/トウダイグサ属)は独特の[杯状花序](https://ja.wikipedia.org/wiki/杯状花序)（Cyathium）をもつ。

| 構造 | 説明 |
|------|------|
| 総苞 | 杯状に合着した苞葉 |
| 腺体 | 総苞縁に蜜腺（三日月形など） |
| 雄花 | 多数、雄しべ1本ずつが1花に相当 |
| 雌花 | 中央に1個、3心皮の子房 |

### 4-3. スミレ科 Violaceae

#### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 1属 |
| 日本産種数 | 約60種 |
| 世界の属数 | 約25属 |
| 世界の種数 | 約800種 |

#### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 多年草（まれに1年草） |
| 葉 | 互生または根生、托葉あり |

| 分類群 | 特徴 | 代表種 |

| 無茎種 | 地上茎なし、葉は根生 | スミレ、ノジスミレ |

| 和名 | 学名 | 分布 | 特徴 |

| [ツボスミレ](https://ja.wikipedia.org/wiki/ツボスミレ) | *Viola verecunda* | 全国 | 白花に紫条、湿った場所 |

## ウリ科の子房下位（試験重要度 ★★★★）

> **問：ウリ科の子房の位置は上位か下位か？これは試験でどのように問われるか？**
> → **下位子房**（果実が「萼の下」に形成）。子房下位を持つ重要な科として頻出。
> 「ウリ科＝下位子房」はアブラナ科（上位子房）との比較で出題される。

### ウリ科の重要特徴まとめ

| 特徴 | 内容 | 試験重要度 |
|------|------|------------|
| 子房の位置 | **下位** | ★★★★★ |
| 花の性 | **単性花**（雌雄同株が多い、一部異株） | ★★★★ |
| 巻きひげ | 葉腋から出る（葉の変形） | ★★★ |
| 花弁 | **合弁花**、5裂（または5枚） | ★★★ |
| 果実 | **液果**（ウリ状果）、大型のものが多い | ★★★ |

## スミレ科の詳細（試験重要度 ★★★★）

### 距（きょ）とは

> **問：スミレ科の花の最も重要な識別特徴は何か？**
> → **距（きょ）**。下弁（唇弁）の基部が袋状・筒状に伸びた構造で、蜜を分泌する。距の有無・形・色がスミレ属の識別に重要。

### タチツボスミレ vs スミレ（試験頻出）

| 特徴 | タチツボスミレ（*Viola grypoceras*） | スミレ（*Viola mandshurica*） |
|------|--------------------------------------|-------------------------------|
| 地上茎 | **あり**（有茎種） | **なし**（無茎種、葉は根生） |
| 葉の形 | **丸い心形** | **細長い披針形〜細長三角形** |
| 葉柄の翼 | なしまたは微翼 | **翼が明瞭** |
| 托葉 | 羽状に深裂する | 全縁〜細鋸歯 |
| 花色 | 淡紫色 | **濃紫色** |
| 分布 | 全国（最普通種） | 全国（道端・草地） |
| 距の色 | 淡紫色 | 同色 |

> **問：タチツボスミレとスミレは何で区別できるか？最も重要な特徴を答えよ。**
> → **地上茎の有無**（タチツボスミレ＝有茎種、スミレ＝無茎種）と**葉の形**（タチツボスミレは丸い、スミレは細長い）。

### 閉鎖花（閉鎖性花・へいさか）

> **問：スミレ科の多くの種が形成する「閉鎖花」とは何か？その意義を述べよ。**
> → **閉鎖花**：蕾のまま開かずに自家受粉して結実する花。春の開放花が受粉できないときの保険として、夏〜秋に閉鎖花を大量に作り確実に種子を残す。種子散布はアリが行う（myrmecochory：アリ散布）。

### エイザンスミレの識別

| 部位 | 特徴 |
|------|------|
| 葉 | **掌状に3深裂**（さらに各裂片が羽状に裂ける）→ 独特の形 |
| 花 | 淡紅色、距は白色 |
| 分布 | 本州〜九州（比叡山に多い） |

## フウロソウ科の詳細（試験重要度 ★★★）

### 嘴状果実の構造

フウロソウ科の果実は**蒴果で嘴状に長く伸びる**。熟すと下から上へ反り返り種子を弾き飛ばす。

> **問：属名 Geranium（ゲラニウム）とはどういう意味で、何に由来するか？**
> → **ギリシャ語で「鶴」の意**。果実の嘴状の形が鶴のくちばしに似ることから。

### ゲンノショウコ詳細

### [ゲンノショウコ](https://ja.wikipedia.org/wiki/ゲンノショウコ) *Geranium thunbergii*（試験重要度 ★★★）

| 部位/特徴 | 内容 |
|-----------|------|
| 薬効 | **下痢止め・整腸薬**（葉・茎を乾燥させた生薬「現草」） |
| 名前の由来 | 「現の証拠」＝飲むとすぐに効く（目に見える証拠） |
| 花の色 | 西日本は**白花**が多く、東日本は**紅紫色**が多い傾向 |
| 果実 | 熟すと**神輿（みこし）のような形**に5分果が跳ね上がる。「みこし草」の別名 |
| 分布 | 全国（草地・道端） |

> **問：ゲンノショウコの薬効は何か？また「現の証拠」という名前の由来を説明せよ。**
> → **整腸薬・下痢止め**として使用。「飲めば現（すぐ）に効果が証明される薬草」という意味。

## ミソハギ科の詳細（試験重要度 ★★★）

### サルスベリ詳細

### [サルスベリ](https://ja.wikipedia.org/wiki/サルスベリ) *Lagerstroemia indica*（試験重要度 ★★★）

| 部位/特徴 | 内容 |
|-----------|------|
| 樹皮 | **なめらか・つるつる**（「サルも滑る」が名前の由来） |
| 花 | **皺のある花弁**（縮れた縁）が特徴。紅・白・紫色。夏〜秋に長期間開花 |
| 花弁数 | 6枚（ミソハギ科の特徴を示す） |
| 葉序 | **コクサギ型葉序**（2枚ずつ同側につく互生）を示す |
| 原産 | 中国原産、栽培・逸出 |
| 別名 | 百日紅（ひゃくじつこう）＝長期間咲く |

> **問：サルスベリの樹皮の特徴と名前の由来を説明せよ。**
> → 樹皮が**滑らか（つるつる）**で皮が剥がれる。猿も滑って登れないほど滑らかなことから「猿滑り」。

### ミソハギ科の花弁の特徴

> **問：ミソハギ科（Lythraceae）の花弁の形態的特徴を答えよ。**
> → **花弁に皺（しわ）がある**（crumpled petals）。蕾の中で折り畳まれているため。サルスベリの縮れた花弁が典型例。
