## はじめに

[被子植物](https://ja.wikipedia.org/wiki/被子植物)（Angiosperms）は、現在の地球上で最も繁栄している植物群で、約30万種以上が知られています。日本には約5,500種の被子植物が自生し、生物分類技能検定の専門問題の中心となります。

---

## 被子植物の特徴

### 基本的な特徴

| 特徴 | 内容 |
|------|------|
| 種子形成 | 胚珠が子房に包まれる（「被子」の由来） |
| [重複受精](https://ja.wikipedia.org/wiki/重複受精) | 精細胞が卵細胞と中央細胞の両方と受精 |
| [胚乳](https://ja.wikipedia.org/wiki/胚乳) | 三倍体の胚乳を形成（種子の栄養組織） |
| [導管](https://ja.wikipedia.org/wiki/道管) | 道管を持つ（仮道管のみの種もある） |
| 花 | 花被・雄しべ・雌しべからなる生殖器官 |

### [裸子植物](https://ja.wikipedia.org/wiki/裸子植物)との違い

| 特徴 | 被子植物 | 裸子植物 |
|------|----------|----------|
| 胚珠 | 子房に包まれる | 裸出 |
| 受精 | 重複受精 | 単受精 |
| 胚乳 | 三倍体（3n） | 単相体（n） |
| 導管 | 道管を持つ | [仮道管](https://ja.wikipedia.org/wiki/仮道管)のみ |
| 花被 | あり | なし（多くの場合） |

## 被子植物の進化と多様化

### 進化の歴史

| 時代 | 出来事 |
|------|--------|
| 約1億4000万年前 | 被子植物の出現（白亜紀前期） |
| 約1億年前 | 急速な多様化（被子植物革命） |
| 約6600万年前 | K-Pg境界後さらに多様化 |
| 現在 | 約30万種以上が存在 |

### 繁栄の要因

1. **重複受精と胚乳**
   - 効率的な種子の栄養供給

2. **花と送粉者の共進化**
   - 昆虫、鳥類との相互作用

3. **多様な生活型**
   - 草本、木本、つる性、水生など

4. **道管の発達**
   - 効率的な水分輸送

## 被子植物の形態的多様性

### 生活型

| 生活型 | 特徴 | 例 |
|--------|------|-----|
| 高木 | 幹を持つ木本、6m以上 | サクラ、ブナ |
| 低木 | 幹を持つ木本、6m以下 | ツツジ、アジサイ |
| つる性 | 他物に絡まる | フジ、アケビ |
| 多年草 | 地上部が枯れても地下部が残る | ススキ、ユリ |
| 一年草 | 1年で生活環を完了 | イネ、アサガオ |
| 水生植物 | 水中または水辺に生育 | スイレン、ハス |

### 花の多様性

| 特徴 | 変異 | 例 |
|------|------|-----|
| 対称性 | 放射相称、左右相称 | キク、スミレ |
| 花被 | 萼と花弁が分化、未分化 | サクラ、チューリップ |
| 合着 | 離弁、合弁 | バラ、キキョウ |
| 花序 | 単生、総状、頭状など | モクレン、フジ、タンポポ |

## 試験対策のポイント

### 最重要ポイント

1. **APG体系の系統関係**
   - アンボレラが最基部
   - 真正双子葉類の三溝粒花粉
   - バラ類とキク類の区別

2. **科の再編成**
   - ユリ科の分割
   - ゴマノハグサ科の再編成
   - カエデ科・トチノキ科の統合

3. **主要科の特徴**
   - 形態的特徴
   - 代表種
   - 学名（ラテン語）

### 学習のコツ

- 系統樹を視覚的に覚える
- 「目」のレベルで科をまとめて覚える
- 代表種から科の特徴を連想

## 確認問題

### 問1
被子植物と裸子植物の最も重要な違いは何か。

<details>
<summary>解答を見る</summary>

**胚珠が子房に包まれるかどうか**

被子植物は胚珠が子房に包まれ、裸子植物は胚珠が裸出している。

</details>

### 問2
被子植物に特有の受精様式は何か。

<details>
<summary>解答を見る</summary>

**重複受精**

一方の精細胞が卵細胞と受精して胚を形成し、もう一方の精細胞が中央細胞と受精して胚乳（3n）を形成する。

</details>

### 問3
APG IV体系で被子植物の最基部に位置する科は何か。

<details>
<summary>解答を見る</summary>

**アンボレラ科（Amborellaceae）**

ニューカレドニア固有の1属1種。

</details>

### 問4
真正双子葉類の花粉の特徴は何か。

<details>
<summary>解答を見る</summary>

**三溝粒（tricolpate）または派生型**

花粉に3つの溝（または孔）を持つ。

</details>
