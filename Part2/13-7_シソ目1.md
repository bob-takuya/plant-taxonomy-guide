# 13-7. シソ目1（シソ科・クマツヅラ科）

## はじめに

[シソ目](https://ja.wikipedia.org/wiki/シソ目)（Lamiales）は、シソ類で最大の目で、シソ科、ゴマノハグサ科、オオバコ科など多様な科を含みます。APG体系で旧ゴマノハグサ科が大幅に再編成されました。

---

## シソ目の概要

| 項目 | 内容 |
|------|------|
| 日本産科数 | 約13科 |
| 世界の科数 | 約25科 |
| 世界の種数 | 約23,000種 |
| 主な特徴 | 合弁花、左右相称の花が多い、イリドイドを含む |

## 1. シソ科 Lamiaceae（Labiatae）

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約40属 |
| 日本産種数 | 約150種 |
| 世界の属数 | 約240属 |
| 世界の種数 | 約7,000種 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 草本（まれに木本） |
| 茎 | **四角形**（断面が四角） |
| 葉 | **対生**、単葉 |
| 芳香 | 精油を含み芳香があるものが多い |
| 花 | 合弁、**唇形花冠**（上唇・下唇に分かれる） |
| 萼 | 筒状、5裂 |
| 雄しべ | 4（2強雄しべ）または2 |
| 子房 | 上位、4室（4分果となる） |
| 果実 | **4分果**（4つの小堅果に分かれる） |

> 📝 **識別キーワード**：四角い茎 ＋ 対生葉 ＋ 唇形花冠 ＋ 4分果

## 2. クマツヅラ科 Verbenaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約10属 |
| 日本産種数 | 約20種 |
| 世界の種数 | 約800種 |

> 📝 **分類の変更**：APG体系で多くの属がシソ科に移動し、縮小された

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 草本または木本 |
| 茎 | 四角形のものが多い |
| 葉 | 対生 |
| 花 | 合弁、左右相称 |
| 果実 | 核果または4分果 |

### 旧クマツヅラ科からシソ科へ移動した属

| 属名 | 現在の科 | 代表種 |

| ムラサキシキブ属 | シソ科 | ムラサキシキブ |

| クサギ属 | シソ科 | クサギ |
| ハマゴウ属 | シソ科 | ハマゴウ |

> 📝 **ムラサキシキブとコムラサキの違い**：
> - **ムラサキシキブ**：果実がまばらにつく、自生種
> - **コムラサキ**：果実が密集、庭木に多い（園芸では「ムラサキシキブ」と呼ばれることも）

## 確認問題

### 問1
シソ科の形態的特徴を4つ述べよ。

<details>
<summary>解答を見る</summary>

- **茎が四角形**
- **葉が対生**
- **唇形花冠**
- **果実が4分果**

</details>

### 問2
旧クマツヅラ科からシソ科に移動した属を2つ挙げよ。

<details>
<summary>解答を見る</summary>

以下から2つ：
- **ムラサキシキブ属**
- **クサギ属**
- **カリガネソウ属**
- **ハマゴウ属**

</details>

### 問3
シソ科の「ホトケノザ」と春の七草の「ホトケノザ」の違いを述べよ。

<details>
<summary>解答を見る</summary>

- **シソ科のホトケノザ**：*Lamium amplexicaule*、早春の紅紫色の花
- **春の七草のホトケノザ**：キク科のコオニタビラコ（*Lapsanastrum apogonoides*）

</details>

## 暗記用まとめ

### シソ科の識別

```
四角い茎 ＋ 対生葉 ＋ 唇形花冠 ＋ 4分果 → シソ科
```

### シソ科のハーブ

| 用途 | 植物名 |
|------|--------|
| イタリア料理 | バジル、オレガノ |
| 香辛料 | ローズマリー、セージ、タイム |
| 芳香・入浴 | ラベンダー |
| 飲料 | ミント |
| 和食 | シソ（紫蘇） |

### 分類変更のまとめ

| 属名 | 旧科 | 現科 |
|------|------|------|
| ムラサキシキブ属 | クマツヅラ科 | シソ科 |
| クサギ属 | クマツヅラ科 | シソ科 |
| ハマゴウ属 | クマツヅラ科 | シソ科 |
