## はじめに

[モクレン目](https://ja.wikipedia.org/wiki/モクレン目)は、モクレン類（Magnoliids）の中核をなす目です。かつては「原始的双子葉類」とされていましたが、APG体系では真正双子葉類とは別系統であることが明らかになりました。

---

## モクレン目の概要

| 項目 | 内容 |
|------|------|
| 目名 | Magnoliales |
| 日本産科数 | 1科（バンレイシ科は栽培のみ） |
| 主な特徴 | 多数の花被片・雄しべ、らせん配列 |

## [モクレン科](https://ja.wikipedia.org/wiki/モクレン科) Magnoliaceae

| 項目 | 内容 |
|------|------|
| 日本産属数 | 2属 |
| 日本産種数 | 約15種 |
| 世界の属数/種数 | 2属/約300種 |
| 分布 | 東アジア、北中米 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活型 | 落葉または常緑の高木・低木 |
| 葉 | 互生、単葉、全縁、托葉あり（落ちやすい） |
| 花 | 大型、花被片多数（らせん配列）、両性花 |
| 雄しべ | 多数、らせん配列 |
| 雌しべ | 多数、離生 |
| 果実 | 袋果が集まった集合果 |

### 重要種の詳細

#### [ホオノキ](https://ja.wikipedia.org/wiki/ホオノキ)（*Magnolia obovata*）

- **葉**：日本産広葉樹で最大級（長さ20〜40cm）
- **花**：白色で芳香、直径15cm以上
- **用途**：朴葉味噌、朴葉焼き、下駄の材料
- **分布**：北海道〜九州の山地

#### [コブシ](https://ja.wikipedia.org/wiki/コブシ)（*Magnolia kobus*）

- **開花期**：3〜4月（葉の展開前）
- **花**：白色、花弁状の花被片6枚
- **名前の由来**：果実が握りこぶしに似る
- **見分け方**：花の下に1枚の葉がつく

#### [シデコブシ](https://ja.wikipedia.org/wiki/シデコブシ)（*Magnolia stellata*）

- **分布**：[東海地方](https://ja.wikipedia.org/wiki/東海地方)の限られた湿地
- **保全状況**：**環境省レッドリスト絶滅危惧II類（VU）**
- **特徴**：花被片12〜18枚（コブシより多い）
- **脅威**：湿地開発、園芸採取

#### [オガタマノキ](https://ja.wikipedia.org/wiki/オガタマノキ)（*Magnolia compressa*）

- **特徴**：モクレン科で日本唯一の常緑樹
- **用途**：神社の神木（招霊の木）
- **分布**：関東以西〜沖縄の暖地

## モクレン類全体の系統的位置

```
モクレン類（Magnoliids）
│
├── センリョウ目 Chloranthales
│
├── モクレン目 Magnoliales ←【本章】
│   ├── モクレン科
│   └── バンレイシ科
│
├── クスノキ目 Laurales
│
└── コショウ目 Piperales
```

### 真正双子葉類との違い

| 特徴 | モクレン類 | 真正双子葉類 |
|------|------------|--------------|
| 花粉 | 単溝粒 | **三溝粒** |
| 花被片 | らせん配列が多い | 輪生が多い |
| 花弁数 | 不定 | 4または5の倍数 |

## 確認問題

### 問1
日本産モクレン科で最大の葉を持つ種は何か。

<details>
<summary>解答を見る</summary>

**ホオノキ（*Magnolia obovata*）**

葉の長さは20〜40cmにもなり、日本産広葉樹で最大級。

</details>

### 問2
東海地方の湿地に限定分布し、絶滅危惧種に指定されているモクレン科の植物は何か。

<details>
<summary>解答を見る</summary>

**シデコブシ（*Magnolia stellata*）**

環境省レッドリスト絶滅危惧II類（VU）。

</details>

### 問3
モクレン科で日本唯一の常緑樹は何か。

<details>
<summary>解答を見る</summary>

**オガタマノキ（*Magnolia compressa*）**

神社の神木として植えられる。

</details>

### 問4
コブシとタムシバの見分け方を述べよ。

<details>
<summary>解答を見る</summary>

**コブシ**は花の下に1枚の小さな葉がつくが、**タムシバ**にはつかない。また、タムシバの葉を噛むと甘い。

</details>
