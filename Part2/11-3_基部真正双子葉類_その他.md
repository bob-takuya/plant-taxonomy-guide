## ヤマモガシ目 Proteales

[ヤマモガシ目](https://ja.wikipedia.org/wiki/ヤマモガシ目)は、APG体系で大きく再編成された目です。

---

### [ハス科](https://ja.wikipedia.org/wiki/ハス科) Nelumbonaceae

| 項目 | 内容 |
|------|------|
| 日本産属数 | 1属 |
| 日本産種数 | 1種 |
| 世界の属数/種数 | 1属/2種 |
| 分布 | アジア、北米 |

| 種名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ハス](https://ja.wikipedia.org/wiki/ハス) | *Nelumbo nucifera* | 日本全土（栽培・野生化） | 大型の花、蓮根 |

**ハスの詳細**
- **史前帰化植物**と考えられる
- 根茎は[蓮根](https://ja.wikipedia.org/wiki/レンコン)として食用
- 種子は[大賀ハス](https://ja.wikipedia.org/wiki/大賀ハス)（2000年前）で有名
- [仏教](https://ja.wikipedia.org/wiki/仏教)と密接な関係
- **スイレン科とは別科**（APG体系で明確に）

**ハスとスイレンの違い**

| 特徴 | ハス（ハス科） | スイレン（スイレン科） |
|------|----------------|------------------------|
| 葉 | 水面より上に出る | 水面に浮く |
| 花托 | 蜂の巣状 | − |
| 根茎 | 蓮根（食用） | 食用としない |
| 系統 | ヤマモガシ目 | スイレン目 |

## ヤマグルマ目 Trochodendrales

[ヤマグルマ目](https://ja.wikipedia.org/wiki/ヤマグルマ目)は、原始的な形質を持つ木本植物の目です。

---

### [ヤマグルマ科](https://ja.wikipedia.org/wiki/ヤマグルマ科) Trochodendraceae

| 項目 | 内容 |
|------|------|
| 日本産属数 | 1属 |
| 日本産種数 | 1種 |
| 世界の属数/種数 | 2属/2種 |
| 分布 | 東アジア |

| 種名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ヤマグルマ](https://ja.wikipedia.org/wiki/ヤマグルマ) | *Trochodendron aralioides* | 本州〜沖縄 | 常緑高木、**道管なし** |

**ヤマグルマの特徴**
- **被子植物で道管を持たない**珍しい例
- 仮道管のみで水を通す（原始的形質）
- 日本固有種
- 常緑で輪生状の葉
- 着生することもある

## 関連植物：コショウ目の重要種（参考）

> ⚠️ 以下の植物はAPG体系では**モクレン類コショウ目（Piperales）**に属し、基部真正双子葉類ではありません。しかし、旧分類では近縁として扱われることが多く、試験でも問われる重要種のため参考として記載します。

### [ドクダミ](https://ja.wikipedia.org/wiki/ドクダミ) *Houttuynia cordata*

| 項目 | 内容 |
|------|------|
| 科 | ドクダミ科（Saururaceae） |
| 目 | コショウ目（Piperales） |
| クレード | **モクレン類**（Magnoliids） |
| 分布 | 日本全土（北海道〜沖縄）、東アジア |
| 生活型 | 多年草、地下茎で旺盛に繁殖 |

#### 形態的特徴

| 部位/特徴 | 内容 |
|-----------|------|
| 高さ | 20〜50cm |
| 葉 | 心形、互生、独特の臭気がある |
| 花 | 穂状花序（真の花）＋白い総苞片4枚（花弁状） |
| 花弁 | **花弁はなし**（白い部分は総苞片） |
| 雄しべ | 3本 |
| 臭気成分 | **デカノイルアセトアルデヒド**（主要臭気成分）、ラウリルアルデヒド等 |
| 別名 | **十薬**（じゅうやく）、ドクダメ、シブキ |

#### なぜ重要か（試験重要度 ★★★★）

- **「十薬」として有名**：10種類の薬効（利尿・解毒・排膿・抗菌等）があるとされる
- **臭気の正体**：主成分はデカノイルアセトアルデヒド（抗菌作用を持つ）
- 乾燥すると臭気成分が分解され、茶（どくだみ茶）として利用される
- 名前の由来：「毒を矯める（毒だみ）」→「ドクダミ」
- **白い部分は花弁ではなく総苞片**（試験頻出！）
- 生薬名「魚腥草（ぎょせいそう）」としても知られる

#### 試験頻出問題

> **問：** ドクダミの白い部分（白い花弁に見える部分）は何か。
> → **答え：** **総苞片（そうほうへん）**4枚。真の花は中央の穂状花序部分で、花弁を持たない。

> **問：** ドクダミの別名「十薬」の由来は何か。
> → **答え：** 十種類の薬効があることに由来するとされる（利尿・解毒・排膿・抗菌等）。

> **問：** ドクダミの主要臭気成分は何か。
> → **答え：** **デカノイルアセトアルデヒド**。抗菌作用を持つが、乾燥すると分解されて臭気が減少する。

## ハス科の重要種詳細

### [ハス](https://ja.wikipedia.org/wiki/ハス) *Nelumbo nucifera*

#### 形態的特徴

| 部位/特徴 | 内容 |
|-----------|------|
| 科・目 | ハス科（Nelumbonaceae）・ヤマモガシ目 |
| 葉の位置 | **水面より高く立ち上がる**（スイレンとの最大の違い） |
| 葉の形 | 盾形（葉柄が中央につく）、撥水性が強い（ロータス効果） |
| 花 | 大型（径20〜25cm）、白〜淡紅色 |
| 花托 | **蜂の巣状**（果実が埋まる独特の形状） |
| 果実 | 堅果、花托の穴に1個ずつ入る |
| 根茎 | **蓮根**として食用 |
| 種子 | 発芽力が長期間持続（2000年前の種子からの発芽記録：大賀ハス） |

#### 試験頻出問題

> **問：** ハスはAPG体系でどの目に分類されるか。また、スイレンとはどの目が異なるか。
> → **答え：** ハスは**ヤマモガシ目（Proteales）ハス科**。スイレンは**スイレン目（Nymphaeales）スイレン科**。全く異なる目に属する。

---

## 確認問題（4択形式）

**Q1.** ドクダミについて正しいものはどれか。
① 白い花弁が4枚ある　② APG体系では基部真正双子葉類に分類される　③ 白く見える部分は総苞片で、花弁は持たない　④ 臭気成分は加熱しても変化しない

<details><summary>答え</summary>**③ 白く見える部分は総苞片で、花弁は持たない** — ドクダミの白い部分（4枚）は総苞片。真の花は中央の穂状花序で花弁を欠く。APG体系ではコショウ目（モクレン類）に属する。</details>

**Q2.** ハスとスイレンの違いについて正しいものはどれか。
① ハスはスイレン目ハス科、スイレンはスイレン目スイレン科　② ハスの葉は水面に浮き、スイレンの葉は水面より上に出る　③ ハスはヤマモガシ目ハス科で根茎が食用になる　④ どちらも同じ目に属する

<details><summary>答え</summary>**③ ハスはヤマモガシ目ハス科で根茎が食用になる** — ハス（ヤマモガシ目ハス科）の葉は水面より上に立ち上がり、根茎（蓮根）が食用。スイレン（スイレン目スイレン科）の葉は水面に浮く。</details>

**Q3.** ヤマグルマについて正しいものはどれか。
① 被子植物で唯一花粉を持たない　② 被子植物で道管を持たず仮道管のみを持つ珍しい種　③ スイレン目に分類される　④ 北海道のみに分布する日本固有種

<details><summary>答え</summary>**② 被子植物で道管を持たず仮道管のみを持つ珍しい種** — ヤマグルマ（ヤマグルマ目ヤマグルマ科）は被子植物で道管を持たない。アンボレラ科も同様だが、ヤマグルマは日本産唯一の例。</details>

## 確認問題

### 問1
ハスとスイレンの違いを2つ述べよ。

<details>
<summary>解答を見る</summary>

- **葉の位置**：ハスは水面より上、スイレンは水面に浮く
- **分類**：ハスはヤマモガシ目ハス科、スイレンはスイレン目スイレン科

その他、ハスの花托は蜂の巣状になる。

</details>

### 問2
被子植物で道管を持たない珍しい例として知られる植物は何か。

<details>
<summary>解答を見る</summary>

**ヤマグルマ（*Trochodendron aralioides*）**

仮道管のみで水を通す原始的形質を持つ。ヤマグルマ科ヤマグルマ目。

</details>

### 問3
櫛や印鑑の材料として使われるツゲ科の植物は何か。

<details>
<summary>解答を見る</summary>

**ツゲ（*Buxus microphylla* var. *japonica*）**

材は緻密で硬く、細工物に適する。

</details>

### 問4
特定外来生物に指定されているアリノトウグサ科の水生植物は何か。

<details>
<summary>解答を見る</summary>

**オオフサモ（*Myriophyllum aquaticum*）**

南アメリカ原産。水路を閉塞するなど被害を与える。

</details>
