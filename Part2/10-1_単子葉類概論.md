## はじめに

[単子葉類](https://ja.wikipedia.org/wiki/単子葉類)（Monocots）は、被子植物の主要な系統群の一つで、日本には約35科1,500種が分布します。[イネ科](https://ja.wikipedia.org/wiki/イネ科)、[ラン科](https://ja.wikipedia.org/wiki/ラン科)、[ユリ科](https://ja.wikipedia.org/wiki/ユリ科)など、人間生活に密接に関わる植物を多く含みます。

---

## 単子葉類の基本的特徴

### 形態的特徴

| 特徴 | 内容 |
|------|------|
| 子葉 | 1枚（「単子葉」の由来） |
| 葉脈 | 平行脈（まれに網状脈） |
| 茎 | 維管束が散在、形成層なし |
| 根 | ひげ根系（主根なし） |
| 花 | 3数性（花被片・雄しべが3の倍数） |
| 花粉 | 単溝粒 |

### 真正双子葉類との比較

| 特徴 | 単子葉類 | 真正双子葉類 |
|------|----------|--------------|
| 子葉 | 1枚 | 2枚 |
| 葉脈 | 平行脈 | 網状脈 |
| 維管束 | 散在 | 環状配列 |
| 形成層 | なし | あり（木本） |
| 根系 | ひげ根 | 主根系 |
| 花被片 | 3の倍数 | 4または5の倍数 |
| 花粉 | 単溝粒 | 三溝粒 |

## ==単子葉類の共有派生形質（Synapomorphies）（超頻出）==

### 主要な共有派生形質

単子葉類を単系統群として特徴づける形質：

| 形質 | 説明 | 例外 |
|------|------|------|
| ==**単子葉**== | ==胚が1枚の子葉をもつ== | ほとんど例外なし |
| ==**平行脈**== | ==葉脈が平行に走る== | ラン科、サトイモ科など一部は網状脈 |
| ==**散在維管束**== | ==茎の維管束が不規則に配列== | ヤシ科など一部は環状に近い |
| ==**ひげ根系**== | ==主根が早期に退化== | - |
| ==**3数性の花**== | ==花の部品が3またはその倍数== | 4数性の例もある（マイヅルソウなど） |
| ==**単溝型花粉**== | ==花粉粒に1本の溝== | ほとんど例外なし |

### ==試験で問われる形質（Q37対応）==

**問：単子葉類の共有派生形質でないものはどれか**

以下は単子葉類の特徴だが、**すべてが共有派生形質とは限らない**：

| 形質 | 単子葉類の特徴か | 共有派生形質か | 説明 |
|------|------------------|----------------|------|
| 平行脈 | ✅ YES | ✅ YES | 最も典型的な特徴だが例外あり |
| 散在維管束 | ✅ YES | ✅ YES | 茎の断面で維管束が散在 |
| 3数性の花 | ✅ YES | ✅ YES | 花被片・雄しべが3の倍数 |
| 単子葉 | ✅ YES | ✅ YES | 名前の由来、最も確実 |
| ひげ根系 | ✅ YES | ✅ YES | 主根が機能せず不定根が発達 |
| **維管束形成層なし** | ✅ YES | ⚠️ **注意** | 二次成長しないことが多いが、ヤシ科など例外あり |
| **草本性** | ⚠️ 多い | ❌ NO | ヤシ科など木本もある |
| **風媒花** | ⚠️ 多い | ❌ NO | ラン科など虫媒花も多い |

> 📝 **試験ポイント**: 「草本性」や「風媒花」は単子葉類に多いが、**共有派生形質ではない**（木本のヤシや虫媒のランがいる）

### 単子葉類の定義に最も重要な形質

最も確実な共有派生形質は：

1. **単子葉**（子葉1枚）
2. **単溝型花粉**（花粉粒に1本の溝）
3. **散在維管束**（茎の断面）

これらは例外がほとんどなく、分子系統学的にも支持される。

## 単子葉類の進化

### 起源と多様化

| 時代 | 出来事 |
|------|--------|
| 約1億3000万年前 | 単子葉類の出現（白亜紀前期） |
| 約1億年前 | 主要系統の分化 |
| 約6600万年前 | [K-Pg境界](https://ja.wikipedia.org/wiki/K-Pg境界)後さらに多様化 |
| 約3000万年前 | イネ科の急速な多様化 |

### 単子葉類の適応放散

1. **水生環境への適応**
   - オモダカ目の多くが水生・湿地性
   - ショウブ目も湿地性

2. **草原環境への適応**
   - イネ科の繁栄
   - 単子葉類の茎の構造が草原火災への耐性

3. **着生への適応**
   - ラン科の多様化
   - [CAM型光合成](https://ja.wikipedia.org/wiki/CAM型光合成)の獲得

## 単子葉類の識別ポイント

### 野外での識別

1. **葉脈を確認**
   - 平行脈 → 単子葉類の可能性
   - ただし、ヤマノイモ科など網状脈もある

2. **茎の断面**
   - 維管束が散在 → 単子葉類

3. **花の構造**
   - 3数性（花被片3または6枚）

4. **根の形態**
   - ひげ根系

### 主要科の識別キー

```
1. 水生または湿地性 ............... 2
1. 陸生 ........................... 4

2. 浮葉または沈水性 ............... オモダカ目の科
2. 抽水性 ......................... 3

3. 穂状花序 ....................... ショウブ科
3. 総状・円錐花序 ................. オモダカ科・トチカガミ科

4. つる性 ......................... ヤマノイモ科・サルトリイバラ科
4. 非つる性 ....................... 5

5. イネ型の葉（細長く、葉鞘あり）.. 6
5. その他の葉形 ................... 7

6. 茎は中空、節あり ............... イネ科
6. 茎は充実、節なし ............... カヤツリグサ科

7. 花が鮮やかで目立つ ............. ユリ科・ラン科・アヤメ科等
7. 花が小さく目立たない ........... 各種
```

## 4択確認問題

**Q1.** 単子葉類の特徴として、「共有派生形質」ではないものはどれか。
① 単子葉（子葉1枚）　② 散在維管束　③ 草本性　④ 単溝型花粉

<details><summary>答え</summary>
**③ 草本性** — 草本性は単子葉類に多い特徴だが、ヤシ科など木本のグループもあるため「共有派生形質（synapomorphy）」ではない。①②④はいずれも単子葉類の共有派生形質。
</details>

---

**Q2.** APG体系で単子葉類の最基部に位置する目はどれか。
① オモダカ目　② ユリ目　③ ショウブ目　④ キジカクシ目

<details><summary>答え</summary>
**③ ショウブ目**（Acorales） — ショウブ属1属のみからなり、単子葉類の最も早い分岐系統。かつてはサトイモ科に含められていたが、分子系統解析で最基部と判明。
</details>

---

**Q3.** 旧ユリ科から分割された科として正しいものはどれか。
① タコノキ科　② ショウブ科　③ ヒガンバナ科　④ イグサ科

<details><summary>答え</summary>
**③ ヒガンバナ科**（Amaryllidaceae） — APG体系により旧広義ユリ科は大幅に分割され、ヒガンバナ科・キジカクシ科・シュロソウ科・イヌサフラン科・ユリ科（狭義）等に再編された。
</details>

---

**Q4.** イネ科とカヤツリグサ科を識別する最も重要な形質の組み合わせはどれか。
① イネ科は茎中空・節あり、カヤツリグサ科は茎充実・節なし　② イネ科は茎充実・節なし、カヤツリグサ科は茎中空・節あり　③ 両者とも茎は中空　④ どちらも3列葉

<details><summary>答え</summary>
**① イネ科は茎中空・節あり、カヤツリグサ科は茎充実・節なし** — これが最重要の識別点。さらにカヤツリグサ科の茎断面は三稜形（三角形）、イネ科は円形。葉配列はイネ科2列、カヤツリグサ科3列。
</details>
