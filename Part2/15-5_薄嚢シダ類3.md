# 15-5. 薄嚢シダ類3（ウラボシ科・その他の科）

## はじめに

本章では、ウラボシ科を中心に、残りの薄嚢シダ類を解説します。ウラボシ科は着生シダに多様化した科で、ノキシノブなど身近な種を含みます。

---

## 1. ウラボシ科 Polypodiaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約15属 |
| 日本産種数 | 約50種 |
| 世界の種数 | 約1,600種 |
| 特徴 | 多くが着生性、包膜を欠く |

### 形態的特徴

| 部位 | 特徴 |

| 根茎 | 匍匐、鱗片を密生 |
| 葉 | 単葉～羽状複葉 |
| ソーラス | 円形～線形 |

## 2. ゼンマイ科 Osmundaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 2属 |
| 日本産種数 | 約5種 |
| 世界の種数 | 約20種 |
| 特徴 | 大型、栄養葉と胞子葉が分離 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 根茎 | 太く直立、塊状 |
| 葉 | 大型、2回羽状複葉 |
| 胞子葉 | 栄養葉と別、または羽片が変形 |
| 胞子嚢 | 大型、環帯が不完全 |

### 日本産主要種

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ゼンマイ](https://ja.wikipedia.org/wiki/ゼンマイ) | *Osmunda japonica* | 北海道～沖縄 | 山菜、綿毛を被る、二形性 |
| [ヤマドリゼンマイ](https://ja.wikipedia.org/wiki/ヤマドリゼンマイ) | *Osmundastrum cinnamomeum* | 北海道～九州 | 湿地、胞子葉が中央 |
| [シロヤマゼンマイ](https://ja.wikipedia.org/wiki/シロヤマゼンマイ) | *Osmunda banksiifolia* | 本州～沖縄 | 常緑、白っぽい |

### ゼンマイの特徴

| 項目 | 内容 |
|------|------|
| 食用 | 若芽を乾燥保存、煮物など |
| 綿毛 | 若芽を覆う綿毛は除去して食用 |
| 二形性 | 栄養葉（緑）と胞子葉（褐色）が別 |
| 胞子葉 | 「男ゼンマイ」と呼ばれ、採らない |

> 📝 **ゼンマイの採取**：胞子葉（男ゼンマイ）を残し、栄養葉（女ゼンマイ）を採取するのが持続的な利用法

## 5. コケシノブ科 Hymenophyllaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約5属 |
| 日本産種数 | 約25種 |
| 世界の種数 | 約600種 |
| 特徴 | 葉が1層の細胞、コケ状 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 葉 | 細胞層が**1層**、半透明 |
| 根茎 | 細く匍匐 |
| 生活形 | 湿潤な環境に生育 |
| ソーラス | 葉縁に、筒状の包膜 |

### 日本産主要種

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [コケシノブ](https://ja.wikipedia.org/wiki/コケシノブ) | *Hymenophyllum wrightii* | 本州～九州 | 小型、湿潤な岩上 |
| [ウチワゴケ](https://ja.wikipedia.org/wiki/ウチワゴケ) | *Gonocormus minutus* | 本州～沖縄 | 扇形、1細胞層 |
| [アオホラゴケ](https://ja.wikipedia.org/wiki/アオホラゴケ) | *Vandenboschia orientalis* | 北海道～九州 | やや大型 |
| [ホソバコケシノブ](https://ja.wikipedia.org/wiki/ホソバコケシノブ) | *Hymenophyllum polyanthos* | 本州～沖縄 | 細い裂片 |
| [チチブホラゴケ](https://ja.wikipedia.org/wiki/チチブホラゴケ) | *Vandenboschia kalamocarpa* | 本州～九州 | 秩父産 |

> 📝 **コケシノブ科**：葉身が1層の細胞からなるため、光を透かすと半透明に見える特異なシダ

## 6. ウラジロ科 Gleicheniaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 2属 |
| 日本産種数 | 約5種 |
| 世界の種数 | 約160種 |
| 特徴 | 二叉分岐、葉裏が白い |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 根茎 | 長く匍匐 |
| 葉 | 反復二叉分岐 |
| 葉裏 | 白色（蝋質） |
| ソーラス | 円形、包膜なし |

### 日本産主要種

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ウラジロ](https://ja.wikipedia.org/wiki/ウラジロ) | *Gleichenia japonica* | 本州～沖縄 | **正月飾り**、葉裏白色 |
| [コシダ](https://ja.wikipedia.org/wiki/コシダ) | *Dicranopteris linearis* | 本州～沖縄 | 先駆植物、荒地に群生 |
| [カネコシダ](https://ja.wikipedia.org/wiki/カネコシダ) | *Gleichenia laevissima* | 本州～沖縄 | 葉裏緑色 |

### ウラジロの文化的利用

| 項目 | 内容 |
|------|------|
| 正月飾り | 鏡餅の下に敷く |
| 意味 | 葉裏の白色→「裏を返しても心は白い」→清廉 |
| 意味 | 二叉分岐→夫婦円満 |
| 意味 | 長く伸びる→長寿 |

## 確認問題

### 問1
ウラボシ科の包膜の特徴を述べよ。

<details>
<summary>解答を見る</summary>

**包膜を欠く**

</details>

### 問2
ゼンマイの採取で「男ゼンマイ」と呼ばれる部分は何か。なぜ採取しないのか。

<details>
<summary>解答を見る</summary>

- **胞子葉**を「男ゼンマイ」と呼ぶ
- 胞子葉を残すことで翌年の再生産を確保するため

</details>

### 問3
コケシノブ科の葉の細胞層の特徴を述べよ。

<details>
<summary>解答を見る</summary>

**1層の細胞からなる**（半透明）

</details>

### 問4
ウラジロが正月飾りに使われる理由を述べよ。

<details>
<summary>解答を見る</summary>

- 葉裏の白色→**清廉潔白**の象徴
- 二叉分岐→**夫婦円満**
- 長く伸びる→**長寿**

</details>
