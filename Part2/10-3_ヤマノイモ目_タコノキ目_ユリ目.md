## ヤマノイモ目 Dioscoreales

[ヤマノイモ目](https://ja.wikipedia.org/wiki/ヤマノイモ目)は、つる性植物を多く含む目です。日本には2科が分布します。

---

### [ヤマノイモ科](https://ja.wikipedia.org/wiki/ヤマノイモ科) Dioscoreaceae

| 項目 | 内容 |
|------|------|
| 日本産属数 | 1属 |
| 日本産種数 | 約10種 |
| 世界の属数/種数 | 4属/約870種 |
| 分布 | 世界の熱帯〜温帯 |

#### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活型 | つる性多年草 |
| 葉 | 互生または対生、心形、**網状脈**（単子葉類では稀） |
| 茎 | 右巻きまたは左巻き |
| 花 | 小型、雌雄異株が多い |
| 果実 | 蒴果（3翼あり） |
| 地下部 | 塊根または塊茎 |

#### 日本産の主要種

##### [ヤマノイモ属](https://ja.wikipedia.org/wiki/ヤマノイモ属) *Dioscorea*

| 種名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ヤマノイモ](https://ja.wikipedia.org/wiki/ヤマノイモ) | *D. japonica* | 本州〜九州 | 自然薯、対生葉 |
| [ナガイモ](https://ja.wikipedia.org/wiki/ナガイモ) | *D. batatas* | 栽培（中国原産） | 長芋、とろろ |
| [オニドコロ](https://ja.wikipedia.org/wiki/オニドコロ) | *D. tokoro* | 北海道〜九州 | 互生葉、有毒 |
| [カエデドコロ](https://ja.wikipedia.org/wiki/カエデドコロ) | *D. quinquelobata* | 本州〜九州 | 掌状葉 |
| [タチドコロ](https://ja.wikipedia.org/wiki/タチドコロ) | *D. gracillima* | 本州〜九州 | 細い葉 |
| [キクバドコロ](https://ja.wikipedia.org/wiki/キクバドコロ) | *D. septemloba* | 本州〜九州 | 掌状に深裂 |
| [ウチワドコロ](https://ja.wikipedia.org/wiki/ウチワドコロ) | *D. nipponica* | 北海道〜九州 | 円形の葉 |

**ヤマノイモ（自然薯）の詳細**
- **別名**：[自然薯](https://ja.wikipedia.org/wiki/自然薯)（じねんじょ）
- **特徴**：
  - 葉は対生（他種は互生が多い）
  - 塊根は粘りが強い
  - 珠芽（むかご）を葉腋につける
- **用途**：食用（とろろ）、薬用
- **識別**：茎は右巻き

**ヤマノイモとオニドコロの見分け方**

| 特徴 | ヤマノイモ | オニドコロ |
|------|------------|------------|
| 葉のつき方 | 対生 | 互生 |
| 珠芽（むかご） | あり | なし |
| 塊根 | 食用可 | 有毒 |

## タコノキ目 Pandanales

[タコノキ目](https://ja.wikipedia.org/wiki/タコノキ目)は、熱帯性の植物を含む目です。日本には1科が分布します。

---

### [タコノキ科](https://ja.wikipedia.org/wiki/タコノキ科) Pandanaceae

| 項目 | 内容 |
|------|------|
| 日本産属数 | 1属 |
| 日本産種数 | 3種 |
| 世界の属数/種数 | 5属/約900種 |
| 分布 | 旧世界熱帯 |

#### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活型 | 常緑の低木〜高木 |
| 葉 | 線形、3列螺旋状、縁に鋸歯 |
| 支柱根 | 発達（タコの足状） |
| 花 | 雌雄異株、肉穂花序 |
| 果実 | 集合果 |

#### 日本産の種

##### [タコノキ属](https://ja.wikipedia.org/wiki/タコノキ属) *Pandanus*

| 種名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [タコノキ](https://ja.wikipedia.org/wiki/タコノキ) | *P. boninensis* | **小笠原固有** | 固有種、支柱根発達 |
| [アダン](https://ja.wikipedia.org/wiki/アダン) | *P. odoratissimus* | 沖縄〜奄美 | 海岸、果実は集合果 |
| [ビヨウタコノキ](https://ja.wikipedia.org/wiki/ビヨウタコノキ) | *P. utilis* | 栽培（マダガスカル原産） | 観賞用 |

**タコノキの詳細**
- **小笠原固有種**
- 支柱根がタコの足のように見える
- [小笠原諸島](https://ja.wikipedia.org/wiki/小笠原諸島)の代表的植物

**アダンの詳細**
- 南西諸島の海岸に普通
- 果実はパイナップル状
- 防風・防潮林として利用
- [ヤシガニ](https://ja.wikipedia.org/wiki/ヤシガニ)の食物

## 主要種詳細解説

### [カタクリ](https://ja.wikipedia.org/wiki/カタクリ) *Erythronium japonicum*

#### 形態的特徴

| 部位/特徴 | 内容 |
|-----------|------|
| 生活型 | 落葉多年草（早春植物＝スプリング・エフェメラル） |
| 鱗茎 | 長さ4〜5cm、白色、デンプン豊富 |
| 葉 | 1〜2枚、まだら模様あり（落葉樹林の暗い場所への適応） |
| 花 | 3〜5月、紅紫色、花被片6枚、強く反り返る |
| 果実 | 蒴果、アリが種子を散布（elaiosome） |
| 分布 | 北海道〜九州の落葉樹林林床 |
| 発芽〜開花 | **種子から開花まで8〜9年** |

#### なぜ重要か（試験重要度 ★★★★）

- スプリング・エフェメラルの代表種
- 「片栗粉」の名前の由来（かつて鱗茎からデンプンを採取）
- 現在市販の片栗粉はジャガイモデンプン
- 種子にエライオソームがつき、アリが散布（蟻散布）

#### 試験頻出問題

> **問：** カタクリを「スプリング・エフェメラル」と呼ぶ理由は？
> → **答え：** 早春（3〜5月）に地上部を出して開花・結実し、夏には地上部が完全に枯れて**鱗茎で休眠**するため。「春の短命植物」とも呼ぶ。

> **問：** 現在市販されている片栗粉の原料は何か？
> → **答え：** **ジャガイモデンプン**。かつてはカタクリの鱗茎からデンプンを採取したが、現在は主にジャガイモを原料とする。

| 鱗茎 | 大型、直径10〜15cm |
| 葉 | 互生、披針形 |
| 花 | 7〜8月、白色に黄色の縦筋と紅紫色の斑点、直径20〜25cm |
| 芳香 | **非常に強い甘い香り**（日本のユリの中で最強） |
| 分布 | 本州（東北〜近畿）の山地 |
| 固有性 | **日本固有種** |

#### なぜ重要か（試験重要度 ★★★★）

- 「ユリの王様」と呼ばれる日本最大級のユリ
- 日本固有種で世界的に有名
- 西洋で育種され「オリエンタルハイブリッド」の親

#### 試験頻出問題

> **問：** ヤマユリの特徴を3つ述べよ。
> → **答え：** ①日本固有種、②花が非常に大型（直径20cm超）、③強い甘い芳香。

### オニユリとコオニユリの違い（試験頻出）

| 特徴 | オニユリ *L. lancifolium* | コオニユリ *L. leichtlinii* |
|------|------------------------|--------------------------|
| むかご | **あり**（葉腋につく黒いむかご） | **なし** |
| 葉 | 線状披針形、葉腋にむかご | 線状披針形、むかごなし |
| 花色 | 橙色、紫黒色の斑点 | 橙色、斑点あり |
| 繁殖 | むかごで栄養繁殖 | 種子繁殖 |

#### 試験頻出問題

> **問：** オニユリとコオニユリの最も簡単な識別点は？
> → **答え：** **むかごの有無**。オニユリには葉腋に黒いむかごがつくが、コオニユリにはない。

### ムカゴ（珠芽）の生態

**ムカゴ**（珠芽・むかご）は茎の葉腋にできる栄養繁殖器官。

| 植物 | ムカゴの特徴 |

| オニユリ | 葉腋に黒色の小さなムカゴ |
| ヌカボシクリハラン（シダ） | シダでもムカゴを形成する例 |

## 確認問題

**Q1.** カタクリについて正しいものはどれか。
① 球茎で越冬する　② 種子から開花まで2〜3年　③ 種子から開花まで8〜9年かかる　④ 現在も鱗茎から片栗粉を製造している

<details><summary>答え</summary>
**③ 種子から開花まで8〜9年かかる** — カタクリは**鱗茎**（球茎ではない）で越冬し、種子から開花まで8〜9年要する。現在の片栗粉はジャガイモデンプン。
</details>

---

**Q2.** ヤマノイモとオニドコロの識別に最も有効な特徴はどれか。
① 花の色　② 葉のつき方（対生か互生か）　③ 果実の形　④ つるの巻き方

<details><summary>答え</summary>
**② 葉のつき方（対生か互生か）** — ヤマノイモは**対生**、オニドコロは**互生**。ヤマノイモにはむかごがあり食用可だが、オニドコロは有毒。
</details>

---

**Q3.** オニユリとコオニユリの最大の違いはどれか。
① 花の色　② むかごの有無　③ 葉の形　④ 分布域

<details><summary>答え</summary>
**② むかごの有無** — オニユリは葉腋に黒いむかごがつく（栄養繁殖）が、コオニユリにはむかごがない。
</details>

---

**Q4.** タコノキ（小笠原固有種）の形態的特徴として正しいものはどれか。
① 掌状複葉　② 支柱根（タコの足状）がある　③ 平行脈でない　④ 浮葉を形成する

<details><summary>答え</summary>
**② 支柱根（タコの足状）がある** — タコノキはタコノキ科で支柱根が発達し、タコの足のように見えることが名前の由来。小笠原固有種。
</details>

**Q5.** スプリング・エフェメラルとして分類される植物はどれか。
① ヤマユリ　② カタクリ　③ オニユリ　④ テッポウユリ

<details><summary>答え</summary>
**② カタクリ** — 早春に地上部を出し開花・結実した後、夏に地上部が枯れて休眠する「春植物（スプリング・エフェメラル）」の代表種。
</details>

## 確認問題

### 問1
ヤマノイモ科の特徴として、単子葉類では稀な形質は何か。

<details>
<summary>解答を見る</summary>

**網状脈**

単子葉類は通常平行脈だが、ヤマノイモ科は網状脈を持つ。

</details>

### 問2
ヤマノイモとオニドコロの見分け方を述べよ。

<details>
<summary>解答を見る</summary>

- **ヤマノイモ**：葉は対生、珠芽（むかご）あり、塊根は食用
- **オニドコロ**：葉は互生、珠芽なし、塊根は有毒

</details>

### 問3
タコノキの名前の由来は何か。

<details>
<summary>解答を見る</summary>

**支柱根がタコの足のように見える**ため

タコノキは小笠原固有種で、支柱根が発達する。

</details>

### 問4
カタクリのような早春に開花し、夏には地上部が枯れる植物群を何と呼ぶか。

<details>
<summary>解答を見る</summary>

**スプリング・エフェメラル（春植物）**

早春に開花・結実し、夏までに地上部が枯れて休眠する。

</details>

### 問5
バイケイソウやコバイケイソウと誤食されやすい植物は何か。

<details>
<summary>解答を見る</summary>

**ギボウシ類（オオバギボウシなど）**

若芽の段階では見分けが困難で、誤食事故が発生している。バイケイソウ類は猛毒。

</details>
