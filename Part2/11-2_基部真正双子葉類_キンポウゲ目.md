## キンポウゲ目 Ranunculales

[キンポウゲ目](https://ja.wikipedia.org/wiki/キンポウゲ目)は、基部真正双子葉類で最大の目で、多数の雄しべと離生心皮を持つ「原始的」な花の形態が特徴です。

**トリカブトの毒性**
- [アコニチン](https://ja.wikipedia.org/wiki/アコニチン)を含む猛毒
- 全草有毒、特に根が強毒
- [ニリンソウ](https://ja.wikipedia.org/wiki/ニリンソウ)との誤食事故
- 古来より毒矢に使用

#### 🔴 詳細：トリカブトの毒性（試験重要度 ★★★★★）

| 項目 | 内容 |
|------|------|
| 主要毒成分 | **アコニチン**（ジテルペン系アルカロイド） |
| その他の毒成分 | メサコニチン、ヒパコニチン |
| 有毒部位 | **全草**（特に塊根が最強毒） |
| 塊根の名称 | 母根を「烏頭（うず）」、子根を「附子（ぶし・ぶす）」 |
| 中毒症状 | 口唇・舌の灼熱感→麻痺、悪心・嘔吐、四肢の痺れ、不整脈、呼吸困難 |
| 毒性のメカニズム | **電位依存性Naチャネルを持続的に開放**→神経・心筋の機能麻痺 |
| 誤食事例 | **ニリンソウ**の若葉と誤食する死亡事故が毎年発生 |
| 伝統的利用 | アイヌ民族が根を用いた毒矢（オコイヤ）でクマ狩りに使用 |
| 薬用 | 漢方では附子・烏頭として使用（適切に処理して鎮痛・強心薬） |
| 分類 | 日本三大有毒植物（トリカブト・ドクウツギ・ドクゼリ）の一つ |

#### 試験頻出問題

> **問：** トリカブトの主要毒成分は何アルカロイドか。
> → **答え：** **アコニチン**（ジテルペン系アルカロイド）。電位依存性Naチャネルに作用して神経・心筋を麻痺させる。

> **問：** トリカブトと誤食されやすい山菜は何か。なぜ間違えやすいか。
> → **答え：** **ニリンソウ**。春の芽吹き時期に同じ沢沿いに生え、若葉の形態が似ているため。ニリンソウは食用だが、トリカブト混入による死亡事故が発生している。

> **問：** トリカブトの根（子根）の漢方名は何か。
> → **答え：** **附子（ぶし・ぶす）**。母根は烏頭（うず）と呼ぶ。

#### 混同しやすい種との違い（ニリンソウとの識別）

| 特徴 | トリカブト | ニリンソウ |
|------|------------|------------|
| 毒性 | **猛毒（全草）** | 無毒（食用山菜） |
| 葉の形 | 掌状に5〜7深裂 | 3深裂〜浅裂 |
| 根 | **塊根**（太い） | 細い根茎 |
| 花 | 兜形、紫色、5萼片 | 白い花弁状萼、5〜6枚 |
| 開花時期 | 夏〜秋 | 春（3〜5月）|
| 生育地 | 沢沿い・湿地 | 沢沿い（同所的に生える！）|

### [メギ科](https://ja.wikipedia.org/wiki/メギ科) Berberidaceae

!!! warning "メギ科の葯は「弁開葯」（2022年度問37・2023年度問42で出題）"
    - **弁開葯（valvate dehiscence）**：葯壁が弁のように上向きに開いて花粉を放出する。
    - メギ科（メギ・ナンテン等）の決定的な特徴。
    - **孔開葯（poricidal dehiscence）**：孔から花粉が出る（ツツジ科・ナス科等）
    - 2022年度問37「弁開葯をもつ科」→ **メギ科**（正解1）
    - 2023年度問42「科と形態的特徴の組み合わせで誤り」→ 「メギ科−孔開葯」が誤り

| 世界の属数/種数 | 約15属/約700種 |

| **葯の開裂様式** | **弁開葯（valvate）**← 頻出！|

## ケシ科の重要種詳細

### [ケシ](https://ja.wikipedia.org/wiki/ケシ) *Papaver somniferum*

#### 形態的特徴

| 部位/特徴 | 内容 |
|-----------|------|
| 生活型 | 一年草 |
| 高さ | 60〜150cm |
| 葉 | 互生、葉は茎を抱く、灰緑色 |
| 花 | 大型（径5〜10cm）、花弁4枚、白〜赤紫〜紫 |
| 雄しべ | **多数** |
| 果実 | 球形の蒴果（ケシ坊主） |
| 乳液 | 未熟果に傷をつけると**白色乳液（アヘン）**が出る |

#### なぜ重要か（試験重要度 ★★★★）

- **麻薬成分**：モルヒネ、コデイン、テバイン等のアルカロイドを含む
- **法的規制**：あへん法により、日本では**栽培禁止**（研究・薬用は許可制）
- 乳液の収集から**アヘン・ヘロイン**が製造される
- モルヒネは医療用鎮痛薬として現在も使用
- ケシ科の特徴である**乳液・大型花弁・多数雄しべ**の典型種

#### 試験頻出問題

> **問：** ケシ（*Papaver somniferum*）が法的に規制されている理由と根拠法令は何か。
> → **答え：** 未熟な果実から採れる乳液（アヘン）に**モルヒネ・コデイン**等の麻薬成分が含まれるため。**あへん法**により日本では栽培が原則禁止されている。

### [クサノオウ](https://ja.wikipedia.org/wiki/クサノオウ) *Chelidonium majus*

#### 形態的特徴

| 部位/特徴 | 内容 |
|-----------|------|
| 生活型 | 二年草〜多年草 |
| 分布 | 日本全土（道端・人里） |
| 葉 | 羽状深裂、粉白緑色 |
| 花 | 黄色、花弁4枚 |
| 乳液 | 傷つけると**橙黄色の乳液**が出る |
| 毒成分 | ベルベリン、コプチシン、ケリドニン等のアルカロイド |

#### なぜ重要か（試験重要度 ★★★）

- **ケシ科の識別に重要**：乳液の色が橙黄色（タケニグサは同じく橙色）
- 全草有毒だが、古来より皮膚病の民間薬として使用
- 「草の王」の名は薬効が高いことから（「瘡（くさ）の王」説もあり）

### ケシ科全体の形態的特徴まとめ

| 特徴 | 詳細 |
|------|------|
| 乳液 | 多くの種が有色の乳液（白・橙・黄）を持つ |
| 花弁 | 大型、2+2の4枚が基本（花蕾時は萼片2枚がすぐ落ちる） |
| 雄しべ | **多数** |
| 心皮 | 合生（子房は1室〜多室） |
| 果実 | 蒴果（長角果状または球状） |

## 確認問題（4択形式）

**Q1.** トリカブトの主要毒成分として正しいものはどれか。
① テトロドトキシン　② アコニチン　③ ソラニン　④ アマニチン

<details><summary>答え</summary>**② アコニチン** — ジテルペン系アルカロイドで、電位依存性Naチャネルに作用して神経・心筋を麻痺させる猛毒。</details>

**Q2.** キンポウゲ科の花の特徴として正しいものはどれか。
① 花弁は合着して筒状になる　② 雄しべは通常5本　③ 心皮は多数で離生する　④ 子房は下位

<details><summary>答え</summary>**③ 心皮は多数で離生する** — キンポウゲ科は多数の雄しべと多数の離生心皮を持つ「原始的」な花の形態が特徴。</details>

**Q3.** センニンソウとボタンヅルの識別点として正しいものはどれか。
① センニンソウは鋸歯ある葉、ボタンヅルは全縁の葉　② センニンソウは全縁の葉、ボタンヅルは鋸歯ある葉　③ センニンソウは黄色い花、ボタンヅルは白い花　④ センニンソウは木本、ボタンヅルは草本

<details><summary>答え</summary>**② センニンソウは全縁の葉、ボタンヅルは鋸歯ある葉** — センニンソウ（全縁）、ボタンヅル（鋸歯あり・ボタン葉状）。どちらも白い花のつる性草本。</details>

**Q4.** ケシ（*Papaver somniferum*）に含まれる麻薬成分として正しいものはどれか。
① カフェイン　② モルヒネ　③ ニコチン　④ カプサイシン

<details><summary>答え</summary>**② モルヒネ** — ケシの未熟果実から採れる乳液（アヘン）にモルヒネ・コデイン・テバイン等が含まれる。あへん法により栽培は原則禁止。</details>

**Q5.** 北岳固有種で絶滅危惧IA類に指定されるキンポウゲ科植物はどれか。
① コマクサ　② ニリンソウ　③ キタダケソウ　④ フクジュソウ

<details><summary>答え</summary>**③ キタダケソウ**（*Callianthemum hondoense*） — 南アルプス北岳のみに自生する氷河期遺存種。国内希少野生動植物種に指定。</details>
