## [ナデシコ科](https://ja.wikipedia.org/wiki/ナデシコ科) Caryophyllaceae

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約25属 |
| 日本産種数 | 約80種 |
| 世界の属数/種数 | 約100属/約2,200種 |
| 分布 | 北半球温帯を中心 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 茎 | **節が著しく膨らむ**（タデ科との識別に重要） |
| 葉 | **対生**、単葉、全縁、**托葉なし** |
| 分枝 | **二叉分枝**（フォーク状に分かれる）が多い |
| 花 | 放射相称、花弁5（または4） |
| 花弁 | 先端が深く裂けるものが多い（例：ハコベは花弁10枚に見えるが実際は5枚の深裂） |
| 萼 | 合着または離生 |
| 雄しべ | 10本（または5本） |
| 子房 | 上位、1室 |
| 果実 | 蒴果（頂端で歯状に裂開） |

> **ナデシコ科の科の識別ポイント**：① 節が著しく膨らむ、② 葉が対生・全縁・托葉なし、③ 二叉分枝 の3点で確認。

## [タデ科](https://ja.wikipedia.org/wiki/タデ科) Polygonaceae

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約10属 |
| 日本産種数 | 約60種 |
| 世界の属数/種数 | 約50属/約1,200種 |
| 分布 | 世界中 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 茎 | **節が著しく膨らむ** |
| 葉 | 互生、単葉 |
| **托葉鞘** | **筒状の膜質の托葉（鞘状）が節を包む** ← 科の同定に最重要 |
| 花 | 小型、花被片3〜6（花弁状または緑色） |
| 果実 | **痩果、しばしば3稜形**（三角形断面） |

#### 🔴 托葉鞘（たくようしょう）詳細（試験重要度 ★★★★★）

タデ科の**最重要識別形質**。

| 項目 | 内容 |
|------|------|
| 構造 | 節を包む**筒状・膜質**の托葉 |
| 学術名 | **ochrea（オクレア）** |
| 形態 | 透明〜褐色の薄膜状で茎を囲む |
| 種による違い | 縁が毛状に裂けるもの（例：ミゾソバ）、無毛のもの（イタドリ）等 |
| 鑑定ポイント | 他の科には見られないタデ科固有の構造 |

> **試験頻出**：野外で茎の節部に筒状の膜質構造を見たら→タデ科と判断できる。

#### 🌍 イタドリの世界的侵略問題（試験重要度 ★★★★）

| 項目 | 内容 |
|------|------|
| 学名（現在） | *Fallopia japonica*（= *Reynoutria japonica*） |
| 英名 | **Japanese knotweed** |
| 侵略問題の背景 | 19世紀に観賞・飼料用として欧米に導入されたが急速に野生化 |
| 繁殖力 | **地下茎（根茎）が非常に強靭**で、5〜7mの深さまで伸びる |
| 再生能力 | 5gの根茎断片からでも再生可能 |
| 分布 | ヨーロッパ・北米・オーストラリアに侵入 |
| 法的規制 | イギリスでは**Wildlife and Countryside Act**で拡散が違法 |
| 「世界最悪の侵略的外来種100選」| **IUCNに掲載**（哺乳類・植物・菌類等のリスト） |
| 日本国内 | 日本原産、在来種として普通に分布。若芽は山菜として食用 |
| 茎の特徴 | 中空、節明瞭、酸味（シュウ酸含有） |

#### 試験頻出問題

> **問：** イタドリ（Japanese knotweed）が欧米で特に問題となる理由は何か。
> → **答え：** **根茎による強力な繁殖力**のため駆除が極めて困難で、生態系・建物基礎・道路インフラにまで被害を与える。IUCN「世界最悪の侵略的外来種100選」に選ばれている。

## 確認問題（4択形式）

**Q1.** タデ科の識別に最も重要な構造はどれか。
① 葉が対生である　② 節を包む筒状の膜質の托葉（托葉鞘）がある　③ 花弁が5枚で離弁花　④ 乳液が出る

<details><summary>答え</summary>**② 節を包む筒状の膜質の托葉（托葉鞘）がある** — タデ科固有の構造で、ochrea（オクレア）とも呼ばれる。これを確認すればタデ科と同定できる。</details>

**Q2.** イタドリについて正しいものはどれか。
① 日本原産で欧米に侵入し「世界最悪の侵略的外来種100選」に選ばれている　② アフリカ原産で日本に侵入した外来種　③ 日本では特定外来生物に指定されている　④ 雌雄異株で果実は食用にできない

<details><summary>答え</summary>**① 日本原産で欧米に侵入し「世界最悪の侵略的外来種100選」に選ばれている** — 英名 Japanese knotweed。強靭な根茎で駆除困難。日本では在来種として普通に生育し、若芽が食用山菜として利用される。</details>

**Q3.** ナデシコ科の形態的特徴の組み合わせとして正しいものはどれか。
① 節が膨らむ・葉が対生・托葉鞘がある　② 節が膨らむ・葉が対生・托葉なし　③ 節が膨らまない・葉が互生・托葉あり　④ 節が膨らむ・葉が互生・托葉なし

<details><summary>答え</summary>**② 節が膨らむ・葉が対生・托葉なし** — ナデシコ科の3大特徴。「節が膨らむ・托葉鞘あり」はタデ科。</details>

**Q4.** スイバ（*Rumex acetosa*）の特徴として正しいものはどれか。
① 両性花で酸味がない　② 雌雄異株で葉に酸味がある（シュウ酸）　③ 花弁は白色で5枚　④ 水中に生育する沈水植物

<details><summary>答え</summary>**② 雌雄異株で葉に酸味がある（シュウ酸）** — スイバ＝「酸い葉」の意。雌雄異株。葉の基部は矢じり形。ギシギシは両性花で苦い。</details>

**Q5.** 「大和撫子」の語源となった秋の七草のナデシコ科植物はどれか。
① タカネナデシコ　② カワラナデシコ　③ フシグロセンノウ　④ ハマナデシコ

<details><summary>答え</summary>**② カワラナデシコ**（*Dianthus superbus* var. *longicalycinus*） — 秋の七草の一つで「大和撫子」の語源。河原に生え、ピンク色の深裂した花弁が特徴。</details>

タデ科の特徴的な構造で、茎を囲む筒状の托葉を何と呼ぶか。
