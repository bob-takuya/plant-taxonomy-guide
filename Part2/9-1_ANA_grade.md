# 9-1. 基部被子植物（ANA grade）

[← 目次に戻る](../NEW_00_目次.md) | [← 前：日本産被子植物全科一覧](./8-2_日本産被子植物全科一覧.md)

---

## はじめに

基部被子植物（ANA grade）は、被子植物の系統樹で最も早く分岐したグループです。**アンボレラ目**（**A**mborellales）、**スイレン目**（**N**ymphaeales）、**アウストロバイレヤ目**（**A**ustrobaileyales）の頭文字をとって「ANA grade」と呼ばれます。

---

## アンボレラ目 Amborellales

### [アンボレラ科](https://ja.wikipedia.org/wiki/アンボレラ科) Amborellaceae

**試験最頻出！必ず覚える**

| 項目 | 内容 |
|------|------|
| 日本産 | **なし**（ニューカレドニア固有） |
| 属数/種数 | 1属1種 |
| 唯一の種 | [アンボレラ](https://ja.wikipedia.org/wiki/アンボレラ)（*Amborella trichopoda*） |

#### 重要な特徴

| 特徴 | 内容 | 試験での重要度 |
|------|------|----------------|
| 系統的位置 | **被子植物の最基部** | ★★★★★ |
| 維管束 | **仮道管のみ（道管なし）** | ★★★★★ |
| 花 | 花被片未分化、雌雄異株 | ★★★ |
| 分布 | [ニューカレドニア](https://ja.wikipedia.org/wiki/ニューカレドニア)固有 | ★★★ |

#### 試験頻出問題

> **問：APG体系で被子植物の最も原始的な科は？**
> → **Amborellaceae（アンボレラ科）**

> **問：被子植物で道管を持たず仮道管のみを持つ科は？**
> → **Amborellaceae（アンボレラ科）**

## スイレン目 Nymphaeales

### [スイレン科](https://ja.wikipedia.org/wiki/スイレン科) Nymphaeaceae

| 項目 | 内容 |
|------|------|
| 日本産属数 | 3属 |
| 日本産種数 | 約10種 |
| 生活型 | 水生植物（浮葉性・沈水性） |

#### 日本産の属と代表種

| 属名 | 学名 | 代表種 | 特徴 |
|------|------|--------|------|
| スイレン属 | *Nymphaea* | [ヒツジグサ](https://ja.wikipedia.org/wiki/ヒツジグサ)（*N. tetragona*） | 白い花、浮葉 |
| コウホネ属 | *Nuphar* | [コウホネ](https://ja.wikipedia.org/wiki/コウホネ)（*N. japonica*） | 黄色い花、浮葉 |
| | | [オゼコウホネ](https://ja.wikipedia.org/wiki/オゼコウホネ)（*N. pumila* var. *ozeensis*） | 尾瀬に分布 |
| オニバス属 | *Euryale* | [オニバス](https://ja.wikipedia.org/wiki/オニバス)（*E. ferox*） | 大型、トゲあり、絶滅危惧 |

#### 形態的特徴

- **葉**：浮葉性、円形〜楕円形、切れ込みあり
- **花**：水面上に開花、多数の花被片・雄しべ
- **果実**：[液果](https://ja.wikipedia.org/wiki/液果)状、水中で成熟
- **根茎**：泥中を這う

#### オニバスの保全状況

[オニバス](https://ja.wikipedia.org/wiki/オニバス)は環境省レッドリストで**絶滅危惧II類（VU）**に指定されています。
- 湿地の開発により減少
- 葉の直径は1mを超えることも

### [ハゴロモモ科](https://ja.wikipedia.org/wiki/ハゴロモモ科) Cabombaceae

| 項目 | 内容 |
|------|------|
| 日本産属数 | 2属 |
| 日本産種数 | 2種 |
| 生活型 | 水生植物（沈水性） |

#### 日本産の属と代表種

| 属名 | 学名 | 代表種 | 特徴 |
|------|------|--------|------|
| ジュンサイ属 | *Brasenia* | [ジュンサイ](https://ja.wikipedia.org/wiki/ジュンサイ)（*B. schreberi*） | 食用、ぬめりあり |
| フサジュンサイ属 | *Cabomba* | [フサジュンサイ](https://ja.wikipedia.org/wiki/フサジュンサイ)（*C. caroliniana*） | 帰化種、羽状の葉 |

#### ジュンサイの特徴

- 若芽は食用（「じゅんさい」として流通）
- 寒天質の粘液で覆われる
- 浮葉は楯形（葉柄が葉の中央につく）
- 花は小型で紫褐色

## アウストロバイレヤ目 Austrobaileyales

### [マツブサ科](https://ja.wikipedia.org/wiki/マツブサ科) Schisandraceae

| 項目 | 内容 |
|------|------|
| 日本産属数 | 3属 |
| 日本産種数 | 約5種 |
| 生活型 | つる性木本、常緑または落葉低木 |

**注意：旧シキミ科はマツブサ科に統合されました**

#### 日本産の属と代表種

| 属名 | 学名 | 代表種 | 特徴 |
|------|------|--------|------|
| マツブサ属 | *Schisandra* | [マツブサ](https://ja.wikipedia.org/wiki/マツブサ)（*S. repanda*） | つる性、赤い果実 |
| | | [チョウセンゴミシ](https://ja.wikipedia.org/wiki/チョウセンゴミシ)（*S. chinensis*） | 薬用、五味子 |
| サネカズラ属 | *Kadsura* | [サネカズラ](https://ja.wikipedia.org/wiki/サネカズラ)（*K. japonica*） | つる性、赤い集合果 |
| シキミ属 | *Illicium* | [シキミ](https://ja.wikipedia.org/wiki/シキミ)（*I. anisatum*） | **有毒**、仏事に使用 |

#### シキミの重要性

[シキミ](https://ja.wikipedia.org/wiki/シキミ)（*Illicium anisatum*）

- **全草有毒**（特に果実）：[アニサチン](https://ja.wikipedia.org/wiki/アニサチン)を含む
- 「[悪しき実](https://ja.wikipedia.org/wiki/シキミ)」が名前の由来
- 仏壇・墓前に供える
- 果実は八角（[トウシキミ](https://ja.wikipedia.org/wiki/トウシキミ)）に似るが食用不可
- **植物で唯一の[劇物](https://ja.wikipedia.org/wiki/劇物)指定**

#### サネカズラの特徴

[サネカズラ](https://ja.wikipedia.org/wiki/サネカズラ)（*Kadsura japonica*）

- 別名：ビナンカズラ（美男葛）
- 樹皮の粘液を整髪料に使用
- [万葉集](https://ja.wikipedia.org/wiki/万葉集)に詠まれる
- 赤い集合果が美しい

## 基部被子植物の試験対策まとめ

### 最重要ポイント

| ポイント | 内容 |
|----------|------|
| アンボレラ科 | 被子植物最基部、道管なし |
| スイレン科 | 水生、コウホネ・オニバス |
| マツブサ科 | シキミを含む（有毒、劇物） |

### 学名一覧

| 和名 | 学名 | 覚え方 |
|------|------|--------|
| アンボレラ科 | **Amborellaceae** | Ambo-最基部 |
| スイレン科 | **Nymphaeaceae** | Nymph-水の精 |
| ハゴロモモ科 | **Cabombaceae** | Cabo-ジュンサイ |
| マツブサ科 | **Schisandraceae** | Schis-裂ける |

## 確認問題

### 問1
APG体系で被子植物の最も原始的とされる科の学名を答えよ。

<details>
<summary>解答を見る</summary>

**Amborellaceae（アンボレラ科）**

ニューカレドニア固有の1属1種。被子植物で唯一、道管を持たない。

</details>

### 問2
シキミはAPG体系で何科に分類されるか。

<details>
<summary>解答を見る</summary>

**マツブサ科（Schisandraceae）**

旧シキミ科はマツブサ科に統合された。

</details>

### 問3
食用になる水生植物で、若芽が寒天質の粘液で覆われる種は何か。

<details>
<summary>解答を見る</summary>

**ジュンサイ（*Brasenia schreberi*）**

ハゴロモモ科。浮葉は楯形。

</details>

### 問4
日本産のスイレン科で、黄色い花を咲かせる属は何か。

<details>
<summary>解答を見る</summary>

**コウホネ属（*Nuphar*）**

コウホネ、オゼコウホネなどが含まれる。

</details>
