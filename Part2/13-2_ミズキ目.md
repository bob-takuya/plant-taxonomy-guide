## はじめに

[ミズキ目](https://ja.wikipedia.org/wiki/ミズキ目)（Cornales）は、キク上群の基部に位置する目で、ミズキ科とアジサイ科が日本で重要です。APG体系で旧ユキノシタ科の一部がアジサイ科として独立しました。

---

## ミズキ目の概要

| 項目 | 内容 |
|------|------|
| 日本産科数 | 4科 |
| 世界の科数 | 7科 |
| 世界の種数 | 約600種 |
| 主な特徴 | 対生または互生葉、核果が多い |

## 1. ミズキ科 Cornaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 3属 |
| 日本産種数 | 約15種 |
| 世界の属数 | 約2属（狭義） |
| 世界の種数 | 約85種 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 落葉または常緑の木本 |
| 葉 | 対生（まれに互生）、単葉、全縁 |
| 葉脈 | 側脈が葉縁に沿って弓なりに走る |
| 花 | 小型、4数性、集散花序 |
| 子房 | 下位 |
| 果実 | [核果](https://ja.wikipedia.org/wiki/核果) |

### 日本産主要種（ミズキ属 Cornus）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ミズキ](https://ja.wikipedia.org/wiki/ミズキ) | *Cornus controversa* | 北海道～九州 | 枝が階段状に出る、白い散房花序 |
| [クマノミズキ](https://ja.wikipedia.org/wiki/クマノミズキ) | *Cornus macrophylla* | 本州～沖縄 | ミズキより開花が遅い、葉が対生 |
| [ヤマボウシ](https://ja.wikipedia.org/wiki/ヤマボウシ) | *Cornus kousa* | 本州～九州 | 白い大きな総苞片4枚、赤い集合果 |
| [ハナミズキ](https://ja.wikipedia.org/wiki/ハナミズキ) | *Cornus florida* | 栽培（北米原産） | 総苞片がピンク～白、街路樹 |
| [サンシュユ](https://ja.wikipedia.org/wiki/サンシュユ) | *Cornus officinalis* | 栽培（中国原産） | 早春の黄花、赤い果実は薬用 |
| [ゴゼンタチバナ](https://ja.wikipedia.org/wiki/ゴゼンタチバナ) | *Cornus canadensis* | 北海道～本州 | 草本状、高山植物、白い苞 |
| [ヒメサンシュユ](https://ja.wikipedia.org/wiki/ヒメサンシュユ) | *Cornus chinensis* | 栽培 | サンシュユより小型 |

### 日本産主要種（その他）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [アオキ](https://ja.wikipedia.org/wiki/アオキ) | *Aucuba japonica* | 本州～沖縄 | 常緑、赤い果実、日陰に強い |
| [ヒメアオキ](https://ja.wikipedia.org/wiki/ヒメアオキ) | *Aucuba japonica* var. *borealis* | 北海道～本州（日本海側） | 小型、多雪地適応 |

> 📝 **アオキの分類**：アオキ属はかつてミズキ科に含まれていたが、APG体系ではガリア目アオキ科（またはガリア科）とされることも

### ミズキとクマノミズキの違い

| 特徴 | ミズキ | クマノミズキ |
|------|--------|--------------|
| 葉序 | 互生 | 対生 |
| 開花時期 | 5月 | 6～7月 |
| 枝の出方 | 階段状 | あまり階段状でない |

### ヤマボウシとハナミズキの違い

| 特徴 | ヤマボウシ | ハナミズキ |
|------|------------|------------|
| 原産地 | 日本・東アジア | 北アメリカ |
| 総苞片の先端 | 尖る | 丸い、凹む |
| 開花時期 | 5～6月（葉が出た後） | 4～5月（葉より先） |
| 果実 | 集合果（食用可） | 個々の核果が集まる |

### ==ヤマボウシの偽花構造（2024年度問60 完全出題）==

!!! danger "2024年度問60で全5問出題された超重要トピック"
    
    「この植物は4枚の白い花弁をもつ花が咲いているように見えるが……」という問題文が2024年度問60に出題。
    
    | 問い | 答え |
    |------|------|
    | 花弁に見える白い4枚のものは？ | **総苞片（A→正解4：団集花序）** |
    | 中心部の花序は？ | **団集（団散）花序**（20〜30個の小花が頭状集合） |
    | 花弁状のものの正体（B）は？ | **総苞片** |
    | 果実表面の突起（C）は？ | **花柱の残骸** |
    | 同じ科の種は？ | **ゴゼンタチバナ（ミズキ科）** |
    
    **偽花（pseudanthium）の仕組み**：
    - 中心部＝**団集花序**（20〜30個の両性花が頭状に集まる）
    - 周囲の白い4枚＝**総苞片（大型化した苞）**←花弁ではない！
    - 複合果（集合果）表面の突起＝**花柱の残骸**
    - ノウゼンカズラ科のハナミズキ観察でも同様の問いが想定される
    
    **偽花をもつ植物**：ヤマボウシ（ミズキ科）・トウダイグサ属（杯状花序）・マツムシソウ（頭花）等

## 2. アジサイ科 Hydrangeaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 6属 |
| 日本産種数 | 約40種 |
| 世界の属数 | 約17属 |
| 世界の種数 | 約250種 |

> 📝 **分類の変更**：かつてユキノシタ科に含まれていたが、APG体系でアジサイ科として独立

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 木本（低木が多い）、まれに草本 |
| 葉 | 対生、単葉、鋸歯縁 |
| 花 | 両性花と装飾花をもつものが多い |
| 装飾花 | 萼片が大きく発達、不稔 |
| 子房 | 下位～半下位 |
| 果実 | 蒴果 |

### 日本産主要種（アジサイ属 Hydrangea）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ガクアジサイ](https://ja.wikipedia.org/wiki/ガクアジサイ) | *Hydrangea macrophylla* f. *normalis* | 本州～四国 | 周辺に装飾花、中心に両性花 |
| [アジサイ](https://ja.wikipedia.org/wiki/アジサイ) | *Hydrangea macrophylla* f. *macrophylla* | 栽培 | 全て装飾花の園芸品種 |
| [ヤマアジサイ](https://ja.wikipedia.org/wiki/ヤマアジサイ) | *Hydrangea serrata* | 本州～九州 | 小型、山地性、変異多い |
| [エゾアジサイ](https://ja.wikipedia.org/wiki/エゾアジサイ) | *Hydrangea serrata* var. *megacarpa* | 北海道～本州 | ヤマアジサイの北方型 |
| [コアジサイ](https://ja.wikipedia.org/wiki/コアジサイ) | *Hydrangea hirta* | 本州～九州 | 装飾花なし、芳香あり |
| [タマアジサイ](https://ja.wikipedia.org/wiki/タマアジサイ) | *Hydrangea involucrata* | 本州～九州 | つぼみが球状 |
| [ノリウツギ](https://ja.wikipedia.org/wiki/ノリウツギ) | *Hydrangea paniculata* | 北海道～九州 | 円錐花序、樹皮から糊を採る |
| [カシワバアジサイ](https://ja.wikipedia.org/wiki/カシワバアジサイ) | *Hydrangea quercifolia* | 栽培（北米原産） | 葉がカシワ状 |

### 日本産主要種（その他の属）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ツルアジサイ](https://ja.wikipedia.org/wiki/ツルアジサイ) | *Hydrangea petiolaris* | 北海道～九州 | つる性、気根で登攀 |
| [イワガラミ](https://ja.wikipedia.org/wiki/イワガラミ) | *Schizophragma hydrangeoides* | 北海道～九州 | つる性、装飾花の萼片1枚 |
| [ウツギ](https://ja.wikipedia.org/wiki/ウツギ) | *Deutzia crenata* | 北海道～九州 | 「卯の花」、髄が中空 |
| [マルバウツギ](https://ja.wikipedia.org/wiki/マルバウツギ) | *Deutzia scabra* | 本州～九州 | 葉が丸い |
| [ヒメウツギ](https://ja.wikipedia.org/wiki/ヒメウツギ) | *Deutzia gracilis* | 本州～九州 | 小型、岩場 |
| [バイカウツギ](https://ja.wikipedia.org/wiki/バイカウツギ) | *Philadelphus satsumi* | 本州～九州 | 白い4弁花、芳香 |
| [サツマウツギ](https://ja.wikipedia.org/wiki/サツマウツギ) | *Philadelphus satsumi* | 本州～九州 | バイカウツギと同種扱いも |

### アジサイの花の色

[アジサイ](https://ja.wikipedia.org/wiki/アジサイ)の花（装飾花の萼）の色は土壌のpHによって変化します。

| 土壌pH | 色 | 理由 |
|--------|-----|------|
| 酸性 | 青色 | アルミニウムイオンが吸収される |
| 中性～アルカリ性 | ピンク～赤色 | アルミニウムが吸収されない |

### ツルアジサイとイワガラミの違い

| 特徴 | ツルアジサイ | イワガラミ |
|------|--------------|------------|
| 装飾花の萼片 | 3～4枚 | 1枚 |
| 葉の形 | 卵形～円形 | 卵形、より尖る |
| 分布 | 全国 | 全国 |

## 確認問題

### 問1
ミズキ科の葉脈の特徴を述べよ。

<details>
<summary>解答を見る</summary>

**側脈が葉縁に沿って弓なりに走る**

</details>

### 問2
ミズキとクマノミズキの葉序の違いを述べよ。

<details>
<summary>解答を見る</summary>

- **ミズキ**：互生
- **クマノミズキ**：対生

</details>

### 問3
アジサイの花の色が土壌pHによって変わる仕組みを説明せよ。

<details>
<summary>解答を見る</summary>

- **酸性土壌**：アルミニウムイオンが溶け出し、花に吸収されて**青色**になる
- **アルカリ性土壌**：アルミニウムが吸収されず、**ピンク～赤色**になる

</details>

### 問4
ツルアジサイとイワガラミの装飾花の違いを述べよ。

<details>
<summary>解答を見る</summary>

- **ツルアジサイ**：装飾花の萼片が**3～4枚**
- **イワガラミ**：装飾花の萼片が**1枚**

</details>
