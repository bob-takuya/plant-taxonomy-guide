## はじめに

[ツツジ目](https://ja.wikipedia.org/wiki/ツツジ目)（Ericales）は、キク上群の基部に位置する大きな目で、ツツジ科、ツバキ科、サクラソウ科など日本で馴染み深い植物を多く含みます。本章ではツツジ科を中心に解説します。

---

| 項目 | 内容 |
|------|------|
| 日本産科数 | 7科 |
| 世界の科数 | 22科 |
| 世界の種数 | 約12,000種 |

## 1. ツツジ科 Ericaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約20属 |
| 日本産種数 | 約100種 |
| 世界の属数 | 約125属 |
| 世界の種数 | 約4,000種 |

> 📝 **分類の変更**：APG体系で旧イチヤクソウ科、旧ガンコウラン科がツツジ科に統合された

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 木本（低木が多い）、まれに草本 |
| 葉 | 互生、単葉、革質が多い |
| 花 | 合弁花、花冠は壺形・筒形・漏斗形 |
| 雄しべ | 花弁の2倍（10本が多い）、葯は頂孔開裂 |
| 果実 | 蒴果または液果 |
| 菌根 | [エリコイド菌根](https://ja.wikipedia.org/wiki/菌根#エリコイド菌根)を形成（酸性土壌適応） |

## 確認問題

### 問1
ツツジ科の雄しべの特徴を述べよ。

<details>
<summary>解答を見る</summary>

- **雄しべは花弁の2倍**（通常10本）
- **葯は頂孔開裂**

</details>

### 問2
APG体系でツツジ科に統合された旧科を2つ挙げよ。

<details>
<summary>解答を見る</summary>

- **旧イチヤクソウ科**
- **旧ガンコウラン科**

</details>

### 問3
レンゲツツジに含まれる有毒成分は何か。

<details>
<summary>解答を見る</summary>

**グラヤノトキシン**

</details>

### 問4
ミツバツツジとシロヤシオの葉の共通点を述べよ。

<details>
<summary>解答を見る</summary>

**3～5枚の葉が輪生状につく**

</details>
