# 16-1. コケ植物概論

## はじめに

[コケ植物](https://ja.wikipedia.org/wiki/コケ植物)（Bryophytes）は、維管束をもたない陸上植物の一群です。蘚類・苔類・ツノゴケ類の3群からなり、世界に約20,000種、日本に約1,800種が知られています。

---

## コケ植物の概要

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産種数 | 約1,800種 |
| 世界の種数 | 約20,000種 |
| 3つの群 | 蘚類、苔類、ツノゴケ類 |

### コケ植物の特徴

| 特徴 | 内容 |
|------|------|
| 維管束 | **なし** |
| 根 | 仮根（rhizoid）のみ |
| 世代交代 | 配偶体優占型 |
| 生殖 | 精子は水中を泳いで受精 |
| 生育環境 | 湿潤な環境を好む |

---

## コケ植物の体制

### 配偶体と胞子体

| 世代 | 特徴 |
|------|------|
| 配偶体（n） | 主要な植物体、光合成を行う |
| 胞子体（2n） | 配偶体上に寄生、胞子を作る |

> 📝 **世代交代の違い**：シダや種子植物では胞子体（2n）が優占するが、コケ植物では配偶体（n）が優占

### 生殖器官

| 器官 | 機能 |
|------|------|
| 造卵器 | 卵を作る |
| 造精器 | 精子を作る |
| 蒴（さく） | 胞子嚢、胞子を作る |

---

## コケ植物の分類

### 3つの門

| 門 | 和名 | 特徴 | 種数（世界） |
|----|------|------|--------------|
| Bryophyta | [蘚類](https://ja.wikipedia.org/wiki/蘚類) | 茎葉体、蒴柄が発達 | 約12,000種 |
| Marchantiophyta | [苔類](https://ja.wikipedia.org/wiki/苔類) | 葉状体または茎葉体 | 約7,000種 |
| Anthocerotophyta | [ツノゴケ類](https://ja.wikipedia.org/wiki/ツノゴケ類) | 葉状体、角状の胞子体 | 約300種 |

### 系統関係

```
陸上植物
│
├── ツノゴケ類 Anthocerotophyta
│
├── 苔類 Marchantiophyta
│
├── 蘚類 Bryophyta
│
└── 維管束植物（シダ＋種子植物）
```

> 📝 **注意**：コケ植物が単系統群かどうかは議論がある。分子系統解析では、蘚類・苔類・ツノゴケ類は各々独立の系統とする見解もある

## 1. 蘚類 Bryophyta

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産種数 | 約1,000種 |
| 体制 | 茎葉体 |
| 蒴 | 蒴柄の先端につく |
| 蓋 | 蓋（ふた）がある |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 茎 | 直立または匍匐 |
| 葉 | 螺旋状または2列に配列 |
| 中肋 | 葉の中央に中肋をもつものが多い |
| 蒴柄 | 発達、蒴を持ち上げる |
| 蓋 | 成熟すると外れて胞子散布 |
| 蒴歯 | 蓋の下に歯状構造 |

### 主要な科

| 科名 | 特徴 | 代表種 |
|------|------|--------|
| [スギゴケ科](https://ja.wikipedia.org/wiki/スギゴケ科) | 大型、直立 | スギゴケ、オオスギゴケ |
| [シラガゴケ科](https://ja.wikipedia.org/wiki/シラガゴケ科) | 白っぽい、クッション状 | アラハシラガゴケ |
| [ハイゴケ科](https://ja.wikipedia.org/wiki/ハイゴケ科) | 匍匐性 | ハイゴケ |
| [カガミゴケ科](https://ja.wikipedia.org/wiki/カガミゴケ科) | 光沢、羽状分枝 | カガミゴケ |
| [チョウチンゴケ科](https://ja.wikipedia.org/wiki/チョウチンゴケ科) | 林床 | コツボゴケ |
| [ギボウシゴケ科](https://ja.wikipedia.org/wiki/ギボウシゴケ科) | 岩上 | ギボウシゴケ |
| [ホウオウゴケ科](https://ja.wikipedia.org/wiki/ホウオウゴケ科) | 2列葉、鳳凰の羽状 | ホウオウゴケ |
| [シッポゴケ科](https://ja.wikipedia.org/wiki/シッポゴケ科) | 大型、羽状分枝 | シッポゴケ |

## 2. 苔類 Marchantiophyta

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産種数 | 約700種 |
| 体制 | 葉状体または茎葉体 |
| 蒴 | 蓋がない、裂開 |
| 油体 | 細胞内に油体をもつ |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 体制 | 葉状体（ゼニゴケ型）または茎葉体（ムチゴケ型） |
| 葉 | 2～3列、腹葉をもつものも |
| 油体 | 細胞内に特有の油体 |
| 蒴 | 蓋がなく、4裂または不規則に裂開 |
| 弾糸 | 胞子散布を助ける |

### 主要な科

| 科名 | 体制 | 代表種 |
|------|------|--------|
| [ゼニゴケ科](https://ja.wikipedia.org/wiki/ゼニゴケ科) | 葉状体 | ゼニゴケ |
| [ジャゴケ科](https://ja.wikipedia.org/wiki/ジャゴケ科) | 葉状体 | ジャゴケ |
| [ウロコゴケ科](https://ja.wikipedia.org/wiki/ウロコゴケ科) | 茎葉体 | ウロコゴケ |
| [ムチゴケ科](https://ja.wikipedia.org/wiki/ムチゴケ科) | 茎葉体 | ムチゴケ |
| [ヤスデゴケ科](https://ja.wikipedia.org/wiki/ヤスデゴケ科) | 茎葉体 | ヤスデゴケ |
| [ツキヌキゴケ科](https://ja.wikipedia.org/wiki/ツキヌキゴケ科) | 茎葉体 | ツキヌキゴケ |
| [クサリゴケ科](https://ja.wikipedia.org/wiki/クサリゴケ科) | 茎葉体 | クサリゴケ |

## 3. ツノゴケ類 Anthocerotophyta

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産種数 | 約20種 |
| 体制 | 葉状体 |
| 胞子体 | 角状、基部から成長 |
| 葉緑体 | 1細胞に1個 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 配偶体 | 薄い葉状体 |
| 胞子体 | **角状**、先端から裂開 |
| 成長 | 胞子体は基部から継続成長 |
| 葉緑体 | 1細胞に1個（他のコケは多数） |

### 主要な種

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ナガサキツノゴケ](https://ja.wikipedia.org/wiki/ナガサキツノゴケ) | *Anthoceros punctatus* | 本州～沖縄 | 水田、畑地 |
| [ニワツノゴケ](https://ja.wikipedia.org/wiki/ニワツノゴケ) | *Phaeoceros laevis* | 全国 | 庭、路傍 |

## コケ植物の生態

### 生育環境

| 環境 | 代表的なコケ |
|------|--------------|
| 日向・乾燥地 | ギンゴケ、ハマキゴケ |
| 林床 | スギゴケ、コツボゴケ |
| 岩上 | ギボウシゴケ、イワダレゴケ |
| 樹幹 | ヤスデゴケ、キヌゴケ |
| 湿原 | ミズゴケ |
| 水辺 | カワゴケ |

### コケ植物の役割

| 役割 | 内容 |
|------|------|
| 水分保持 | スポンジ状に水を貯える |
| 土壌形成 | 岩石の風化を促進 |
| 遷移の先駆 | 裸地に最初に定着 |
| 環境指標 | 大気汚染に敏感 |

## 確認問題

### 問1
コケ植物の世代交代で、優占する世代は何か。

<details>
<summary>解答を見る</summary>

**配偶体（n）**

維管束植物では胞子体（2n）が優占

### 問2
コケ植物を構成する3つの群を挙げよ。

<details>
<summary>解答を見る</summary>

- **蘚類**（Bryophyta）
- **苔類**（Marchantiophyta）

</details>

### 問3
苔類の細胞内に特有の構造は何か。

<details>
