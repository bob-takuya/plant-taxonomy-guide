## はじめに

本章では、シダ植物のうち小葉植物（ヒカゲノカズラ類、イワヒバ類、ミズニラ類）とトクサ類を解説します。これらは薄嚢シダ類とは系統的に離れた古い系統群です。

## 1. 小葉植物（リコプシダ綱）

### 概要

| 項目 | 内容 |
|------|------|
| 特徴 | 小さな葉（小葉）をもつ |
| 葉脈 | 分岐しない1本の葉脈 |
| 石炭紀 | 石炭紀に大繁栄（巨大な樹木も） |
| 現存群 | 草本のみ |

### 1-2. イワヒバ科 Selaginellaceae

!!! danger "イワヒバ科は異形胞子性（2022年度問21で出題）"
    「**同形胞子性である**」→ **誤り**（これが試験の正解＝誤りを選ぶ問題）
    
    イワヒバ科は**異形胞子性（heterosporous）**：**大胞子（megaspore）**と**小胞子（microspore）**の2種類を作る。
    大胞子→雌性配偶体、小胞子→雄性配偶体。ミズニラ科も同様に異形胞子性。
    ヒカゲノカズラ科は**同形胞子性**（1種類の胞子のみ）。
    
    また、イワヒバ科の葉には**小舌（ligule）**がある（ミズニラ科も同様）。「小舌がない」は誤り。

!!! info "真葉植物（Euphyllophyta）とは（2022年度問20で出題）"
    被子植物・裸子植物・シダ植物（薄嚢シダ類・分岐シダ類）は「**真葉植物**」（大型葉・megaphyllをもつ）。
    **石松類（ヒカゲノカズラ科・イワヒバ科・ミズニラ科）は真葉植物ではない**（小型葉・microphyllのみ）。
    2022年度問20：「真葉植物に分類される科」→ **ゼンマイ科**（シダ植物）が正解。

#### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 1属 |
| 日本産種数 | 約20種 |
| 特徴 | **異型胞子性**（大胞子・小胞子）、葉に**小舌あり** |

#### 日本産主要種

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [イワヒバ](https://ja.wikipedia.org/wiki/イワヒバ) | *Selaginella tamariscina* | 本州～九州 | 岩上、乾燥に強い、園芸品種多い |
| [クラマゴケ](https://ja.wikipedia.org/wiki/クラマゴケ) | *Selaginella remotifolia* | 北海道～九州 | 這性、林床 |
| [カタヒバ](https://ja.wikipedia.org/wiki/カタヒバ) | *Selaginella involvens* | 本州～沖縄 | 岩上、乾燥すると巻く |
| [コンテリクラマゴケ](https://ja.wikipedia.org/wiki/コンテリクラマゴケ) | *Selaginella uncinata* | 帰化 | 青みがかった葉、観賞用 |
| [ヒメクラマゴケ](https://ja.wikipedia.org/wiki/ヒメクラマゴケ) | *Selaginella heterostachys* | 本州～九州 | 小型 |

> 📝 **異型胞子性**：大胞子（雌）と小胞子（雄）を別々に作る

## 2. トクサ類（トクサ綱）

### 2-1. トクサ科 Equisetaceae

#### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 1属 |
| 日本産種数 | 約10種 |
| 特徴 | 茎に節、中空、珪酸を含む |

#### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 茎 | 中空、節がある、縦の稜がある |
| 葉 | 退化して鞘状 |
| 胞子嚢穂 | 茎頂に形成 |
| 珪酸 | 表皮に珪酸を含む（研磨に使用） |

#### 日本産主要種

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [トクサ](https://ja.wikipedia.org/wiki/トクサ) | *Equisetum hyemale* | 北海道～本州 | 常緑、研磨材、庭園植物 |
| [スギナ](https://ja.wikipedia.org/wiki/スギナ) | *Equisetum arvense* | 全国 | 栄養茎と胞子茎が分離、ツクシは胞子茎 |
| [イヌスギナ](https://ja.wikipedia.org/wiki/イヌスギナ) | *Equisetum palustre* | 北海道～本州 | 湿地性 |
| [イヌドクサ](https://ja.wikipedia.org/wiki/イヌドクサ) | *Equisetum ramosissimum* | 本州～沖縄 | 常緑、分枝する |
| [ミズドクサ](https://ja.wikipedia.org/wiki/ミズドクサ) | *Equisetum fluviatile* | 北海道～本州 | 水生 |
| [フサスギナ](https://ja.wikipedia.org/wiki/フサスギナ) | *Equisetum sylvaticum* | 北海道～本州 | 林床、側枝がさらに分枝 |
| [ヤチスギナ](https://ja.wikipedia.org/wiki/ヤチスギナ) | *Equisetum palustre* | 北海道 | 湿地 |

### スギナとツクシ

| 部位 | 説明 |
|------|------|
| スギナ | 緑色の栄養茎、光合成を行う |
| ツクシ | 褐色の胞子茎、春に出る、食用 |

> 📝 **ツクシの名の由来**：「土筆」と書くのは、筆が地面から生えているように見えることから

## 暗記用まとめ

### 小葉植物の科

| 科名 | 特徴 | 代表種 |
|------|------|--------|
| ヒカゲノカズラ科 | 同型胞子 | ヒカゲノカズラ |
| イワヒバ科 | 異型胞子 | イワヒバ |
| ミズニラ科 | 水生 | ミズニラ |

### トクサ科の識別

| 種名 | 特徴 |
|------|------|
| トクサ | 常緑、分枝しない |
| スギナ | 栄養茎（スギナ）と胞子茎（ツクシ）が分離 |
| イヌドクサ | 分枝する |

### 石炭紀の繁栄

| 群 | 石炭紀の形態 | 現在 |
|----|--------------|------|
| 小葉植物 | 巨大な樹木（鱗木類など） | 草本のみ |
| トクサ類 | 大型の樹木（ロボク類） | 草本のみ |
