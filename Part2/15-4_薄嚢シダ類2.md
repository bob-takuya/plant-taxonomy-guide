# 15-4. 薄嚢シダ類2（ウラボシ目2：オシダ科・メシダ科）

## はじめに

本章では、薄嚢シダ類の中でも最も大きな科である[オシダ科](https://ja.wikipedia.org/wiki/オシダ科)と[メシダ科](https://ja.wikipedia.org/wiki/メシダ科)を解説します。日本のシダの多くがこれらの科に属します。

---

## 1. オシダ科 Dryopteridaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約15属 |
| 日本産種数 | 約150種 |
| 世界の種数 | 約2,100種 |
| 特徴 | 日本最大のシダ科 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 根茎 | 短く直立または斜上、鱗片を密生 |
| 葉 | 1～3回羽状複葉 |
| 葉柄基部 | 鱗片が多い |
| ソーラス | 円形 |
| 包膜 | 円腎形（D字型）、ソーラスの側面につく |

## 2. メシダ科 Athyriaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約8属 |
| 日本産種数 | 約80種 |
| 世界の種数 | 約650種 |
| 特徴 | 葉が軟質、包膜がJ字型～馬蹄形 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 根茎 | 短く直立～斜上、または匍匐 |
| 葉 | 2～3回羽状複葉 |
| 葉質 | 軟質、草質 |
| ソーラス | 円形または線形 |
| 包膜 | J字形、馬蹄形、または欠く |

### 日本産主要種（メシダ属 Athyrium）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [メシダ](https://ja.wikipedia.org/wiki/メシダ) | *Athyrium melanolepis* | 北海道～九州 | オシダに対する「雌シダ」 |
| [ヤマイヌワラビ](https://ja.wikipedia.org/wiki/ヤマイヌワラビ) | *Athyrium vidalii* | 北海道～九州 | 山地林床 |
| [イヌワラビ](https://ja.wikipedia.org/wiki/イヌワラビ) | *Athyrium niponicum* | 北海道～九州 | 普通種、変異多い |
| [カラクサイヌワラビ](https://ja.wikipedia.org/wiki/カラクサイヌワラビ) | *Athyrium clivicola* | 本州～九州 | 唐草模様の斑入り |
| [ホソバイヌワラビ](https://ja.wikipedia.org/wiki/ホソバイヌワラビ) | *Athyrium iseanum* | 本州～九州 | 細い葉身 |
| [サトメシダ](https://ja.wikipedia.org/wiki/サトメシダ) | *Athyrium deltoidofrons* | 本州～沖縄 | 里山に多い |
| [ヘビノネゴザ](https://ja.wikipedia.org/wiki/ヘビノネゴザ) | *Athyrium yokoscense* | 北海道～九州 | 鉱山跡地、重金属耐性 |
| [ミヤマメシダ](https://ja.wikipedia.org/wiki/ミヤマメシダ) | *Athyrium melanolepis* | 北海道～本州 | 山地性 |

### 日本産主要種（シケシダ属 Deparia）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [クサソテツ](https://ja.wikipedia.org/wiki/クサソテツ) | *Matteuccia struthiopteris* | 北海道～九州 | 若芽は**コゴミ**として食用 |

### イヌワラビ類の変異

| 特徴 | 内容 |
|------|------|
| 変異 | 非常に多様、多数の品種 |
| 園芸 | 「ニシキシダ」として観賞用に栽培 |
| 色彩 | 赤紫色～銀白色の斑入りも |

### クサソテツ（コゴミ）

| 項目 | 内容 |
|------|------|
| 和名 | クサソテツ |
| 食用名 | **コゴミ**（若芽） |
| 特徴 | ゼンマイに似るが毛がない |
| 調理 | アク抜き不要、おひたし等 |
| 分布 | 北海道～九州の山地 |

> 📝 **コゴミ**：クサソテツの若芽。ワラビと異なりアク抜き不要で食べやすい山菜

## オシダ・メシダの語源

| 名称 | 意味 |
|------|------|
| オシダ（雄羊歯） | 大型で剛強なシダの意 |
| メシダ（雌羊歯） | オシダに比べ軟らかいシダの意 |

## 暗記用まとめ

### オシダ科の主要属

| 属名 | 特徴 | 代表種 |
|------|------|--------|
| オシダ属 | 鱗片多い、円腎形包膜 | オシダ、ベニシダ |
| ヤブソテツ属 | 1回羽状、羽片全縁～鋸歯 | ヤブソテツ |
| イノデ属 | 鱗片多い、羽片鋸歯 | イノデ、ジュウモンジシダ |
| カナワラビ属 | 2～3回羽状 | リョウメンシダ |

### メシダ科の主要属

| 属名 | 特徴 | 代表種 |
|------|------|--------|
| メシダ属 | 軟質、J字形包膜 | イヌワラビ |
| シケシダ属 | 湿地性 | シケシダ |
| クサソテツ属 | 二形性、食用 | クサソテツ（コゴミ） |

### 食用シダのまとめ

| 食用名 | 正式和名 | 科名 | 注意点 |
|--------|----------|------|--------|
| ワラビ | ワラビ | コバノイシカグマ科 | 要アク抜き |
| ゼンマイ | ゼンマイ | ゼンマイ科 | 乾燥保存 |
| コゴミ | クサソテツ | メシダ科 | アク抜き不要 |

### オシダとメシダの比較

| 特徴 | オシダ科 | メシダ科 |
|------|----------|----------|
| 葉質 | 革質～紙質 | 軟質～草質 |
| 包膜 | 円腎形（D字型） | J字形～馬蹄形 |
| 鱗片 | 多い | 少ない |
