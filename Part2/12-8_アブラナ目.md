## はじめに

[アブラナ目](https://ja.wikipedia.org/wiki/アブラナ目)（Brassicales）は、アオイ類に属する目で、アブラナ科を中心に経済的に重要な野菜・香辛料植物を多く含みます。[グルコシノレート](https://ja.wikipedia.org/wiki/グルコシノレート)（からし油配糖体）を含むことが特徴です。

| 世界の科数 | 17科 |

## 1. アブラナ科 Brassicaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約50属 |
| 日本産種数 | 約100種 |
| 世界の属数 | 約320属 |
| 世界の種数 | 約3,700種 |

> 📝 **旧名**：十字花科（Cruciferae）とも呼ばれる

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 草本（1年草または多年草） |
| 味・臭い | グルコシノレートによる辛味・刺激臭 |
| 葉 | 互生、托葉なし |
| 花 | **十字形花冠**（花弁4枚が十字状） |
| 萼片 | 4枚 |
| 雄しべ | **6本**（4強雄しべ：長い4本＋短い2本） |
| 子房 | 上位、2心皮 |
| 果実 | [角果](https://ja.wikipedia.org/wiki/角果)（長角果または短角果） |

### 角果の分類（詳細比較表）

| 種類 | 定義 | 形状 | 代表種（属） | 試験重要度 |
|------|------|------|--------------|------------|
| **長角果**（silique） | 長さが幅の**3倍以上** | 細長い | ナタネ、キャベツ、タネツケバナ属、ハタザオ属 | ★★★★ |

!!! warning "ダイコン属の果実は不裂開（2022年度問47で出題）"
    ダイコン属（*Raphanus*）の果実は通常の長角果と異なり、**海綿状で成熟しても裂開しない**（不裂開果）。
    くびれのある形状で、1〜数個の種子を含む区画に分かれる（節果・ロメントに近い構造）。
    「アブラナ科で裂開しない果実をもつ属」→ **ダイコン属**（2022年度問47 正解1）。
| **短角果**（silicle） | 長さが幅の**3倍未満** | 丸〜楕円・三角形 | ナズナ（三角形）、イヌナズナ（楕円）、グンバイナズナ（軍配形）、マメグンバイナズナ（円形） | ★★★★ |

> **問：ナズナ（*Capsella bursa-pastoris*）の果実の型と形を答えよ。**
> → **短角果**。逆三角形（ハート形）で上が広く下が狭い。

> **問：アブラナ科で「軍配（ぐんばい）の形をした果実」をもつ種は何か？**
> → **グンバイナズナ**（*Thlaspi arvense*）。軍配形の短角果に翼がある。

### ワサビの詳細（試験重要度 ★★★★）

### [ワサビ](https://ja.wikipedia.org/wiki/ワサビ) *Eutrema japonicum*

| 部位/特徴 | 内容 |
|-----------|------|
| 生育環境 | **清流沿いの岩場や渓流（山葵田）** |
| 利用部位 | **根茎（株）**を磨り下ろして香辛料に |
| 辛味成分 | **アリルイソチオシアネート**（allyl isothiocyanate）|
| 成分の生成 | 細胞が破壊されたとき、グルコシノレート（シニグリン）が**ミロシナーゼ**酵素で加水分解されて生成 |
| 葉 | 腎形〜円形の大きな根生葉 |
| 花 | 白い十字形花、4強雄しべ |
| 分布 | 日本固有種（日本・朝鮮半島） |

> **問：ワサビの辛味はどのように生成されるか？その化学物質名を含めて説明せよ。**
> → 根茎の細胞が破壊（磨り下ろす）されると、グルコシノレートの一種**シニグリン**が酵素**ミロシナーゼ**と反応し、**アリルイソチオシアネート**が生成されて辛味と揮発性の香気が発生する。

### セイヨウワサビ（ホースラディッシュ）との比較

| 項目 | ワサビ（*Eutrema japonicum*） | セイヨウワサビ（*Armoracia rusticana*） |
|------|-------------------------------|----------------------------------------|
| 別名 | - | ホースラディッシュ、西洋ワサビ |
| 辛味成分 | アリルイソチオシアネート | 同成分（シニグリン由来） |
| 香り | **独特の爽やかな香り**（さわやかで複雑） | **強烈でシンプルな辛み**（香りは単調） |
| 原産 | 日本・朝鮮半島 | 東ヨーロッパ・西アジア |
| 粉ワサビ | 多くがホースラディッシュ+着色料 | - |

> **問：市販の「粉ワサビ（チューブワサビ）」の多くに使用されている植物は何か？**
> → **セイヨウワサビ（ホースラディッシュ、*Armoracia rusticana*）**が主原料で、着色料（緑色）を添加。本物のワサビは高価で希少。

## ケシ目（Papaverales）の廃止と再配置

> 📝 **分類の変更**：旧来の分類体系（クロンキスト体系など）では**ケシ目（Papaverales）**という目が存在し、ケシ科・フウチョウソウ科・アブラナ科などを含んでいた。

> **問：APG体系でケシ科（Papaveraceae）はどの目に位置付けられるか？**
> → **キンポウゲ目（Ranunculales）**。バラ類ではなく、キク上群（Asterids）でも真正双子葉類の基部グループに再配置された。旧ケシ目は廃止。

> **問：APG体系でアブラナ科はケシ科と同じ目に属するか？**
> → **否**。アブラナ科はアブラナ目（Brassicales）に属しバラ類（アオイ類）。ケシ科はキンポウゲ目（Ranunculales）に属す。旧ケシ目は廃止された。

## 試験頻出ポイント（アブラナ科総まとめ）

> **問：アブラナ科の「4強雄しべ」とは何か説明せよ。**
> → 雄しべが6本あり、そのうち**長い4本と短い2本**に分かれる構造。長い4本が花の中心部に、短い2本が外側につく。「4強」とは4本が特に発達しているという意味。

> **問：春の七草に含まれるアブラナ科の3種を七草名・和名・学名で答えよ。**
> → ①**ナズナ**（ナズナ、*Capsella bursa-pastoris*）②**スズナ**（カブ、*Brassica rapa* var. *rapa*）③**スズシロ**（ダイコン、*Raphanus sativus*）
