## はじめに

[ユキノシタ目](https://ja.wikipedia.org/wiki/ユキノシタ目)（Saxifragales）は、中核真正双子葉類の基部に位置する目で、形態的・生態的に多様な科を含みます。試験ではユキノシタ科、ベンケイソウ科、マンサク科、カツラ科が頻出です。

| 世界の科数 | 約15科 |
| 世界の種数 | 約2,500種 |
| 主な特徴 | 草本～木本まで多様、多くは2心皮 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 多年草（まれに1年草・低木） |
| 葉 | 互生または根生、単葉、鋸歯縁が多い |

**ユキノシタの詳細**
- 別名「虎耳草」（葉の形から）
- 民間薬として中耳炎などに使用
- 葉は天ぷらにして食用可
- 匍匐枝（ランナー）で栄養繁殖

#### 🔴 ユキノシタの詳細（試験重要度 ★★★★★）

| 部位/特徴 | 内容 |
|-----------|------|
| 学名 | *Saxifraga stolonifera* |
| 分布 | 本州〜九州（湿った岩場・林縁） |
| 葉 | 円形〜腎形、**脈に沿って白い斑（まだら模様）**、葉裏は赤紫色 |
| 走出枝 | **細い走出枝（ランナー）**を出して栄養繁殖（stolonifera = ランナーを出す） |
| 花 | 白色、5弁花 |
| **花の重要形質** | **花弁が不等長**：上の3枚が小さい卵形、**下の2枚が著しく長く垂れる** |
| 花弁サイズ | 上3枚：約5mm、下2枚：**約15〜20mm**（3〜4倍長い） |
| 食用 | **葉を天ぷらにして食用**（山菜として利用） |
| 民間薬 | 中耳炎・火傷・虫刺されに生葉汁を使用 |
| 英名 | Strawberry Geranium（ストロベリーゼラニウム） |
| 別名 | 虎耳草（こじそう）、ユキノシタソウ |

#### 試験頻出問題

> **問：** ユキノシタの花弁の特徴（不等長）を説明せよ。
> → **答え：** 花弁5枚のうち、**上の3枚は短く（約5mm）**、**下の2枚は著しく長く垂れ下がる（約15〜20mm）**。この不等花弁がユキノシタの最重要な識別特徴。

> **問：** ユキノシタの走出枝の役割は何か。学名との関係は。
> → **答え：** 走出枝（ランナー）を伸ばして新たな株を作る**栄養繁殖**。学名の種小名「stolonifera」は「走出枝を持つ」の意。

> **問：** ユキノシタの食用方法は何か。
> → **答え：** **葉を天ぷら**にして食用。軽く湯通しして和え物にもする。

| 和名 | 学名 | 分布 | 特徴 |

| [アテツマンサク](https://ja.wikipedia.org/wiki/アテツマンサク) | *H. japonica* var. *bitchuensis* | 本州（中国地方） | 萼の内側も黄色 |

**マンサクの名前の由来**
- 「まず咲く」が転じた説
- 「満作」（豊作の前兆）説

#### 🔴 マンサクの詳細（試験重要度 ★★★★）

| 部位/特徴 | 内容 |
|-----------|------|
| 学名 | *Hamamelis japonica* |
| 生活型 | 落葉低木〜小高木（高さ2〜5m） |
| 花期 | **早春（1〜3月）**：葉が展開する前に開花（日本最早の花木の一つ） |
| 花弁 | **4枚、細長いリボン状（線形）**、長さ約1〜1.5cm、黄色 |
| 花弁の機能 | **低温でも枯れにくい特殊な構造**：気温が下がると丸まって自己保護し、温かくなると再び開く |
| 萼 | 4枚、内側が暗紫色（重要な識別ポイント） |
| 果実 | 木質の蒴果、熟すと2つに裂けて種子を飛ばす（最大数m） |
| 英名 | Japanese witch hazel（ウィッチヘーゼル） |
| 薬用 | 樹皮・葉の抽出物が収斂剤・化粧品に使用（西洋マンサク *H. virginiana*） |

#### 試験頻出問題

> **問：** マンサクの花弁の形態的特徴と、それが早春に機能する上での利点を説明せよ。
> → **答え：** 花弁は**細長いリボン状（線形）で4枚**。低温になると丸まって自己保護し、気温が上がると伸びる性質がある。雪や霜があっても花弁が枯死しにくく、早春の開花に適応している。

> **問：** マンサク属の萼の内側の色は何色か（識別に重要）。
> → **答え：** **暗紫色**（赤紫色）。萼の外側は黄褐色。これがシナマンサクとの識別にも使われる（アテツマンサクは内側も黄色）。

## 参考：アジサイ（旧ユキノシタ科→現ミズキ目）

> ⚠️ アジサイはAPG体系で**ミズキ目アジサイ科**に移動しました。ユキノシタ目ではありませんが、旧分類（ユキノシタ科）の重要種として試験に頻出のため、ここで詳細を記載します。

### [アジサイ](https://ja.wikipedia.org/wiki/アジサイ) *Hydrangea macrophylla*

| 項目 | 内容 |
|------|------|
| 現在の分類 | **ミズキ目（Cornales）アジサイ科（Hydrangeaceae）** |
| 旧分類 | ユキノシタ科 |
| 分布 | 本州（太平洋岸）〜九州（山地）、広く栽培 |
| 生活型 | 落葉低木（高さ1〜2m） |
| 花期 | 6〜7月（梅雨の花） |

#### 形態的特徴（最重要：装飾花と両性花の区別）

| 花の種類 | 特徴 | 機能 |
|----------|------|------|
| **装飾花（中性花）** | 大型の花弁状萼片（4〜5枚）、雄しべ・雌しべが退化 | **昆虫への誘引**（花粉・蜜はなし） |
| **両性花（真の花）** | 小型、花弁5枚・雄しべ10本・雌しべあり | **実際の繁殖** |

#### ガクアジサイとアジサイ（ホンアジサイ）の違い

| 特徴 | ガクアジサイ（原種） | アジサイ（ホンアジサイ）（園芸品種） |
|------|---------------------|--------------------------------------|
| 学名 | *H. macrophylla* f. *normalis* | *H. macrophylla* f. *macrophylla* |
| 花序の形 | **周縁に装飾花、中央に両性花**（額縁状） | **全体が装飾花**（玉状） |
| 結実 | 結実する | **結実しない**（両性花が退化） |
| 野生型 | ○ 本来の形 | × 栽培選抜 |
| 別名 | ガクアジサイ（がく咲き） | テマリアジサイ |

#### なぜ重要か（試験重要度 ★★★★）

- **装飾花と両性花の区別**は試験最頻出事項
- ガクアジサイが原種で、アジサイ（ホンアジサイ）は栽培選抜種
- 花の色が変わる理由：**土壌のpH**（酸性→青、アルカリ性→赤）とアルミニウムの吸収量
- 全草（特に葉・若芽）が**有毒**（青酸配糖体を含む）→不用意に食べない
- **旧ユキノシタ科から現ミズキ目アジサイ科へ**の分類変更（試験頻出！）

#### 試験頻出問題

> **問：** アジサイの「装飾花（中性花）」と「両性花（真の花）」の違いを説明せよ。
> → **答え：** 装飾花は大型の花弁状萼片（4〜5枚）を持つが雄しべ・雌しべが退化しており結実しない。昆虫への**誘引機能**を持つ。両性花は小型で目立たないが、花弁・雄しべ・雌しべを全て持ち**実際の繁殖**を担う。

> **問：** ガクアジサイとアジサイ（ホンアジサイ）の最大の形態的違いは何か。
> → **答え：** ガクアジサイは花序の**周縁のみに装飾花**があり中央に両性花（結実する）。アジサイ（ホンアジサイ）は**花序全体が装飾花**のみ（結実しない）。

> **問：** アジサイはAPG体系でどの目・科に分類されるか。
> → **答え：** **ミズキ目（Cornales）アジサイ科（Hydrangeaceae）**。旧分類ではユキノシタ科だったが再編成された。

## 確認問題（4択形式）

**Q1.** ユキノシタの花弁の特徴として正しいものはどれか。
① 花弁は5枚で全て同じ大きさ　② 花弁は上2枚が長く下3枚が短い　③ 花弁は上3枚が短く下2枚が著しく長く垂れる　④ 花弁は4枚でリボン状

<details><summary>答え</summary>**③ 花弁は上3枚が短く下2枚が著しく長く垂れる** — ユキノシタ（*Saxifraga stolonifera*）の最重要識別特徴。不等花弁で下の2枚が上の3枚より約3〜4倍長い。</details>

**Q2.** マンサクについて正しいものはどれか。
① 夏に最初に開花する落葉樹　② 早春（1〜3月）に細長いリボン状の花弁を持つ花を咲かせる　③ 花弁は3枚で白色　④ 熱帯原産で日本では栽培のみ

<details><summary>答え</summary>**② 早春（1〜3月）に細長いリボン状の花弁を持つ花を咲かせる** — マンサク（*Hamamelis japonica*）。葉が展開する前に黄色い細長い花弁4枚の花を咲かせる。低温で丸まって自己保護する特性がある。</details>

**Q3.** アジサイ（ホンアジサイ）とガクアジサイの違いとして正しいものはどれか。
① アジサイは両性花のみで種子ができる　② ガクアジサイは全体が装飾花で種子ができない　③ アジサイは花序全体が装飾花で種子ができず、ガクアジサイは周縁のみ装飾花で両性花も持ちます　④ 両者に形態的な違いはない

<details><summary>答え</summary>**③ アジサイは花序全体が装飾花で種子ができず、ガクアジサイは周縁のみ装飾花で両性花も持つ** — ガクアジサイが原種。アジサイ（ホンアジサイ）は栽培で全花が装飾花となった品種で結実しない。</details>

**Q4.** APG体系でアジサイ属はどの目・科に分類されるか。
① ユキノシタ目ユキノシタ科　② ミズキ目アジサイ科　③ バラ目バラ科　④ ナデシコ目ナデシコ科

<details><summary>答え</summary>**② ミズキ目（Cornales）アジサイ科（Hydrangeaceae）** — 旧分類ではユキノシタ科だったが、APG体系で大幅に再編成された。</details>

**Q5.** ベンケイソウ科の多くの種が行う光合成の型として正しいものはどれか。
① C3型光合成　② C4型光合成（トウモロコシ型）　③ CAM型光合成（夜間CO₂固定）　④ 光合成を行わない（腐生植物）

<details><summary>答え</summary>**③ CAM型光合成（夜間CO₂固定）** — ベンケイソウ科は多肉植物が多く、夜間に気孔を開いてCO₂を取り込み昼間に光合成する（CAM: Crassulacean Acid Metabolism）。乾燥適応の戦略。</details>
