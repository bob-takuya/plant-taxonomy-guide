# 14-2. マツ科

## はじめに

[マツ科](https://ja.wikipedia.org/wiki/マツ科)（Pinaceae）は、裸子植物の中で最も重要な科の一つで、北半球の森林を構成する主要な樹種を含みます。日本にはマツ属、モミ属、トウヒ属、ツガ属、カラマツ属などが自生します。

---

## マツ科の概要

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 7属 |
| 日本産種数 | 約20種 |
| 世界の属数 | 11属 |
| 世界の種数 | 約230種 |
| 分布 | 北半球 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 常緑高木（カラマツ属のみ落葉） |
| 葉 | 針形、束生または螺旋状 |
| 球果 | 木質、種鱗と苞鱗をもつ |
| 種子 | 翼をもつものが多い |
| 樹脂 | 樹脂道を持ち、松脂を分泌 |

## 1. マツ属 Pinus

### 概要

| 項目 | 内容 |
|------|------|
| 日本産種数 | 約8種 |
| 世界の種数 | 約100種 |
| 特徴 | 針葉が2～5本束生（短枝上） |

### 二葉松と五葉松

| 型 | 針葉の数 | 代表種 |
|----|----------|--------|
| 二葉松 | 2本 | アカマツ、クロマツ |
| 五葉松 | 5本 | ゴヨウマツ、ハイマツ |

### 日本産主要種（二葉松）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [アカマツ](https://ja.wikipedia.org/wiki/アカマツ) | *Pinus densiflora* | 北海道～九州 | 樹皮が赤褐色、女松 |
| [クロマツ](https://ja.wikipedia.org/wiki/クロマツ) | *Pinus thunbergii* | 本州～九州 | 樹皮が黒灰色、海岸、雄松 |
| [リュウキュウマツ](https://ja.wikipedia.org/wiki/リュウキュウマツ) | *Pinus luchuensis* | 九州～沖縄 | 琉球列島固有 |

### 日本産主要種（五葉松）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ゴヨウマツ](https://ja.wikipedia.org/wiki/ゴヨウマツ) | *Pinus parviflora* | 本州～九州 | 山地性、盆栽 |
| [ハイマツ](https://ja.wikipedia.org/wiki/ハイマツ) | *Pinus pumila* | 北海道～本州（高山） | 這性、高山帯・亜高山帯 |
| [キタゴヨウ](https://ja.wikipedia.org/wiki/キタゴヨウ) | *Pinus parviflora* var. *pentaphylla* | 北海道～本州 | ゴヨウマツの変種 |
| [チョウセンゴヨウ](https://ja.wikipedia.org/wiki/チョウセンゴヨウ) | *Pinus koraiensis* | 栽培 | 松の実を食用 |

### アカマツとクロマツの違い

| 特徴 | アカマツ | クロマツ |
|------|----------|----------|
| 樹皮 | 赤褐色 | 黒灰色 |
| 冬芽 | 赤褐色 | 白色 |
| 生育環境 | 内陸の山地 | 海岸 |
| 別名 | 女松（メマツ） | 雄松（オマツ） |
| 針葉 | やや細く柔らかい | やや太く硬い |

> 📝 **マツ枯れ**：[マツ材線虫病](https://ja.wikipedia.org/wiki/マツ材線虫病)（マツノザイセンチュウ）により多くのマツ林が被害を受けている

## 2. モミ属 Abies

### 概要

| 項目 | 内容 |
|------|------|
| 日本産種数 | 5種 |
| 特徴 | 針葉が扁平、球果が直立し成熟後に崩壊 |

### 日本産主要種

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [モミ](https://ja.wikipedia.org/wiki/モミ) | *Abies firma* | 本州～九州 | 低山帯、葉先が2裂 |
| [ウラジロモミ](https://ja.wikipedia.org/wiki/ウラジロモミ) | *Abies homolepis* | 本州～九州 | 山地帯、葉裏が白い |
| [シラビソ](https://ja.wikipedia.org/wiki/シラビソ) | *Abies veitchii* | 本州 | 亜高山帯、オオシラビソと混生 |
| [オオシラビソ](https://ja.wikipedia.org/wiki/オオシラビソ) | *Abies mariesii* | 本州 | 亜高山帯、アオモリトドマツとも |
| [トドマツ](https://ja.wikipedia.org/wiki/トドマツ) | *Abies sachalinensis* | 北海道 | 北海道の代表的な針葉樹 |

### モミ属の垂直分布

```
高山帯
↓
亜高山帯：シラビソ、オオシラビソ
↓
山地帯：ウラジロモミ
↓
低山帯：モミ
```

### 概要

| 項目 | 内容 |
|------|------|

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|

## 🔴 過去問頻出：属ごとの識別ポイント（試験最重要）

### 葉枕（ようちん）の有無

葉枕は枝上に残る葉の基部の突起で、葉の脱落後も残る。

| 属 | 葉枕の有無 | 備考 |
|----|-----------|------|
| **トウヒ属** | **あり**（目立つ） | 葉が落ちると葉枕が突起として残る |
| **ツガ属** | **なし** | 葉は有柄だが、葉柄が脱落すると跡が残らない |
| マツ属 | 短枝が残る | 長枝に短枝が生じ、短枝先に針葉が束生 |
| モミ属 | 脱落跡は円形くぼみ | 葉跡が丸くくぼむ |

> 📝 **識別ポイント**：トウヒ属は**葉枕あり**（枝が粗面に見える）、ツガ属は**葉枕なし**（葉は有柄）。これは試験で頻繁に問われる重要な違い。

### 球果の成熟年（試験頻出）

| 属 | 球果成熟年 | 補足 |
|----|-----------|------|
| 属・種 | 球果成熟年 | 補足 |
|--------|----------|------|
| **アカマツ・クロマツ** | **翌年**（開花の翌年秋に成熟） | 2年目に成熟 |
| **ハイマツ** | 翌年秋 | 種子に翼なし（⚠️頻出）|
| **ゴヨウマツ** | **翌々年**（開花の3年目に成熟） | ⚠️アカマツと区別！ |
| **モミ属**（モミ・ウラジロモミ・トドマツ等） | **当年**（開花年の秋に成熟） | 球果は直立、成熟後崩壊 |
| **トウヒ属**（エゾマツ等） | **当年**（年内に成熟） | 球果は下垂、崩壊しない |
| **ツガ属** | 当年 | |
| **カラマツ属** | 当年 | |
| **スギ・ヒノキ科** | 当年 | |

!!! danger "ゴヨウマツの球果成熟は翌々年（3年目）"
    - アカマツ・クロマツ：**翌年**（2年目）
    - **ゴヨウマツ：翌々年（3年目）**← ここが試験の落とし穴
    - 2021年度問53-5・2022年度問33-3で繰り返し出題。「ゴヨウマツの球果は開花の翌年に成熟する」は**誤り**。
    
    語呂合わせ：**「ゴ（5本葉）ヨウ（翌々）マツ」→ 翌々年成熟**

### マツ属の葉の形態

マツ属の葉は**2型**（長枝の葉と短枝の針形葉）がある：
- **長枝**：鱗片状の葉（褐色の膜質）
- **短枝**：針形の葉が**2〜5本束生**

> ⚠️ **注意**：「針形葉は2〜10本束生」は誤り。正しくは2〜5本（日本産種では2本または5本）。

### 各種の詳細識別情報

#### ハイマツ（*Pinus pumila*）

| 特徴 | 内容 |
|------|------|
| 生活形 | **低木**（這性）。高木にはならない |
| 分布 | 北海道・本州の**高山帯** |
| 針葉 | 短枝に**5本束生**（五葉松） |
| 球果 | 卵形、紫褐色に熟す |
| 種子 | **翼がない**（他のマツ属種と異なる） |

> ⚠️ **重要**：ハイマツの種子には**翼（よく）がない**。他のマツ属種（アカマツ、ゴヨウマツ等）は種子に翼がある。

#### カラマツ（*Larix kaempferi*）

| 特徴 | 内容 |
|------|------|
| 生活形 | **落葉高木**（低木ではない） |
| 分布 | 本州中部の亜高山帯に自生、植林に広く利用 |
| 葉 | 短枝に多数束生（20〜40本） |

> ⚠️ **注意**：カラマツは**高木**（低木ではない）。落葉針葉樹だが樹高20m以上になる大木。

#### コメツガ（*Tsuga diversifolia*）

| 特徴 | 内容 |
|------|------|
| 生活形 | 常緑高木 |
| 分布 | **本州・四国の亜高山帯**（北海道には分布しない） |
| 葉 | 線形、2型、短い葉柄あり、葉枕なし |

> ⚠️ **重要**：コメツガは**北海道に分布しない**。北海道の亜高山帯を代表するのはエゾマツ・トドマツ。

#### トウヒ属（*Picea*）の詳細

| 特徴 | 内容 |
|------|------|
| 葉枕 | **あり**（明瞭に突起する） |
| 短枝 | **なし** |
| 苞鱗 | **微小**（種鱗より著しく短い）|
| 球果 | **下垂**、崩壊しない |
| エゾマツ球果 | 年内（当年）に成熟 |

#### ツガ属（*Tsuga*）の詳細

| 特徴 | 内容 |
|------|------|
| 葉枕 | **なし** |
| 葉 | 扁平、**有柄**（短い葉柄あり） |
| 葉は2型 | 主枝の葉と側枝の葉で大きさが異なる |
| 球果 | 小型、下垂 |

## 確認問題

### 問1
マツ属の二葉松と五葉松の違いを述べよ。

<details>
<summary>解答を見る</summary>

- **二葉松**：針葉が2本束生（例：アカマツ、クロマツ）
- **五葉松**：針葉が5本束生（例：ゴヨウマツ、ハイマツ）

</details>

### 問2
アカマツとクロマツの樹皮の色の違いを述べよ。

<details>
<summary>解答を見る</summary>

- **アカマツ**：樹皮が赤褐色
- **クロマツ**：樹皮が黒灰色

</details>

### 問3
モミ属とトウヒ属の球果の違いを述べよ。

<details>
<summary>解答を見る</summary>

- **モミ属**：球果が直立し、成熟後に崩壊する
- **トウヒ属**：球果が下垂し、崩壊しない

</details>

### 問4
マツ科で唯一の落葉針葉樹は何か。

<details>
<summary>解答を見る</summary>

**カラマツ**

</details>
