## はじめに

[キク目](https://ja.wikipedia.org/wiki/キク目)（Asterales）は、キキョウ類に属する目で、被子植物最大の科であるキク科を含みます。本章ではキク科の概論とタンポポ亜科を解説します。

---

## キク目の概要

| 項目 | 内容 |
|------|------|
| 日本産科数 | 3科 |
| 世界の科数 | 11科 |
| 世界の種数 | 約26,000種 |

### 日本産キク目の科

| 科名 | 日本産種数 | 特徴 |
|------|------------|------|
| [キク科](https://ja.wikipedia.org/wiki/キク科) | 約360種 | 被子植物最大科 |
| [キキョウ科](https://ja.wikipedia.org/wiki/キキョウ科) | 約30種 | 乳液、鐘形花 |
| [ミツガシワ科](https://ja.wikipedia.org/wiki/ミツガシワ科) | 約5種 | 水生植物 |

## 1. キク科 Asteraceae（Compositae）

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約110属 |
| 日本産種数 | 約360種 |
| 世界の属数 | 約1,600属 |
| 世界の種数 | 約23,000種 |

> 📝 **被子植物最大の科**：キク科は被子植物で最も種数が多い科

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 草本（まれに木本） |
| 葉 | 互生または対生、托葉なし |
| 花序 | **[頭状花序](https://ja.wikipedia.org/wiki/頭状花序)**（頭花） |
| 総苞 | 頭状花序を包む苞葉 |
| 小花 | 管状花と舌状花の2型 |
| 子房 | 下位、1室1胚珠 |
| 果実 | [痩果](https://ja.wikipedia.org/wiki/痩果)（cypsel） |
| 冠毛 | 萼が変化した毛状の付属体（種子散布） |

### 頭状花序の構造

```
      舌状花（周辺）
         ↓
    ●●●●●●●
   ●○○○○○○●  ← 管状花（中央）
   ●○○○○○○●
   ●○○○○○○●
    ●●●●●●●
         ↑
       総苞片
```

### 小花の2型

| 小花の型 | 特徴 | 例 |
|----------|------|-----|
| [管状花](https://ja.wikipedia.org/wiki/管状花) | 筒状、5裂、両性 | ヒマワリの中央 |
| [舌状花](https://ja.wikipedia.org/wiki/舌状花) | 平らに開く、通常5歯 | タンポポ、ヒマワリの周辺 |

### キク科の主要亜科

| 亜科名 | 小花の構成 | 代表的な属 |
|--------|------------|------------|
| [タンポポ亜科](https://ja.wikipedia.org/wiki/タンポポ亜科) | 舌状花のみ、乳液あり | タンポポ属、ニガナ属 |
| [アザミ亜科](https://ja.wikipedia.org/wiki/アザミ亜科) | 管状花のみ | アザミ属、ゴボウ属 |
| [キク亜科](https://ja.wikipedia.org/wiki/キク亜科) | 周辺に舌状花＋中央に管状花 | キク属、ヒマワリ属 |

## 2. タンポポ亜科 Cichorioideae

### 特徴

| 特徴 | 内容 |
|------|------|
| 小花 | **舌状花のみ**で構成 |
| 乳液 | 白い乳液を含む |
| 代表的な属 | タンポポ属、ニガナ属、ノゲシ属 |

### 日本産主要種（タンポポ属 Taraxacum）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [カントウタンポポ](https://ja.wikipedia.org/wiki/カントウタンポポ) | *Taraxacum platycarpum* | 関東 | 在来種、総苞外片が反り返らない |
| [カンサイタンポポ](https://ja.wikipedia.org/wiki/カンサイタンポポ) | *Taraxacum japonicum* | 西日本 | 在来種、総苞外片が反り返らない |
| [トウカイタンポポ](https://ja.wikipedia.org/wiki/トウカイタンポポ) | *Taraxacum longeappendiculatum* | 東海地方 | 在来種 |
| [エゾタンポポ](https://ja.wikipedia.org/wiki/エゾタンポポ) | *Taraxacum venustum* | 北海道 | 在来種 |
| [シロバナタンポポ](https://ja.wikipedia.org/wiki/シロバナタンポポ) | *Taraxacum albidum* | 西日本 | 在来種、白花 |
| [セイヨウタンポポ](https://ja.wikipedia.org/wiki/セイヨウタンポポ) | *Taraxacum officinale* | 帰化 | **総苞外片が反り返る**、無性生殖 |
| [アカミタンポポ](https://ja.wikipedia.org/wiki/アカミタンポポ) | *Taraxacum laevigatum* | 帰化 | 痩果が赤褐色 |

### 在来タンポポと外来タンポポの見分け方

| 特徴 | 在来種 | セイヨウタンポポ |
|------|--------|------------------|
| 総苞外片 | **反り返らない** | **反り返る** |
| 繁殖様式 | 有性生殖 | 無性生殖（単為生殖） |
| 開花時期 | 主に春 | 通年 |
| 生育環境 | 里山、草地 | 都市部、撹乱地 |

> 📝 **雑種の問題**：近年、在来種と外来種の雑種が増加している

## 確認問題

### 問1
キク科の花序の特徴を述べよ。

<details>
<summary>解答を見る</summary>

**頭状花序**（多数の小花が花床に密集し、1つの花のように見える）

</details>

### 問2
キク科の小花の2型を述べよ。

<details>
<summary>解答を見る</summary>

- **管状花**：筒状、5裂
- **舌状花**：平らに開く、5歯

</details>

### 問3
タンポポ亜科の特徴を2つ述べよ。

<details>
<summary>解答を見る</summary>

- **舌状花のみで構成される**
- **白い乳液を含む**

</details>

### 問4
在来タンポポとセイヨウタンポポの見分け方を述べよ。

<details>
<summary>解答を見る</summary>

- **在来種**：総苞外片が反り返らない
- **セイヨウタンポポ**：総苞外片が反り返る

</details>

### 問5
春の七草の「ホトケノザ」は何科の何という植物か。

<details>
<summary>解答を見る</summary>

**キク科のコオニタビラコ**（*Lapsanastrum apogonoides*）

シソ科のホトケノザ（*Lamium amplexicaule*）とは別種

</details>

## 暗記用まとめ

### キク科の基本構造

| 構造 | 内容 |
|------|------|
| 花序 | 頭状花序 |
| 小花 | 管状花、舌状花 |
| 果実 | 痩果 |
| 冠毛 | 萼由来、種子散布に関与 |

### キク科の亜科

| 亜科 | 小花の構成 |
|------|------------|
| タンポポ亜科 | 舌状花のみ |
| アザミ亜科 | 管状花のみ |
| キク亜科 | 舌状花＋管状花 |

### タンポポの見分け方

```
総苞外片が反り返らない → 在来種
総苞外片が反り返る → セイヨウタンポポ
```

### 春の七草（キク科）

| 七草名 | 和名 | 学名 |
|--------|------|------|
| ホトケノザ | コオニタビラコ | *Lapsanastrum apogonoides* |
| ゴギョウ | ハハコグサ | *Pseudognaphalium affine* |
