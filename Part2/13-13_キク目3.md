## ==1b. キク科の特殊種・試験頻出種（⚠️要注意）==

### ==センボンヤリ（千本槍）*Leibnitzia anandria* （キク科）==

> ⚠️ **2021年度問41で直接出題。「春の頭花は閉鎖花である」→ 誤り**

| 特徴 | 内容 |
|------|------|
| 科・目 | キク科、キク目 |
| 学名 | *Leibnitzia anandria* |
| 生活形 | 多年草 |
| 分布 | 北海道〜九州、日当たりの良い草地 |
| 最大の特徴 | **春と秋で全く異なる頭花をつける「二型花序性」** |

#### 春の頭花（開放花）
- **管状花＋舌状花**の両方からなる通常の頭花
- 花は**開放花**（開いて送粉者を誘引する）
- **閉鎖花ではない**（⚠️試験頻出誤答：「春の頭花は閉鎖花」→ 誤り）

#### 秋の頭花（閉鎖花）
- **管状花のみ**で構成
- 花は**閉鎖花**（開かずに自家受粉する）
- 長い果柄（ヤリのように直立する→名前の由来）

#### センボンヤリの試験ポイントまとめ

| 選択肢 | 正誤 |
|--------|------|
| 春の頭花は筒状花と舌状花からなる | ✅ 正しい |
| 秋の頭花は筒状花だけからなる | ✅ 正しい |
| 筒状花も舌状花も2唇形である | ✅ 正しい（キク科の管状花は5裂、下側2枚が目立つ2唇形） |
| **春の頭花は閉鎖花である** | ❌ **誤り**（閉鎖花は秋の頭花） |

### ==キッコウハグマ（亀甲白熊）*Ainsliaea apiculata* （キク科）==

> ⚠️ **2021年度問30で直接出題。「頭花が1個の小花からなる種」として出題**

| 特徴 | 内容 |
|------|------|
| 科・目 | キク科（モミジガサ連）、キク目 |
| 学名 | *Ainsliaea apiculata* |
| 生活形 | 多年草 |
| 分布 | 本州〜九州、林床 |
| 最大の特徴 | **頭花が1個の小花（管状花）のみからなる**（きわめて稀な構造） |

#### キッコウハグマの識別ポイント

- 頭花は管状花**1個のみ**（キク科の最小単位）
- 1つの頭花に多数の小花を持つキク科の中で例外的
- 葉は亀甲模様状に浅く切れ込む（亀甲→名前の由来）
- 秋に白〜淡紅色の花を咲かせる

#### キク科の頭花の小花数比較（試験頻出）

| 植物名 | 頭花の小花数 | タイプ |
|--------|------------|--------|
| **キッコウハグマ** | **1個** | 管状花のみ1個（特異） |
| モミジガサ | 5〜7個 | 管状花のみ |
| クサヤツデ | 2〜8個 | 管状花のみ |
| サワヒヨドリ | 5〜7個 | 管状花のみ |
| ノアザミ | 多数（全て管状花） | 管状花のみ |
| タンポポ | 多数（全て舌状花） | 舌状花のみ |
| ヒマワリ | 多数（外側＋中央） | 舌状花＋管状花 |
