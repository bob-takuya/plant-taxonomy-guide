### ミズゴケの生態と利用

| 項目 | 内容 |
|------|------|
| 保水性 | 自重の20倍以上の水を吸収 |
| 酸性化 | 周囲を酸性化する |
| 泥炭 | 堆積して泥炭（ピート）を形成 |
| 園芸利用 | 鉢植えの用土、蘭栽培 |
| 医療 | かつて包帯材として使用 |

> 📝 **高層湿原**：ミズゴケが堆積して形成される。尾瀬、釧路湿原など

### [ミズゴケ](https://ja.wikipedia.org/wiki/ミズゴケ属) *Sphagnum* spp.（詳細解説）

#### 形態的特徴

| 部位/特徴 | 内容 |
|-----------|------|
| 生活形 | 蘚類（茎葉体）、直立し頭部が頭状に密集 |
| 大きさ | 茎の高さ数cm〜20cm以上、群落として湿地を覆う |
| 葉の細胞構造 | **ヒアロサイト**（大型の透明死細胞）と**クロロサイト**（葉緑細胞）が交互に配列 |
| ヒアロサイト | 多数の穴（気孔）があり、大量の水を吸収・貯蔵する死細胞 |
| 保水力 | 乾燥重量の**20倍以上**の水を吸収（最大30倍とも） |
| 蒴 | **球形**、熟すと圧力差で爆発的に胞子を放出 |
| 生育環境 | 高層湿原・低層湿原・湿った林床・池沼のほとり |
| 分布 | 北海道〜九州（日本に約40種、世界に約300種） |

#### 高層湿原の形成メカニズム（ミズゴケが作る地形）

1. ミズゴケが繁茂し、周囲を**酸性化**（H⁺イオンを放出）
2. 酸性 + 過湿（低酸素）環境により**微生物による分解が著しく抑制**
3. 枯死体が分解されずに積み重なり**泥炭（ピート）**を形成
4. 泥炭が堆積して地面が盛り上がり→**高層湿原（raised bog）**が形成
5. 高層湿原は周囲の地下水と切り離され、**雨水のみで維持される貧栄養環境**になる

> 📝 **代表的な高層湿原**：尾瀬ヶ原（群馬・福島）、釧路湿原（北海道）、霧ヶ峰八島湿原（長野）

#### なぜ重要か（試験重要度 ★★★★★）

- **ヒアロサイト（透明細胞・貯水細胞）による吸水機構**が最重要試験ポイント
- **泥炭形成の主役**：地球の炭素固定（炭素貯蔵庫）として気候変動対策に注目される
- 泥炭（ピート）は燃料・園芸用土・ウイスキー製造（スモーキーフレーバー）にも利用
- かつて戦時中に**包帯材**として利用（高い吸水性と軽い抗菌性）
- 蘚類の中で唯一、独自の目（ミズゴケ目）を形成する系統的に孤立したグループ

#### 試験頻出問題

> **問：ミズゴケの葉に特徴的な2種類の細胞を述べ、それぞれの機能を説明せよ。**
> → **答え：** ①**ヒアロサイト（透明細胞・貯水細胞）**：大型の死細胞で、細胞壁に多数の穴があり、大量の水（乾燥重量の20倍以上）を吸い込み貯蔵する。②**クロロサイト（葉緑細胞）**：光合成を行う小型の生細胞。

> **問：高層湿原がミズゴケによって形成されるメカニズムを説明せよ。**
> → **答え：** ①ミズゴケが周囲を酸性化し、②過湿・低酸素環境で微生物による枯死体の分解が抑制される。③分解されない枯死体が泥炭として堆積し、④地面が周囲より盛り上がった高層湿原が形成される。

> **問：ミズゴケの蒴（胞子嚢）の形と、胞子の散布方法を答えよ。**
> → **答え：** 蒴は**球形**。成熟すると蒴内部の圧力が高まり、蓋が外れて**爆発的に胞子を放出**する。

#### 混同しやすい特徴の整理

| 比較項目 | ミズゴケ | 他の蘚類（例：スギゴケ） |
|---------|---------|------------------------|
| 葉の細胞 | ヒアロサイト（透明）＋クロロサイト | 均一な葉緑細胞 |
| 保水力 | 乾燥重量の20倍以上 | 普通の保水力 |
| 生育環境 | 湿地・高層湿原 | 林床・土上 |
| 蒴の形 | 球形（爆発的放出） | 角柱状・楕円形など |
| 系統 | ミズゴケ目（孤立） | 各種の目 |

### [スギゴケ](https://ja.wikipedia.org/wiki/スギゴケ) *Polytrichum juniperinum*（詳細解説）

#### 形態的特徴

| 部位/特徴 | 内容 |
|-----------|------|
| 生活形 | 蘚類（茎葉体）、直立性 |
| 大きさ | 高さ3〜10cm（オオスギゴケ *P. commune* は20cm以上） |
| 葉 | 線形〜披針形、基部に卵楕円形の**鞘部（しょうぶ）**があり茎を抱く |
| **薄板（lamellae）** | 葉の腹面（上側）の中肋上に多数の**縦の薄板**が並ぶ（1細胞層の板状突起） |
| 中肋 | 発達し、葉先から**赤褐色の芒（のぎ）**状に突出 |
| 蒴 | 4〜6角形の**角柱状**、**冠状の帽（かさ）**に覆われる |
| 雌雄性 | **雌雄異株**。雄株の先端に赤い**造精器盤**が形成される |
| 生育環境 | 日本庭園、林縁、明るい林床（土上・崖上） |
| 分布 | 北海道〜九州（乾燥にも比較的強い） |

#### 薄板（lamellae）の詳細

薄板はスギゴケ科（Polytrichaceae）に特有の構造で、葉の腹面（上側）の中肋上に縦走する1細胞層の板状の突起。

- **機能1**：光合成面積の拡大（薄板の側面の細胞が光合成を行う）
- **機能2**：毛管現象による水の保持（薄板間の空間に水を保つ）
- **識別点**：薄板の高さ（何細胞分か）と頂端細胞の形（フラスコ形・平ら・波打つ）が種の識別に重要
- スギゴケ *P. juniperinum*：薄板は高さ6〜8細胞、頂端細胞は横断面でフラスコ形

#### なぜ重要か（試験重要度 ★★★★）

- **スギゴケ科の最重要形質「薄板」**：他の蘚類には見られない独自構造（試験頻出）
- **雌雄異株**：雄株先端の赤い造精器盤は「コケの花」と呼ばれるほど目立つ
- 日本庭園（苔庭）の代表的なコケ：金閣寺・西芳寺（苔寺）など
- 蘚類の中で最も大型のグループに属し、道管に似た水分輸送細胞を持つ（高等植物に近い）

#### 試験頻出問題

> **問：スギゴケ科（Polytrichaceae）の葉に特有の構造を述べ、その機能を答えよ。**
> → **答え：** **薄板（lamellae）**。葉の腹面（上側）の中肋上に縦走する1細胞層の板状突起。光合成面積の拡大と毛管現象による水分保持に機能する。

> **問：スギゴケ（*Polytrichum juniperinum*）の雌雄性と、雄株の先端に見られる構造を答えよ。**
> → **答え：** **雌雄異株**。雄株の茎頂端には造精器が集まった**造精器盤**が形成され、赤い花のように見える。

> **問：スギゴケ科の蒴（胞子嚢）の形態的特徴を答えよ。**
> → **答え：** **角柱状（4〜6角形）**で、上部を覆う**冠状の帽（かさ）**がある。この帽は雌株の造卵器から発達する。

#### スギゴケ属の主要種の比較

| 種名 | 学名 | 大きさ | 特徴 |
|------|------|--------|------|
| スギゴケ | *P. juniperinum* | 3〜10cm | 薄板頂端フラスコ形、葉縁内折 |
| オオスギゴケ | *P. commune* | 10〜20cm以上 | 最大種、湿地に生育 |
| セイタカスギゴケ | *P. formosum* | 5〜15cm | 林床に普通 |
| コスギゴケ | *Pogonatum inflexum* | 2〜5cm | 小型、崩壊地・路傍 |

## 確認問題

### 問1（4択）
ミズゴケ（*Sphagnum*）の葉に特有の透明細胞の名称はどれか。

**a) クロロサイト　b) ヒアロサイト　c) トレボウキシア　d) アレオール**

<details>
<summary>解答を見る</summary>

**b) ヒアロサイト（透明細胞・貯水細胞）**

大型の死細胞で、細胞壁に多数の穴があり乾燥重量の20倍以上の水を吸収・貯蔵する。

</details>

### 問2（4択）
スギゴケ科（Polytrichaceae）の葉に特有の構造はどれか。

**a) 翼細胞　b) 薄板（lamellae）　c) 油体　d) ピレノイド**

<details>
<summary>解答を見る</summary>

**b) 薄板（lamellae）**

葉の腹面の中肋上に縦走する1細胞層の板状突起。光合成面積拡大と保水に機能する。

</details>

### 問3
ミズゴケが高層湿原を形成するメカニズムを3ステップで述べよ。

<details>
<summary>解答を見る</summary>

1. ミズゴケが周囲を**酸性化**する（H⁺を放出）
2. 酸性・過湿・低酸素条件で枯死体の**分解が抑制**される
3. 分解されない枯死体が**泥炭として堆積**し、地面が盛り上がって高層湿原が形成される

</details>

### 問4（4択）
スギゴケ（*Polytrichum juniperinum*）の雌雄性として正しいものはどれか。

**a) 雌雄同株　b) 雌雄異株　c) 雄株のみ（雌株なし）　d) 性別はない**

<details>
<summary>解答を見る</summary>

**b) 雌雄異株**

雄株の茎頂端に赤い造精器盤が形成される（「コケの花」）。

</details>

### 問5
ホウオウゴケ科の葉の配列の特徴を述べよ。

<details>
<summary>解答を見る</summary>

**2列**につく（他の蘚類は螺旋状に多列で配列することが多い）。「鳳凰の羽」のように扁平に見える。

</details>

### 問6（4択）
苔玉に最も利用される蘚類はどれか。

**a) スギゴケ　b) ミズゴケ　c) ハイゴケ　d) シラガゴケ**

<details>
<summary>解答を見る</summary>

**c) ハイゴケ**（*Hypnum plumaeforme*）

匍匐性で柔軟なため、根を包む苔玉に最適。光沢のある絹糸状の外観も美しい。

</details>

## 暗記用まとめ

### 蘚類の主要科の特徴（識別ポイント付き）

| 科名 | 最重要識別特徴 | 代表種 | 生育環境 |
|------|--------------|--------|---------|
| ミズゴケ科 | **ヒアロサイト**（透明細胞）、球形蒴 | オオミズゴケ | 湿原 |
| スギゴケ科 | **薄板**（中肋の板状突起）、角柱状蒴、**雌雄異株** | スギゴケ | 土上・林縁 |
| シラガゴケ科 | **白色**（中肋が葉を占める）、クッション状 | シラガゴケ | 林床 |
| ハイゴケ科 | **匍匐性**、絹糸状光沢、鎌形の葉 | ハイゴケ | 林床・石垣 |
| ホウオウゴケ科 | 葉が**2列**（扁平）、葉の3部構造 | ホウオウゴケ | 渓流沿い |
| ギボウシゴケ科 | **岩上性**、乾燥耐性、葉先に毛 | スナゴケ | 岩上・砂地 |

### ミズゴケとスギゴケの比較（最頻出）

| 比較項目 | ミズゴケ（*Sphagnum*） | スギゴケ（*Polytrichum*） |
|---------|----------------------|------------------------|
| 大きさ | 中型（数〜20cm） | 中〜大型（3〜20cm超） |
| 葉の特徴 | **ヒアロサイト＋クロロサイト** | **薄板**が中肋上に縦走 |
| 蒴の形 | **球形**（爆発的放出） | **角柱状**（帽あり） |
| 雌雄性 | 雌雄同株 | **雌雄異株** |
| 生育環境 | 湿地・高層湿原 | 土上・林縁 |
| 主な利用 | 園芸用土・泥炭 | 苔庭（日本庭園） |

### 園芸利用される蘚類

| 用途 | 適した種 | 理由 |
|------|----------|------|
| 苔庭 | スギゴケ | 直立性、見栄えが良い |
| 苔玉 | ハイゴケ | 匍匐性で根を包みやすい |
| 盆栽 | シラガゴケ | 白みがかった外観、クッション状 |
| 蘭栽培 | ミズゴケ | 優れた保水性と通気性 |
| 屋上緑化 | スナゴケ | 乾燥耐性が非常に高い |

### 特殊環境に生育する蘚類

| 環境 | 種名 | 理由 |
|------|------|------|
| 高層湿原 | ミズゴケ類 | ヒアロサイトで保水、酸性化で泥炭形成 |
| 火事跡 | ヒョウタンゴケ | 焼け跡の裸地に真っ先に定着 |
| コンクリート・都市 | ギンゴケ | 乾燥・汚染耐性が高い |
| 砂地・乾燥地 | スナゴケ | 極度の乾燥耐性 |
| 渓流・滝のそば | ホウオウゴケ | 常湿環境が必要 |
