## ショウブ目 Acorales

[ショウブ目](https://ja.wikipedia.org/wiki/ショウブ目)は、単子葉類の最基部に位置する目で、[ショウブ科](https://ja.wikipedia.org/wiki/ショウブ科)1科のみで構成されます。

---

### [ショウブ科](https://ja.wikipedia.org/wiki/ショウブ科) Acoraceae

| 項目 | 内容 |
|------|------|
| 日本産属数 | 1属 |
| 日本産種数 | 2種 |
| 世界の属数/種数 | 1属/2〜4種 |
| 分布 | 北半球の温帯〜熱帯 |

#### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活型 | 多年草、根茎あり |
| 葉 | 線形、2列互生、芳香 |
| 花序 | 肉穂花序 |
| 花 | 両性花、花被片6 |
| 果実 | 液果 |

#### 日本産の種

| 種名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ショウブ](https://ja.wikipedia.org/wiki/ショウブ) | *Acorus calamus* | 北海道〜九州 | 端午の節句、薬用 |
| [セキショウ](https://ja.wikipedia.org/wiki/セキショウ) | *A. gramineus* | 本州〜九州 | 小型、渓流沿い |

#### ショウブの詳細

- **別名**：ニオイショウブ
- **注意**：[ハナショウブ](https://ja.wikipedia.org/wiki/ハナショウブ)（アヤメ科）とは別種
- **用途**：
  - [端午の節句](https://ja.wikipedia.org/wiki/端午の節句)の菖蒲湯
  - 根茎は生薬「白菖」
- **特徴**：
  - 葉に強い芳香
  - 肉穂花序は葉状の苞から斜上

#### 系統的位置

かつてはサトイモ科に含められていましたが、分子系統解析により単子葉類の最基部であることが判明し、独立の科・目となりました。

## ショウブとセキショウの詳細比較

### [ショウブ](https://ja.wikipedia.org/wiki/ショウブ) *Acorus calamus*

#### 形態的特徴

| 部位/特徴 | 内容 |
|-----------|------|
| 生活型 | 多年草（根茎で繁殖） |
| 葉 | 線形、長さ50〜100cm、幅1〜2cm、中肋明瞭 |
| 葉の芳香 | 根茎・葉ともに**強い芳香**（β-アサロン等の精油成分） |
| 花序 | 肉穂花序、葉状苞から斜上 |
| 花 | 両性花、花被片6 |
| 分布 | 北海道〜九州、湿地・池畔 |
| 生薬 | 根茎は「白菖（びゃくしょう）」として薬用 |

#### なぜ重要か（試験重要度 ★★★★）

- 単子葉類最基部（ショウブ目）の唯一の科
- **端午の節句の菖蒲湯**（5月5日）に使う植物として文化的に重要
- 「ショウブ」と「ハナショウブ（アヤメ科）」の混同に注意！
- 根茎にβ-アサロンなどの精油成分を含む

#### 試験頻出問題

> **問：** 端午の節句に菖蒲湯として使われるショウブは何科か？
> → **答え：** **ショウブ科（Acoraceae）**。アヤメ科のハナショウブとは全く別の植物（科も形態も異なる）。

> **問：** ショウブ目が単子葉類の最基部とされる根拠は？
> → **答え：** **分子系統解析**。形態的には単子葉類の典型を示し、最も早く他の単子葉類から分岐した系統と判明した。

### [セキショウ](https://ja.wikipedia.org/wiki/セキショウ) *Acorus gramineus*

#### 形態的特徴

| 部位/特徴 | 内容 |
|-----------|------|
| 生活型 | 常緑多年草（根茎で繁殖） |
| 葉 | 線形、長さ20〜50cm（ショウブより小型）、幅2〜5mm（非常に細い） |
| 生育地 | 渓流沿いの湿った岩上・石垣 |
| 分布 | 本州〜九州 |

#### ショウブとセキショウの比較

| 特徴 | ショウブ | セキショウ |
|------|---------|-----------|
| 大きさ | 大型（葉長50〜100cm） | 小型（葉長20〜50cm） |
| 葉の幅 | 幅広い（1〜2cm） | 非常に細い（2〜5mm） |
| 生育地 | 湿地・池 | 渓流・岩上 |

#### 試験頻出問題

> **問：** ショウブとセキショウの形態的な違いを述べよ。
> → **答え：** ショウブは葉が幅広く大型（葉長50〜100cm）で湿地に生育する。セキショウは葉が非常に細く小型で渓流沿いの岩上に生育する。

## オモダカ目 Alismatales

[オモダカ目](https://ja.wikipedia.org/wiki/オモダカ目)は、水生・湿地性の植物を多く含む目です。日本には10科が分布します。

---

### [オモダカ科](https://ja.wikipedia.org/wiki/オモダカ科) Alismataceae

| 項目 | 内容 |
|------|------|
| 日本産属数 | 3属 |
| 日本産種数 | 約10種 |
| 世界の属数/種数 | 約15属/約100種 |
| 分布 | 世界の温帯〜熱帯 |

#### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活型 | 水生または湿地性の多年草 |
| 葉 | 根生、葉身は矢じり形〜楕円形 |
| 花 | 放射相称、花弁3枚（白色） |
| 雄しべ | 6〜多数 |
| 果実 | 痩果が多数集まる |

#### 日本産の主要種

##### [オモダカ属](https://ja.wikipedia.org/wiki/オモダカ属) *Sagittaria*

| 種名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [オモダカ](https://ja.wikipedia.org/wiki/オモダカ) | *S. trifolia* | 北海道〜沖縄 | 水田雑草、矢じり形の葉 |
| [アギナシ](https://ja.wikipedia.org/wiki/アギナシ) | *S. aginashi* | 本州〜九州 | 殖芽で越冬 |
| [ウリカワ](https://ja.wikipedia.org/wiki/ウリカワ) | *S. pygmaea* | 本州〜沖縄 | 小型、線形の葉 |

**オモダカの特徴**
- 葉は矢じり形（人の顔に見える→「面高」）
- 水田の代表的な雑草
- [クワイ](https://ja.wikipedia.org/wiki/クワイ)はオモダカの栽培品種

### [オモダカ](https://ja.wikipedia.org/wiki/オモダカ) *Sagittaria trifolia* 詳細

#### 形態的特徴

| 部位/特徴 | 内容 |
|-----------|------|
| 生活型 | 多年草（塊茎で越冬） |
| 葉 | **矢尻形（矢じり形）**の基部が左右に突き出る |
| 葉身の変異 | 沈水葉は線形、浮葉・気中葉は矢尻形 |
| 花 | 白色3弁花、雌雄同株、上部に雄花・下部に雌花 |
| 果実 | 痩果が多数集まる集合果 |
| 分布 | 北海道〜沖縄の水田・湿地 |

#### なぜ重要か（試験重要度 ★★★）

- **クワイ**はオモダカの栽培品種で食用（正月料理）
- 水田の代表的な難防除雑草
- 葉の形が「矢尻形」という独特の形態
- 「面高（おもだか）」は矢尻形の葉が人の顔に見えるという説

#### 試験頻出問題

> **問：** クワイとオモダカの関係は？
> → **答え：** クワイはオモダカ（*Sagittaria trifolia*）の**栽培品種**。大きな塊茎を食用にする。正月料理の縁起物。

##### [サジオモダカ属](https://ja.wikipedia.org/wiki/サジオモダカ) *Alisma*

| 種名 | 学名 | 分布 | 特徴 |
|------|------|------|------|

### [サトイモ科](https://ja.wikipedia.org/wiki/サトイモ科) Araceae

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約15属 |
| 日本産種数 | 約50種 |
| 世界の属数/種数 | 約140属/約3,800種 |
| 分布 | 世界の熱帯〜温帯 |

#### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活型 | 多年草、多くは塊茎または根茎 |
| 葉 | 単葉〜複葉、網状脈 |
| 花序 | **肉穂花序**と**仏炎苞** |
| 花 | 小型、肉穂花序に密生 |
| 果実 | 液果 |

#### 肉穂花序と仏炎苞

サトイモ科の最大の特徴は、**[肉穂花序](https://ja.wikipedia.org/wiki/肉穂花序)**（小さな花が太い軸に密生）と**[仏炎苞](https://ja.wikipedia.org/wiki/仏炎苞)**（花序を包む大型の苞）です。

## 4択確認問題

**Q1.** 端午の節句の「菖蒲湯」に使われるショウブが属する科はどれか。
① アヤメ科　② サトイモ科　③ ショウブ科　④ カキツバタ科

<details><summary>答え</summary>
**③ ショウブ科** — ショウブ（*Acorus calamus*）はショウブ科。アヤメ科のハナショウブとは全く別の植物。単子葉類の最基部に位置する。
</details>

---

**Q2.** オモダカ（*Sagittaria trifolia*）の栽培品種で食用になるものはどれか。
① ゴボウ　② クワイ　③ 菊芋　④ コンニャク

<details><summary>答え</summary>
**② クワイ** — クワイはオモダカの栽培品種で塊茎を食用にする。正月料理の縁起物。「芽が出る」ことから縁起がよいとされる。
</details>

---

**Q3.** 世界最小の被子植物とされるウキクサ科（APG体系ではサトイモ科）の植物はどれか。
① ウキクサ　② アオウキクサ　③ ミジンコウキクサ　④ ジュンサイ

<details><summary>答え</summary>
**③ ミジンコウキクサ**（*Wolffia globosa*） — 直径約1mm、根を持たず水面に浮遊する世界最小の被子植物。
</details>

---

**Q4.** テンナンショウ属（サトイモ科）の特徴として正しいものはどれか。
① 雌雄同株で性転換しない　② 栄養状態によって性転換する　③ 平行脈を持つ　④ 毒性はない

<details><summary>答え</summary>
**② 栄養状態によって性転換する** — テンナンショウ属は栄養状態が良い個体（大きい個体）は雌性、小さい個体は雄性になる性転換を行う。全株有毒（シュウ酸カルシウム含有）。
</details>

## 確認問題

### 問1
ショウブとハナショウブの違いを述べよ。

<details>
<summary>解答を見る</summary>

**ショウブ**はショウブ科（旧サトイモ科）で、肉穂花序を持ち、葉に芳香がある。端午の節句の菖蒲湯に使う。

**ハナショウブ**はアヤメ科で、大型で美しい花を咲かせる観賞植物。

両者は全く別の科に属し、外見も大きく異なる。

</details>

### 問2
サトイモ科の花序の特徴を述べよ。

<details>
<summary>解答を見る</summary>

**肉穂花序**と**仏炎苞**

肉穂花序は太い軸に小さな花が密生し、仏炎苞はそれを包む大型の苞である。

</details>

### 問3
世界最小の被子植物は何か。

<details>
<summary>解答を見る</summary>

**ミジンコウキクサ（*Wolffia globosa*）**

直径約1mmで、根を持たない。ウキクサ科（サトイモ科）。

</details>

### 問4
ザゼンソウが持つ珍しい生理的特徴は何か。

<details>
<summary>解答を見る</summary>

**発熱（サーモジェネシス）**

開花時に肉穂花序が発熱し、周囲の雪を溶かす。温度は25℃以上に達することもある。

</details>
