# 14-4. その他の裸子植物（イチイ科・マキ科・ソテツ科・イチョウ科）

## はじめに

本章では、マツ科・ヒノキ科・スギ科以外の日本産裸子植物（イチイ科、マキ科、ソテツ科、イチョウ科）を解説します。

---

## 1. イチイ科 Taxaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 2属 |
| 日本産種数 | 約5種 |
| 世界の種数 | 約30種 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 常緑高木または低木 |
| 葉 | 線形、扁平、螺旋状配列 |
| 球果 | 形成しない |
| 種子 | 肉質の[仮種皮](https://ja.wikipedia.org/wiki/仮種皮)に包まれる |
| 雌雄性 | 雌雄異株が多い |

### 日本産主要種（イチイ属 Taxus）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [イチイ](https://ja.wikipedia.org/wiki/イチイ) | *Taxus cuspidata* | 北海道～九州 | 赤い仮種皮、アララギとも |
| [キャラボク](https://ja.wikipedia.org/wiki/キャラボク) | *Taxus cuspidata* var. *nana* | 本州（日本海側高山） | イチイの矮性変種、這性 |
| [セイヨウイチイ](https://ja.wikipedia.org/wiki/セイヨウイチイ) | *Taxus baccata* | 栽培 | 抗がん剤（タキソール）原料 |

### イチイの特徴

| 項目 | 内容 |
|------|------|
| 分布 | 北海道～九州の山地 |
| 名前の由来 | 貴族の笏（しゃく）の材料→「一位」 |
| 仮種皮 | 赤色、甘い、食用可 |
| 種子 | **有毒**（タキシン） |
| 用途 | 彫刻材、庭園木 |

> ⚠️ **注意**：仮種皮は食用可だが、種子は有毒（タキシン）

### 日本産主要種（カヤ属 Torreya）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [カヤ](https://ja.wikipedia.org/wiki/カヤ) | *Torreya nucifera* | 本州～九州 | 碁盤の材、種子は食用 |
| [チャボガヤ](https://ja.wikipedia.org/wiki/チャボガヤ) | *Torreya nucifera* var. *radicans* | 本州（日本海側） | カヤの矮性変種 |

### カヤの特徴

| 項目 | 内容 |
|------|------|
| 分布 | 本州～九州の山地 |
| 種子 | 食用可（榧の実）、油も採取 |
| 材 | 碁盤・将棋盤の最高級材 |
| 葉 | 先端が鋭く刺さる |

## 2. マキ科 Podocarpaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 2属 |
| 日本産種数 | 約5種 |
| 世界の種数 | 約170種 |
| 分布 | 主に南半球 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 常緑高木 |
| 葉 | 線形または広線形 |
| 種子 | 肉質の[種托](https://ja.wikipedia.org/wiki/種托)の上につく |

### 日本産主要種（マキ属 Podocarpus）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [イヌマキ](https://ja.wikipedia.org/wiki/イヌマキ) | *Podocarpus macrophyllus* | 本州～沖縄 | 暖地性、生垣、種托は食用可 |
| [ラカンマキ](https://ja.wikipedia.org/wiki/ラカンマキ) | *Podocarpus macrophyllus* var. *maki* | 栽培 | イヌマキの変種、葉が小さい |

### 日本産主要種（ナギ属 Nageia）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ナギ](https://ja.wikipedia.org/wiki/ナギ) | *Nageia nagi* | 本州～沖縄 | 広い葉、神木として植栽 |

### ナギの特徴

| 項目 | 内容 |
|------|------|
| 葉 | 針葉樹としては広く、平行脈 |
| 分布 | 本州（紀伊半島以西）～沖縄 |
| 信仰 | 熊野神社などで神木 |
| 用途 | 葉は縁結びのお守り |

> 📝 **ナギの葉の特徴**：裸子植物としては珍しく幅広い葉をもち、一見被子植物のよう

## 3. ソテツ科 Cycadaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 1属 |
| 日本産種数 | 1種 |
| 世界の種数 | 約100種 |
| 分布 | 熱帯・亜熱帯 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 常緑の木本状 |
| 茎 | 太い幹、分枝しない |
| 葉 | 羽状複葉、頂生 |
| 雌雄性 | **雌雄異株** |
| 精子 | **鞭毛をもつ精子**で受精 |

### 日本産種

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ソテツ](https://ja.wikipedia.org/wiki/ソテツ) | *Cycas revoluta* | 九州～沖縄 | 日本唯一の自生ソテツ |

### ソテツの特徴

| 項目 | 内容 |
|------|------|
| 自生地 | 九州南部～沖縄の海岸 |
| 葉 | 羽状複葉、革質 |
| 種子 | 赤色、大型、有毒（サイカシン） |
| 精子 | 1896年に[池野成一郎](https://ja.wikipedia.org/wiki/池野成一郎)が発見 |
| 食用 | 幹のデンプンを救荒食に利用（毒抜き必要） |

> 📝 **ソテツの精子の発見**：1896年、池野成一郎がソテツの精子に鞭毛があることを発見。イチョウでも平瀬作五郎が同年発見。裸子植物で精子が鞭毛をもつことの発見は世界的に大きな意義があった。

> ⚠️ **ソテツ地獄**：沖縄で戦後の食糧難の際、毒抜きが不十分なソテツを食べて中毒死する事故が多発した

## 4. イチョウ科 Ginkgoaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 世界の属数 | 1属 |
| 世界の種数 | 1種 |
| 分布 | 中国原産（現在は世界中で栽培） |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | **落葉高木** |
| 葉 | 扇形、二叉脈、切れ込みあり |
| 雌雄性 | **雌雄異株** |
| 精子 | **鞭毛をもつ精子**で受精 |
| 種子 | 黄色の外種皮（悪臭）、白い内種皮（銀杏） |

### 日本における存在

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [イチョウ](https://ja.wikipedia.org/wiki/イチョウ) | *Ginkgo biloba* | 栽培 | 街路樹、神社仏閣、銀杏を食用 |

### イチョウの特徴

| 項目 | 内容 |
|------|------|
| 原産地 | 中国 |
| 葉 | 扇形、秋に黄葉 |
| 種子 | ギンナン、食用（多量摂取は有毒） |
| 精子 | 1896年、平瀬作五郎が発見 |
| 化石 | 中生代に繁栄、「生きた化石」 |
| 薬用 | 葉エキスが血流改善に使用 |

> 📝 **生きた化石**：イチョウは中生代（恐竜時代）に繁栄し、現在は1属1種のみが残存。野生は絶滅したとされ、中国で人為的に保護されてきた。

### イチョウの雌株と雄株

| 特徴 | 雄株 | 雌株 |
|------|------|------|
| 花 | 尾状の雄花 | 胚珠が2個 |
| 種子 | つけない | ギンナンをつける |
| 街路樹 | 好まれる | 悪臭のため避けられる |

> 📝 **街路樹としてのイチョウ**：雌株は種子の外種皮が悪臭を放つため、街路樹には雄株が選ばれることが多い

## 確認問題

### 問1
イチイ科の種子の特徴を述べよ。

<details>
<summary>解答を見る</summary>

**肉質の仮種皮に包まれる**（球果を形成しない）

</details>

### 問2
ソテツとイチョウに共通する受精様式の特徴は何か。

<details>
<summary>解答を見る</summary>

**鞭毛をもつ精子で受精する**

</details>

### 問3
イチョウの精子を発見した日本人研究者は誰か。

<details>
<summary>解答を見る</summary>

**平瀬作五郎**（1896年）

ソテツの精子は池野成一郎が同年発見

</details>

### 問4
「生きた化石」と呼ばれる裸子植物を2種挙げよ。

<details>
<summary>解答を見る</summary>

- **イチョウ**
- **メタセコイア**

</details>
