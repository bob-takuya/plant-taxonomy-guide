## はじめに

[マツムシソウ目](https://ja.wikipedia.org/wiki/マツムシソウ目)（Dipsacales）は、キキョウ類に属する目で、スイカズラ科、レンプクソウ科、マツムシソウ科などを含みます。APG体系で旧スイカズラ科が再編成されました。

---

## マツムシソウ目の概要

| 項目 | 内容 |
|------|------|
| 日本産科数 | 3科 |
| 世界の科数 | 7科 |
| 世界の種数 | 約1,100種 |
| 主な特徴 | 対生葉、合弁花 |

## 2. レンプクソウ科 Adoxaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 2属 |
| 日本産種数 | 約10種 |
| 世界の種数 | 約200種 |

> 📝 **分類の変更**：APG体系で旧スイカズラ科からガマズミ属、ニワトコ属が移動

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 木本または草本 |
| 葉 | 対生、単葉または複葉 |
| 花 | 小型、5数性（レンプクソウは4～5数性） |
| 果実 | 核果 |

### 日本産主要種（ガマズミ属 Viburnum）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ガマズミ](https://ja.wikipedia.org/wiki/ガマズミ) | *Viburnum dilatatum* | 北海道～九州 | 白い散房花序、赤い実 |
| [コバノガマズミ](https://ja.wikipedia.org/wiki/コバノガマズミ) | *Viburnum erosum* | 本州～九州 | 葉が小さい |
| [ミヤマガマズミ](https://ja.wikipedia.org/wiki/ミヤマガマズミ) | *Viburnum wrightii* | 北海道～九州 | 葉に光沢 |
| [オトコヨウゾメ](https://ja.wikipedia.org/wiki/オトコヨウゾメ) | *Viburnum phlebotrichum* | 本州～九州 | 葉脈が目立つ |
| [ヤブデマリ](https://ja.wikipedia.org/wiki/ヤブデマリ) | *Viburnum plicatum* var. *tomentosum* | 本州～九州 | 周辺に装飾花 |
| [オオデマリ](https://ja.wikipedia.org/wiki/オオデマリ) | *Viburnum plicatum* var. *plicatum* | 栽培 | 全て装飾花の園芸品種 |
| [オオカメノキ](https://ja.wikipedia.org/wiki/オオカメノキ) | *Viburnum furcatum* | 北海道～九州 | ムシカリとも、周辺に装飾花 |
| [カンボク](https://ja.wikipedia.org/wiki/カンボク) | *Viburnum opulus* var. *sargentii* | 北海道～本州 | 3裂葉、周辺に装飾花 |
| [ケナシカンボク](https://ja.wikipedia.org/wiki/ケナシカンボク) | *Viburnum opulus* var. *calvescens* | 本州 | 毛が少ない |

| [サンゴジュ](https://ja.wikipedia.org/wiki/サンゴジュ) | *Viburnum odoratissimum* var. *awabuki* | 本州～沖縄 | 常緑、生垣、防火樹 |

**ガマズミ属の装飾花の有無（試験頻出）**

| 種名 | 装飾花の有無 |
|------|------------|
| ガマズミ | **なし**（すべて両性花） |
| コバノガマズミ | なし |
| ヤブデマリ | **あり**（周辺に装飾花） |
| オオカメノキ（ムシカリ） | **あり**（周辺に装飾花） |
| カンボク | **あり**（周辺に装飾花） |
| **ハクサンボク** | **なし**（装飾花をもたない、常緑） |

> ⚠️ **試験頻出**（2021年問29）：「ガマズミ属で装飾花をもたない種」→ **ハクサンボク**（常緑、暖地性）  
> オオカメノキ、カンボク、ヤブデマリはいずれも装飾花をもつ。
