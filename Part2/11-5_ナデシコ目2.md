**ヨウシュヤマゴボウの注意**
- 全草有毒（特に根）
- 赤い果実が子供に誤食される
- 漬物のヤマゴボウ（モリアザミ）とは別種

#### 🔴 ヨウシュヤマゴボウの毒性詳細（試験重要度 ★★★★）

| 項目 | 内容 |
|------|------|
| 学名 | *Phytolacca americana* |
| 原産地 | 北アメリカ（帰化植物） |
| 毒成分 | **フィトラッカトキシン**（ポリペプチド系）、**フィトラッカシン**等 |
| 有毒部位 | **全草**（根が最強）、果実・種子・葉も有毒 |
| 誤食の危険 | 紫黒色の果実が美しく、子どもが誤食する事故が多発 |
| 中毒症状 | 嘔吐・下痢・腹痛・頭痛、重症例では呼吸困難・痙攣・昏睡 |
| 混同注意 | 「ヤマゴボウ漬け」はキク科モリアザミの根で別植物 |
| 外来種問題 | 大型で繁殖力が強く、在来植物の生育地を侵食 |
| 名前の由来 | 「洋種」の「山牛蒡（ヤマゴボウ）」。根がゴボウに似るが有毒 |

> **危険な外見の組み合わせ**：大型植物（1〜2m）＋紫黒色の美しい果実（ブドウに類似）＋子供の目線の高さに果実がなる

#### 試験頻出問題

> **問：** ヨウシュヤマゴボウの有毒部位はどこか。また混同しやすい食用植物は何か。
> → **答え：** **全草有毒**（根が最強）。漬物「ヤマゴボウ」はキク科モリアザミ（*Cirsium dipsacoides*）の根で別種。

## ナデシコ目のAPG体系における位置まとめ

| 特徴 | 内容 |
|------|------|
| クレード | **中核真正双子葉類（Core Eudicots）** |
| 系統的位置 | バラ類・キク類のいずれにも属さない独立したグループ |
| 共通特徴 | ベタレイン色素を持つ種が多い（アントシアニンの代わり） |
| 代表的科 | ナデシコ科・タデ科・ヒユ科・モウセンゴケ科 |

## 確認問題（4択形式）

**Q1.** ヨウシュヤマゴボウについて正しいものはどれか。
① 根のみが有毒で果実は食べられる　② 全草が有毒で、漬物「ヤマゴボウ」のヤマゴボウとは別の植物　③ 日本原産の在来植物　④ ヒユ科に属する

<details><summary>答え</summary>**② 全草が有毒で、漬物「ヤマゴボウ」のヤマゴボウとは別の植物** — ヨウシュヤマゴボウ（北米原産帰化植物、ヤマゴボウ科）は全草有毒。漬物「ヤマゴボウ」はキク科モリアザミの根。</details>

**Q2.** APG体系でヒユ科に統合された旧科はどれか。
① ナデシコ科　② アカザ科　③ タデ科　④ ヤマゴボウ科

<details><summary>答え</summary>**② アカザ科** — APG体系で旧アカザ科はヒユ科（Amaranthaceae）に統合。ホウレンソウ・テンサイ・シロザ・アカザなどを含む。</details>

**Q3.** 食虫植物であるモウセンゴケの捕虫方法はどれか。
① 落とし穴式（ウツボカズラ型）　② 粘着式（粘毛で虫を捕らえる）　③ 吸い込み式（タヌキモ型）　④ 挟み込み式（ハエトリソウ型）

<details><summary>答え</summary>**② 粘着式（粘毛で虫を捕らえる）** — 葉の表面の腺毛から粘液を分泌して虫を捕らえ、葉が巻いて消化・吸収する。</details>

**Q4.** スベリヒユについて正しいものはどれか。
① 有毒で食べられない　② 多肉質の葉を持ち食用可でオメガ3脂肪酸を含む　③ 水生植物で沼地に生育　④ ナデシコ科に属する

<details><summary>答え</summary>**② 多肉質の葉を持ち食用可でオメガ3脂肪酸を含む** — スベリヒユ（スベリヒユ科）。多肉質で乾燥に強い畑雑草。山形の伝統食「ひょっこ」としても知られる。</details>

| 科名 | 代表属 | 特徴 |
|------|--------|------|
| ナデシコ科 | ナデシコ、ハコベ | 対生葉、節膨らむ |

| ヒユ科 | アカザ、ホウレンソウ | 旧アカザ科含む |
