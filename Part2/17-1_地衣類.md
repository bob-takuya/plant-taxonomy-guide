## はじめに

[地衣類](https://ja.wikipedia.org/wiki/地衣類)（Lichens）は、菌類と藻類（または藍藻）が共生して形成する複合生物です。世界に約20,000種、日本に約1,800種が知られ、岩上や樹皮など様々な環境に生育します。

---

## 地衣類の概要

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産種数 | 約1,800種 |
| 世界の種数 | 約20,000種 |
| 構成 | 菌類＋藻類（または藍藻） |
| 分類 | 菌類の一部として扱う |

### 地衣類の定義

| 項目 | 内容 |
|------|------|
| 共生体 | 菌類（主に子嚢菌）と藻類（緑藻または藍藻） |
| 菌類の役割 | 形態形成、水分・無機塩の吸収 |
| 藻類の役割 | 光合成による有機物生産 |
| 関係 | **共生**（相利共生） |

> 📝 **地衣類の分類**：地衣類は独立した分類群ではなく、菌類（主に子嚢菌門）の中で藻類と共生する種をまとめた生態的なグループ

## 地衣類の構造

### 地衣体の構造

| 部位 | 特徴 |
|------|------|
| 上皮層 | 菌糸が密に詰まった保護層 |
| 藻類層 | 藻類（共生藻）が分布 |
| 髄層 | 菌糸が疎に分布 |
| 下皮層 | 基物に接する面 |
| 仮根 | 基物に付着する構造 |

### 共生藻のタイプ

| タイプ | 共生藻 | 特徴 |
|--------|--------|------|
| 緑藻地衣 | [トレボウキシア](https://ja.wikipedia.org/wiki/トレボウキシア属)など | 最も多い |
| 藍藻地衣 | [ネンジュモ](https://ja.wikipedia.org/wiki/ネンジュモ属)など | 窒素固定能 |
| 混合地衣 | 緑藻＋藍藻 | 両方をもつ |

## 地衣類の繁殖

### 有性繁殖

| 構造 | 特徴 |
|------|------|
| 子器 | 胞子を作る器官 |
| 裸子器 | 円盤状、縁が地衣体と異なる |
| 被子器 | 縁が地衣体と同質 |
| 子嚢胞子 | 有性胞子、菌類のみを繁殖 |

### 無性繁殖

| 構造 | 特徴 |
|------|------|
| 粉芽 | 菌糸と藻類の塊、風で散布 |
| 裂芽 | 地衣体の一部が分離 |
| 断片化 | 地衣体の破片による繁殖 |

> 📝 **有性繁殖の問題**：子嚢胞子は菌類のみを繁殖させるため、発芽後に適切な藻類と出会う必要がある

## 主要種 詳細解説

### [ウメノキゴケ](https://ja.wikipedia.org/wiki/ウメノキゴケ) *Parmotrema tinctorum*

#### 形態的特徴

| 部位/特徴 | 内容 |
|-----------|------|
| 生活形 | 葉状地衣 |
| 大きさ | 直径5〜20cm（最大約25cm） |
| 地衣体 | 灰緑色〜灰白色、縁は波打つ裂片に分かれる（裂片幅5〜20mm、先端は円形） |
| 裂芽 | 中央部に密生する粒状〜円筒状の裂芽（高さ最大2.5mm）が特徴 |
| 裏面 | 縁が淡褐色〜淡赤褐色で光沢、中央部は黒色、まばらな偽根 |
| 生育環境 | 樹皮上（ウメ・サクラ・コナラなど）、清浄な空気が必要 |
| 分布 | 全国の山地〜低地（大気汚染の影響で都市部では減少・消滅） |
| 共生藻 | *Trebouxia*属の緑藻 |
| 地衣成分 | アトラノリン・クロロアトラノリン（皮層）、レカノール酸（髄層）、オルセイン（紫色染料） |

#### なぜ重要か（試験重要度 ★★★★★）

- **大気汚染の生物指標として最重要の地衣類**。年間平均SO₂濃度20ppb（0.02ppm）以上で生育できなくなる
- 汚染環境ではアトラノリンが減少し、地衣体が変色・漂白される（ピンク色のクロロシスが生じる）
- 大気汚染改善後に都市部へ再定着する事例がある（回復指標にもなる）
- 紫色染料「オルセイン」の原料（細胞核の染色にも利用される）
- 裂芽（isidiia）による無性繁殖が活発。「梅の木に生えるゴケ」が名前の由来

#### 試験頻出問題

> **問：ウメノキゴケを大気汚染の指標として使う根拠を答えよ。**
> → **答え：** 二酸化硫黄（SO₂）に非常に敏感であり、年間平均SO₂濃度が20ppb（約0.02ppm）以上の地域では生育できない。清浄な空気がなければ生存できないため、在否が大気質の指標となる。

> **問：ウメノキゴケの地衣体中央に密生する粒状の構造は何か。その役割も答えよ。**
> → **答え：** 裂芽（isidiia）。菌糸と共生藻が一体となった粒状〜円筒状の塊で、地衣体ごと散布されて無性繁殖（栄養繁殖）する。

#### 混同しやすい種との違い

| 特徴 | ウメノキゴケ | サルオガセ | カラタチゴケ |
|------|-------------|-----------|-------------|
| 生活形 | 葉状 | 糸状（樹状） | 葉状〜樹状 |
| 垂れ下がるか | しない（葉状に広がる） | する（長く垂下） | しない |
| 汚染感受性 | 高い（主要指標種） | 高い | 中程度 |
| 付着 | 葉状体全体で付着 | 樹幹に基部固着 | 葉状体で付着 |

### [チズゴケ](https://ja.wikipedia.org/wiki/チズゴケ) *Rhizocarpon geographicum*

#### 形態的特徴

| 部位/特徴 | 内容 |
|-----------|------|
| 生活形 | 痂状地衣（crustose）—岩面に密着し剥がれない |
| 大きさ | 最大約15cmの斑点を形成（モザイク状に広がる） |
| 地衣体 | 明るい黄緑色〜鮮黄緑色のアレオール（小区画、0.2〜1.8mm）が密集 |
| プロタルス | アレオール間を埋める**黒色の菌糸体**が地図のような模様を作る |
| 子器 | 小さい黒色（直径1.5mm未満）、アレオール上に分布 |
| 胞子 | 褐黒色、多細胞（石垣状 muriform）、5〜22細胞 |
| 生育環境 | 山岳地帯の清浄な岩石上（花崗岩・珪岩など）、低汚染地域のみ |
| 分布 | 北半球の山岳地帯（日本では亜高山帯以上の岩石上） |

#### なぜ重要か（試験重要度 ★★★★）

- **ライケノメトリー（地衣年代測定・lichenometry）**に最もよく使われる種
- 成長速度が年間0.3〜0.9mmと**極めて遅い**、かつ長寿命
- 氷河堆積物・モレーン・落石・建築物などの年代特定に有効（放射性炭素が難しい500年未満の年代に特に有効）
- 黄緑色のアレオールと黒いプロタルスが**地図のような模様**を形成することが名前の由来
- 痂状地衣は葉状・樹状より大気汚染に耐性があるが、チズゴケは清浄な山岳環境のみ

#### 試験頻出問題

> **問：チズゴケが地質学的年代測定（ライケノメトリー）に用いられる理由を答えよ。**
> → **答え：** 成長速度が年間0.3〜0.9mmと非常に遅く、長寿命で安定して成長するため。岩石上の最大個体の直径から露出年代を推定できる。放射性炭素年代測定が不得意な500年未満の年代特定に有効。

> **問：チズゴケの地図状の模様は、何と何が作り出すか答えよ。**
> → **答え：** 黄緑色の**アレオール**（地衣化した小区画）の間を埋める**黒色のプロタルス**（藻類を含まない非地衣化菌糸組織）が境界線となり、地図のような模様になる。

### ハナゴケ *Cladonia stellaris*（ミヤマハナゴケ）

#### 形態的特徴

| 部位/特徴 | 内容 |
|-----------|------|
| 生活形 | 樹状地衣（fruticose）—直立〜クッション状 |
| 大きさ | 高さ5〜10cm、クッション状の塊（直径10cm以上になることも） |
| 地衣体 | 白色〜灰色（ウスニン酸により黄色みを帯びる場合あり） |
| 分枝 | 分岐点で**3〜4本に分かれるカリフラワー状**の緻密な分枝 |
| 子柄 | 内部が中空の樹状構造（*Cladonia*属に共通） |
| 生育環境 | 酸性土壌、寒冷地、ツンドラ草原、北方林（タイガ）の林床 |
| 分布 | 北半球の周極地域；日本では本州中部以北〜北海道の高山帯 |
| 英名 | Star reindeer lichen |

#### なぜ重要か（試験重要度 ★★★★）

- **トナカイ・カリブーの最重要食料**（特に冬季の雪下）
- トナカイはUV（紫外線）を認識できる特殊な視覚でハナゴケを雪中から発見する
- 北方林・ツンドラで広大なマット状群落を形成し、北方生態系の重要な一次生産者
- *Cladonia*属は日本でも多様（コアカミゴケ・ジョウゴゴケ・ヤリノホゴケなど）

#### 試験頻出問題

> **問：ハナゴケ（*Cladonia stellaris*）がトナカイにとって重要な食料である理由を答えよ。**
> → **答え：** 北方林やツンドラで広大な群落を形成し、冬季に雪下でも生育しているため、積雪期の貴重な食料源となる。トナカイはUV視覚でハナゴケを雪の下から発見できる。

> **問：ハナゴケ *Cladonia stellaris* の生活形と分枝の特徴を答えよ。**
> → **答え：** 樹状地衣（fruticose）。分岐点で3〜4本に分かれるカリフラワー状の緻密な分枝を形成する。内部が中空の子柄からなる。

### [サルオガセ](https://ja.wikipedia.org/wiki/サルオガセ) *Usnea* spp.

#### 形態的特徴

| 部位/特徴 | 内容 |
|-----------|------|
| 生活形 | 糸状〜樹状地衣（fruticose）—樹幹から垂れ下がる |
| 大きさ | 数cm〜3m以上（ナガサルオガセ *U. longissima* は特に長大） |
| 地衣体 | 灰緑色〜黄緑色、円筒形のひも状 |
| 内部構造 | **弾力性のある白い中央軸**（髄層）を持つ（断面で確認できる重要な識別点） |
| 生育環境 | 霧が多く清浄な高山の針葉樹林（霧や露から水分を吸収） |
| 分布 | 日本全国の山地〜高山（日本・台湾に約43種） |
| 地衣成分 | **ウスニン酸**（usnic acid）を含む |
| 別名 | 霧藻（きりも） |

#### なぜ重要か（試験重要度 ★★★）

- **ウスニン酸（usnic acid）**：強力な抗菌成分。グラム陽性菌（黄色ブドウ球菌・MRSA・レンサ球菌）に有効
- ウスニン酸は抗炎症・抗ウイルス・抗がん・抗酸化作用も報告されている（研究中）
- **霧の多い清浄な空気に完全依存**：根がなく大気中の水分を体全体で吸収するため、霧・露が不可欠
- 大気汚染に弱く、汚染の指標種。サルオガセ属の消失は大気汚染を示す
- 中央軸の断面構造が近縁種との識別に重要

#### 試験頻出問題

> **問：サルオガセ属（*Usnea*）が含む特徴的な地衣成分とその作用を答えよ。**
> → **答え：** **ウスニン酸（usnic acid）**。グラム陽性菌に対する強力な抗菌作用を持つ。黄色ブドウ球菌・レンサ球菌・MRSA（メチシリン耐性黄色ブドウ球菌）にも有効。

> **問：サルオガセが霧の多い地域にのみ生育する理由を答えよ。**
> → **答え：** 根を持たず、大気中の水分（霧・露）を地衣体全体で直接吸収するため。安定した大気中水分の供給がなければ生存できない。また大気汚染（SO₂等）にも弱いため、清浄かつ湿潤な環境が必要。

### [イワタケ](https://ja.wikipedia.org/wiki/イワタケ) *Umbilicaria esculenta*

#### 形態的特徴

| 部位/特徴 | 内容 |
|-----------|------|
| 生活形 | 葉状地衣（臍状付着型 umbilicate） |
| 大きさ | 直径数cm〜10cm（大きいものは30cmに達する） |
| 地衣体上面 | 灰色、平滑 |
| 地衣体下面 | **黒色でとげ状の毛が密生** |
| 付着 | 葉状体の**中央1点（臍状体・umbilicus）**のみで岩に固着、縁は自由 |
| 質感 | 革質。乾燥するともろく、濡れると灰緑色に変化 |
| 成長速度 | **非常に遅い（年間約1mm）** |
| 生育環境 | 日当たりの良い深山の断崖絶壁（花崗岩・珪岩上） |
| 分布 | 北海道〜九州の深山（現在は全国的に個体数が激減） |

#### なぜ重要か（試験重要度 ★★★）

- **食用地衣類の代表種**（東アジア固有の食文化）：天ぷら・酢の物・和え物
- 断崖絶壁での採取が困難・危険なため**高級食材**として高値で取引される
- 「臍（へそ）付着型（umbilicate）」葉状地衣の代表例として形態分類上重要
- 年間約1mmの極めて遅い成長速度と断崖という厳しい環境を選ぶことで競合を避ける
- 大気汚染も影響し、全国的に個体数が激減

#### 試験頻出問題

> **問：イワタケ（*Umbilicaria esculenta*）の付着様式の特徴を答えよ。**
> → **答え：** 葉状体の**中央1点（臍状体）のみ**で岩に固着する（umbilicate型）。縁は岩から離れており自由になっている。この特徴で他の葉状地衣（全面付着）と区別できる。

> **問：イワタケが高級食材である理由を2つ答えよ。**
> → **答え：** ①成長が年間約1mmと非常に遅く量が少ない。②断崖絶壁という採取困難・危険な場所にしか生育しない。

### [リトマスゴケ](https://ja.wikipedia.org/wiki/リトマスゴケ) *Roccella tinctoria*

#### 形態的特徴

| 部位/特徴 | 内容 |
|-----------|------|
| 生活形 | 樹状地衣（fruticose）—直立〜垂れ下がる |
| 大きさ | 高さ約10cm |
| 地衣体 | 灰白色〜灰褐色、円筒形〜扁平な分枝 |
| 生育環境 | 地中海沿岸・西アフリカの海岸の岩石・樹木 |
| 分布 | **日本には自生しない**（地中海沿岸・カナリア諸島・マデイラ島など） |
| 地衣成分 | **オルシノール**（フェノール性化合物）→発酵・アンモニア処理でリトマス色素が生成 |
| 別名 | オーシェルウィード（Orchella weed） |

#### なぜ重要か（試験重要度 ★★★）

- **リトマス試験紙の歴史的原料**：最も知名度の高い地衣類の利用例
- 利用の歴史：14世紀頃のフィレンツェで羊毛染色に使用。1300年頃、スペインの医師アルナルドゥス・デ・ビラ・ノバが記述
- **リトマス色素の変色**：pH4.5以下→赤色、pH8.3以上→青色（試験に出る！）
- 「リトマス（litmus）」の語源：古ノルド語の「lit（染料）+ mosi（コケ）」
- 染料「オーセイル（orchil）＝ロイヤルパープル」の製造にも使用された
- 日本に自生しないが、ウメノキゴケなど他の地衣類も同様の色素を含む

#### 試験頻出問題

> **問：リトマス試験紙の原料となる地衣類の学名と、その産地を答えよ。**
> → **答え：** *Roccella tinctoria*（リトマスゴケ）。地中海沿岸・西アフリカに分布。日本には自生しない。

> **問：リトマスゴケから得られるリトマス色素のpHによる変色を答えよ。**
> → **答え：** pH4.5以下の酸性では**赤色**に、pH8.3以上のアルカリ性では**青色**に変化する。

## 地衣類と環境

### 大気汚染の指標

| 汚染レベル（SO₂濃度の目安） | 見られる地衣類の生活形 | 残存する代表種 |
|------------|------------|----------------|
| **清浄**（SO₂ < 20ppb） | 葉状・樹状・糸状・痂状すべて | ウメノキゴケ・サルオガセ・チズゴケ |
| **軽度汚染** | 樹状・痂状（葉状は減少） | ハナゴケ・ヘリトリゴケ |
| **中度汚染** | 痂状のみ | モジゴケ・ロウソクゴケ |
| **重度汚染**（SO₂ > 100ppb以上） | 地衣類なし | （消滅） |

> 📝 **大気汚染指標の原理**：地衣類は葉状体全体でガス（SO₂・NOx等）を吸収する。排出器官がないため有害ガスを蓄積し、共生藻の光合成が阻害されて死滅する。葉状・糸状地衣が最も敏感で、痂状地衣が最も耐性を持つ。

> 📝 **SO₂規制後の回復**：日本では1970〜80年代の大気汚染規制後、都市部でウメノキゴケの再定着が観察されている（回復指標としても機能）

### 環境モニタリング

| 方法 | 内容 |
|------|------|
| 在否調査 | 指標種の有無を調査 |
| 被度調査 | 地衣類の被覆率を測定 |
| 化学分析 | 地衣体中の重金属を分析 |

地衣類の4つの生活形を述べよ。

### 問3
地衣類の無性繁殖構造を2つ挙げよ。それぞれの特徴を述べよ。

<details>
<summary>解答を見る</summary>

- **粉芽**：菌糸と藻類が塊状になったごく小さな構造。表面からパウダー状に飛散
- **裂芽**（isidiia）：菌糸と藻類からなる粒状〜円筒状の突起。地衣体表面に密生し脱落して散布

</details>

### 問4（4択）
大気汚染の指標として最も重要で、SO₂濃度20ppb以上で消滅する地衣類はどれか。

**a) チズゴケ　b) ウメノキゴケ　c) モジゴケ　d) ロウソクゴケ**

<details>
<summary>解答を見る</summary>

**b) ウメノキゴケ**（*Parmotrema tinctorum*）

葉状地衣類は痂状地衣類より大気汚染への感受性が高く、ウメノキゴケはSO₂に特に敏感。

</details>

### 問5（4択）
チズゴケ（*Rhizocarpon geographicum*）が地質学的年代測定（ライケノメトリー）に使われる最大の理由はどれか。

**a) 色が鮮やかで遠くから見える　b) 成長速度が年間0.3〜0.9mmと極めて遅い　c) 岩を溶かして深く固着する　d) 500年以上生きられない**

<details>
<summary>解答を見る</summary>

**b) 成長速度が年間0.3〜0.9mmと極めて遅い**

成長が遅く安定しているため、最大個体の直径から岩石が露出してからの年数を推定できる（ライケノメトリー）。

</details>

### 問6（4択）
サルオガセ属（*Usnea*）が含む抗菌性地衣成分はどれか。

**a) レカノール酸　b) オルシノール　c) ウスニン酸　d) アトラノリン**

<details>
<summary>解答を見る</summary>

**c) ウスニン酸（usnic acid）**

グラム陽性菌に対する強力な抗菌作用を持つ。MRSAにも有効。

</details>

### 問7（4択）
イワタケ（*Umbilicaria esculenta*）の形態的特徴として正しいものはどれか。

**a) 樹幹から垂れ下がる糸状の地衣体　b) 岩面に密着して剥がれない痂状地衣　c) 葉状体の中央1点（臍状体）で岩に固着する　d) 黄緑色の地衣体に黒い境界線がある**

<details>
<summary>解答を見る</summary>

**c) 葉状体の中央1点（臍状体）で岩に固着する**

umbilicate型（臍状付着型）が*Umbilicaria*属の特徴。縁は自由になっている。

</details>

## 暗記用まとめ

### 地衣類の生活形

| 生活形 | 特徴 | 代表例 | 汚染感受性 |
|--------|------|--------|-----------|
| 痂状 | 基物に密着、剥がれない | チズゴケ、モジゴケ | 低い（最も耐性） |
| 葉状 | 葉のように広がる | ウメノキゴケ、イワタケ | **高い** |

### 主要種 識別ポイント一覧

| 種名 | 学名 | 生活形 | 最重要特徴 | 利用・意義 |
|------|------|--------|-----------|-----------|
| ウメノキゴケ | *Parmotrema tinctorum* | 葉状 | 裂芽密生、SO₂感受性 | 大気汚染指標、オルセイン染料 |

### 大気汚染と地衣類の詳細対応表

| SO₂濃度 | 汚染度 | 残存する生活形 | 残存する代表種 |
|---------|--------|--------------|--------------|
