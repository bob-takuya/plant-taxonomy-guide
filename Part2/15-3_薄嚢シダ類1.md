# 15-3. 薄嚢シダ類1（ウラボシ目1）

[← 目次に戻る](../NEW_00_目次.md) | [← 前：小葉植物・トクサ類](./15-2_小葉植物_トクサ類.md)

## はじめに

本章から[薄嚢シダ類](https://ja.wikipedia.org/wiki/薄嚢シダ類)（Leptosporangiate ferns）を解説します。薄嚢シダ類は現生シダ類の大部分を占める群で、日本産シダの約95%がこのグループに属します。

---

## 薄嚢シダ類の概要

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産科数 | 約25科 |
| 日本産種数 | 約600種 |
| 世界の種数 | 約10,000種 |
| 特徴 | 胞子嚢壁が1層の薄い細胞からなる |

### 厚嚢シダ類との比較

| 特徴 | 薄嚢シダ類 | 厚嚢シダ類 |
|------|------------|------------|
| 胞子嚢壁 | 1層 | 多層 |
| 胞子数/嚢 | 64個（または32個） | 多数 |
| 環帯 | 発達 | なし～不完全 |
| 多様性 | 大（約10,000種） | 小（約200種） |

## 1. コバノイシカグマ科 Dennstaedtiaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約6属 |
| 日本産種数 | 約15種 |
| 世界の種数 | 約270種 |
| 特徴 | 長い根茎、ソーラスは葉縁近くに |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 根茎 | 長く匍匐、毛を密生 |
| 葉 | 多くは2～3回羽状複葉 |
| ソーラス | 葉縁近くにつく |
| 包膜 | 葉縁が折れ返った偽包膜、または杯状 |

### 日本産主要種

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [ワラビ](https://ja.wikipedia.org/wiki/ワラビ) | *Pteridium aquilinum* var. *latiusculum* | 全国 | 山菜、アク抜き必要 |
| [イシカグマ](https://ja.wikipedia.org/wiki/イシカグマ) | *Microlepia strigosa* | 本州～沖縄 | 偽包膜が杯状 |
| [コバノイシカグマ](https://ja.wikipedia.org/wiki/コバノイシカグマ) | *Dennstaedtia scabra* | 本州～沖縄 | 小型 |
| [フモトシダ](https://ja.wikipedia.org/wiki/フモトシダ) | *Microlepia marginata* | 本州～沖縄 | 林縁、路傍 |
| [オオフジシダ](https://ja.wikipedia.org/wiki/オオフジシダ) | *Monachosorum henryi* | 本州～九州 | 石灰岩地 |
| [フジシダ](https://ja.wikipedia.org/wiki/フジシダ) | *Monachosorum nipponicum* | 本州～九州 | 2回羽状 |
| [ユノミネシダ](https://ja.wikipedia.org/wiki/ユノミネシダ) | *Histiopteris incisa* | 本州～沖縄 | 白緑色、熱帯性 |

### ワラビの特徴

| 項目 | 内容 |
|------|------|
| 分布 | 全国の日当たりの良い草地 |
| 利用 | 春の山菜、若芽を食用 |
| 毒性 | [プタキロシド](https://ja.wikipedia.org/wiki/プタキロシド)を含む、発がん性 |
| 調理 | 必ず灰汁抜きが必要 |

> ⚠️ **ワラビの毒性**：プタキロシドは発がん性物質。必ず十分なアク抜き（重曹や灰を使用）が必要

## 2. イノモトソウ科 Pteridaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約10属 |
| 日本産種数 | 約40種 |
| 世界の種数 | 約1,100種 |
| 特徴 | 多様な形態、乾燥地適応種も |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 根茎 | 短く斜上または匍匐 |
| 葉 | 単葉～数回羽状複葉、多様 |
| ソーラス | 葉縁に沿う、または葉裏全体 |
| 包膜 | 葉縁が折れ返る偽包膜 |

### 日本産主要種（イノモトソウ属 Pteris）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [イノモトソウ](https://ja.wikipedia.org/wiki/イノモトソウ) | *Pteris multifida* | 本州～沖縄 | 井戸の周りに生育、中軸に翼 |
| [オオバノイノモトソウ](https://ja.wikipedia.org/wiki/オオバノイノモトソウ) | *Pteris cretica* | 本州～沖縄 | 大型、石灰岩地 |
| [アマクサシダ](https://ja.wikipedia.org/wiki/アマクサシダ) | *Pteris semipinnata* | 本州～沖縄 | 葉身基部が左右非対称 |
| [マツザカシダ](https://ja.wikipedia.org/wiki/マツザカシダ) | *Pteris nipponica* | 本州～九州 | 石灰岩地 |
| [モエジマシダ](https://ja.wikipedia.org/wiki/モエジマシダ) | *Pteris vittata* | 本州～沖縄 | ヒ素を吸収（ファイトレメディエーション） |
| [ナチシダ](https://ja.wikipedia.org/wiki/ナチシダ) | *Pteris wallichiana* | 本州～沖縄 | 大型、掌状複葉 |

### 日本産主要種（その他の属）

| 和名 | 学名 | 分布 | 特徴 |
|------|------|------|------|
| [タチシノブ](https://ja.wikipedia.org/wiki/タチシノブ) | *Onychium japonicum* | 本州～沖縄 | 細裂、日当たりの良い草地 |
| [ホウライシダ](https://ja.wikipedia.org/wiki/ホウライシダ) | *Adiantum capillus-veneris* | 本州～沖縄 | 扇形の羽片、石灰岩湿地 |
| [クジャクシダ](https://ja.wikipedia.org/wiki/クジャクシダ) | *Adiantum pedatum* | 北海道～九州 | 掌状複葉、山地林床 |
| [ハコネシダ](https://ja.wikipedia.org/wiki/ハコネシダ) | *Adiantum monochlamys* | 本州～九州 | 小型、岩上 |
| [シシラン](https://ja.wikipedia.org/wiki/シシラン) | *Vittaria flexuosa* | 本州～沖縄 | 単葉、線形、着生 |
| [エビガラシダ](https://ja.wikipedia.org/wiki/エビガラシダ) | *Cheilanthes argentea* | 北海道～九州 | 葉裏に白粉 |

### ホウライシダ属（Adiantum）の特徴

| 項目 | 内容 |
|------|------|
| 和名 | カラクサシダ、アジアンタムとも |
| 特徴 | 羽片が扇形、葉柄が黒色で光沢 |
| 園芸 | 観葉植物として多くの種が栽培 |
| 水 | 水をはじく（水滴がつかない） |

> 📝 **アジアンタム**：属名 Adiantum は「濡れない」を意味するギリシャ語に由来。葉が水をはじく

## 4. シノブ科 Davalliaceae

## 確認問題

### 問1
薄嚢シダ類の胞子嚢壁の特徴を述べよ。

<details>
<summary>解答を見る</summary>

**1層の薄い細胞からなる**（厚嚢シダ類は多層）

</details>

### 問2
ワラビを食用にする際に必要な処理は何か。また、その理由を述べよ。

<details>
<summary>解答を見る</summary>

- **必要な処理**：灰汁（アク）抜き
- **理由**：発がん性物質である**プタキロシド**を含むため

</details>

### 問3
チャセンシダ科のソーラスと包膜の特徴を述べよ。

<details>
<summary>解答を見る</summary>

- **ソーラス**：脈に沿って**線形**
- **包膜**：脈の**片側に開く**

</details>

### 問4
シノブの文化的利用を述べよ。

<details>
<summary>解答を見る</summary>

根茎を球形に仕立てて**吊りシノブ**（釣り忍）として、夏に風鈴とともに吊るす

</details>
