## バラ目 Rosales

[バラ目](https://ja.wikipedia.org/wiki/バラ目)は、マメ類（Fabids）に属する目で、[バラ科](https://ja.wikipedia.org/wiki/バラ科)を中心とする重要な目です。

| 項目 | 内容 |
|------|------|
| 日本産科数 | 6科 |
| 日本産種数 | 約350種 |
| 主な科 | バラ科、イラクサ科、ニレ科、アサ科、クワ科、グミ科 |

## [バラ科](https://ja.wikipedia.org/wiki/バラ科) Rosaceae

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約40属 |
| 日本産種数 | 約200種 |
| 世界の属数/種数 | 約90属/約3,000種 |
| 分布 | 北半球温帯を中心 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 葉 | 互生、単葉〜複葉、**托葉あり** |
| 花 | 放射相称、**5数性** |
| 萼 | 5裂、しばしば副萼あり |
| 花弁 | 5枚（八重咲き品種は多数） |
| 雄しべ | **多数** |
| 心皮 | 1〜多数 |
| 花托 | 様々に発達（ナシ状果、イチゴ状など） |

### 亜科と果実の分類

| 亜科 | 果実の型 | 代表属 |
|------|----------|--------|
| サクラ亜科 | **核果**（モモ型） | サクラ、ウメ、モモ |
| ナシ亜科 | **ナシ状果** | リンゴ、ナシ、ビワ |
| バラ亜科 | **痩果集合** | バラ、イチゴ、キイチゴ |
| シモツケ亜科 | **袋果** | シモツケ、ユキヤナギ |

**ナナカマドの詳細**
- 「七竈」＝七回竈に入れても燃えない
- 赤い果実と紅葉が美しい
- 落葉高木
- 葉は**対生ではなく互生**（奇数羽状複葉）
- 花弁は**白色**
- 果実は**ナシ状果**（梨状果・偽果）、核果ではない
- 花序は**散房花序**または複散房花序
- 常緑高木ではない（落葉性）

> ⚠️ **試験頻出**（2021年問43より）：ナナカマド属の正しい記述は「花弁は白色」。  
> 常緑高木×、葉は対生×、果実は核果×。

!!! info "イラクサ科の花の特徴（2021年度問38で出題）"
    - 多くは草本、単性花（雌雄異花）
    - 花被片2〜5個、**雄蕊は花被片と対生**（向かい合う）
    - **花糸は開花時に反り返って花粉を弾き飛ばす**（風媒花の適応）
    - この「花被片と対生する雄蕊＋花糸の反り返り」がイラクサ科の識別ポイント
    - アサ科・タデ科・ヒユ科と混同しないこと

### [アサ科](https://ja.wikipedia.org/wiki/アサ科) Cannabaceae

!!! warning "APGによる科の再編：エノキ・ムクノキはアサ科へ"
    旧エングラー体系ではエノキ・ムクノキ・ケヤキ等は全て**ニレ科（Ulmaceae）**に含まれていた。  
    APG分類体系では：
    - **ニレ科**（Ulmaceae）：ニレ属・ケヤキ属等のみ
    - **アサ科**（Cannabaceae）：**エノキ属・ムクノキ属**＋アサ・ホップ等
    
    2022年度問53-2で直接出題：「エノキ・ムクノキ・ウラジロエノキ・エゾエノキが属する科 → Cannabaceae（アサ科）」が正解。

## バラ科の果実タイプ詳細比較（試験最重要）

### 果実タイプ比較表

| 果実の型 | 構造 | 亜科 | 代表種 | 試験重要度 |
|----------|------|------|--------|------------|
| **核果**（drupe） | 内果皮が硬い「核（石）」になる。中に種子1個 | サクラ亜科 | サクラ、モモ、ウメ、アンズ、スモモ | ★★★★★ |
| **仁果/ナシ状果**（pome） | 花托・萼筒が肥大して果肉になる**偽果**。食べる部分は花托 | ナシ亜科 | リンゴ、ナシ、ビワ、カリン | ★★★★★ |
| **集合痩果**（etaerio of achenes） | 花托が肥大し表面に痩果が並ぶ偽果 | バラ亜科 | イチゴ（食べる赤い部分＝花托） | ★★★★ |
| **偽果**（バラ型） | 萼筒が肥大してローズヒップになり、内部に痩果 | バラ亜科 | バラ（Rosa属） | ★★★ |
| **袋果**（follicle） | 縫合線で裂開する単心皮果 | シモツケ亜科 | シモツケ、ユキヤナギ | ★★ |

> **問：リンゴを食べるとき、実際に食べている部分（果肉）は植物学的に何が変化したものか？**
> → **花托（花床）と萼筒**が肥大した**偽果**（仁果）。本来の果実（子房）は芯の部分。

> **問：イチゴの表面についている粒々（種のように見える部分）は植物学的に何か？**
> → **痩果**（そうか）。これが本来の果実で、赤く肥大した部分は**花托**（偽果）。

> **問：モモとリンゴの果実の型をそれぞれ答えよ。**
> → モモ＝**核果**（drupeで内果皮が硬い石）。リンゴ＝**仁果/ナシ状果**（花托が発達した偽果）。

## ソメイヨシノ詳細（試験重要度 ★★★★★）

### [ソメイヨシノ](https://ja.wikipedia.org/wiki/ソメイヨシノ) *Cerasus* × *yedoensis*

#### 形態的特徴

| 部位/特徴 | 内容 |
|-----------|------|
| 親 | **エドヒガン**（*C. itosakura*）× **オオシマザクラ**（*C. speciosa*） |
| 葉 | 展葉前か同時に開花（先行または同時型） |
| 花 | 淡紅色、径3〜4cm、1花柄に3〜5個 |
| 果実 | ほとんど結実しない（不稔性） |
| 増殖 | **接ぎ木**のみで増殖→日本全国のソメイヨシノは**同一クローン** |
| 托葉蜜腺 | **托葉の基部に蜜腺**（托葉蜜腺）がある（サクラ属の特徴） |

> **問：ソメイヨシノが日本全国で同時に一斉開花する理由を述べよ。**
> → 日本全国のソメイヨシノはすべて**同一クローン**（接ぎ木で増殖）のため、遺伝的に同一。開花は気温（積算温度）に依存するため、気温条件が揃えば一斉に咲く。

> **問：サクラ属（Cerasus / Prunus）の葉柄に見られる特徴的な構造は何か？**
> → **托葉蜜腺**（托葉の変形した蜜腺）。葉柄基部や托葉に蜜腺がある。アリなどを誘引して防衛する間接防衛の仕組み。
