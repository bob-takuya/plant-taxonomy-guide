# 13-9. セリ目

## はじめに

[セリ目](https://ja.wikipedia.org/wiki/セリ目)（Apiales）は、キキョウ類に属する目で、セリ科とウコギ科を含みます。春の七草のセリをはじめ、食用・薬用・有毒植物を多く含む重要な目です。

---

## セリ目の概要

| 項目 | 内容 |
|------|------|
| 日本産科数 | 2科 |
| 世界の科数 | 7科 |
| 世界の種数 | 約5,500種 |
| 主な特徴 | 散形花序、分果、精油を含む |

---

## 1. セリ科 Apiaceae（Umbelliferae）

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約50属 |
| 日本産種数 | 約100種 |
| 世界の属数 | 約450属 |
| 世界の種数 | 約3,700種 |

> 📝 **旧名**：傘形科（Umbelliferae）とも呼ばれる

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 草本（まれに木本） |
| 茎 | 中空のものが多い |
| 葉 | 互生、複葉（羽状または掌状）、葉柄基部が鞘状 |
| 花序 | **散形花序**（複散形花序） |
| 花 | 小型、5数性、花弁先端が内曲 |
| 子房 | 下位、2室 |
| 果実 | **分果**（2分果が合着、熟すと分離） |
| 化学成分 | 精油（芳香）、フラノクマリンなど |

> 📝 **識別キーワード**：散形花序 ＋ 分果 ＋ 複葉 ＋ 芳香

### セリとドクゼリの見分け方

| 特徴 | セリ | ドクゼリ |
|------|------|----------|
| 根茎 | 細い、横走 | 太く、中空、節あり |
| 香り | 良い芳香 | 不快臭 |
| 生育環境 | 湿地、水田 | 湿地、沼 |
| 大きさ | 小型～中型 | 大型（1m以上） |

## 2. ウコギ科 Araliaceae

### 基本情報

| 項目 | 内容 |
|------|------|
| 日本産属数 | 約15属 |
| 日本産種数 | 約30種 |
| 世界の属数 | 約50属 |

### 形態的特徴

| 部位 | 特徴 |
|------|------|
| 生活形 | 木本（まれに草本） |

## 確認問題

### 問1
セリ科の花序と果実の特徴を述べよ。

<details>
<summary>解答を見る</summary>

- **花序**：散形花序（複散形花序）
- **果実**：分果（2分果が熟すと分離）

</details>

### 問2
日本三大有毒植物に含まれるセリ科植物は何か。

<details>
<summary>解答を見る</summary>

**ドクゼリ**

</details>

### 問3
春の七草に含まれるセリ目の植物は何か。

<details>
<summary>解答を見る</summary>

**セリ**

</details>

### 問4
ウコギ科で食用として人気のある山菜を3つ挙げよ。

<details>
<summary>解答を見る</summary>

- **タラノキ（タラの芽）**
- **コシアブラ**
- **ウド**

</details>

## 暗記用まとめ

### セリ科の識別

```
散形花序 ＋ 分果 ＋ 複葉 ＋ 芳香 → セリ科
```

### セリ科の有毒植物

| 植物名 | 有毒成分 | 似ている食用植物 |
|--------|----------|------------------|
| ドクゼリ | シクトキシン | セリ |
| ドクニンジン | コニイン | シャク、ヤブニンジン |

### セリ科の経済植物

| 用途 | 植物名 |
|------|--------|
| 野菜 | ニンジン、セロリ |
| 香味野菜 | パセリ、ミツバ |
| 香辛料 | コリアンダー、クミン、ディル |
| 山菜 | セリ、シャク |

### ウコギ科の山菜

| 植物名 | 特徴 |
|--------|------|
| タラノキ | 山菜の王様、棘多い |
| コシアブラ | 近年人気、棘なし |
| ウド | 草本、大型 |
