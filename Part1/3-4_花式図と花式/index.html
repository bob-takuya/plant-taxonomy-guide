    3-4. 花式図と花式¶

    ← 目次に戻る | ← 前：生殖器官

    はじめに¶

    花式（Floral Formula）と花式図（Floral Diagram）は、花の構造を簡潔に表現する方法です。試験では主要科の花式を理解していることが求められます。

    花式（Floral Formula）¶

    花式の記号¶

    記号 意味
    K 萼（Kalyx）
    C 花冠（Corolla）
    P 花被（Perianth）※萼と花弁が区別できない場合
    A 雄しべ（Androecium）
    G 雌しべ（Gynoecium）

    数と修飾記号¶

    記号 意味
    数字 各器官の数
    ∞ 多数
    ( ) 合着
    上線 Ḡ 子房上位
    下線 G̲ 子房下位
    ⊕ または ✳ 放射相称
    ↑ または % 左右相称
    ♂ 雄花
    ♀ 雌花
    ⚥ 両性花

    主要科の花式¶

    ==バラ科== Rosaceae¶

    ✳ K5 C5 A∞ Ḡ1-∞
    要素 説明
    ✳ 放射相称
    K5 萼片5枚
    C5 花弁5枚
    A∞ 雄しべ多数
    Ḡ1-∞ 子房上位、心皮1〜多数

    ==アブラナ科== Brassicaceae¶

    ✳ K4 C4 A2+4 Ḡ(2)
    要素 説明
    K4 萼片4枚
    C4 花弁4枚（十字形）
    A2+4 雄しべ6本（2短+4長）==四強雄蕊==
    Ḡ(2) 子房上位、2心皮が合着

    ==マメ科== Fabaceae¶

    ↑ K(5) C5 A(9)+1 または A(10) Ḡ1
    要素 説明
    ↑ 左右相称（蝶形花）
    K(5) 萼片5枚が合着
    C5 花弁5枚（旗弁・翼弁・竜骨弁）
    A(9)+1 9本合着+1本離生（2体雄蕊）
    Ḡ1 子房上位、1心皮

    ==シソ科== Lamiaceae¶

    ↑ K(5) C(5) A4 または A2+2 Ḡ(2)
    要素 説明
    ↑ 左右相称（唇形花）
    K(5) 萼片5枚が合着
    C(5) 花弁5枚が合着（唇形）
    A2+2 雄しべ4本（2長+2短）==二強雄蕊==
    Ḡ(2) 子房上位、2心皮が合着

    ==キク科== Asteraceae¶

    管状花¶

    ✳ K0 または pappus C(5) A(5) G̲(2)

    舌状花¶

    ↑ K0 または pappus C(5) A(5) G̲(2)
    要素 説明
    K0/pappus 萼は冠毛に変形または退化
    C(5) 花弁5枚が合着
    A(5) 雄しべ5本、葯が合着（集約雄蕊）
    G̲(2) ==子房下位==、2心皮が合着

    ==セリ科== Apiaceae¶

    ✳ K5 C5 A5 G̲(2)
    要素 説明
    K5 萼片5枚（小さいか退化）
    C5 花弁5枚
    A5 雄しべ5本
    G̲(2) ==子房下位==、2心皮が合着（→2分果）

    ==ナス科== Solanaceae¶

    ✳ K(5) C(5) A5 Ḡ(2)
    要素 説明
    K(5) 萼片5枚が合着
    C(5) 花弁5枚が合着
    A5 雄しべ5本
    Ḡ(2) 子房上位、2心皮が合着

    ==ユリ科== Liliaceae（狭義）¶

    ✳ P3+3 A3+3 Ḡ(3)
    要素 説明
    P3+3 花被片6枚（外花被3+内花被3）
    A3+3 雄しべ6本
    Ḡ(3) 子房上位、3心皮が合着

    ==ラン科== Orchidaceae¶

    ↑ P3+3 A1-2 G̲(3)
    要素 説明
    ↑ 左右相称
    P3+3 花被片6枚（唇弁を含む）
    A1-2 雄しべ1-2本（蕊柱に合着）
    G̲(3) 子房下位、3心皮が合着

    花式図（Floral Diagram）¶

    花式図の基本¶

    花式図は花を上から見た断面図で表します。

    花式図の基本配置

          苞（または軸）
              ↑
            ○○○  ← 萼片
           ●●●●● ← 花弁
          ◎◎◎◎◎  ← 雄しべ
             ◉     ← 雌しべ
              ↓
           花柄側

    記号の意味¶

    記号 意味
    ○ 萼片
    ● 花弁
    ◎ 雄しべ
    ◉ 雌しべ（子房）
    — 器官の合着

    主要科の花式図¶

    バラ科の花式図¶

            苞
         │
    ○ ○ ○ ○ ○    ← 萼片5
   ● ● ● ● ●     ← 花弁5
  ◎◎◎◎◎◎◎◎◎◎  ← 雄しべ多数
       ◉◉◉        ← 雌しべ1〜多数

    アブラナ科の花式図¶

            苞
         │
      ○   ○       ← 萼片4
    ○       ○
      ●   ●       ← 花弁4（十字形）
    ●       ●
     ◎ ◎◎◎ ◎     ← 雄しべ6（2短+4長）
       ◎  ◎
        ◉         ← 雌しべ（2心皮）

    キク科（管状花）の花式図¶

            苞
         │
      冠毛         ← 萼（変形）
    ●────●        ← 花弁5（合着）
   ◎─◎─◎─◎─◎    ← 雄しべ5（葯が合着）
       ◉          ← 雌しべ（子房下位）

    花式の読み方・書き方のコツ¶

    花を見て花式を書く手順¶

    1. 対称性を確認（放射相称 or 左右相称）
    2. 萼片の数を数える
    3. 花弁の数を数える（合着しているか確認）
    4. 雄しべの数を数える（長さの違いがあるか確認）
    5. 雌しべの構造を確認（子房の位置、心皮数）

    花式から科を推定する¶

    花式の特徴 該当する科
    A∞（雄しべ多数） バラ科
    A2+4（四強雄蕊） アブラナ科
    A(9)+1（2体雄蕊） マメ科
    A2+2（二強雄蕊） シソ科
    A(5)（集約雄蕊） キク科
    G̲（子房下位） キク科、セリ科、ウリ科
    ↑（左右相称） マメ科、シソ科、ラン科

    確認問題¶

    問1¶

    アブラナ科の花式で、雄しべの数と配列を表す記号は何か。

    解答 **A2+4** または **A6** 2本が短く4本が長い（四強雄蕊）。

    問2¶

    次の花式はどの科のものか：↑ K(5) C5 A(9)+1 Ḡ1

    解答 **マメ科** 左右相称、蝶形花、2体雄蕊、1心皮が特徴。

    問3¶

    キク科の雄しべの特徴を表す用語は何か。

    解答 **集約雄蕊** 5本の雄しべの葯が合着して筒状になる。

    問4¶

    花式で子房下位を表す記号は何か。

    解答 **G̲**（Gの下に線）

    問5¶

    放射相称と左右相称を表す記号をそれぞれ答えよ。

    解答 - **放射相称**：✳ または ⊕ - **左右相称**：↑ または %

    主要科の花式¶

    科名 花式 特徴
    バラ科 ✳K5C5A∞Ḡ1-∞ 雄しべ多数
    アブラナ科 ✳K4C4A2+4Ḡ(2) 四強雄蕊
    マメ科 ↑K(5)C5A(9)+1Ḡ1 蝶形花、2体雄蕊
    シソ科 ↑K(5)C(5)A2+2Ḡ(2) 唇形花、二強雄蕊

    セリ科 ✳K5C5A5G̲(2) 子房下位
