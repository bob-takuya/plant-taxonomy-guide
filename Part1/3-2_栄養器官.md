[← 目次に戻る](../NEW_00_目次.md) | [← 前：形態学総論](./3-1_形態学総論.md)

## はじめに

栄養器官である根・茎・葉は植物の同定において最も重要な特徴です。特に**葉序**（対生・互生）と**葉の形態**は科の識別に必須です。

# 第1部：根（Root）

## 根の基本構造

### 根の各部名称

```
    ┌─────────────────┐
    │     主根        │
    │  （taproot）    │
    └────────┬────────┘
             │
    ┌────────┼────────┐
    │        │        │
  側根    側根     側根
（lateral roots）
             │
    ┌────────┼────────┐
    │   根毛帯       │
    │（root hair zone）│
    └────────┼────────┘
             │
         伸長帯
             │
         分裂帯
             │

```

## 根の種類

### 根系の型

| 型 | 特徴 | 分布 |
|-----|------|------|
| **主根系** | 主根が太く発達 | ==双子葉類== |
| **ひげ根系** | 多数の細い根が束状 | ==単子葉類== |

### 根の起源による分類

| 種類 | 説明 | 例 |
|------|------|-----|
| **定根** | 種子の幼根に由来 | 主根、側根 |
| **不定根** | 茎・葉から発生 | サツマイモ、ラン類の気根 |

## 特殊な根

| 根の型 | 説明 | 代表例 |
|--------|------|--------|
| **貯蔵根** | 養分を蓄える | ダイコン、ニンジン、サツマイモ |
| **気根** | 空気中に出る | ガジュマル、ラン類、マングローブ |
| **支柱根** | 茎を支える | トウモロコシ、タコノキ |
| **板根** | 板状に発達 | 熱帯樹木（サキシマスオウノキ） |
| **呼吸根** | 酸素を取り込む | マングローブ（オヒルギ） |
| **吸盤根** | 他の植物に着生 | ツタ、テイカカズラ |
| **寄生根** | 養分を吸収 | ネナシカズラ、ヤドリギ |

### ==マングローブの根（頻出）==

| 種類 | 特徴 | 代表種 |
|------|------|--------|
| **支柱根** | 幹から斜めに伸びる | ヤエヤマヒルギ |
| **膝根** | 膝のように曲がる | オヒルギ |
| **筍根** | タケノコ状に立ち上がる | メヒルギ |

## 葉の基本構造

### 葉の各部名称

```
        ┌──────────┐
        │   葉身   │
        │ (lamina) │
        └────┬─────┘
             │
        ┌────┴────┐
        │  葉柄   │
        │(petiole)│
        └────┬────┘
             │

```

| 部位 | 説明 | 備考 |
|------|------|------|

| **托葉** | 葉柄基部の付属体 | ==バラ科の特徴== |

## ==葉の内部構造・気孔（試験頻出）==

### 気孔（Stomata）の構造

> ⚠️ **2021年度問25で直接出題。誤りの選択肢として「孔辺細胞に葉緑体はない」が登場。**

```
気孔の断面（上から見た図）

    副細胞  孔辺細胞  副細胞
    ┌────┐┌──────┐┌────┐
    │    ││  ◯◯  ││    │
    │    ││  開口  ││    │
    └────┘└──────┘└────┘
         孔辺細胞   副細胞
```

| 部位 | 説明 | 重要ポイント |
|------|------|-------------|
| **気孔（stomata）** | 表皮にある開口部、ガス交換・蒸散を担う | 通常は葉の**下面に多い** |
| **孔辺細胞（guard cells）** | 気孔を囲む2個の特殊細胞 | **葉緑体をもつ**（⚠️試験頻出：「葉緑体がない」は誤り） |
| **副細胞（subsidiary cells）** | 孔辺細胞を取り囲む隣接細胞 | あることが多い（一般的な表皮細胞と区別される） |

#### 気孔の分布（試験頻出）

| 植物の種類 | 気孔の分布 |
|-----------|------------|
| 多くの陸上植物 | **下面に多い**（蒸散抑制のため） |
| 浮葉性水生植物（ハス・スイレン等） | **上面のみ**（水面に浮くため） |
| 沈水植物（オオカナダモ等） | **気孔なし** |
| イネ科草本 | 上下両面に分布 |

#### 孔辺細胞と葉緑体（⚠️頻出誤答パターン）

> **「孔辺細胞に葉緑体はない」→ 誤り**  
> 孔辺細胞は**葉緑体をもつ**。光合成によって糖を生産し、浸透圧を変化させることで気孔の開閉を行う。これが孔辺細胞が葉緑体を必要とする理由。  
> 通常の表皮細胞には葉緑体がないが、孔辺細胞は例外的に葉緑体をもつ。

#### 気孔開閉のメカニズム（概要）

| 条件 | 孔辺細胞の状態 | 気孔 |
|------|--------------|------|
| 光照射・CO₂減少 | 浸透圧上昇→吸水・膨圧上昇 | **開く** |
| 暗・CO₂増加・乾燥 | 浸透圧低下→脱水・膨圧低下 | **閉じる** |

## ==葉序（Phyllotaxis）（超頻出）==

### 葉序の種類

| 葉序 | 説明 | 代表的な科 |
|------|------|------------|
| ==**互生**== | 1節に1枚ずつ交互 | バラ科、キク科、ブナ科、マメ科、アブラナ科 |
| ==**対生**== | 1節に2枚向かい合う | ==シソ科==、アカネ科、アジサイ科、オオバコ科 |
| **輪生** | 1節に3枚以上 | アカネ科（ヤエムグラ）、トチノキ |
| **根生** | 根元に集まる | タンポポ、オオバコ |

### ==コクサギ型葉序（頻出）==

| 説明 | 該当植物 |
|------|----------|
| 2枚ずつ交互につく（互生と対生の中間） | コクサギ（ミカン科） |

```
エリカ葉の断面
      葉の縁が巻き込む
        ↓
    ───┐ ┌───
       │ │
       └─┘
      気孔は内側
```

|------|------|

</details>

### 問4
単子葉類と双子葉類で葉脈のパターンはどう異なるか。

<details>
<summary>解答</summary>

- **単子葉類**：平行脈
- **双子葉類**：網状脈

</details>

### 問5
イネ科の葉で、葉鞘と葉身の境にある膜状の構造を何というか。

<details>
<summary>解答</summary>

**葉舌（ligule）**

</details>
