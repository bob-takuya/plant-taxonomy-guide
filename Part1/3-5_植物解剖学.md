# 3-5. 植物解剖学（Plant Anatomy）

[← 目次に戻る](../NEW_00_目次.md) | [← 前：花式図と花式](./3-4_花式図と花式.md)

---

## はじめに

植物解剖学は、植物の内部構造と組織を顕微鏡レベルで研究する分野です。**気孔の構造**、**維管束の配列**、**組織系の分化**は、植物の同定や分類において重要な特徴となります。

---

# 第1部：植物の組織系

## 組織系の基本

### 3つの組織系

| 組織系 | 機能 | 主な構成組織 |
|--------|------|--------------|
| **表皮系** | 保護、ガス交換 | 表皮、気孔、毛茸 |
| **維管束系** | 物質輸送 | 木部、師部 |
| **基本組織系** | 光合成、貯蔵、支持 | 柔組織、厚角組織、厚壁組織 |

---

# 第2部：表皮系（Dermal Tissue System）

## 表皮（Epidermis）

### 表皮の基本構造

| 特徴 | 説明 |
|------|------|
| **細胞配列** | 通常1層（まれに多層） |
| **細胞壁** | 外壁が厚く、クチクラ層で覆われる |
| **葉緑体** | 通常は持たない（例外：シダ類の気孔） |
| **機能** | 保護、水分蒸散の調節 |

### クチクラ（Cuticle）

| 項目 | 説明 |
|------|------|
| **構成成分** | クチン（脂質の重合体） |
| **機能** | 水分の蒸散を防ぐ |
| **厚さ** | 環境により変化（乾燥地では厚い） |

---

## ==気孔（Stomata）（超頻出）==

### 気孔の基本構造

```
        表皮細胞
   ┌──────┬──────┐
   │              │
   │              │
   │   ┌──┐   │
   │   │孔│   │  ← 気孔孔（stomatal pore）
   │   └──┘   │
   │              │
   └──────┴──────┘

   ○ = 孔辺細胞（guard cell）
   □ = 副細胞（subsidiary cell）
```

### ==気孔の構成要素（必須暗記）==

| 構造 | 説明 | 特徴 |
|------|------|------|
| **孔辺細胞** | 気孔孔を囲む2個の細胞 | ==**葉緑体を持つ**==（例外的） |
| **気孔孔** | ガス交換の開口部 | 孔辺細胞の膨圧変化で開閉 |
| **副細胞** | 孔辺細胞に隣接する細胞 | ==数や配列は分類形質== |
| **気室** | 気孔孔の下の空隙 | 葉内部とつながる |

### ==気孔の重要特性（頻出）==

| 特性 | 内容 |
|------|------|
| **孔辺細胞の数** | ==**1個の気孔に2個**== |
| **葉緑体** | ==**孔辺細胞は葉緑体を持つ**==（表皮細胞は通常持たない） |
| **分布** | ==**葉の下面＞上面**==（通常） |
| **開閉機構** | 孔辺細胞の膨圧変化による |

### 気孔の分布パターン

| 葉の種類 | 上面 | 下面 | 例 |
|----------|------|------|-----|
| **両面葉** | 少ない | 多い | 多くの双子葉類 |
| **等面葉** | ほぼ同数 | ほぼ同数 | イネ科など |
| **浮葉** | 多い | なし（または少ない） | スイレン、ハス |
| **沈水葉** | なし～非常に少ない | なし～非常に少ない | 沈水植物 |

## 気孔の型（副細胞の配列）

### ==主要な気孔型（分類形質）==

#### 1. 不斉細胞型（Anomocytic）

```
  □ □ □ □
  □ ○ ○ □  ← 副細胞が孔辺細胞と区別できない
  □ □ □ □
```

| 特徴 | 分布 |
|------|------|
| 副細胞が特定できない | ナデシコ科、キンポウゲ科 |

#### 2. 横列細胞型（Paracytic）

```
  □ □ □ □
  ■ ○ ○ ■  ← 孔辺細胞の両側に副細胞が平行配列
  □ □ □ □
```

| 特徴 | 分布 |
|------|------|
| 2個の副細胞が孔辺細胞に平行 | アブラナ科、マメ科 |

#### 3. 周囲細胞型（Cyclocytic）

```
    ■ ■ ■
  ■ ○ ○ ■  ← 孔辺細胞を副細胞が取り囲む
    ■ ■ ■
```

| 特徴 | 分布 |
|------|------|
| 4個以上の副細胞が輪状に囲む | ヤシ科、ソテツ科 |

#### 4. 十字型（Cruciferous）

```
  □ ■ ■ □
  ■ ○ ○ ■  ← 副細胞が十字状配列
  □ ■ ■ □
```

| 特徴 | 分布 |
|------|------|
| 副細胞が十字状 | アブラナ科の一部 |

#### 5. イネ型（Gramineous）

```
  □ □ □ □
  ■ ○ ○ ■  ← ダンベル型の孔辺細胞
  □ □ □ □
```

| 特徴 | 分布 |
|------|------|
| 孔辺細胞がダンベル型 | ==イネ科、カヤツリグサ科== |
| 副細胞2個（ドーム状） | 単子葉類に特徴的 |

## 毛茸（Trichomes）

### 毛茸の種類

| 種類 | 特徴 | 例 |
|------|------|-----|
| **単細胞毛** | 1つの細胞からなる | キュウリ、オクラ |
| **多細胞毛** | 複数の細胞からなる | トマト、シソ |
| **腺毛** | 分泌物を出す | ハッカ、モウセンゴケ |
| **星状毛** | 星形に分岐（放射状に広がる） | アブラナ科・**コナラ属・シナノキ属・ウツギ属・ムラサキシキブ属** |
| **丁字状毛** | T字形に分岐 | ムクロジ目の一部 |
| **盾状鱗片** | 盾状の構造 | ブロメリア科・グミ科 |
| **刺毛** | 硬くて鋭い | イラクサ科 |

!!! warning "星状毛の出題パターン（2023年度問40で出題）"
    「コナラ属・シナノキ属・ウツギ属・ムラサキシキブ属の葉にみられる毛のタイプ」
    → 正解：**星状毛**（2023年度問40 正解1）
    
    星状毛をもつ主要科・属：
    - コナラ属（ブナ科）・アジサイ科（ウツギ属等）・アオイ科・シナノキ属・ムラサキシキブ属（シソ科）等

### 毛茸の機能

| 機能 | 説明 |
|------|------|
| **蒸散抑制** | 表面積を増やし境界層を厚くする |
| **紫外線防御** | 光を反射 |
| **草食動物からの防御** | 刺毛、有毒成分の分泌 |
| **種子散布** | 綿毛（タンポポなど） |

# 第3部：維管束系（Vascular Tissue System）

## 木部（Xylem）

### 木部の構成要素

| 構成要素 | 機能 | 特徴 |
|----------|------|------|
| **道管** | 水・無機養分の輸送 | 細胞壁が木化、細胞は死んでいる |
| **仮道管** | 水の輸送（裸子植物） | 道管より原始的 |
| **木部柔細胞** | 貯蔵 | 生きている細胞 |
| **木部繊維** | 支持 | 厚壁、木化 |

### 道管の穿孔板

| 型 | 特徴 | 分布 |
|-----|------|------|
| **単穿孔** | 1つの開口部 | 進化的に新しい（双子葉類の多く） |
| **階段穿孔** | 多数の横棒 | 原始的（モクレン類、バラ類の一部） |
| **網状穿孔** | 網目状 | 中間的 |

---

## 師部（Phloem）

### 師部の構成要素

| 構成要素 | 機能 | 特徴 |
|----------|------|------|
| **師管要素** | 光合成産物の輸送 | 核を失うが生きている |
| **伴細胞** | 師管要素のサポート | 師管要素と発生的に連結 |
| **師部柔細胞** | 貯蔵 | 生きている細胞 |
| **師部繊維** | 支持 | 靭皮繊維として利用（アサ、カラムシ） |

---

## 維管束の配列

### ==茎の維管束配列（分類形質）==

#### 双子葉類の茎

```
    表皮
     ○
  ○     ○
○ ●─●─● ○  ← 維管束が環状配列
  ○     ○     ● = 維管束
     ○         ○ = 皮層
    中央 = 髄
```

| 特徴 | 説明 |
|------|------|
| 配列 | ==環状配列== |
| 形成層 | ==維管束形成層あり== |
| 二次成長 | ==可能==（木本化） |

#### 単子葉類の茎

```
    表皮
  ○ ● ○ ●
 ● ○ ● ○ ●  ← 維管束が散在
  ○ ● ○ ●
   ● ○ ●
```

| 特徴 | 説明 |
|------|------|
| 配列 | ==散在配列==（不規則） |
| 形成層 | ==なし==（通常） |
| 二次成長 | ==不可能==（草本性） |

---

### ==根の維管束配列==

#### 双子葉類の根

```
  表皮
   ○
 ○ X ○  ← 中心に放射状配列
○ XPX ○    X = 木部（放射状）
 ○ X ○     P = 師部（交互配列）
   ○
```

| 特徴 | 説明 |
|------|------|
| 中心柱 | 放射中心柱 |
| 配列 | 木部と師部が交互（通常2～4原型） |

#### 単子葉類の根

```
  表皮
   ○
 ○○○○○  ← 多原型（多数の木部）
○XPXPX○
○PXPXP○
 ○○○○○
```

| 特徴 | 説明 |
|------|------|
| 中心柱 | 多原型が多い（6原型以上） |
| 髄 | 中心に髄をもつことが多い |

# 第4部：基本組織系（Ground Tissue System）

## 基本組織の種類

### 柔組織（Parenchyma）

| 特徴 | 機能 |
|------|------|
| 薄い細胞壁 | 光合成、貯蔵、分泌 |
| 生きた細胞 | 多能性を保持（再生能力） |
| 細胞間隙あり | ガス交換 |

### 厚角組織（Collenchyma）

| 特徴 | 機能 |
|------|------|
| 細胞壁が不均一に厚化 | 成長中の器官の支持 |
| 細胞壁は柔軟（木化しない） | 伸長可能 |
| 生きた細胞 | 葉柄、若い茎に多い |

### 厚壁組織（Sclerenchyma）

| 種類 | 特徴 | 例 |
|------|------|-----|
| **石細胞** | 等径的、厚壁、木化 | ナシの果肉のざらつき |
| **繊維** | 細長い、厚壁、木化 | 木部繊維、師部繊維 |

# 第5部：葉の解剖学

## 葉の断面構造

### ==双子葉類の葉（典型的な二面葉）==

```
    上表皮（気孔少ない）
  ───────────────
    柵状組織（光合成）
    ■■■■■■■

    海綿状組織
    ○ ○ ○ ○ ○
     ○ ○ ○ ○   ← 細胞間隙大
    ○ ○ ○ ○ ○
  ───────────────
    下表皮（気孔多い）
      [気孔]
```

| 組織 | 特徴 | 機能 |
|------|------|------|
| **柵状組織** | 細長い細胞が並ぶ、葉緑体多い | ==主要な光合成の場== |
| **海綿状組織** | 不規則な配列、細胞間隙大 | ガス交換、光合成 |

### ==イネ科の葉（等面葉）==

```
    上表皮
  ───────────────
    葉緑体を持つ細胞が均一分布
    ○ ● ○ ● ○
    ● ○ ● ○ ●
    ○ ● ○ ● ○
  ───────────────
    下表皮
```

| 特徴 | 説明 |
|------|------|
| 柵状組織と海綿状組織の区別なし | 葉緑体が均一分布 |
| 上面・下面ともに気孔あり | 等面葉の特徴 |

---

## ==C3・C4・CAM植物の葉解剖（頻出）==

### C4植物の葉（クランツ構造）

```
      維管束鞘細胞
        (大きい葉緑体)
          │
    ┌─────┴─────┐
    │ 維管束   │
    │          │
    └─────┬─────┘
          │
      葉肉細胞
    (通常の葉緑体)
```

| 特徴 | 説明 |
|------|------|
| **クランツ構造** | ==維管束鞘細胞が発達== |
| 葉緑体の分化 | 維管束鞘と葉肉で異なる |
| 代表的な科 | ==イネ科（トウモロコシ、サトウキビ）、ヒユ科== |

### C3・C4・CAMの比較

| 項目 | C3植物 | C4植物 | CAM植物 |
|------|--------|--------|---------|
| **解剖学的特徴** | 通常の葉構造 | ==クランツ構造== | 多肉組織 |
| **光合成効率** | 低い | 高い | 中程度 |
| **環境適応** | 温帯 | ==熱帯・乾燥地== | ==極乾燥地== |
| **代表例** | イネ、コムギ | ==トウモロコシ、サトウキビ== | ==サボテン、ベンケイソウ== |

# 第6部：根の解剖学

## 根の一次構造

### 根の断面（外側から内側へ）

```
表皮 → 皮層 → 内皮 → 中心柱
        ↓      ↓
      通気組織  カスパリー線
```

### ==内皮（Endodermis）==

| 特徴 | 機能 |
|------|------|
| **カスパリー線** | スベリン帯が細胞壁を横切る |
| 選択的透過 | 水・養分の吸収を制御 |
| 位置 | 皮層と中心柱の境界 |

### 根の成熟段階

| 領域 | 特徴 |
|------|------|
| **根冠** | 保護、粘液分泌 |
| **分裂帯** | 細胞分裂が盛ん |
| **伸長帯** | 細胞が伸長 |
| **成熟帯** | 根毛が発達、吸収 |

# 第7部：二次成長（Secondary Growth）

## 形成層（Cambium）

### 維管束形成層（Vascular Cambium）

| 特徴 | 機能 |
|------|------|
| 分裂組織 | 二次木部・二次師部を形成 |
| 位置 | 木部と師部の間 |
| 分布 | ==双子葉類、裸子植物==（単子葉類には通常ない） |

### コルク形成層（Cork Cambium / Phellogen）

| 特徴 | 機能 |
|------|------|
| 分裂組織 | コルク層を形成 |
| 位置 | 表皮の内側または皮層 |
| 機能 | 表皮に代わる保護組織 |

---

## 年輪（Annual Rings）

### 年輪の形成

| 部位 | 特徴 | 形成時期 |
|------|------|----------|
| **春材** | 道管が大きい、壁が薄い | 春～初夏 |
| **秋材** | 道管が小さい、壁が厚い | 夏～秋 |
| **年輪境界** | 秋材と次の春材の境 | 明瞭な線 |

### 年輪による年代測定

| 用語 | 説明 |
|------|------|
| **年輪年代学** | 年輪パターンから年代を特定 |
| **クロスデーティング** | 複数の試料を照合 |
| **気候復元** | 年輪幅から過去の気候を推定 |

# 第8部：特殊な解剖学的適応

## 水生植物の解剖学

### 通気組織（Aerenchyma）

```
  細胞
   ○
○     ○  ← 大きな細胞間隙
   ○
  空隙
   ○
○     ○
   ○
```

| 特徴 | 機能 | 分布 |
|------|------|------|
| 大きな細胞間隙 | 酸素の貯蔵・輸送 | ==水生植物、湿地植物== |
| スポンジ状 | 浮力を提供 | ハス、ガマ、イネ科の一部 |

---

## 乾燥地植物の解剖学

### 乾生構造（Xeromorphic Features）

| 適応 | 例 |
|------|-----|
| **クチクラ層の肥厚** | サボテン |
| **多肉組織** | ベンケイソウ科、サボテン科 |
| **気孔の陥没** | マツ、トウヒ |
| **毛茸の発達** | ヨモギ、エーデルワイス |
| **葉の縮小・退化** | サボテン（茎が光合成） |

## 確認問題

### 問1
1個の気孔を構成する孔辺細胞は何個か。

<details>
<summary>解答を見る</summary>

==**2個**==

</details>

### 問2
気孔の孔辺細胞の特徴を3つ挙げよ。

<details>
<summary>解答を見る</summary>

1. ==**葉緑体を持つ**==（表皮細胞は通常持たない）
2. 膨圧変化により気孔孔を開閉する
3. 通常2個で1つの気孔を構成

</details>

### 問3
気孔は通常、葉の上面と下面のどちらに多いか。

<details>
<summary>解答を見る</summary>

==**下面**==（下面＞上面が一般的）

例外：浮葉植物（スイレンなど）は上面に多い

</details>

### 問4
双子葉類と単子葉類で、茎の維管束配列はどう異なるか。

<details>
<summary>解答を見る</summary>

- **双子葉類**：==環状配列==、維管束形成層あり、二次成長可能
- **単子葉類**：==散在配列==、維管束形成層なし、二次成長不可

</details>

### 問5
C4植物の葉に見られる特徴的な解剖学的構造を何というか。

<details>
<summary>解答を見る</summary>

==**クランツ構造（Kranz anatomy）**==

維管束鞘細胞が発達し、葉肉細胞とともに光合成に関与する。

</details>

### 問6
根の内皮細胞に見られる、スベリン帯からなる構造を何というか。

<details>
<summary>解答を見る</summary>

==**カスパリー線（Casparian strip）**==

水や養分の選択的透過を制御する。

</details>

### 問7
イネ科の気孔型の名称と、孔辺細胞の形状を答えよ。

<details>
<summary>解答を見る</summary>

- **名称**：==イネ型（Gramineous型）==
- **孔辺細胞の形状**：==ダンベル型==

</details>

### 問8
二次成長が可能な植物群を2つ挙げよ。

<details>
<summary>解答を見る</summary>

1. ==**双子葉類**==
2. ==**裸子植物**==

（単子葉類は通常、維管束形成層がないため二次成長しない）

</details>
