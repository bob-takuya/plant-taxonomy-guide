## 学習の全体像

### 合格に必要な知識量

| 分野 | 覚えるべき内容 | 目安量 |
|------|----------------|--------|
| 科の学名 | 日本産全科のラテン語名 | 約200科 |
| 科の特徴 | 形態・分布・代表種 | 主要80科は詳細に |
| 属・種 | 代表的な種の所属科 | 約500種 |
| 分類学史 | 重要な学者と功績 | 約15人 |
| 法規 | 法律名と対象種 | 約5法規 |
| 分布境界線 | 名称と位置 | 約10本 |

### 学習優先順位

```
【最優先】
  ↓  APG分類体系の理解
  ↓  主要科の学名と特徴（約50科）
  ↓  法規と保全（特定外来生物、種の保存法）
  ↓  分類学史（リンネ、ダーウィン、木村資生等）
  ↓
【高優先】
  ↓  全科の学名（約200科）
  ↓  写真同定の訓練
  ↓  分布境界線と植物地理
  ↓
【中優先】
  ↓  シダ植物・コケ植物
  ↓  裸子植物
  ↓
【低優先】
  ↓  菌類・地衣類・藻類
```

## 推奨学習スケジュール

### 3ヶ月計画（仕事をしながらの場合）

#### 1ヶ月目：基礎固め

| 週 | 内容 | 学習時間目安 |
|-----|------|--------------|
| 1週目 | 過去問1回分を解く（現状把握）+ 本書Part1第1-2章 | 平日1h/日、休日4h |
| 2週目 | APG分類体系の理解 + 系統樹の暗記 | 平日1h/日、休日4h |
| 3週目 | 主要科の学名暗記開始（バラ類20科） | 平日1.5h/日、休日5h |
| 4週目 | 主要科の学名暗記（キク類20科）+ 法規 | 平日1.5h/日、休日5h |

**1ヶ月目の到達目標：**
- APG体系の主要クレードを説明できる
- 主要40科の学名を書ける
- 過去問正答率40%以上

#### 2ヶ月目：知識拡充

| 週 | 内容 | 学習時間目安 |
|-----|------|--------------|
| 1週目 | 過去問2回分 + 単子葉類の科 | 平日1.5h/日、休日5h |
| 2週目 | 裸子植物・シダ植物 | 平日1.5h/日、休日5h |
| 3週目 | コケ・地衣類・菌類・藻類 | 平日2h/日、休日6h |
| 4週目 | 過去問3回分 + 弱点補強 | 平日2h/日、休日6h |

**2ヶ月目の到達目標：**
- 全科の学名を認識できる（約200科）
- 写真同定で科レベルの判断ができる
- 過去問正答率55%以上

#### 3ヶ月目：実践演習

| 週 | 内容 | 学習時間目安 |
|-----|------|--------------|
| 1週目 | 過去問4回分（本番形式） | 平日2h/日、休日7h |
| 2週目 | 写真同定の集中訓練 + フィールドワーク | 平日2h/日、休日8h |
| 3週目 | 弱点の徹底補強 + 学名の総復習 | 平日2h/日、休日8h |
| 4週目 | 最終確認 + 本番形式演習 | 平日1h/日、休日4h |

**3ヶ月目の到達目標：**
- 過去問正答率70%以上を安定して取れる
- 苦手分野がない状態
- 時間配分の感覚をつかむ

## 効率的な暗記法

### 科の学名の覚え方

#### 1. 語源から覚える

| 科名 | 語源 | 覚え方 |
|------|------|--------|
| Rosaceae | Rosa（バラ属）+ -aceae | バラ属から派生 |
| Asteraceae | Aster（キク属）+ -aceae | 星形の花 |
| Poaceae | Poa（イチゴツナギ属）+ -aceae | 草の意味 |
| Fagaceae | Fagus（ブナ属）+ -aceae | ブナ属から派生 |
| Orchidaceae | Orchis（ラン属）+ -aceae | 精巣の意味（根の形から） |

#### 2. グループで覚える

**バラ類の主要科（マメ類）**
```
マメ目：Fabaceae（マメ科）
バラ目：Rosaceae, Ulmaceae, Cannabaceae, Urticaceae...
ブナ目：Fagaceae, Betulaceae, Juglandaceae...
```

**キク類の主要科（シソ類）**
```
シソ目：Lamiaceae, Oleaceae, Plantaginaceae...
ナス目：Solanaceae, Convolvulaceae...
キク目：Asteraceae, Campanulaceae...
```

#### 3. 語呂合わせ

| 覚える内容 | 語呂合わせ |
|------------|------------|
| APG基部被子植物 | 「アンボスイ」（アンボレラ→スイレン→アウストロバイレヤ） |
| 分布境界線（北→南） | 「宮八ブラ渡三」（宮部線→八田線→ブラキストン線→渡瀬線→三宅線） |
| 五界説 | 「モ原菌植動」（モネラ→原生生物→菌→植物→動物） |

### 形態用語の覚え方

#### 図解で理解

葉序、花式、果実型などは必ず図で確認しながら覚える。

```
葉序の種類：
  互生        対生        輪生
   ↓          ↓↓         ↓↓↓
  ○          ○○         ○○○
   |          ||          \|/
```

## 過去問の活用法

### 過去問演習の3段階

#### 第1段階：全体把握（1周目）

- 時間を気にせず解く
- 解答を見ながら学習
- 出題傾向を把握する

#### 第2段階：弱点分析（2周目）

- 時間を計って解く
- 間違えた問題をリスト化
- 弱点分野を特定

#### 第3段階：本番対策（3周目以降）

- 本番と同じ条件で解く
- 合格点（70点）を目標に
- 時間配分を意識

### 間違いノートの作り方

| 項目 | 記入内容 |
|------|----------|
| 問題番号 | 2023年度 問15 |
| 分野 | APG分類体系 |
| 間違えた理由 | キク類とバラ類の所属を混同 |
| 正解の知識 | セリ科はキク類（セリ目） |
| 関連知識 | ウコギ科もセリ目 |

## フィールドワークの重要性

### 写真同定対策としてのフィールドワーク

- 週末に近くの植物園や野山へ
- スマートフォンで植物を撮影
- 図鑑やアプリで同定を試みる

### 観察のポイント

| 観察項目 | 確認すること |
|----------|--------------|
| 葉 | 葉序、葉形、鋸歯、毛の有無 |
| 花 | 花弁数、対称性、雄しべ・雌しべの数 |
| 果実 | 果実の型（液果、乾果等） |
| 茎 | 断面（円形・四角形）、毛の有無 |
| 全体 | 草本/木本、生育環境 |

### おすすめの観察地

- 植物園（種名ラベルあり）
- 大学のキャンパス
- 里山・雑木林
- 海岸植生（ハマミズナ科等）
- 湿地（カヤツリグサ科等）

## 参考教材の使い方

### 必須教材

1. **過去問題集**（最新4年分）
   - 最優先で入手
   - 最低3周は解く

2. **本書（この参考書）**
   - 体系的な知識習得に
   - 学名リストの暗記に

3. **植物図鑑**
   - 写真同定の訓練に
   - 推奨：山渓ハンディ図鑑シリーズ

### 補助教材

- APG原色牧野植物大図鑑（高価だが網羅的）
- 日本の野生植物シリーズ（平凡社）
- スマートフォンアプリ（PlantNet等）

## 直前期の過ごし方

### 試験1週間前

- 新しい知識を入れない
- 既習内容の総復習
- 学名リストの最終確認
- 睡眠時間を確保

### 試験前日

- 軽い復習のみ
- 持ち物の確認
- 試験会場へのアクセス確認
- 早めに就寝

### 試験当日

- 朝食をしっかり取る
- 余裕を持って会場入り
- 深呼吸してリラックス
