# 6-3. レッドリストと絶滅危惧種

[← 目次に戻る](../NEW_00_目次.md) | [← 前：外来生物法](./6-2_外来生物法.md)

## はじめに

レッドリストのカテゴリーと代表的な絶滅危惧植物は試験で頻出です。環境省レッドリストとIUCNレッドリストの両方を理解しておきましょう。

---

## 環境省レッドリスト

### 概要

| 項目 | 内容 |
|------|------|
| 作成者 | 環境省 |
| 対象 | 日本に生息・生育する野生生物 |
| 更新 | 概ね5年ごとに見直し |
| 最新版 | 第5次レッドリスト（植物・菌類）2024年 |

### カテゴリーの定義

| カテゴリー | 略号 | 意味 |
|------------|------|------|
| **絶滅** | EX (Extinct) | 日本では既に絶滅 |
| **野生絶滅** | EW (Extinct in the Wild) | 飼育・栽培下のみで存続 |
| **絶滅危惧IA類** | CR (Critically Endangered) | **絶滅の危機に瀕している** |
| **絶滅危惧IB類** | EN (Endangered) | 絶滅危機が増大している |
| **絶滅危惧II類** | VU (Vulnerable) | 絶滅の危険が増大 |
| **準絶滅危惧** | NT (Near Threatened) | 将来的に絶滅危惧になりうる |
| **情報不足** | DD (Data Deficient) | 評価に必要な情報が不足 |
| **絶滅のおそれのある地域個体群** | LP | 地域的に孤立した個体群 |

### カテゴリーの関係

```
    絶滅危惧種
       │
┌──────┴──────┐
│             │
絶滅危惧I類    絶滅危惧II類
│             (VU)
├── IA類(CR)
│
└── IB類(EN)
```

**絶滅危惧I類** = IA類(CR) + IB類(EN)

## IUCNレッドリスト

### 概要

| 項目 | 内容 |
|------|------|
| 作成者 | IUCN（国際自然保護連合） |
| 対象 | 世界中の野生生物 |
| 掲載種数 | 約46,000種以上（2024年） |

### カテゴリー

環境省レッドリストとほぼ同じカテゴリーを使用。

---

## 絶滅危惧種の減少要因

### 主な要因

| 要因 | 具体例 | 影響を受ける植物 |
|------|--------|------------------|
| **生息地の破壊** | 湿地の埋め立て、森林伐採 | 湿地性植物、森林植物 |
| **採取圧** | 園芸目的の採取 | ラン科植物 |
| **外来種** | 競合、捕食 | 小笠原固有種 |
| **環境汚染** | 水質汚濁、大気汚染 | 水生植物 |
| **気候変動** | 生息適地の変化 | 高山植物 |

### 特に影響を受けやすい植物

1. **湿地性植物**
   - 湿地の減少が著しい
   - サギソウ、トキソウ、カキランなど

2. **ラン科植物**
   - 園芸目的の採取圧
   - 特定の菌根菌に依存

3. **高山植物**
   - 分布が限定的
   - 気候変動の影響

4. **島嶼固有種**
   - 小笠原、南西諸島など
   - 外来種の影響大

## 保全の取り組み

### 法的保護

| 制度 | 内容 |
|------|------|
| 種の保存法 | 国内希少野生動植物種の指定 |
| 自然公園法 | 特別保護地区での採取規制 |
| 文化財保護法 | 天然記念物の指定 |

### 保全活動

| 活動 | 内容 |
|------|------|
| 生息地保全 | 保護区の設定、環境管理 |
| 人工繁殖 | 種子バンク、組織培養 |
| 普及啓発 | 希少種の周知、盗掘防止 |
| モニタリング | 個体数の継続調査 |

## 確認問題

### 問1
環境省レッドリストで「絶滅の危機に瀕している種」を表すカテゴリーは何か。

<details>
<summary>解答を見る</summary>

**絶滅危惧IA類（CR: Critically Endangered）**

絶滅危惧I類の中でも、より危機的な状況にある種。

</details>

### 問2
レッドリストのカテゴリーで、EXは何を意味するか。

<details>
<summary>解答を見る</summary>

**絶滅（Extinct）**

日本では既に絶滅した種。

</details>

### 問3
絶滅危惧種が多い植物の科を1つ挙げよ。

<details>
<summary>解答を見る</summary>

**ラン科**

園芸目的の採取圧が高く、多くの種が絶滅危惧種に指定されている。

</details>

### 問4
湿地の減少により絶滅が危惧されているラン科の植物を2つ挙げよ。

<details>
<summary>解答を見る</summary>

**サギソウ、トキソウ、カキラン**などから2つ

いずれも湿地性のラン科植物。

</details>

### 問5
IUCNとは何か。

<details>
<summary>解答を見る</summary>

**国際自然保護連合（International Union for Conservation of Nature）**

世界的なレッドリストを作成している国際機関。

</details>
