# 6-2. 外来生物法と特定外来生物

[← 目次に戻る](../NEW_00_目次.md) | [← 前：種の保存法](./6-1_種の保存法.md)

---

## はじめに

外来生物法と特定外来生物に関する問題は共通問題で頻出です。特に**特定外来生物に指定されている植物**を具体的に覚えておく必要があります。

---

## 外来生物法の概要

### 正式名称と基本情報

| 項目 | 内容 |
|------|------|
| 正式名称 | 特定外来生物による生態系等に係る被害の防止に関する法律 |
| 制定 | 2004年（平成16年） |
| 施行 | 2005年（平成17年） |
| 所管 | 環境省 |
| 目的 | 特定外来生物による生態系等への被害防止 |

### 規制の対象となる行為

**特定外来生物について以下の行為が原則禁止：**

1. **飼養・栽培・保管・運搬**
2. **輸入**
3. **野外への放出・植栽・播種**
4. **譲渡し・引渡し**

---

## 外来生物のカテゴリー

### 特定外来生物

| 項目 | 内容 |
|------|------|
| 定義 | 生態系・農林水産業・人の生命身体に被害を及ぼす（おそれがある）外来生物 |
| 規制 | 飼育・栽培・保管・運搬・輸入・放出が原則禁止 |
| 指定数 | 約160種類（2023年時点） |

### 未判定外来生物

- 特定外来生物と類似の外来生物
- 輸入にあたっては届出が必要

### 生態系被害防止外来種リスト

- 環境省作成の注意喚起リスト
- 法的規制はないが、注意が必要な種

| 植物名 | 学名 | 原産地 | 特徴・被害 |
|--------|------|--------|------------|
| **オオキンケイギク** | *Coreopsis lanceolata* | 北アメリカ | **黄色い花、道路法面に繁殖** |
| **オオハンゴンソウ** | *Rudbeckia laciniata* | 北アメリカ | 高さ3mにも達する大型種 |
| **ミズヒマワリ** | *Gymnocoronis spilanthoides* | 中南米 | 水辺に繁殖、水路閉塞 |

- 道路脇でよく見かける黄色い花

### ウリ科

**アレチウリの特徴：**
- 1年生つる性植物
- 河川敷や空き地で大繁殖

## 外来生物の侵入経路

### 意図的導入

| 目的 | 例 |
|------|-----|
| 観賞用 | オオキンケイギク、ボタンウキクサ |
| 緑化用 | オオキンケイギク |
| 農業用 | − |
| 水質浄化 | オオフサモ |

### 非意図的導入

| 経路 | 例 |
|------|-----|
| 輸入穀物への混入 | 各種外来雑草 |
| バラスト水 | 水生植物 |
| 園芸植物への付着 | − |

## 防除と駆除

### 防除の原則

1. **侵入防止**：水際での阻止
2. **早期発見・早期駆除**：定着前に対処
3. **分布拡大防止**：拡散を抑制
4. **駆除**：定着した個体群の除去

### 駆除上の注意

- 特定外来生物の運搬も禁止
- 駆除した植物も適切に処理が必要
- 根からの再生に注意（ナガエツルノゲイトウ等）

---

## 罰則

### 違反時の罰則

| 違反内容 | 罰則 |
|----------|------|
| 飼養・栽培・保管・運搬の禁止違反 | 3年以下の懲役または300万円以下の罰金 |
| 輸入の禁止違反 | 3年以下の懲役または300万円以下の罰金 |
| 野外への放出の禁止違反 | 3年以下の懲役または300万円以下の罰金 |
| 法人の違反 | 1億円以下の罰金 |

## 確認問題

### 問1
次のうち、特定外来生物に指定されている植物はどれか。
1. セイタカアワダチソウ
2. オオキンケイギク
3. ハルジオン
4. シロツメクサ

<details>
<summary>解答を見る</summary>

**2. オオキンケイギク**

黄色い花が美しいが、繁殖力が強く生態系に影響を与えるため特定外来生物に指定。
セイタカアワダチソウは要注意外来生物だが、特定外来生物では**ない**。

</details>

### 問2
特定外来生物について禁止されている行為を3つ挙げよ。

<details>
<summary>解答を見る</summary>

**飼育・栽培・保管・運搬・輸入・野外への放出**のうち3つ

譲渡し・引渡しも禁止されている。

</details>

### 問3
河川敷で大繁殖し、つる性で在来植物を覆いつくす特定外来生物は何か。

<details>
<summary>解答を見る</summary>

**アレチウリ**

ウリ科の1年生つる性植物。北アメリカ原産。

</details>

### 問4
外来生物法の正式名称は何か。

<details>
<summary>解答を見る</summary>

**特定外来生物による生態系等に係る被害の防止に関する法律**

2004年制定、2005年施行。

</details>

### 問5
水面を覆い尽くす浮葉性の特定外来生物で、サトイモ科の植物は何か。

<details>
<summary>解答を見る</summary>

**ボタンウキクサ**

南アメリカ原産。ウォーターレタスとも呼ばれる。

</details>
