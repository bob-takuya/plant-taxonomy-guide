# 6-1. 種の保存法

## はじめに

共通問題では、**野生生物の保全に関する法規**が頻出です。特に「種の保存法」に関する問題が多く出題されます。

---

## 種の保存法の概要

### 正式名称と基本情報

| 項目 | 内容 |
|------|------|
| 正式名称 | 絶滅のおそれのある野生動植物の種の保存に関する法律 |
| 制定 | 1992年（平成4年） |
| 施行 | 1993年（平成5年） |
| 所管 | 環境省 |
| 目的 | 絶滅のおそれのある野生動植物の種の保存 |

### 法の構成

1. **国内希少野生動植物種の保護**
2. **国際希少野生動植物種の規制**
3. **生息地等保護区の指定**
4. **保護増殖事業の実施**

---

## 指定カテゴリー

### 国内希少野生動植物種

**最重要カテゴリー。頻出！**

| 項目 | 内容 |
|------|------|
| 対象 | 日本国内に生息・生育する絶滅のおそれのある野生動植物 |
| 指定数 | 約430種（2023年時点） |
| 規制内容 | **捕獲・採取・殺傷・損傷が原則禁止** |

**規制の内容：**
- 捕獲、採取、殺傷、損傷の禁止
- 譲渡し、譲受け、引渡し、引取りの禁止
- 輸出入の規制

### 特定国内希少野生動植物種

| 項目 | 内容 |
|------|------|
| 対象 | 国内希少野生動植物種のうち、商業的に繁殖されたもの |
| 規制内容 | 販売・頒布目的の陳列・広告の規制 |
| 例外 | 登録された個体は譲渡等が可能 |

### 国際希少野生動植物種

| 項目 | 内容 |
|------|------|
| 対象 | ワシントン条約（CITES）附属書I掲載種など |
| 規制内容 | 国際取引の規制、国内での譲渡規制 |
| 指定数 | 約800種 |

## 生息地等保護区

### 制度の概要

種の保存法に基づき、国内希少野生動植物種の生息・生育地を保護区として指定できます。

| 区分 | 内容 |
|------|------|
| 管理地区 | 開発行為が原則禁止 |
| 監視地区 | 開発行為に届出が必要 |

### 指定されている保護区の例

| 保護区名 | 対象種 | 所在地 |
|----------|--------|--------|
| レブンアツモリソウ保護区 | レブンアツモリソウ | 北海道礼文島 |
| キタダケソウ保護区 | キタダケソウ | 山梨県北岳 |

---

## 保護増殖事業

### 概要

国内希少野生動植物種の個体の繁殖の促進、生息地・生育地の整備その他の保護増殖のための事業。

### 植物の保護増殖事業の例

| 対象種 | 事業内容 |
|--------|----------|
| レブンアツモリソウ | 人工繁殖、生息地保全 |
| アツモリソウ | 人工繁殖、盗掘防止 |
| キタダケソウ | モニタリング、生息地保全 |

---

## 罰則

### 違反時の罰則

| 違反内容 | 罰則 |
|----------|------|
| 捕獲・採取・殺傷・損傷 | 5年以下の懲役または500万円以下の罰金 |
| 譲渡し等の禁止違反 | 5年以下の懲役または500万円以下の罰金 |
| 法人の違反 | 1億円以下の罰金 |

## 確認問題

### 問1
日本に生息・生育する絶滅のおそれのある野生動植物を保護する法律は何か。

<details>
<summary>解答を見る</summary>

**種の保存法（絶滅のおそれのある野生動植物の種の保存に関する法律）**

1992年制定、1993年施行。

</details>

### 問2
種の保存法で保護される種のうち、日本国内に生息・生育するものを何というか。

<details>
<summary>解答を見る</summary>

**国内希少野生動植物種**

捕獲・採取・殺傷・損傷が原則禁止。

</details>

### 問3
国内希少野生動植物種に指定されている植物で、北海道礼文島に固有のラン科植物は何か。

<details>
<summary>解答を見る</summary>

**レブンアツモリソウ**

黄色い花を咲かせるアツモリソウの変種。

</details>

### 問4
種の保存法で、国内希少野生動植物種に最も多く指定されている植物の科は何か。

<details>
<summary>解答を見る</summary>

**ラン科**

園芸目的の採取圧が高く、多くの種が指定されている。

</details>
