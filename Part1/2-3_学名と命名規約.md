# 2-3. 学名と国際植物命名規約

[← 目次に戻る](../NEW_00_目次.md) | [← 前：APG分類体系](./2-2_APG分類体系の理解.md)

## はじめに

生物分類技能検定では、**学名のつけ方**や**命名規約のルール**が頻出です。また、植物部門では**科名をラテン語で答える問題**が多数出題されるため、学名の基礎知識は必須です。

---

## 二名法（binomial nomenclature）

### 概要

| 項目 | 内容 |
|------|------|
| 確立者 | **カール・フォン・リンネ（1753年）** |
| 構成 | **属名 + 種小名** |
| 言語 | ラテン語（またはラテン語化した語） |

### 学名の書き方

```
Prunus serrulata Lindl.
  ↑       ↑        ↑
 属名   種小名   命名者
```

#### 書式ルール

1. **属名**：大文字で始める、イタリック体
2. **種小名**：小文字、イタリック体
3. **命名者**：通常体（イタリックにしない）

**例：**
- *Ginkgo biloba* L. （イチョウ）
- *Oryza sativa* L. （イネ）
- *Cryptomeria japonica* (L.f.) D.Don （スギ）

### 命名者の略記

| 略記 | 正式名 | 備考 |
|------|--------|------|
| **L.** | Linnaeus（リンネ） | 最も多く使用される |
| **Thunb.** | Thunberg | 日本の植物を多数記載 |
| **Siebold** | Siebold | 日本の植物を記載 |
| **Makino** | 牧野富太郎 | 日本の植物学の父 |
| **Nakai** | 中井猛之進 | 朝鮮・満州の植物 |

## 国際藻類・菌類・植物命名規約（ICN）

### 概要

| 項目 | 内容 |
|------|------|
| 旧称 | 国際植物命名規約（ICBN） |
| 現行版 | 深セン規約（2018年） |
| 適用対象 | 藻類、菌類、植物 |

### 主要な原則

#### 1. 先取権の原則（principle of priority）

**最も重要な原則！**

- 同一の分類群に複数の名前がある場合、**最も古い正当に発表された名前が有効**
- 植物の学名の出発点：リンネ『植物の種』(1753年)

**例：**
ある植物に2つの名前が付けられた場合
- 1800年に発表された名前A
- 1850年に発表された名前B
→ 名前Aが有効（先に発表されたため）

#### 2. 模式法（typification）

- 全ての学名は**タイプ標本**に基づく
- 科 → 属 → 種の階層でタイプが指定される

#### 3. ラテン語記載の原則

- 2012年以前：ラテン語での記載が必須
- 2012年以降：ラテン語または英語での記載が可能

## 種下分類群

### 階級と表記

| 階級 | 略記 | 意味 | 例 |
|------|------|------|-----|
| **亜種** | subsp. / ssp. | 地理的隔離による分化 | *Rosa rugosa* subsp. *rugosa* |
| **変種** | var. | 形態的差異のある集団 | *Camellia japonica* var. *spontanea* |
| **品種** | f. | 個体レベルの変異 | *Acer palmatum* f. *atropurpureum* |

### 亜種・変種・品種の違い

| 階級 | 特徴 | 遺伝的差異 |
|------|------|------------|
| **亜種** | 地理的に隔離された集団 | 大きい |
| **変種** | 同所的だが形態差がある | 中程度 |
| **品種** | 散発的に出現する変異 | 小さい |

### 植物の分類階級と標準語尾

| 階級 | 語尾 | 例 |
|------|------|-----|
| 門 | **-phyta** | Magnoliophyta（被子植物門） |
| 亜門 | -phytina | - |
| 綱 | **-opsida**（または -atae） | Magnoliopsida（モクレン綱） |
| 亜綱 | -idae | Rosidae（バラ亜綱） |
| 目 | **-ales** | Rosales（バラ目） |
| 科 | **-aceae** | Rosaceae（バラ科） |
| 亜科 | -oideae | Rosoideae（バラ亜科） |
| 連 | -eae | Roseae（バラ連） |
| 属 | （規定なし） | Rosa（バラ属） |
| 種 | （規定なし） | Rosa rugosa（ハマナス） |

| アブラナ科 | Brassicaceae | Cruciferae |
| マメ科 | Fabaceae | Leguminosae |

| シソ科 | Lamiaceae | Labiatae |

**注意：** 試験では正式名（-aceae語尾）で解答することが多い

## シノニム（synonym：異名）

### 定義

- 同一の分類群に対して付けられた複数の学名
- 先取権の原則により、**最も古い名前が正名（accepted name）**
- それ以外はシノニム

### 種類

| 種類 | 説明 |
|------|------|
| **同物異名** | 同一の分類群に別の名前がつけられた |
| **異物同名** | 異なる分類群に同じ名前がつけられた（後のものは無効） |

---

## バシオニム（basionym：基礎異名）

### 定義

- 分類群が別の属や階級に移動された際の、元の名前
- 命名者名が括弧に入れられる

### 例

```
スギの学名の変遷：

1. 最初の命名
   Cupressus japonica L.f. (1781)
   ↓
2. 別属に移動
   Cryptomeria japonica (L.f.) D.Don (1839)
```

- バシオニム：*Cupressus japonica* L.f.
- 現在の学名：*Cryptomeria japonica* (L.f.) D.Don
- (L.f.) は元の命名者、D.Don は組み替えを行った人

## 確認問題

### 問1
二名法を確立した人物と、その出版年を答えよ。

<details>
<summary>解答を見る</summary>

**リンネ（Linnaeus）、1753年**

『植物の種』（Species Plantarum）で二名法を体系化した。

</details>

### 問2
同一の分類群に複数の学名がある場合、どの名前が有効となるか。

<details>
<summary>解答を見る</summary>

**最も古く正当に発表された名前**

これを先取権の原則（principle of priority）という。

</details>

### 問3
次の学名の表記で、正しいものはどれか。
1. *prunus Serrulata* Lindl.
2. *Prunus serrulata* Lindl.
3. *PRUNUS SERRULATA* Lindl.
4. *Prunus Serrulata* lindl.

<details>
<summary>解答を見る</summary>

**2. *Prunus serrulata* Lindl.**

- 属名は大文字で始める
- 種小名は小文字
- 命名者は通常体（非イタリック）

</details>

### 問4
科の学名に用いられる語尾は何か。

<details>
<summary>解答を見る</summary>

**-aceae**

例：Rosaceae（バラ科）、Asteraceae（キク科）

</details>

### 問5
*Cryptomeria japonica* (L.f.) D.Don において、(L.f.) が括弧に入っている理由は何か。

<details>
<summary>解答を見る</summary>

**バシオニム（基礎異名）の命名者を示すため**

L.f.（リンネの息子）が最初に別の属（Cupressus）で命名し、D.Donがその後Cryptomeria属に移動させた。

</details>
