## 試験基本情報

### 実施概要（2025年度）

| 項目 | 内容 |
|------|------|
| **試験日** | 10月18日（土）14:00〜16:00 |
| **試験時間** | 120分 |
| **試験形式** | CBT方式（全国のテストセンターで受験） |
| **出題形式** | 4択選択式が中心 |
| **満点** | 100点 |
| **合格基準** | **70点以上**（年度により変動あり、65点程度で合格の場合も） |
| **受験料** | 7,700円（税込） |
| **受付期間** | 7月1日〜8月29日 |
| **合格率** | 約10%（非常に難関） |

### 受験資格
特に制限なし。誰でも受験可能。

### 資格の意義
- 環境省「一般競争（指名競争）参加資格申請」の有資格者として認定
- 農水省、国交省、地方自治体等の入札資格として採用
- 環境アセスメント調査・生物調査の専門家としての証明

## 過去問分析に基づく出題傾向

### 【最重要】毎年必ず出題される分野

#### 1. APG分類体系に関する問題
- APG III/IVで最も原始的な科（アンボレラ科）
- 特定の科に最も近縁な科
- ユリ科の分割・再編成
- エングラー体系との違い

#### 2. 科の形態的特徴
- 葉序（対生・互生・輪生）
- 花弁・萼片の数
- 雄しべ・雌しべの構造
- 果実の型

#### 3. 学名（ラテン語）での解答
- 科名をラテン語で選択
- 属名からの科の特定
- 学名の表記ルール

#### 4. 分類学史・歴史的偉人
- リンネの二名法
- ダーウィンの進化論
- 木村資生の中立進化説
- 牧野富太郎の功績

#### 5. 生物地理学
- ブラキストン線、渡瀬線などの分布境界線
- 日本の植物区系
- 植生帯

### 【重要】高頻度で出題される分野

#### 6. 関連法規
- 種の保存法（国内希少野生動植物種）
- 特定外来生物法
- 天然記念物
- レッドリストカテゴリー

#### 7. 写真同定問題
- 植物の写真から種名または科名を同定
- 葉、花、果実の写真からの識別
- カラー口絵として出題

#### 8. シダ植物・コケ植物
- 胞子体と配偶体の違い
- 維管束の有無
- 仮道管を持つ植物
- 世代交代

#### 9. 外来種・絶滅危惧種
- 特定外来生物の具体例
- 侵略的外来種
- レッドリストカテゴリー
- 代表的な絶滅危惧植物

#### 10. 種から科の同定
- 種名から所属する科を答える問題
- 例：「カワヂシャ」→ オオバコ科

## 過去問の具体例

### 例題1：形態と科の同定

> **問題**：葉は対生、花柱は2本または3本の形質を持つ日本の野生植物が属する科をひとつ選びなさい。
>
> 1. Ericaceae
> 2. Hydrangeaceae
> 3. Pinaceae
> 4. Rubiaceae

**正解：2. Hydrangeaceae（アジサイ科）**

**解説：**
- Ericaceae（ツツジ科）：葉は互生が多い
- **Hydrangeaceae（アジサイ科）：葉は対生、花柱は2〜3本**
- Pinaceae（マツ科）：裸子植物、針葉樹
- Rubiaceae（アカネ科）：葉は対生だが、花柱は通常1本

### 例題2：APG分類体系

> **問題**：APG III体系において、被子植物で最も原始的（基部に位置する）とされる科を選びなさい。
>
> 1. Magnoliaceae
> 2. Amborellaceae
> 3. Nymphaeaceae
> 4. Ranunculaceae

**正解：2. Amborellaceae（アンボレラ科）**

**解説：**
- **Amborellaceae（アンボレラ科）：APG体系で被子植物の最基部に位置する単型科**
- Nymphaeaceae（スイレン科）：アンボレラの次に基部に位置
- Magnoliaceae（モクレン科）：モクレン類（Magnoliids）に属する
- Ranunculaceae（キンポウゲ科）：真正双子葉類

### 例題3：分類学者と功績

> **問題**：中立進化説を提唱した日本人科学者は誰か。
>
> 1. 牧野富太郎
> 2. 今西錦司
> 3. 木村資生
> 4. 南方熊楠

**正解：3. 木村資生**

## 得点戦略

### 目標配点

| 分野 | 目標得点 | 戦略 |
|------|----------|------|
| 共通問題 | 15/20点 | 法規・偉人問題は確実に得点 |
| 維管束植物 | 45/55点 | 主要科の特徴を完璧に |
| 写真同定 | 7/10点 | フィールドワークで補強 |
| シダ・コケ他 | 8/15点 | 基本事項のみ押さえる |
| **合計** | **75点** | 余裕を持って合格 |

### 絶対に落とせない分野

- APG分類体系の基本
- 主要科のラテン語名と形態的特徴
- 分類学者の功績
- 法規関連（特定外来生物、種の保存法）
- 分布境界線

### 優先度を下げてよい分野

- 藻類の詳細（主要なもののみ押さえる）
- 菌類の詳細な分類
- コケ植物の細かい同定
