# 5-2. 植生と遷移

[← 目次に戻る](../NEW_00_目次.md) | [← 前：植物生態学](./5-1_植物生態学.md)

---

## はじめに

植生遷移は、ある場所の植物群落が時間とともに変化していく過程です。一次遷移と二次遷移の違い、先駆種と極相種の特徴は試験頻出です。

---

## 遷移の基本概念

### 遷移とは

| 用語 | 定義 |
|------|------|
| **遷移（Succession）** | 植物群落が時間とともに変化する過程 |
| **極相（Climax）** | 遷移の最終段階、安定した群落 |
| **先駆種** | 遷移の初期に現れる種 |
| **極相種** | 極相林を構成する種 |

### 遷移の種類

| 種類 | 開始条件 | 特徴 |
|------|----------|------|
| **一次遷移** | 裸地（土壌なし） | 時間がかかる |
| **二次遷移** | 土壌がある場所 | 比較的速い |

---

## ==一次遷移（頻出）==

### 乾性遷移（陸上の一次遷移）

裸地から始まる遷移。

| 段階 | 特徴 | 代表種 |
|------|------|--------|
| **裸地** | 植生なし | - |
| **地衣類・コケ段階** | ==最初の定着者== | ウメノキゴケ、スナゴケ |
| **一年生草本段階** | 先駆草本 | メヒシバ、エノコログサ |
| **多年生草本段階** | 草原形成 | ススキ、ヨモギ |
| **低木段階** | 木本の侵入 | ヌルデ、アカメガシワ |
| **陽樹林段階** | ==先駆樹種の林== | アカマツ、シラカンバ |
| **陰樹林段階** | ==極相林== | シイ、ブナ |

### 遷移の模式図

```
時間 →
裸地 → 地衣類 → 草本 → 低木 → 陽樹林 → 陰樹林（極相）
      コケ類           ↑        ↑         ↑
                    ヌルデ   アカマツ   シイ・ブナ
                  アカメガシワ シラカンバ
```

### ==湿性遷移（水辺の一次遷移）==

池沼から始まる遷移。

| 段階 | 特徴 | 代表種 |
|------|------|--------|
| **沈水植物段階** | 水中に生育 | クロモ、マツモ |
| **浮葉植物段階** | 葉が水面に浮く | スイレン、ヒシ |
| **抽水植物段階** | 水から茎葉が出る | ヨシ、ガマ |
| **湿地草原段階** | 土壌の発達 | スゲ類、ミゾソバ |
| **湿地林段階** | ハンノキ林など | ハンノキ、ヤナギ類 |
| **乾性林段階** | 乾燥化進行 | 周辺環境の極相林 |

## 二次遷移

### 二次遷移の特徴

| 項目 | 内容 |
|------|------|
| 開始条件 | 伐採跡地、山火事跡地、耕作放棄地 |
| 速度 | ==一次遷移より速い==（土壌・種子バンクあり） |
| 経過 | 草本→低木→陽樹林→陰樹林 |

### 二次遷移の例

| 場所 | 遷移経過 |
|------|----------|
| 伐採跡地 | 萌芽更新 → コナラ林 → シイ林 |
| 山火事跡地 | 先駆草本 → アカマツ林 → 落葉樹林 |
| 耕作放棄地 | 一年草 → 多年草 → 低木 → 樹林 |

### 萌芽更新

| 用語 | 説明 |
|------|------|
| **萌芽更新** | 切り株から新しい芽が出て再生すること |
| 代表種 | コナラ、クヌギ、ヤマザクラ |
| 利用 | 里山林の維持管理（薪炭林） |

---

## ==先駆種と極相種（頻出）==

### 特性の比較

| 項目 | 先駆種（パイオニア種） | 極相種 |
|------|------------------------|--------|
| 光要求性 | ==陽樹==（強光を要求） | ==陰樹==（耐陰性あり） |
| 光補償点 | 高い | 低い |
| 成長速度 | 速い | 遅い |
| 寿命 | 短い（数十年） | 長い（数百年） |
| 種子生産 | 多量・小型 | 少量・大型 |
| 散布方法 | 風散布が多い | 動物散布も多い |

### ==代表的な先駆種==

| 種名 | 科名 | 分布・特徴 |
|------|------|------------|
| **アカマツ** | マツ科 | 乾燥地、山火事跡地 |
| **シラカンバ** | カバノキ科 | 冷温帯の先駆種 |
| **ヌルデ** | ウルシ科 | 林縁、伐採跡地 |
| **アカメガシワ** | トウダイグサ科 | 暖温帯の先駆種 |
| **カラスザンショウ** | ミカン科 | ギャップに侵入 |

### ==代表的な極相種==

| 植生帯 | 極相種 | 科名 |
|--------|--------|------|
| 暖温帯 | **スダジイ** | ブナ科 |
| 暖温帯 | **タブノキ** | クスノキ科 |
| 冷温帯 | **ブナ** | ブナ科 |
| 冷温帯 | **ミズナラ** | ブナ科 |
| 亜高山帯 | **シラビソ** | マツ科 |

## 日本の極相林

### ==植生帯ごとの極相林==

| 植生帯 | 極相林 | 極相種 | 分布 |
|--------|--------|--------|------|
| 亜熱帯 | 亜熱帯多雨林 | ガジュマル、アコウ | 沖縄 |
| 暖温帯 | ==照葉樹林== | スダジイ、タブノキ | 西南日本低地 |
| 冷温帯 | ==ブナ林== | ブナ、ミズナラ | 東北、中部山地 |
| 亜寒帯 | 針葉樹林 | トドマツ、エゾマツ | 北海道 |

### 土壌極相と気候極相

| 種類 | 説明 | 例 |
|------|------|-----|
| **気候極相** | 気候条件で決まる極相 | 各植生帯の極相林 |
| **土壌極相** | 土壌条件で決まる極相 | 蛇紋岩植生、石灰岩植生 |
| **地形極相** | 地形条件で決まる極相 | 崖地の植生 |

## ギャップ更新

### ギャップダイナミクス

| 用語 | 説明 |
|------|------|
| **ギャップ** | 林冠が開いた空間 |
| **ギャップ更新** | ギャップで次世代の樹木が成長すること |
| 形成原因 | 倒木、枯死、台風被害 |

### ギャップの役割

| 役割 | 説明 |
|------|------|
| 光環境の改善 | 林床に光が届く |
| 多様性の維持 | 先駆種も生育可能 |
| 森林の循環 | 極相林の維持機構 |

## 植生の攪乱

### 攪乱の種類

| 攪乱 | 規模 | 頻度 | 結果 |
|------|------|------|------|
| 台風 | 中〜大 | 年数回 | ギャップ形成 |
| 山火事 | 大 | 数十年〜 | 二次遷移開始 |
| 火山噴火 | 大 | 数百年〜 | 一次遷移開始 |
| 伐採 | 中 | 人為的 | 二次遷移開始 |

### 中規模攪乱仮説

| 項目 | 内容 |
|------|------|
| 提唱者 | Connell（1978） |
| 内容 | ==中程度の攪乱で種多様性が最大== |
| 理由 | 先駆種と極相種が共存 |

---

## 人為の影響と植生

### 里山林

| 項目 | 内容 |
|------|------|
| 形成 | 人為的な伐採・管理 |
| 代表種 | コナラ、クヌギ、アカマツ |
| 管理 | 15〜30年周期の伐採（萌芽更新） |
| 利用 | 薪炭材、シイタケ原木、落ち葉堆肥 |

### 放棄された里山の遷移

```
管理停止 → コナラ・クヌギ林の老齢化
        → 常緑樹（シイ・カシ）の侵入
        → 照葉樹林へ遷移
```

## 確認問題

### 問1
一次遷移で最初に定着する生物群は何か。

<details>
<summary>解答</summary>

**地衣類とコケ類**

岩石表面に最初に定着し、土壌形成に寄与。

</details>

### 問2
先駆種（パイオニア種）の光に対する特性を述べよ。

<details>
<summary>解答</summary>

**陽樹である（強い光を必要とする）**

光補償点が高く、日陰では生育できない。

</details>

### 問3
二次遷移が一次遷移より速い理由は何か。

<details>
<summary>解答</summary>

**土壌がすでに存在し、種子バンクや根系が残っているため**

</details>

### 問4
暖温帯の極相林と代表的な極相種を答えよ。

<details>
<summary>解答</summary>

- 極相林：**照葉樹林**
- 極相種：**スダジイ、タブノキ**

</details>

### 問5
中規模攪乱仮説の内容を説明せよ。

<details>
<summary>解答</summary>

**中程度の攪乱で種多様性が最大になる**

攪乱が強すぎると先駆種のみ、弱すぎると極相種のみになるが、中程度で両者が共存。

</details>
