# 3-3. 生殖器官（花・果実・種子）

## はじめに

花・果実・種子の構造は植物分類の最も重要な特徴です。**花弁の数、雄しべの数、子房の位置、果実の型**は科を特定する決め手となります。

# 第1部：花（Flower）

## 花の基本構造

### 花の各部名称

```
            柱頭（stigma）
              │
            花柱（style）
              │
        ┌─────┴─────┐
        │   子房    │（ovary）
        └─────┬─────┘
              │
    葯（anther）──┐
              │  ├── 雄しべ（stamen）
    花糸（filament）┘
              │
         花弁（petal）──┐
              │        ├── 花被（perianth）
         萼片（sepal）──┘
              │
         花托（receptacle）
              │
         花柄（pedicel）
```

### 花器官の機能

| 器官 | 機能 |
|------|------|
| **萼片** | 蕾の保護、光合成補助 |
| **花弁** | 送粉者の誘引 |
| **雄しべ** | 花粉の生産 |
| **雌しべ** | 受粉・受精・種子形成 |
| **花托** | 花器官の付着点 |

## 花被（Perianth）

### 花被の種類

| 種類 | 説明 | 例 |
|------|------|-----|
| **異花被花** | 萼と花弁が区別できる | バラ、サクラ |
| **同花被花** | 萼と花弁の区別がない | ユリ、チューリップ |
| **単花被花** | 花被が1輪のみ | ドクダミ |
| **無花被花** | 花被がない | ヤナギ、クルミ |

### ==離弁花と合弁花（頻出）==

| 種類 | 説明 | 代表的な科 |
|------|------|------------|
| ==**離弁花**== | 花弁が分離 | バラ科、アブラナ科、キンポウゲ科 |
| ==**合弁花**== | 花弁が合着 | キク科、シソ科、ナス科、ツツジ科 |

### 花冠の形

| 形 | 説明 | 代表例 |
|-----|------|--------|
| **漏斗形** | 漏斗状に広がる | アサガオ（ヒルガオ科） |
| **鐘形** | 鐘のような形 | ホタルブクロ（キキョウ科） |
| **壺形** | 壺のような形 | ドウダンツツジ（ツツジ科） |
| **高杯形** | 筒部と広がった部分 | サクラソウ（サクラソウ科） |
| **唇形** | 上下に分かれる | ==シソ科== |
| **舌状** | 舌のように広がる | キク科の舌状花 |
| **蝶形** | 蝶のような形 | ==マメ科== |
| **車形** | 車輪状に平開 | ナス科 |

## 花の対称性

### ==放射相称花と左右相称花（頻出）==

| 種類 | 説明 | 代表的な科 |
|------|------|------------|
| **放射相称花** | どの方向からも対称 | バラ科、キク科（頭花全体）、セリ科 |
| ==**左右相称花**== | 1つの面でのみ対称 | ==マメ科、シソ科、ラン科、ゴマノハグサ科== |

### ==マメ科の蝶形花（頻出）==

```
      旗弁（banner/standard）
           │
    ┌──────┴──────┐
    │            │
   翼弁          翼弁
  (wing)        (wing)
    │            │
    └──────┬──────┘
           │
      竜骨弁（keel）
    （2枚が合着）
```

### シソ科の唇形花

```
     上唇（2裂）
        │
    ────┴────

    ────┬────
        │
     下唇（3裂）
```

## 雄しべ（Stamen）

### ==雄しべの数と配列（頻出）==

| 科名 | 雄しべの数 | 特徴 |
|------|------------|------|
| ==バラ科== | ==多数== | 花托に多数つく |
| ==アブラナ科== | ==6本（4長2短）== | ==四強雄蕊== |
| ==マメ科== | ==10本== | 単体または2体雄蕊 |
| ==シソ科== | ==4本（2長2短）== | ==二強雄蕊== |
| セリ科 | 5本 | 花弁と同数 |
| ==キク科== | ==5本== | ==集約雄蕊==（葯が合着） |
| ユリ科 | 6本 | 花被片と同数 |

### 雄しべの特殊な配列

| 用語 | 説明 | 例 |
|------|------|-----|
| **四強雄蕊** | 4本が長く2本が短い | ==アブラナ科== |
| **二強雄蕊** | 2本が長く2本が短い | ==シソ科== |
| **単体雄蕊** | 全ての花糸が合着 | マメ科（一部） |
| **2体雄蕊** | 9本が合着し1本が離れる | マメ科（多くの種） |
| **集約雄蕊** | 葯が合着して筒状 | ==キク科== |

## 雌しべ（Pistil）と子房

### ==子房の位置（頻出）==

| 位置 | 説明 | 代表的な科 |
|------|------|------------|
| **子房上位** | 子房が花托の上 | バラ科（一部）、マメ科、アブラナ科 |
| **子房下位** | 子房が花托の下 | ==キク科、セリ科、ウリ科== |
| **子房半下位** | 中間 | バラ科（リンゴ亜科）、ユキノシタ科 |

```
子房上位           子房下位           子房半下位

  雌しべ             花被              花被
    │                 │                │
 ┌──┴──┐          ────┼────        ──┼──
 │子房 │              │              │子房
 └──┬──┘           ┌──┴──┐          └┬┘
    │              │子房 │           │
  花托             └─────┘          花托
```

### 心皮（Carpel）

| 用語 | 説明 |
|------|------|
| **心皮** | 雌しべを構成する葉的器官 |
| **合生心皮** | 複数の心皮が合着 |
| **離生心皮** | 心皮が分離（キンポウゲ科など） |

### 胎座（Placentation）

| 種類 | 説明 | 例 |
|------|------|-----|
| **中軸胎座** | 子房室の中心軸に胚珠 | ユリ科 |
| **側膜胎座** | 子房壁（隔壁上）に胚珠 | スミレ科、ウリ科、キク科 |
| **底生胎座** | 子房の底に1つの胚珠 | キク科・タデ科 |
| **独立中央胎座** | 隔壁なく中央柱に胚珠 | サクラソウ科、ナデシコ科 |
| **縁辺胎座** | 1心皮の腹縫合線（へりの部分）に胚珠が並ぶ | **マメ科（豆果）** |

!!! warning "縁辺胎座（2024年度問40で出題）"
    **縁辺胎座（marginal placentation）**：1枚の心皮の腹縫合線（ventralsuture）上に胚珠が2列に並ぶ。
    1心皮性の雌蕊（袋果・豆果）に特有の胎座タイプ。
    「縁辺胎座をもつ種」→ **クズ（マメ科）**（2024年度問40 正解2）
    マメ科の豆果は縁辺胎座の典型。

# 第2部：花序（Inflorescence）

## 花序の種類

### 主要な花序

| 花序 | 説明 | 代表例 |
|------|------|--------|
| **単頂花序** | 茎の先に1つの花 | チューリップ |
| **総状花序** | 軸に沿って有柄の花 | フジ、ナノハナ |
| **穂状花序** | 軸に沿って無柄の花 | オオバコ |
| ==**散形花序**== | 1点から放射状 | ==セリ科== |
| ==**複散形花序**== | 散形花序が集まる | ==セリ科== |
| ==**頭状花序**== | 多数の花が頭状に密集 | ==キク科== |
| **円錐花序** | 総状花序が分枝 | イネ科 |
| **集散花序** | 頂端から順に開花 | ナデシコ科 |
| **尾状花序** | 垂れ下がる穂状 | ヤナギ、クルミ |
| **肉穂花序** | 肉質の軸に密生 | サトイモ科 |

### ==キク科の頭状花序（頻出）==

```
頭状花序の構造

    舌状花（周辺）
      ↓
    ○○○○○
   ○●●●●●○   ← 管状花（中心）
   ○●●●●●○
    ○○○○○
      ↑
    総苞片
```

| 花の種類 | 特徴 | 例 |
|----------|------|-----|
| **舌状花** | 舌のように広がる | タンポポ |
| **管状花** | 筒状 | アザミ |
| **両方あり** | 周辺が舌状、中心が管状 | ヒマワリ |

---

## ==花序と代表的な属の対応表（試験頻出）==

各花序型をもつ代表的な属・科のリスト。

| 花序型 | 定義 | 代表的な属/科 |
|--------|------|---------------|
| ==**総状花序**== | 軸に沿って有柄の花が並ぶ | ==フジ属（マメ科）、アブラナ属（アブラナ科）== |
| ==**穂状花序**== | 軸に沿って無柄の花が並ぶ | ==オオバコ属、ミズヒキ属（タデ科）== |
| ==**散形花序**== | 1点から放射状に花柄が出る | ==サクラソウ属、サクラ属の一部== |
| ==**複散形花序**== | 散形花序が2段階 | ==セリ科全般（セリ、ニンジン、ミツバ）== |
| ==**頭状花序**== | 花が密集して頭状 | ==キク科全般（タンポポ、ヒマワリ、アザミ）== |
| **円錐花序** | 総状花序が分枝 | ==イネ科（イネ、ススキ）、タデ科（イタドリ）== |
| **集散花序** | 頂端から順に開花 | ==ナデシコ科（ナデシコ、ハコベ）、ユキノシタ科== |
| **尾状花序** | 垂れ下がる穂状 | ==ヤナギ属、ハンノキ属、クルミ属== |
| **肉穂花序** | 肉質の軸に密生 | ==サトイモ科（ミズバショウ、マムシグサ）== |

### 有限花序と無限花序（試験頻出）

| 種類 | 説明 | 代表例 |
|------|------|-------|
| **無限花序（求心花序）** | 下（外側）から順に開花、頂端の花が最後 | 総状、穂状、散形、頭状、肉穂、散房花序 |
| **有限花序（遠心花序）** | 頂端（内側）から順に開花 | **集散花序**（二出集散、さそり形、かたつむり形など） |

> 📝 **有限花序の例**：二出集散花序（ナデシコ科）、さそり形花序（ワスレナグサ属など）、かたつむり形花序（ワスレグサ属など）

#### 特殊な花序タイプ

| 花序タイプ | 定義 | 代表属 |
|----------|------|-------|
| **かたつむり形花序** | 集散花序の一種、渦巻き状に巻く | **ワスレグサ属**（ヘメロカリス属）|
| **さそり形花序** | 集散花序の一種、一方向に曲がる | **ワスレナグサ属**、キュウリグサ属 |
| **肉穂花序** | 肉質の軸に無柄の花が密生 | **ショウブ属**、サトイモ科全般 |
| **穂状花序** | 軸に無柄の花が並ぶ | **オカトラノオ属** |
| **杯状花序** | 3〜5個の雄花と1個の雌花が杯状の総苞に包まれる | **トウダイグサ属**（Euphorbia） |

> ⚠️ **試験頻出混同**：  
> - 「かたつむり形花序 - ワスレグサ属」✓ 正しい  
> - 「さそり形花序 - ワスレナグサ属」✓ 正しい  
> - 「肉穂花序 - ショウブ属」✓ 正しい  
> - 「穂状花序 - オカトラノオ属」は**誤り**。オカトラノオは**総状花序**。

#### 散房花序・散形花序・円錐花序の違い

| 花序 | 定義 | 代表例 |
|------|------|-------|
| **散形花序** | 1点から放射状に花柄が出る（等長） | セリ科、ネギ属、サクラ（多くの種） |
| **散房花序** | 総状花序の変形、下の花柄が長く上面が平ら | コデマリ、ノイバラ |
| **円錐花序** | 総状花序が分枝 | ノリウツギ、ススキ |
| **複散形花序** | 散形花序がさらに散形状に集まる | セリ科全般 |

> 📝 **2022年度試験問39より**：
> - 総状花序：オカトラノオ ✓
> - 円錐花序：ノリウツギ ✓  
> - 散房花序：コデマリ ✓
> - 散形花序：トチバニンジン ✓
> - 頭状花序：マツムシソウ ✓
> - 巻散（鎌状集散）花序：キュウリグサ ✓

#### キク科の頭状花序（有限 vs 無限）

| 植物 | 花序の種類 | 補足 |
|------|----------|------|
| **ノアザミ** | 無限花序（頭状花序は無限） | 下から外周から開花 |
| **ヤマボウシ** | **有限花序** | 頭花（頭状に集まった花序）は有限 |

> ⚠️ **試験頻出**：「ノアザミは無限花序、ヤマボウシは有限花序」（2023年度試験問29）

### 花序型の詳細と識別ポイント

#### ==セリ科の複散形花序（超頻出）==

```
複散形花序の構造

      ○ ○ ○   第2段階の散形花序
       \|/
        ○─────┐
      ○ ○ ○     │
       \|/       │  第1段階の散形花序
        ○────┤
      ○ ○ ○     │
       \|/       │
        ○────┘
         │
        茎
```

**セリ科の特徴**:
- 複散形花序が最大の特徴
- 小散形花序の基部に小総苞
- 果実は2分果（2個に分かれる）

#### マメ科の総状花序

| 属名 | 花序の特徴 |
|------|------------|
| ==フジ属== | ==長い垂れ下がる総状花序== |
| ==ハギ属== | 総状花序、腋生 |
| ==ハナズオウ== | 幹生花（幹から直接花が出る） |

#### キク科の頭状花序の多様性

| タイプ | 花の構成 | 代表属 |
|--------|----------|--------|
| ==舌状花のみ== | 全て舌状花 | ==タンポポ属、ニガナ属== |
| ==管状花のみ== | 全て管状花 | ==アザミ属、ヒヨドリバナ属== |
| ==舌状花+管状花== | 周辺が舌状、中心が管状 | ==ヒマワリ属、ハルジオン属== |

#### イネ科の花序

イネ科は特殊で、小穂（しょうすい）が基本単位。

| 属名 | 花序の型 | 特徴 |
|------|----------|------|
| ==イネ属== | 円錐花序 | 小穂が枝分かれした軸につく |
| ==オオムギ属== | 穂状花序 | 小穂が軸に直接つく |
| ==スズメノカタビラ== | 円錐花序 | 小型、分枝多い |

# 第3部：果実（Fruit）

## 果実の分類

### 果実の大分類

```
果実
├── 真果（子房のみが発達）
│   ├── 単果（1つの雌しべから）
│   │   ├── 乾果（乾いた果皮）
│   │   │   ├── 裂開果（熟すと開く）
│   │   │   └── 閉果（開かない）
│   │   └── 液果（多肉質の果皮）
│   │       ├── 漿果
│   │       └── 核果
│   └── 集合果（多数の雌しべから）
│
└── 偽果（子房以外も発達）
    └── 複合果（多数の花から）
```

## ==乾果の種類（頻出）==

### 裂開果

| 種類 | 説明 | 代表例 | 科 |
|------|------|--------|-----|
| ==**莢果**== | 1心皮、2裂開 | サヤエンドウ | ==マメ科== |
| ==**長角果**== | 2心皮、隔壁を残す | ダイコン、ナタネ | ==アブラナ科== |
| **短角果** | 短い長角果 | ナズナ | アブラナ科 |
| **蒴果** | 多心皮、様々な開き方 | アサガオ、ユリ | 多くの科 |
| **袋果** | 1心皮、1裂開 | シキミ、モクレン | モクレン科 |

### 閉果

| 種類 | 説明 | 代表例 | 科 |
|------|------|--------|-----|
| ==**痩果**== | 小型、1種子 | タンポポ、ヒマワリ | ==キク科==、キンポウゲ科 |
| ==**穎果**== | 果皮と種皮が癒着 | イネ、ムギ | ==イネ科== |
| **堅果** | 硬い果皮 | ドングリ、クリ | ブナ科 |
| **翼果** | 翼がある | カエデ、ニレ | ムクロジ科 |
| ==**分果**== | 複数に分かれる | セリ科（2分果）、シソ科（4分果） | ==セリ科、シソ科== |
| **節果** | 節で分かれるが各節は開かない | クサネム属（マメ科） | マメ科（節豆果） |
| **蓋果** | 蓋状に開く（蒴果の一種） | ハコベ、ヒユ科 | ナデシコ科、ヒユ科 |
| **瓠果（かこ）** | 液果の特殊型・子房下位 | キュウリ・スイカ・カボチャ | **ウリ科** |

!!! warning "ウリ科の果実（2021年度問39で出題）"
    ウリ科（Cucurbitaceae）の特徴：巻きひげ・単性花・**子房下位**・果実は**液果（瓠果）または蓋果**
    - **瓠果（pepo）**：子房下位の液果（花床が果皮に癒着）。キュウリ・スイカ・メロン等
    - **蓋果**：カラスウリ等。熟すと蓋が取れる
    
> ⚠️ **節果（せつか）は閉果**：豆果（莢果）が節間で分かれ、各節が開かない形。クサネム属（マメ科）が例。（2024年度問42で出題）
> ⚠️ **カヤツリグサ科の果実**：穎果（イネ科）**ではない**。カヤツリグサ科の果実は**小堅果（痩果様のナッツ）**。「穎果：カヤツリグサ属」は誤り。（2024年度問42で出題）

## 液果の種類

| 種類 | 特徴 | 例 |
|------|------|-----|
| **漿果** | 全体が多肉質 | トマト、ブドウ、カキ |
| **核果** | 内果皮が硬い | モモ、ウメ、サクランボ |

## 裂開果と心皮数の関係（試験頻出）

| 果実タイプ | 心皮数 | 裂開・閉果 | 代表例 |
|----------|-------|----------|-------|
| **袋果**（たいか） | **1心皮（一心皮）** | 裂開果（腹縫線から1裂） | バイカオウレン、シモツケ属、トリカブト属、エゾノリュウキンカ |
| **莢果**（さやか） | 1心皮 | 裂開果（背腹両縫線から2裂） | マメ科 |
| **蒴果**（さくか） | **合心皮雌蕊（複数心皮が合着）** | 裂開果 | ユリ、アジサイ |
| **痩果**（そうか） | 1〜合心皮 | **閉果** | タンポポ、キンポウゲ科 |
| **豆果** | 1心皮 | 裂開果 | マメ科（一般） |

> 📝 **重要識別点**：  
> - **袋果**は「**一心皮雌蕊**」から形成される（合心皮ではない）
> - **蒴果**は「**合心皮雌蕊**」から形成される裂開果
> - 「合心皮雌蕊による裂開果」→ **蒴果** ✓（袋果は一心皮なので不正解）

### バラ科の果実の多様性

| 亜科 | 果実タイプ | 具体例 |
|------|-----------|-------|
| バラ亜科 | **偽果**（花托が肥大）、痩果 | バラ（偽果＋痩果） |
| サクラ亜科 | **核果** | サクラ、モモ、ウメ |
| リンゴ亜科 | **梨状果**（偽果）、子房下位 | リンゴ、ナシ、オオウラジロノキ |
| キジムシロ亜科 | **集合痩果**（花托が肥大） | イチゴ |
| ナナカマド亜科 | 梨状果または **袋果** | シモツケ（袋果）、ナナカマド（梨状果） |

> ⚠️ **袋果をもつバラ科の属**：**シモツケ属**、ホザキナナカマド属など。  
> シモツケ属とナナカマド属を混同しないこと（ナナカマド属は梨状果）。

### 一心皮性の果実をもつ属（試験頻出）

袋果（一心皮裂開果）をもつ代表的な属：
- **バラ科**：シモツケ属、ホザキナナカマド属
- **キンポウゲ科**：トリカブト属、イチリンソウ属（痩果）、エゾノリュウキンカ（袋果）
- **モクレン科**：モクレン属（集合袋果）

> 📝 「バイカオウレン（キンポウゲ科）は一心皮雌蕊をもつ」  
> 📝 「エゾノリュウキンカ（キンポウゲ科）は袋果をつくる」

## 種子の構造

```
種子の構造

   ┌─ 種皮（seed coat）
```

### ==種髪（しゅはつ）と冠毛（かんもう）の区別（⚠️試験頻出混同）==

> **2021年度問32：「種子の附属物である種髪をもたない植物」**  
> 選択肢：アカメヤナギ・シロバナタンポポ・テイカカズラ・ヤナギラン  
> **正解：シロバナタンポポ**（タンポポの綿毛は「冠毛」であり「種髪」ではない）

| 用語 | 定義 | 付着位置 | 代表例 |
|------|------|---------|--------|
| **種髪（しゅはつ / coma）** | **種皮（種子そのもの）**から生えた毛状の附属体 | 種子に直接付着 | **ヤナギ属・ポプラ属**（ヤナギ科）、**ヤナギラン・フウロ属**（アカバナ科）、**テイカカズラ**（キョウチクトウ科） |
| **冠毛（かんもう / pappus）** | 萼（がく）が変化した毛状〜羽状の構造 | **果実（痩果等）**に付着（種子ではない） | **タンポポ属・アザミ属・ヤマボクチ**（キク科）、**マツムシソウ**（マツムシソウ科） |

#### 見分けのポイント

- **種髪**：種子を割ると毛が出てくる → 種皮の延長
- **冠毛**：果実の頂部についている → 萼片の変形
- タンポポの「綿毛」は厳密には**冠毛**（果実に付着、種子ではない）

#### 種髪をもつ主要植物

| 科 | 種髪をもつ植物 |
|----|--------------|
| ヤナギ科 | アカメヤナギ・シダレヤナギ・ポプラ類（★試験注意） |
| アカバナ科 | ヤナギラン・ミヤマヤナギラン |
| キョウチクトウ科 | テイカカズラ・キョウチクトウ（袋果が裂開し種子に毛） |
| モクレン科 | モクレン属（種皮に絹状の毛） |
