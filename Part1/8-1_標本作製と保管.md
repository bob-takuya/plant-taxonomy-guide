# 8-1. 標本作製と保管

[← 目次に戻る](../NEW_00_目次.md) | [← 前：生物学の基本概念](./7-3_生物学の基本概念.md)

---

## はじめに

生物標本は、分類学研究や種の同定の基礎となる重要な資料です。本章では、植物標本（さく葉標本）と動物標本（液浸標本など）の作製方法と保管について解説します。

---

# 第1部：植物標本（さく葉標本）

## さく葉標本（押し葉標本）とは

| 項目 | 内容 |
|------|------|
| 英名 | Herbarium specimen |
| 定義 | 植物を乾燥・圧平して台紙に貼付した標本 |
| 目的 | 種の同定、分類研究、分布記録 |
| 保存期間 | 適切に保管すれば数百年以上 |

---

## さく葉標本の作製方法

### 1. 採集

| 項目 | 内容 |
|------|------|
| **採集部位** | 栄養器官（葉・茎・根）と生殖器官（花・果実）を含む |
| **個体の選択** | 健全で典型的な個体を選ぶ |
| **採集量** | 台紙に収まるサイズ（30-40cm程度） |
| **採集情報** | 日時、場所、採集者、生態情報を記録 |

> ⚠️ **注意**: 絶滅危惧種や保護地域での採集は許可が必要

### 2. 圧平・乾燥

```
標本の圧平方法

  重し（本・レンガなど）
    ↓
  ═══════════
  新聞紙
  ───────────
  植物体 ← 形を整える
  ───────────
  吸取紙（段ボール）
  ═══════════
  新聞紙
  ───────────
```

| 手順 | 内容 |
|------|------|
| **配置** | 新聞紙に植物を広げ、葉の表裏が見えるように配置 |
| **重石** | 重い本や専用プレスで圧迫 |
| **吸取紙交換** | 毎日1-2回、湿った吸取紙を交換 |
| **乾燥期間** | 通常3-7日（植物の厚さによる） |
| **乾燥の確認** | 完全に乾燥し、触ってパリッとしたら完成 |

### 3. 台紙への固定

| 項目 | 内容 |
|------|------|
| **台紙サイズ** | 通常42cm × 28cm（A3サイズ相当） |
| **固定方法** | 糊付け、または糸・テープで固定 |
| **向き** | 根を下、茎を上にして配置 |

### 4. ラベルの作成

==標本ラベルに記載すべき情報==:

| 項目 | 内容 | 例 |
|------|------|-----|
| **学名** | 属名・種小名・命名者 | *Prunus serrulata* Lindl. |
| **和名** | 日本語名 | ヤマザクラ |
| **科名** | 所属する科 | バラ科 Rosaceae |
| **採集地** | 詳細な場所 | 東京都八王子市高尾山 標高500m |
| **生育環境** | 生育場所の特徴 | 落葉広葉樹林の林縁 |
| **採集日** | 年月日 | 2025年4月15日 |
| **採集者** | 採集者名 | 山田太郎 |
| **同定者** | 同定した人（異なる場合） | 鈴木花子 det. |
| **標本番号** | 個人またはコレクション番号 | YT-2025-0415-001 |

---

## さく葉標本の保管

### 保管環境

| 項目 | 条件 |
|------|------|
| **温度** | 15-20℃が理想 |
| **湿度** | 40-60%（低湿度が重要） |
| **光** | 暗所（退色防止） |
| **害虫対策** | 防虫剤使用、定期的な点検 |

### 標本庫（Herbarium）での管理

| 方式 | 説明 |
|------|------|
| **科・属順配列** | APG分類体系に従って配列 |
| **ファイリング** | 台紙を科・属ごとにまとめて保管 |
| **目録作成** | データベース化して検索可能に |

# 第2部：液浸標本（動物標本）

## ==液浸標本の作製（超頻出）==

動物（特に軟体動物、爬虫類、両生類）を液体中に保存する方法。

### ==固定と保存の2段階（Q13対応）==

```
液浸標本の作製手順

Step 1: 固定
  ホルマリン（ホルムアルデヒド水溶液）
  ↓
  目的：組織を固定し、腐敗を防ぐ
  濃度：5-10%
  期間：数日〜数週間

Step 2: 保存
  エタノール（エチルアルコール）
  ↓
  目的：長期保存
  濃度：70-80%
  期間：永久保存
```

### ==ホルマリンとエタノールを使い分ける理由（超重要）==

| 薬品 | 用途 | 理由 |
|------|------|------|
| ==**ホルマリン**== | ==**固定**== | ==組織のタンパク質を架橋し、形態を保持== |
| ==**エタノール**== | ==**保存**== | ==ホルマリンより安全、標本が硬化しにくい== |

**なぜ最初からエタノールを使わないのか？**

| ホルマリン固定の利点 | エタノール保存の利点 |
|---------------------|---------------------|
| 組織の固定力が強い | 臭気が少ない |
| 形態が崩れにくい | 人体への害が少ない |
| 染色性が良い | 標本が柔軟性を保つ |
| - | DNA抽出に適している（ホルマリンはDNAを損傷） |

> 📝 **試験ポイント**: ホルマリンで固定後、エタノールで保存するのは、**固定力**と**長期保存性・安全性**を両立させるため

### 液浸標本の作製手順

| 手順 | 内容 |
|------|------|
| 1. **麻酔・殺処理** | 動物を苦痛なく殺処理 |
| 2. **ホルマリン固定** | 5-10%ホルマリンに浸す（数日〜数週間） |
| 3. **水洗** | ホルマリンを洗い流す |
| 4. **エタノール置換** | 70-80%エタノールに移す |
| 5. **ラベル添付** | 耐水性ラベルを標本と一緒に保存 |
| 6. **密閉保管** | ガラス瓶に入れて密閉保存 |

# 第3部：その他の標本技法

## 骨格標本

| 項目 | 内容 |
|------|------|
| 対象 | 脊椎動物（魚類、両生類、爬虫類、鳥類、哺乳類） |
| 作製法 | 肉を除去し、骨格のみを組み立てる |
| 用途 | 形態比較、系統研究 |

## 剥製標本

| 項目 | 内容 |
|------|------|
| 対象 | 鳥類、哺乳類 |
| 作製法 | 皮膚を剥ぎ、詰め物をして形を整える |
| 用途 | 展示、教育 |

## 昆虫標本（乾燥標本）

### 展翅標本

| 項目 | 内容 |
|------|------|
| 対象 | チョウ、ガ |
| 作製法 | 翅を広げて板に固定 |
| 乾燥 | 1-2週間 |

### 針刺し標本

| 項目 | 内容 |
|------|------|
| 対象 | 甲虫、ハチなど |
| 作製法 | 昆虫針を体に刺して固定 |
| 保管 | 標本箱に収納、防虫剤使用 |

## 確認問題

### 問1
さく葉標本のラベルに記載すべき必須情報を5つ挙げよ。

<details>
<summary>解答を見る</summary>

1. **学名**（属名・種小名）
2. **採集地**（詳細な場所）
3. **採集日**（年月日）
4. **採集者**（氏名）
5. **科名**または**和名**

</details>

### 問2
液浸標本で、最初にホルマリンで固定し、その後エタノールで保存する理由を説明せよ。

<details>
<summary>解答を見る</summary>

**固定（ホルマリン）**: 組織のタンパク質を架橋して形態を固定し、腐敗を防ぐ。固定力が強い。

**保存（エタノール）**: ホルマリンより臭気が少なく、人体への害が少ない。長期保存に適し、標本が硬化しにくい。またDNA抽出にも適している。

つまり、**固定力の強さ**と**長期保存性・安全性**を両立させるため。

</details>

### 問3
さく葉標本を乾燥させる際、なぜ吸取紙を毎日交換するのか。

<details>
<summary>解答を見る</summary>

**湿気を除去するため**。

植物体から水分が蒸発し、吸取紙が湿ると、それ以上水分を吸収できなくなる。湿った状態が続くとカビが発生するため、毎日乾いた吸取紙と交換して迅速に乾燥させる必要がある。

</details>

### 問4
標本の保管で、害虫対策が重要な理由を説明せよ。

<details>
<summary>解答を見る</summary>

**標本を食害する昆虫（カツオブシムシ、シバンムシなど）が繁殖すると、貴重な標本が破壊されるため**。

特にさく葉標本や昆虫標本は、これらの害虫の餌となりやすい。防虫剤の使用と定期的な点検が不可欠。

</details>
