# 4-2. 日本の分布境界線

[← 目次に戻る](../NEW_00_目次.md) | [← 前：植物区系と分布](./4-1_植物区系と分布.md)

## はじめに

日本列島は南北に長く、様々な**分布境界線**が存在します。これらの境界線と、それぞれで分断される代表的な生物を覚えましょう。共通問題で頻出です。

## ブラキストン線（Blakiston's Line）

**最重要！必ず出題される**

| 項目 | 内容 |
|------|------|
| 位置 | **津軽海峡（北海道と本州の間）** |
| 発見者 | トーマス・ブレーキストン（イギリス人、1883年） |
| 意義 | 哺乳類・鳥類の分布境界線 |
| 別名 | 津軽海峡線 |

### ブラキストン線で分かれる動物

| 北海道側のみ | 本州側のみ |
|--------------|------------|
| ヒグマ | ツキノワグマ |
| エゾシカ | ニホンジカ |
| エゾリス | ニホンリス |
| シマフクロウ | フクロウ |
| エゾライチョウ | ライチョウ |
| ナキウサギ | − |
| エゾモモンガ | ニホンモモンガ |

### 植物の分布

- ブラキストン線は**主に動物**の境界線
- 植物では明確な境界として機能しない種も多い
- ただし、一部の植物では北海道と本州で異なる種がある

| 北海道 | 本州 |
|--------|------|
| エゾノウワミズザクラ | ウワミズザクラ |
| エゾエンゴサク | ヤマエンゴサク |
| エゾヤマザクラ | ヤマザクラ |

**頻出問題パターン：**
> 「ブラキストン線が分布の境界になって**いない**種の組み合わせを選べ」

→ 両側に分布する種（キツネ、タヌキなど）や、植物種を問う問題

## 渡瀬線（Watase's Line）

**重要！頻出**

| 項目 | 内容 |
|------|------|
| 位置 | **トカラ海峡（悪石島と小宝島の間）** |
| 発見者 | 渡瀬庄三郎（日本人動物学者、1912年） |
| 意義 | **旧北区と東洋区の境界** |
| 別名 | トカラ構造線、青木線 |

### 渡瀬線の特徴

- 屋久島・種子島は旧北区（日本本土側）
- 奄美大島以南は東洋区
- **爬虫類・両生類で顕著な違い**

### 渡瀬線で分かれる動物

| 北側（旧北区） | 南側（東洋区） |
|----------------|----------------|
| ニホンヤモリ | オキナワヤモリ |
| アオダイショウ | ハブ |
| カジカガエル | リュウキュウカジカガエル |
| ヤマカガシ | − |
| − | イシカワガエル |

### 植物の分布

- 北側：落葉広葉樹林が優占（照葉樹林北部）
- 南側：亜熱帯常緑広葉樹林へ移行
- 渡瀬線付近：熱帯性植物の分布北限

| 渡瀬線以南に分布 | 科名 |
|------------------|------|
| ガジュマル | クワ科 |
| アコウ | クワ科 |
| ソテツ（自生） | ソテツ科 |
| ヘゴ | ヘゴ科 |
| ビロウ | ヤシ科 |

## 八田線（蕪栗線）

| 項目 | 内容 |
|------|------|
| 位置 | 宗谷海峡（北海道とサハリンの間） |
| 発見者 | 八田三郎（1910年） |
| 意義 | 両生類・爬虫類の分布境界線 |
| 別名 | 宗谷線 |

### 特徴

- 両生類・爬虫類および淡水産無脊椎動物の境界
- サハリンには日本本土の両生類の多くが分布しない

---

## 宮部線

| 項目 | 内容 |
|------|------|
| 位置 | 択捉島と得撫島の間 |
| 発見者 | 宮部金吾（植物学者） |
| 意義 | **植物の分布境界線** |

### 特徴

- 主に植物の境界線として重要
- トドマツ、エゾマツ、ミズナラなどの分布北限
- この線から南の植物相は北海道のものと一致

---

## 対馬線

| 項目 | 内容 |
|------|------|
| 位置 | 対馬海峡（九州と朝鮮半島の間） |
| 意義 | 大陸系動物の分布境界線 |

### 特徴

- 大陸（朝鮮半島）に分布する動物の日本への侵入境界
- ツシマヤマネコ、ツシマテンなどは対馬にのみ分布

---

## 三宅線

| 項目 | 内容 |
|------|------|
| 位置 | 大隅海峡（大隅半島と屋久島の間） |
| 発見者 | 江崎悌三（1929年） |
| 意義 | 昆虫（特にチョウ・ガ類）の分布境界線 |

### 特徴

- 南方系のチョウの北限
- これより北は日本特産の昆虫、南は熱帯型の昆虫が多い

---

## 蜂須賀線

| 項目 | 内容 |
|------|------|
| 位置 | 八重山諸島と台湾の間 |
| 意義 | 東洋区の南限 |

### 特徴

- 八重山諸島と台湾の間の深い海峡
- 動物相・植物相の違い

---

## 世界の動物地理区

### 六大区（ウォーレスによる区分）

| 地理区 | 地域 | 代表的動物 |
|--------|------|------------|
| **旧北区** | ユーラシア北部、北アフリカ、日本 | ヒグマ、オオカミ |
| **新北区** | 北アメリカ | バイソン、アライグマ |
| **東洋区** | 東南アジア、インド | トラ、オランウータン |
| **エチオピア区** | サハラ以南のアフリカ | ゴリラ、キリン |
| **新熱帯区** | 中南米 | ナマケモノ、アルマジロ |
| **オーストラリア区** | オーストラリア、ニューギニア | カンガルー、カモノハシ |

### ウォーレス線

| 項目 | 内容 |
|------|------|
| 位置 | バリ島とロンボク島の間〜ボルネオ島とスラウェシ島の間 |
| 発見者 | アルフレッド・ウォーレス |
| 意義 | 東洋区とオーストラリア区の境界 |

**ウォーレス線の東西で分かれる動物：**
- 西側（東洋区）：有胎盤類（トラ、ゾウなど）
- 東側（オーストラリア区）：有袋類（カンガルー類）

## 確認問題

### 問1
津軽海峡を境界とする分布境界線の名称は何か。

<details>
<summary>解答を見る</summary>

**ブラキストン線**

イギリス人のトーマス・ブレーキストンが発見。北海道と本州の間の哺乳類・鳥類の分布境界線。

</details>

### 問2
トカラ海峡を境界とする分布境界線の名称は何か。

<details>
<summary>解答を見る</summary>

**渡瀬線**

日本人動物学者の渡瀬庄三郎が発見。旧北区と東洋区の境界。

</details>

### 問3
次のうち、ブラキストン線の北側にのみ分布する動物はどれか。
1. ツキノワグマ
2. ヒグマ
3. ニホンジカ
4. ニホンザル

<details>
<summary>解答を見る</summary>

**2. ヒグマ**

ヒグマは北海道にのみ分布。ツキノワグマは本州以南に分布。

</details>

### 問4
バリ島とロンボク島の間を通る、東洋区とオーストラリア区の境界線は何か。

<details>
<summary>解答を見る</summary>

**ウォーレス線**

アルフレッド・ウォーレスが発見した動物地理区の境界線。

</details>

### 問5
渡瀬線を境に、南側（東洋区）にのみ分布する爬虫類は何か。

<details>
<summary>解答を見る</summary>

**ハブ**

渡瀬線より南の奄美大島以南に分布。北側にはアオダイショウなどが分布。

</details>
