# 5-3. 種子散布と送粉

[← 目次に戻る](../NEW_00_目次.md) | [← 前：植生と遷移](./5-2_植生と遷移.md)

## はじめに

種子散布と送粉は植物の繁殖戦略の重要な要素です。散布様式・送粉様式と代表的な植物の対応は試験で頻出です。

---

## 種子散布

### 種子散布の意義

| 意義 | 説明 |
|------|------|
| 分布拡大 | 新しい生育地への進出 |
| 種内競争の回避 | 親植物との競争を避ける |
| 近交弱勢の回避 | 遺伝的多様性の維持 |

---

## ==種子散布様式（頻出）==

### 風散布（Anemochory）

| 適応形態 | 特徴 | 代表種 |
|----------|------|--------|
| **翼果** | 翼状の構造 | ==カエデ属==、シナノキ、ニレ |
| **冠毛** | 綿毛状の構造 | ==タンポポ==、アザミ類、キク科 |
| **微小種子** | 非常に小さい | ==ラン科==、ツツジ科 |
| **房状毛** | 綿状の毛 | ==ヤナギ属==、ガマ |

### ==風散布の代表種==

| 科名 | 代表種 | 散布器官 |
|------|--------|----------|
| キク科 | タンポポ、ノゲシ、アザミ類 | 冠毛（pappus） |
| ムクロジ科 | イロハモミジ、カエデ類 | 翼果 |
| ヤナギ科 | ヤナギ属、ポプラ属 | 綿毛 |
| ラン科 | 全種 | 微小種子 |
| ガガイモ亜科 | イケマ、ガガイモ | 種髪 |

### 動物散布（Zoochory）

#### 被食散布（Endozoochory）

| 特徴 | 内容 |
|------|------|
| 仕組み | 果実を食べた動物が種子を排出 |
| 果実 | 液果・核果が多い |
| 色 | 赤・黒・青など目立つ色 |

| 科名 | 代表種 | 果実の特徴 |
|------|--------|------------|
| バラ科 | サクラ、ノイバラ | 核果、赤い |
| ブドウ科 | ヤマブドウ、ノブドウ | 液果、黒紫 |
| モチノキ科 | クロガネモチ、ソヨゴ | 核果、赤い |
| ツバキ科 | サカキ、ヒサカキ | 液果、黒い |
| ミズキ科 | ミズキ、クマノミズキ | 核果、黒い |

#### 付着散布（Epizoochory）

| 適応形態 | 特徴 | 代表種 |
|----------|------|--------|
| **鉤状構造** | かぎ状のとげ | ==オナモミ==、ヌスビトハギ |
| **粘液** | ねばねばする | ヌスビトハギ、チヂミザサ |
| **芒（のぎ）** | 針状の構造 | イネ科の一部 |

#### 貯食散布（Synzoochory）

| 特徴 | 内容 |
|------|------|
| 仕組み | 動物が貯蔵した種子の一部が発芽 |
| 主な動物 | リス、ネズミ、カケス |

| 科名 | 代表種 | 貯食者 |
|------|--------|--------|
| ブナ科 | ==コナラ、クヌギ、ミズナラ== | リス、ネズミ |
| クルミ科 | オニグルミ | リス |
| マツ科 | チョウセンゴヨウ | ホシガラス |

#### アリ散布（Myrmecochory）

| 特徴 | 内容 |
|------|------|
| 仕組み | ==エライオソーム==をアリが運ぶ |
| エライオソーム | 脂質を含む付属体 |

| 科名 | 代表種 | 特徴 |
|------|--------|------|
| ケシ科 | ==カタクリ==、エンゴサク類 | スプリング・エフェメラル |
| スミレ科 | スミレ属各種 | 閉鎖花も |
| トウダイグサ科 | トウダイグサ | 多様な散布 |

### 水散布（Hydrochory）

| 特徴 | 内容 |
|------|------|
| 仕組み | 水に浮いて運ばれる |
| 適応 | 種子に空隙、コルク質の組織 |

| 環境 | 代表種 | 科名 |
|------|--------|------|
| 海流 | ヤシ科、ハマナタマメ | ヤシ科、マメ科 |
| 河川 | ハンノキ、ヤナギ類 | カバノキ科、ヤナギ科 |
| 湿地 | ミズバショウ、ガマ | サトイモ科、ガマ科 |

### 自動散布（Autochory）

| 機構 | 特徴 | 代表種 |
|------|------|--------|
| **弾き飛ばし** | 果皮が弾けて種子を飛ばす | ==ホウセンカ==、カタバミ |
| **乾燥開裂** | 乾燥で果皮が裂開 | スミレ、エンドウ |

## 送粉（Pollination）

### 送粉の種類

| 種類 | 説明 |
|------|------|
| **自家送粉** | 同一花または同一個体で受粉 |
| **他家送粉** | 異なる個体間で受粉 |

### 自殖と他殖

| 項目 | 自殖 | 他殖 |
|------|------|------|
| 定義 | 自家受精 | 他家受精 |
| 利点 | 確実な種子生産 | 遺伝的多様性 |
| 欠点 | 近交弱勢 | 送粉者が必要 |

---

## ==送粉様式（頻出）==

### 虫媒（Entomophily）

| 特徴 | 内容 |
|------|------|
| 花の特徴 | 目立つ色、香り、蜜 |
| 花粉 | 粘着性あり |
| 代表科 | バラ科、シソ科、キク科 |

#### 送粉者と花の適応

| 送粉者 | 花の特徴 | 代表種 |
|--------|----------|--------|
| **ハチ** | 黄・青・紫、着地場所 | ==マメ科==、シソ科 |
| **チョウ** | 赤・ピンク、長い花筒 | ナデシコ科、ヒガンバナ |
| **ガ（夜行性）** | 白、夜間開花、強香 | ==カラスウリ==、オシロイバナ |
| **ハエ** | 暗色、腐臭 | ウマノスズクサ |
| **甲虫** | 大型、白〜淡色 | モクレン科 |

### 鳥媒（Ornithophily）

| 特徴 | 内容 |
|------|------|
| 花の特徴 | ==赤色==、蜜が多い、無香 |
| 花の構造 | 筒状、丈夫 |
| 日本の鳥媒花 | ツバキ、サザンカ（メジロ） |

| 科名 | 代表種 | 送粉者 |
|------|--------|--------|
| ツバキ科 | ==ヤブツバキ、サザンカ== | メジロ |
| バラ科 | ウメ、サクラ（部分的） | メジロ |

### 風媒（Anemophily）

| 特徴 | 内容 |
|------|------|
| 花の特徴 | 目立たない、蜜なし、無香 |
| 花粉 | 大量、軽い、乾燥 |
| 柱頭 | 羽毛状で大きい |

| 分類群 | 代表種 | 特徴 |
|--------|--------|------|
| イネ科 | ==すべてのイネ科== | 花粉症の原因 |
| カヤツリグサ科 | スゲ属 | 湿地に多い |
| ブナ科 | ==コナラ、クヌギ、ブナ== | 尾状花序 |
| カバノキ科 | シラカンバ、ハンノキ | 尾状花序 |
| マツ科 | マツ属、スギ | 大量の花粉 |

### 水媒（Hydrophily）

| 種類 | 特徴 | 代表種 |
|------|------|--------|
| **水面媒** | 花粉が水面を漂う | キンギョモ |
| **水中媒** | 水中で送粉 | ==アマモ==、セキショウモ |

## 送粉を確実にする仕組み

### ==自家不和合性==

| 項目 | 内容 |
|------|------|
| 定義 | 自己の花粉では受精しない仕組み |
| 機構 | S遺伝子による認識 |
| 代表科 | バラ科、アブラナ科、ナス科 |

### ==雌雄異熟==

| 種類 | 説明 | 代表種 |
|------|------|--------|
| **雄性先熟** | 雄しべが先に成熟 | セリ科、キク科 |
| **雌性先熟** | 雌しべが先に成熟 | オオバコ、イネ科 |

### ==異型花柱性（異形花柱）==

| 種類 | 特徴 | 代表種 |
|------|------|--------|
| **二型花柱** | 長花柱花と短花柱花 | ==サクラソウ==、ミソハギ |
| **三型花柱** | 3タイプの花柱長 | ミソハギ |

---

## 閉鎖花

| 項目 | 内容 |
|------|------|
| 定義 | 開花せずに自家受粉する花 |
| 意義 | 確実な種子生産 |
| 代表種 | ==スミレ属==、ホトケノザ、ミゾソバ |

## 確認問題

### 問1
カエデ属の種子散布様式は何か。

<details>
<summary>解答</summary>

**風散布**

翼果を形成し、風で運ばれる。

</details>

### 問2
エライオソームを持つ種子はどのような散布様式か。

<details>
<summary>解答</summary>

**アリ散布（Myrmecochory）**

エライオソームをアリが餌として運ぶ。カタクリ、スミレなど。

</details>

### 問3
イネ科植物の送粉様式は何か。

<details>
<summary>解答</summary>

**風媒**

目立たない花、大量の花粉、羽毛状の柱頭が特徴。

</details>

### 問4
自家不和合性の意義は何か。

<details>
<summary>解答</summary>

**他家受粉を促進し、遺伝的多様性を維持する**

近交弱勢を防ぐ。

</details>

### 問5
雄しべが先に成熟する現象を何というか。

<details>
<summary>解答</summary>

**雄性先熟**

自家受粉を避ける仕組み。セリ科、キク科に多い。

</details>
