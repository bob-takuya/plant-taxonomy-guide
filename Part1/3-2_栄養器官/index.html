    3-2. 栄養器官（根・茎・葉）¶

    ← 目次に戻る | ← 前：形態学総論

    はじめに¶

    栄養器官である根・茎・葉は植物の同定において最も重要な特徴です。特に葉序（対生・互生）と葉の形態は科の識別に必須です。

    根の種類¶

    根系の型¶

    型 特徴 分布
    主根系 主根が太く発達 ==双子葉類==
    ひげ根系 多数の細い根が束状 ==単子葉類==

    根の起源による分類¶

    種類 説明 例
    定根 種子の幼根に由来 主根、側根
    不定根 茎・葉から発生 サツマイモ、ラン類の気根

    特殊な根¶

    根の型 説明 代表例
    貯蔵根 養分を蓄える ダイコン、ニンジン、サツマイモ
    気根 空気中に出る ガジュマル、ラン類、マングローブ
    支柱根 茎を支える トウモロコシ、タコノキ
    板根 板状に発達 熱帯樹木（サキシマスオウノキ）
    呼吸根 酸素を取り込む マングローブ（オヒルギ）
    吸盤根 他の植物に着生 ツタ、テイカカズラ
    寄生根 養分を吸収 ネナシカズラ、ヤドリギ

    ==マングローブの根（頻出）==¶

    種類 特徴 代表種
    支柱根 幹から斜めに伸びる ヤエヤマヒルギ
    膝根 膝のように曲がる オヒルギ
    筍根 タケノコ状に立ち上がる メヒルギ

    第2部：茎（Stem）¶

    茎の基本構造¶

    茎の各部名称¶

    部位 説明
    節（node） 葉が付く部分
    節間（internode） 節と節の間
    頂芽（terminal bud） 茎の先端の芽
    腋芽（axillary bud） 葉の付け根の芽
    髄（pith） 茎の中心部

    茎の型¶

    地上茎の型¶

    型 説明 例
    直立茎 まっすぐ立つ 多くの木本・草本
    匍匐茎 地表を這う イチゴ
    つる茎 他物に巻き付く アサガオ、フジ
    傾伏茎 斜めに伸びる ツルニチニチソウ

    地下茎の型¶

    型 説明 例
    根茎 横に伸びる タケ、ススキ、ショウガ
    塊茎 養分を蓄えた塊 ==ジャガイモ==（茎の変形）
    球茎 短縮した地下茎 サトイモ、グラジオラス
    鱗茎 鱗片葉に覆われた茎 タマネギ、ユリ（葉の変形）

    ==ジャガイモとサツマイモの違い（頻出）==¶

    項目 ジャガイモ サツマイモ
    器官 ==茎==（塊茎） ==根==（塊根）
    証拠 芽（目）がある 芽がない
    科 ナス科 ヒルガオ科

    茎の断面形状¶

    形状 該当する科・属
    ==四角形== ==シソ科==
    三角形 カヤツリグサ科
    円形 多くの植物
    中空 イネ科、セリ科

    特殊な茎¶

    型 説明 例
    茎棘 棘に変形した茎 サンショウ、サイカチ
    葉状茎 葉のように扁平 ナギイカダ、サボテン
    巻きひげ（茎由来） つる状に変形 ブドウ、パッションフルーツ

    ==葉の内部構造・気孔（試験頻出）==¶

    気孔（Stomata）の構造¶

    ⚠️ 2021年度問25で直接出題。誤りの選択肢として「孔辺細胞に葉緑体はない」が登場。

    気孔の断面（上から見た図）

    副細胞  孔辺細胞  副細胞
    ┌────┐┌──────┐┌────┐
    │    ││  ◯◯  ││    │
    │    ││  開口  ││    │
    └────┘└──────┘└────┘
         孔辺細胞   副細胞
    部位 説明 重要ポイント
    気孔（stomata） 表皮にある開口部、ガス交換・蒸散を担う 通常は葉の下面に多い
    孔辺細胞（guard cells） 気孔を囲む2個の特殊細胞 葉緑体をもつ（⚠️試験頻出：「葉緑体がない」は誤り）
    副細胞（subsidiary cells） 孔辺細胞を取り囲む隣接細胞 あることが多い（一般的な表皮細胞と区別される）

    気孔の分布（試験頻出）¶

    植物の種類 気孔の分布
    多くの陸上植物 下面に多い（蒸散抑制のため）
    浮葉性水生植物（ハス・スイレン等） 上面のみ（水面に浮くため）
    沈水植物（オオカナダモ等） 気孔なし
    イネ科草本 上下両面に分布

    孔辺細胞と葉緑体（⚠️頻出誤答パターン）¶

    「孔辺細胞に葉緑体はない」→ 誤り
    孔辺細胞は葉緑体をもつ。光合成によって糖を生産し、浸透圧を変化させることで気孔の開閉を行う。これが孔辺細胞が葉緑体を必要とする理由。
    通常の表皮細胞には葉緑体がないが、孔辺細胞は例外的に葉緑体をもつ。

    気孔開閉のメカニズム（概要）¶

    条件 孔辺細胞の状態 気孔
    光照射・CO₂減少 浸透圧上昇→吸水・膨圧上昇 開く
    暗・CO₂増加・乾燥 浸透圧低下→脱水・膨圧低下 閉じる

    ==葉序（Phyllotaxis）（超頻出）==¶

    葉序の種類¶

    葉序 説明 代表的な科
    ==互生== 1節に1枚ずつ交互 バラ科、キク科、ブナ科、マメ科、アブラナ科
    ==対生== 1節に2枚向かい合う ==シソ科==、アカネ科、アジサイ科、オオバコ科
    輪生 1節に3枚以上 アカネ科（ヤエムグラ）、トチノキ
    根生 根元に集まる タンポポ、オオバコ

    ==コクサギ型葉序（頻出）==¶

    説明 該当植物
    2枚ずつ交互につく（互生と対生の中間） コクサギ（ミカン科）

    葉身の形態¶

    葉身の形¶

    形 説明 例
    線形 細長い イネ科
    披針形 先がとがった細長い楕円 ヤナギ
    卵形 基部が広い卵型 サクラ
    倒卵形 先が広い卵型 サカキ
    心形 ハート型 ドクダミ、カツラ
    腎形 腎臓型 フキ
    楯形 葉柄が中央につく ハス
    矢じり形 矢じり型 オモダカ
    扇形 扇型 イチョウ

    葉縁の形¶

    形 説明 例
    全縁 切れ込みなし クスノキ
    鋸歯縁 のこぎり状 サクラ
    重鋸歯縁 二重の鋸歯 ケヤキ
    欠刻縁 丸い切れ込み ミツバ
    波状縁 波打つ コナラ

    ==葉脈のパターン（頻出）==¶

    パターン 特徴 該当する群
    ==網状脈== 網目状 ==双子葉類==
    ==平行脈== 平行に走る ==単子葉類==
    掌状脈 掌状に広がる カエデ
    羽状脈 羽状に分かれる ブナ、サクラ

    単葉と複葉¶

    単葉と複葉の区別¶

    種類 説明 例
    単葉 1枚の葉身 サクラ、ブナ
    複葉 複数の小葉 マメ科、セリ科

    複葉の型¶

    型 説明 例
    羽状複葉 羽状に小葉がつく フジ、ヌルデ
    2回羽状複葉 羽状複葉がさらに分かれる セリ科（ニンジン）
    3出複葉 3枚の小葉 クローバー、クズ、カタバミ
    掌状複葉 掌状に小葉がつく トチノキ（5-7枚）
    鳥足状複葉 鳥の足状 ヤブガラシ

    ==複葉を持つ主要な科==¶

    科名 複葉の型 代表種
    マメ科 羽状複葉、3出複葉 フジ、クローバー
    セリ科 2-3回羽状複葉 ニンジン、セリ
    バラ科（一部） 奇数羽状複葉 ナナカマド、キイチゴ
    ミカン科 3出複葉、奇数羽状 サンショウ
    ウルシ科 奇数羽状複葉 ウルシ、ハゼノキ
    ムクロジ科 羽状複葉 ムクロジ、フウセンカズラ

    葉の付属器官¶

    ==托葉（Stipule）（頻出）==¶

    科名 托葉の特徴
    ==バラ科== 葉柄に沿ってつく、多様な形
    マメ科 葉状のものが多い
    ==アカネ科== ==葉間托葉==（対生葉の間につく）
    ==タデ科== ==托葉鞘==（筒状に茎を包む）

    葉耳 葉鞘の上端の耳状突起

    第4部：特殊な葉形態¶

    ==環境適応による葉の特殊化==¶

    針葉（Needle Leaves）¶

    特徴 機能 代表例
    細長い円柱形 表面積縮小、蒸散抑制 ==マツ科、モミ属、トウヒ属==
    クチクラ層厚い 水分保持 裸子植物の多く
    気孔陥没 蒸散抑制 マツ、トウヒ

    ==エリカ葉（Ericoid Leaves）（頻出）==¶

    エリカ葉の断面
      葉の縁が巻き込む
        ↓
    ───┐ ┌───
       │ │
       └─┘
      気孔は内側
    項目 特徴
    定義 ==葉の縁が裏側に巻き込む==
    機能 ==蒸散抑制==（気孔が保護される）
    適応環境 寒冷地、高山、乾燥地
    代表的な科 ==ツツジ科==（特にエリカ属、ガンコウラン属）
    日本産の例 ==ガンコウラン、エゾツツジ、コメバツガザクラ==

    ツツジ科のエリカ葉植物

    和名 学名 分布 特徴
    ガンコウラン Empetrum nigrum var. japonicum 北海道～本州（高山） 黒い液果
    エゾツツジ Therorhodion camtschaticum 北海道～本州（高山） 紅紫色の花
    コメバツガザクラ Harrimanella stelleriana 北海道～本州（高山） 這性低木

    多肉葉（Succulent Leaves）¶

    特徴 機能 代表例
    肥厚した葉 水分貯蔵 ==ベンケイソウ科、ハマミズナ科==
    表皮が厚い 蒸散抑制 アロエ、セダム
    CAM型光合成 水分節約 マンネングサ属

    鱗片葉（Scale Leaves）¶

    特徴 機能 代表例
    退化して小さい鱗片状 保護、貯蔵 ユリの鱗茎、スギノハカズラ
    光合成しない - 球根植物の鱗片

    捕虫葉（Insectivorous Leaves）¶

    型 機構 代表例
    粘着式 粘毛で捕獲 ==モウセンゴケ科==
    落とし穴式 袋状の葉 ==ウツボカズラ科、サラセニア科==
    はさみ式 葉が閉じる ==ハエトリソウ==（ハエトリグサ）
    吸い込み式 小嚢で吸引 ==タヌキモ科==（水中）

    葉の変態（Leaf Modifications）¶

    巻きひげ（Tendrils）¶

    由来 例
    葉全体 エンドウ（マメ科）
    葉の一部 カラスノエンドウ（マメ科）
    托葉 スイートピー（マメ科）

    📝 注意: ブドウの巻きひげは茎由来、ウリ科の巻きひげも茎由来

    棘（Spines）¶

    由来 例
    葉が棘化 サボテン、ヒイラギ
    托葉が棘化 アカシア属の一部

    苞（Bracts）¶

    特徴 例
    花序の基部の変形葉 ハナミズキ、ブーゲンビリア
    色づいて目立つ ポインセチア

    葉の寿命による分類¶

    常緑樹と落葉樹¶

    型 特徴 代表例
    常緑広葉樹 葉が厚く硬い（硬葉樹林） ==シイ、カシ、クスノキ==
    常緑針葉樹 針葉を年中つける ==マツ、スギ、ヒノキ==
    落葉広葉樹 秋に落葉 ==ブナ、コナラ、サクラ==
    落葉針葉樹 秋に落葉（まれ） ==カラマツ、メタセコイア==

    葉の寿命¶

    樹種 葉の寿命
    落葉樹 約半年（春～秋）
    常緑広葉樹 1～3年
    常緑針葉樹 2～5年（マツ類）
    特殊な例 イチョウ：1年（落葉）

    葉の生態的分類¶

    陽葉と陰葉¶

    型 環境 特徴
    陽葉 強光 厚い、柵状組織発達、小型
    陰葉 弱光 薄い、海綿状組織発達、大型

    浮葉と沈水葉¶

    型 特徴 例
    浮葉 水面に浮く、上面に気孔 ==ハス、スイレン、ヒシ==
    沈水葉 水中、気孔なし、薄い ==オオカナダモ、クロモ==
    抽水葉 水面上に出る ==ガマ、アシ（ヨシ）==

    異形葉性（Heterophylly）¶

    例 説明
    ミズニラ 栄養葉と胞子葉が異なる
    ヒルムシロ 浮葉と沈水葉が異なる
    サンショウモ 3枚の葉のうち1枚が水中で根状

    確認問題¶

    問1¶

    ジャガイモとサツマイモは、それぞれ何の器官か。

    解答 - **ジャガイモ**：茎（塊茎） - **サツマイモ**：根（塊根）

    問2¶

    対生の葉序を持つ科を3つ挙げよ。

    解答 **シソ科、アカネ科、アジサイ科**（他にオオバコ科、スイカズラ科など）

    問3¶

    托葉を持つことが特徴的な科を3つ挙げよ。

    解答 **バラ科、マメ科、タデ科、アカネ科**から3つ

    問4¶

    単子葉類と双子葉類で葉脈のパターンはどう異なるか。

    解答 - **単子葉類**：平行脈 - **双子葉類**：網状脈

    問5¶

    イネ科の葉で、葉鞘と葉身の境にある膜状の構造を何というか。

    解答 **葉舌（ligule）**
