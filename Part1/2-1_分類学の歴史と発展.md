共通問題（100点中20点）では、**分類学史・歴史的偉人問題**が頻出です。2018年から2022年にかけて毎年出題されており、確実に得点したい分野です。

## 分類学の発展史

### 古代〜中世

| 時代 | 人物 | 貢献 |
|------|------|------|
| 紀元前4世紀 | **アリストテレス** | 生物を「血のあるもの」と「血のないもの」に二分 |
| 紀元前3世紀 | **テオフラストス** | 「植物学の父」、約500種の植物を記載 |
| 1世紀 | **ディオスコリデス** | 薬用植物約600種を記載（『薬物誌』） |

### 近代分類学の誕生（17〜18世紀）

#### カール・フォン・リンネ（Carl von Linné, 1707-1778）

**最重要人物。必ず覚える。**

| 項目 | 内容 |
|------|------|
| 国籍 | スウェーデン |
| 称号 | **「分類学の父」** |
| 主著 | 『自然の体系』(Systema Naturae, 1735)、『植物の種』(Species Plantarum, 1753) |

**リンネの功績：**

1. **二名法（binomial nomenclature）の確立**
   - 属名＋種小名で生物を命名
   - 例：*Homo sapiens*（ヒト）
   - ラテン語を使用し、国際的に統一

2. **階層的分類体系の確立**
   - 界（Kingdom）→ 門 → 綱 → 目 → 科 → 属 → 種
   - 現在の分類体系の基礎

3. **『植物の種』(1753年)**
   - **植物の学名の出発点（命名規約上）**
   - 約5,900種の植物を記載

4. **三界説**
   - 動物界・植物界・鉱物界
   - 後に生物のみの分類へ発展

**リンネに関する頻出問題：**
- 二名法を確立した人物は？
- 植物学名の出発点となった著作は？
- 「分類学の父」と呼ばれる人物は？

### 進化論と分類学（19世紀）

#### チャールズ・ダーウィン（Charles Darwin, 1809-1882）

| 項目 | 内容 |
|------|------|
| 国籍 | イギリス |
| 主著 | 『種の起源』(On the Origin of Species, 1859) |

**ダーウィンの功績：**

1. **自然選択説（natural selection）**
   - 環境に適応した個体が生き残り、子孫を残す
   - 進化のメカニズムを説明

2. **共通祖先の概念**
   - 全ての生物は共通の祖先から分岐
   - 分類は系統（進化上のつながり）を反映すべき

3. **性選択説**
   - 配偶者獲得のための形質の進化

**ダーウィン以後の分類学：**
- 分類学は「進化の木の影」を学ぶ科学に
- 系統分類学の誕生

#### アルフレッド・ラッセル・ウォーレス（Alfred Russel Wallace, 1823-1913）

| 項目 | 内容 |
|------|------|
| 国籍 | イギリス |
| 称号 | 「生物地理学の父」 |

**ウォーレスの功績：**

1. **自然選択説の独立発見**
   - ダーウィンとほぼ同時期に進化論を提唱

2. **ウォーレス線の発見**
   - 東南アジアにおける動物相の境界線
   - バリ島とロンボク島の間を通る

3. **生物地理学の確立**
   - 世界の動物地理区の設定

### その他の重要な分類学者・進化学者

#### ジャン=バティスト・ラマルク（Jean-Baptiste Lamarck, 1744-1829）

| 項目 | 内容 |
|------|------|
| 国籍 | フランス |
| 学説 | **用不用説（獲得形質の遺伝）** |

**ラマルクの学説：**
- よく使う器官は発達し、使わない器官は退化する
- 獲得した形質は子孫に遺伝する
- 現在では否定されているが、進化論の先駆け

#### ジョルジュ・キュヴィエ（Georges Cuvier, 1769-1832）

| 項目 | 内容 |
|------|------|
| 国籍 | フランス |
| 称号 | 「比較解剖学の父」「古生物学の父」 |
| 学説 | **天変地異説（catastrophism）** |

**キュヴィエの功績：**
- 化石の研究から絶滅の事実を証明
- 比較解剖学を確立
- 動物を4つの門に分類

#### エルンスト・ヘッケル（Ernst Haeckel, 1834-1919）

| 項目 | 内容 |
|------|------|
| 国籍 | ドイツ |
| 造語 | 「生態学」(Ecology)、「系統発生」(Phylogeny) |

**ヘッケルの功績：**
- 反復説（個体発生は系統発生を繰り返す）
- 三界説を生物界・原生生物界・植物界・動物界の四界に拡張
- 系統樹の図示

### 20世紀の進化学

#### 木村資生（きむら もとお, 1924-1994）

**日本人として必ず覚える！**

| 項目 | 内容 |
|------|------|
| 国籍 | 日本 |
| 所属 | 国立遺伝学研究所 |
| 学説 | **中立進化説（neutral theory of molecular evolution）** |

**中立進化説：**
- 分子レベルでの進化の多くは自然選択ではなく、**遺伝的浮動**による
- 有利でも不利でもない中立的な突然変異が蓄積
- 1968年に提唱

**木村資生に関する頻出問題：**
- 中立進化説を提唱した日本人科学者は？
- 分子進化において遺伝的浮動の重要性を主張した学説は？

#### リン・マーグリス（Lynn Margulis, 1938-2011）

| 項目 | 内容 |
|------|------|
| 国籍 | アメリカ |
| 学説 | **細胞内共生説** |

**細胞内共生説：**
- ミトコンドリアと葉緑体は、かつて独立した細菌が共生したもの
- 真核生物の起源を説明

## 分類体系の変遷

### 界の分類の歴史

| 時代 | 提唱者 | 体系名 | 分類 |
|------|--------|--------|------|
| 18世紀 | リンネ | 三界説 | 動物界・植物界・鉱物界 |
| 19世紀 | ヘッケル | 三界説 | 動物界・植物界・原生生物界 |
| 1969年 | ホイタッカー | **五界説** | モネラ界・原生生物界・菌界・植物界・動物界 |
| 1977年 | ウーズ | **三ドメイン説** | 細菌・古細菌・真核生物 |
| 1990年代〜 | 分子系統学 | 系統分類 | 分子データに基づく |

### 五界説（ホイタッカー, 1969年）

**頻出！必ず覚える**

| 界 | 特徴 | 代表例 |
|------|------|--------|
| **モネラ界** | 原核生物、核膜なし | 細菌、シアノバクテリア |
| **原生生物界** | 真核・単細胞（多細胞含む） | アメーバ、ゾウリムシ、藻類 |
| **菌界** | 従属栄養・吸収型 | キノコ、カビ、酵母 |
| **植物界** | 独立栄養・光合成 | コケ、シダ、種子植物 |
| **動物界** | 従属栄養・摂食型 | 無脊椎動物、脊椎動物 |

### 三ドメイン説（ウーズ, 1977年）

| ドメイン | 特徴 | 含まれる生物 |
|----------|------|--------------|
| **細菌（Bacteria）** | 原核生物 | 大腸菌、乳酸菌等 |
| **古細菌（Archaea）** | 原核生物、極限環境 | メタン生成菌、好熱菌等 |
| **真核生物（Eukarya）** | 核膜あり | 動物、植物、菌類、原生生物 |

## 確認問題

### 問1
二名法を確立し、「分類学の父」と呼ばれる人物は誰か。

<details>
<summary>解答を見る</summary>

**カール・フォン・リンネ**

1753年『植物の種』で二名法を体系化し、現在の植物命名法の基礎を築いた。

</details>

### 問2
中立進化説を提唱した日本人科学者は誰か。

<details>
<summary>解答を見る</summary>

**木村資生**

1968年に分子レベルの進化は遺伝的浮動によるものが多いとする中立進化説を提唱した。

</details>

### 問3
用不用説を提唱した人物は誰か。

<details>
<summary>解答を見る</summary>

**ラマルク**

よく使う器官は発達し、使わない器官は退化するという説。現在は否定されている。

</details>

### 問4
五界説を提唱した人物と、その5つの界を答えよ。

<details>
<summary>解答を見る</summary>

**ホイタッカー（1969年）**

1. モネラ界
2. 原生生物界
3. 菌界
4. 植物界
5. 動物界

</details>

### 問5
細胞内共生説を提唱した人物は誰か。

<details>

**リン・マーグリス**

ミトコンドリアと葉緑体が、かつて独立した細菌が共生したものであるという説。
