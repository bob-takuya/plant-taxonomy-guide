## 細胞の構造

### ==原核細胞と真核細胞の違い（頻出）==

| 項目 | 原核細胞 | 真核細胞 |
|------|----------|----------|
| 核膜 | ==なし== | ==あり== |
| ミトコンドリア | なし | あり |
| 葉緑体 | なし | あり（植物） |
| 細胞壁 | ペプチドグリカン | セルロース（植物） |
| 代表生物 | 細菌、シアノバクテリア | 動物、植物、菌類 |

### 植物細胞の特徴

| 構造 | 機能 |
|------|------|
| **細胞壁** | セルロースからなる、支持機能 |
| **葉緑体** | 光合成の場 |
| **液胞** | 浸透圧調節、物質貯蔵 |
| **色素体** | 葉緑体、有色体、白色体 |

### ==葉緑体と光合成（頻出）==

| 項目 | 内容 |
|------|------|
| 構造 | 二重膜、チラコイド、ストロマ |
| 色素 | クロロフィルa, b、カロテノイド |
| 光合成の式 | ==6CO₂ + 6H₂O → C₆H₁₂O₆ + 6O₂== |
| 明反応 | チラコイドで行われる |
| 暗反応 | ストロマで行われる（カルビン回路） |

## 遺伝の基礎

### DNAとRNA

| 項目 | DNA | RNA |
|------|-----|-----|
| 糖 | デオキシリボース | リボース |
| 塩基 | A, T, G, C | A, ==U==, G, C |
| 構造 | 二本鎖 | 一本鎖 |
| 機能 | 遺伝情報の保存 | タンパク質合成 |

### ==メンデルの遺伝の法則（頻出）==

| 法則 | 内容 |
|------|------|
| **優性の法則** | F1では優性形質のみ現れる |
| **分離の法則** | 対立遺伝子は配偶子形成時に分離 |
| **独立の法則** | 異なる遺伝子座は独立に遺伝 |

### 遺伝の分離比

| 交配 | 表現型の比 | 例 |
|------|------------|-----|
| Aa × Aa | ==3:1== | F2の分離比 |
| Aa × aa | 1:1 | 検定交配 |
| AaBb × AaBb | 9:3:3:1 | 二遺伝子交配 |

## 進化

### ==進化の証拠==

| 証拠 | 説明 | 例 |
|------|------|-----|
| 化石 | 過去の生物の痕跡 | 始祖鳥 |
| 相同器官 | 起源が同じで形態が異なる | 前肢（ヒト、クジラ、コウモリ） |
| 痕跡器官 | 退化した器官 | ヒトの虫垂 |
| 発生の相同性 | 胚発生が類似 | 脊椎動物の鰓弓 |

### ダーウィンの自然選択説

| 概念 | 説明 |
|------|------|
| 変異 | 個体間に形質の違いがある |
| 遺伝 | 変異は遺伝する |
| 自然選択 | 環境に適した個体が生存・繁殖 |
| 適応 | 世代を経て集団が変化 |

### ==種分化==

| 種類 | 説明 | 例 |
|------|------|-----|
| **異所的種分化** | 地理的隔離による | 島嶼の固有種 |
| **同所的種分化** | 同じ場所での分化 | 宿主転換 |

## ==地質時代の基礎（頻出）==

### ==主要な地質時代（Q18対応）==

地質時代は古いものから新しいものへと以下のように分けられます。

| 時代 | 紀（代表例） | 年代 | 主な生物 | ==試験重要点== |
|------|--------------|------|----------|----------------|
| **先カンブリア時代** | - | 46億～5.4億年前 | 原核生物、藻類 | 最古の時代 |
| ==**古生代**== | ==カンブリア紀== | ==5.4～4.9億年前== | ==三葉虫の出現== | ==「三葉虫の時代」== |
|  | ==**デボン紀**== | ==4.2～3.6億年前== | ==**魚類が繁栄**==、陸上植物 | ==**「魚類の時代」（Q18）**== |
|  | 石炭紀 | 3.6～3.0億年前 | 大森林、シダ植物 | 石炭の起源 |
| ==**中生代**== | ==**ジュラ紀**== | ==2.0～1.5億年前== | ==恐竜が繁栄==、裸子植物 | ==「恐竜の時代」== |
|  | 白亜紀 | 1.5億～6,600万年前 | ==被子植物の出現==、恐竜絶滅 | 被子植物の時代の始まり |
| ==**新生代**== | ==**古第三紀**== | ==6,600万～2,300万年前== | ==哺乳類の繁栄==、被子植物 | ==「哺乳類の時代」== |
|  | 新第三紀 | 2,300万～258万年前 | 草原の拡大、霊長類 | 現代に近い植生 |
|  | 第四紀 | 258万年前～現在 | ヒト属の出現 | 現代 |

> 📝 **試験ポイント（Q18）**: 「魚類の時代」は==**デボン紀**==
>
> 🔴 **頻出**: 各時代の代表的な生物を覚える
> - カンブリア紀 → 三葉虫
> - ==**デボン紀**== → ==**魚類**==（顎を持つ魚、肺魚）
> - ジュラ紀 → 恐竜（ティラノサウルス、ステゴサウルス）
> - 古第三紀 → 哺乳類

### ==地質時代と植物進化の対応==

| 時代 | 植物の進化 | 代表的な植物群 |
|------|------------|----------------|
| **デボン紀** | ==**陸上植物の進化**==（維管束の発達） | 原始的なシダ植物 |
| **石炭紀** | 大森林の形成 | リンボク、フウインボク |
| **ペルム紀** | 裸子植物の出現 | イチョウ類、針葉樹類 |
| **ジュラ紀** | 裸子植物の繁栄 | ソテツ類、イチョウ類 |
| **白亜紀** | ==**被子植物の出現・拡大**== | 初期の被子植物 |
| **新生代** | 被子植物の多様化 | 現在の植物相の形成 |

### ==地質時代の覚え方==

#### 古生代の順番（古い→新しい）
```
カンブリア紀 → オルドビス紀 → シルル紀 → デボン紀 → 石炭紀 → ペルム紀
```

#### 中生代の順番
```
三畳紀 → ジュラ紀 → 白亜紀
```

#### 新生代の順番
```
古第三紀 → 新第三紀 → 第四紀
```

### ==地質時代に関する重要事項==

| 項目 | 内容 |
|------|------|
| **カンブリア爆発** | カンブリア紀に多様な生物群が出現 |
| **デボン紀** | ==**魚類が大繁栄**==、陸上植物の進化 |
| **石炭紀** | 巨大なシダ植物の森林、酸素濃度が高い |
| **P-T境界** | ペルム紀末の大量絶滅（史上最大） |
| **K-Pg境界** | 白亜紀末の大量絶滅（恐竜絶滅） |
| **第四紀** | 氷河期の繰り返し、ヒトの進化 |

## 生態系の基本

### ==生態系の構成要素==

| 要素 | 役割 | 例 |
|------|------|-----|
| **生産者** | 光合成で有機物生産 | 植物、藻類 |
| **消費者** | 他の生物を食べる | 動物 |
| **分解者** | 有機物を無機物に分解 | 菌類、細菌 |

### 食物連鎖と食物網

| 用語 | 説明 |
|------|------|
| **食物連鎖** | 「食う－食われる」の直線的関係 |
| **食物網** | 複数の食物連鎖が絡み合った構造 |
| **栄養段階** | 食物連鎖上の位置 |

### ==エネルギーピラミッド==

| 原則 | 説明 |
|------|------|
| エネルギー効率 | 次の栄養段階に移るのは約==10%== |
| ピラミッドの形 | 上位ほど小さい |
| 理由 | 呼吸や排泄でエネルギーが失われる |

### 物質循環

| 循環 | 特徴 |
|------|------|
| **炭素循環** | CO₂の吸収（光合成）と放出（呼吸） |
| **窒素循環** | 窒素固定、硝化、脱窒 |
| **水の循環** | 蒸発、降水、流出 |

## 生物間相互作用

### ==共生関係（頻出）==

| 関係 | 説明 | 例 |
|------|------|-----|
| **相利共生** | 両者に利益 | 菌根菌と植物、送粉者と花 |
| **片利共生** | 一方のみ利益、他方に影響なし | 着生植物 |
| **寄生** | 一方が利益、他方が害 | ナンバンギセル |

### 競争

| 種類 | 説明 |
|------|------|
| **種内競争** | 同種間での資源競争 |
| **種間競争** | 異種間での資源競争 |
| **競争排除則** | 同じニッチの2種は共存できない |

### 被食・捕食関係

| 用語 | 説明 |
|------|------|
| 捕食者 | 他の生物を食べる |
| 被食者 | 食べられる側 |
| 個体群動態 | 両者の個体数は周期的に変動 |

## 生物多様性

### ==生物多様性の3つのレベル==

| レベル | 説明 |
|--------|------|
| **遺伝的多様性** | 種内の遺伝子の多様性 |
| **種多様性** | 種の数と量の多様性 |
| **生態系多様性** | 生態系の種類の多様性 |

### 生物多様性の危機

| 原因 | 例 |
|------|-----|
| 生息地の破壊 | 森林伐採、開発 |
| 外来種 | オオクチバス、オオキンケイギク |
| 乱獲 | 過剰な採取 |
| 環境汚染 | 農薬、水質汚濁 |
| 気候変動 | 温暖化 |

## 確認問題

### 問1
原核細胞と真核細胞の最も大きな違いは何か。

<details>
<summary>解答</summary>

**核膜の有無**

原核細胞には核膜がなく、真核細胞には核膜で囲まれた核がある。

</details>

### 問2
メンデルの遺伝でF2の分離比は何対何か。

<details>
<summary>解答</summary>

**3:1**

優性：劣性の比。

</details>

### 問3
食物連鎖でエネルギー効率は約何%か。

<details>
<summary>解答</summary>

**約10%**

残りは呼吸や排泄で失われる。

</details>

### 問4
相利共生の例を1つ挙げよ。

<details>
<summary>解答</summary>

**菌根菌と植物（または送粉者と花）**

両者に利益がある関係。

</details>

### 問5
生物多様性の3つのレベルを挙げよ。

<details>
<summary>解答</summary>

**遺伝的多様性、種多様性、生態系多様性**

</details>
