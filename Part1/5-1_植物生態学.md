## はじめに

植物生態学は、植物と環境の相互作用を研究する分野です。試験では生活形、環境要因への適応、植物群落の構造が頻出テーマです。

---

## 生活形（Life Form）

### ラウンケルの生活形

デンマークの植物学者ラウンケル（Raunkiær）による休眠芽の位置に基づく分類。

| 生活形 | 休眠芽の位置 | 例 |
|--------|--------------|-----|
| **地上植物（Ph）** | 地上30cm以上 | 樹木 |
| **地表植物（Ch）** | 地上0〜30cm | 低木、ササ |
| **半地中植物（H）** | 地表付近 | 多年生草本 |
| **地中植物（Cr）** | 地中 | 球根植物 |
| **一年生植物（Th）** | 種子で越冬 | イネ科一年草 |
| **水生植物（HH）** | 水中 | スイレン |

### ==生活形スペクトル==

| 気候帯 | 優占する生活形 |
|--------|----------------|
| 熱帯多雨林 | 地上植物（Ph）が多い |
| 温帯 | 半地中植物（H）が多い |
| 砂漠 | 一年生植物（Th）が多い |
| ツンドラ | 地表植物（Ch）が多い |

## 環境要因と植物の適応

### 光環境への適応

| 項目 | 陽生植物 | 陰生植物 |
|------|----------|----------|
| 光飽和点 | 高い | 低い |
| 光補償点 | 高い | ==低い== |
| 葉の厚さ | 厚い | 薄い |
| 例 | アカマツ、ススキ | アオキ、シダ類 |

### ==光補償点と光飽和点==

| 用語 | 定義 |
|------|------|
| **光補償点** | 光合成速度＝呼吸速度となる光強度 |
| **光飽和点** | 光合成速度が最大になる光強度 |

```
光合成速度
    ↑
    │        ┌──────── 光飽和点
    │       /
    │      /
    │     /
    │    /
    ├───/─────────── 光補償点
    │  /
 ───┼─/─────────────→ 光強度
    0
```

### 水分環境への適応

| 種類 | 特徴 | 例 |
|------|------|-----|
| **乾生植物** | 乾燥に適応 | サボテン、多肉植物 |
| **中生植物** | 普通の環境 | 多くの陸上植物 |
| **湿生植物** | 湿った環境 | ミズバショウ |
| **水生植物** | 水中生活 | スイレン、コウホネ |

### ==CAM型光合成==

| 項目 | 内容 |
|------|------|
| 特徴 | 夜間にCO₂を取り込み、昼間に光合成 |
| 意義 | 昼間の気孔閉鎖で水分損失を防ぐ |
| 例 | ==サボテン科、ベンケイソウ科、パイナップル科== |

### C4型光合成

| 項目 | 内容 |
|------|------|
| 特徴 | CO₂濃縮機構を持つ |
| 意義 | 高温・乾燥環境で効率的 |
| 例 | ==トウモロコシ、サトウキビ、ススキ== |

## 植物群落の構造

### ==階層構造==

| 層 | 高さ | 構成種 |
|-----|------|--------|
| **高木層** | 15m以上 | 極相種（シイ、ブナ） |
| **亜高木層** | 5〜15m | ヤブツバキ、カクレミノ |

## 植物間の相互作用

### 競争

| 種類 | 説明 |
|------|------|
| 種内競争 | 同種間での資源をめぐる競争 |
| 種間競争 | 異種間での競争 |
| 競争排除則 | 同じニッチの種は共存できない |

### ==共生関係==

| 関係 | 説明 | 例 |
|------|------|-----|
| **相利共生** | 両者に利益 | ==菌根菌==、送粉者 |
| **片利共生** | 一方のみ利益 | 着生植物 |
| **寄生** | 一方が害を受ける | ナンバンギセル |

### 菌根（Mycorrhiza）

| 種類 | 特徴 | 植物例 |
|------|------|--------|
| **外生菌根** | 菌糸が根の外側を覆う | マツ科、ブナ科 |
| **内生菌根（VA菌根）** | 菌糸が細胞内に侵入 | ラン科、多くの草本 |

## 確認問題

### 問1
ラウンケルの生活形で、休眠芽が地中にある植物を何というか。

<details>
<summary>解答</summary>

**地中植物（Cryptophyte, Cr）**

球根植物など。

</details>

### 問2
光合成速度と呼吸速度が等しくなる光強度を何というか。

<details>
<summary>解答</summary>

**光補償点**

これより弱い光では植物は生育できない。

</details>

### 問3
夜間にCO₂を取り込み、昼間に光合成を行う経路を何というか。

<details>
<summary>解答</summary>

**CAM型光合成**

サボテン科、ベンケイソウ科などが持つ。

</details>

### 問4
菌根菌と植物の関係は何共生に分類されるか。

<details>
<summary>解答</summary>

**相利共生**

菌は糖を得て、植物はリン・窒素を得る。

</details>

### 問5
陰生植物の光補償点は陽生植物に比べてどうか。

<details>
<summary>解答</summary>

**低い**

弱い光でも生育可能。

</details>
