# 2-2. APG分類体系の理解

[← 目次に戻る](../NEW_00_目次.md) | [← 前：分類学の歴史と発展](./2-1_分類学の歴史と発展.md)

## はじめに

**APG分類体系に関する問題は毎年出題される最重要分野です。**
2017〜2022年の過去問で毎年出題されており、「APG体系で最も原始的な科」「特定の科に最も近縁な科」などが問われます。

---

## APG分類体系とは

### 概要

| 項目 | 内容 |
|------|------|
| 正式名称 | Angiosperm Phylogeny Group（被子植物系統グループ） |
| 初版 | 1998年（APG I） |
| 最新版 | 2016年（APG IV） |
| 特徴 | **分子系統解析に基づく分類体系** |

### 従来の分類体系との違い

| 分類体系 | 基準 | 特徴 |
|----------|------|------|
| **エングラー体系** | 形態（進化段階） | 単純→複雑、20世紀初頭まで主流 |
| **クロンキスト体系** | 形態（総合的） | 双子葉・単子葉の区分、1980年代主流 |
| **APG体系** | **DNA塩基配列** | 単系統群に基づく、現在の標準 |

### APG体系の版の変遷

| 版 | 年 | 主な変更点 |
|-----|------|------------|
| APG I | 1998 | 初版、基本的な系統樹 |
| APG II | 2003 | 科の再編成 |
| APG III | 2009 | さらなる科の統合・分割 |
| APG IV | 2016 | 最新版、**64目・416科** |

---

## APG体系における被子植物の大分類

### 系統樹の概要

```
被子植物（Angiosperms）
│
├── 基部被子植物（Basal Angiosperms）
│   ├── アンボレラ目（Amborellales）★最基部
│   ├── スイレン目（Nymphaeales）
│   └── アウストロバイレヤ目（Austrobaileyales）
│
├── モクレン類（Magnoliids）
│   ├── カネラ目（Canellales）
│   ├── コショウ目（Piperales）
│   ├── モクレン目（Magnoliales）
│   └── クスノキ目（Laurales）
│
├── 単子葉類（Monocots）
│   └── 〔詳細は第10章〕
│
└── 真正双子葉類（Eudicots）
    ├── 基部真正双子葉類
    │   ├── キンポウゲ目
    │   └── ヤマモガシ目 等
    │
    └── 中核真正双子葉類（Core Eudicots）
        ├── バラ上群（Superrosids）
        │   ├── バラ類（Rosids）
        │   └── マメ類等
        │
        └── キク上群（Superasterids）
            ├── キク類（Asterids）
            └── ナデシコ類等
```

## 基部被子植物（ANA grade）

### アンボレラ目 Amborellales

**最重要！必ず出題される**

| 項目 | 内容 |
|------|------|
| 科 | アンボレラ科（Amborellaceae） |
| 種数 | 1属1種のみ（*Amborella trichopoda*） |
| 分布 | ニューカレドニア固有種 |
| 特徴 | **被子植物の中で最も原始的（最基部）** |

**アンボレラの重要な特徴：**
- **被子植物で唯一、仮道管のみを持つ（道管がない）**
- 雌雄異株
- 花被片は未分化
- 日本には自生しない

**頻出問題：**
> 「APG体系で被子植物の最も原始的な科は？」
> → **Amborellaceae（アンボレラ科）**

### スイレン目 Nymphaeales

| 科 | 代表種 | 特徴 |
|-----|--------|------|
| スイレン科（Nymphaeaceae） | スイレン、コウホネ、オニバス | 水生、浮葉 |
| ハゴロモモ科（Cabombaceae） | ジュンサイ、フサジュンサイ | 水生 |

**特徴：**
- 水生植物
- アンボレラに次いで原始的

### アウストロバイレヤ目 Austrobaileyales

| 科 | 代表種 | 特徴 |
|-----|--------|------|
| マツブサ科（Schisandraceae） | マツブサ、サネカズラ、シキミ | つる性木本 |
| アウストロバイレヤ科 | - | オーストラリア固有 |

**注意：シキミはマツブサ科！**（旧シキミ科は統合された）

## モクレン類（Magnoliids）

### 概要

- 以前は「原始的双子葉類」とされていた
- APG体系では真正双子葉類とは別系統
- 約9,000種
- 花被片の数が不定なものが多い

### 主要な目と科

| 目 | 主な科 | 代表種 |
|-----|--------|--------|
| **モクレン目** | モクレン科（Magnoliaceae） | モクレン、ホオノキ、タイサンボク |
| | バンレイシ科（Annonaceae） | ポポー |
| **クスノキ目** | クスノキ科（Lauraceae） | クスノキ、タブノキ、シナモン、アボカド |
| **コショウ目** | コショウ科（Piperaceae） | コショウ |
| | ドクダミ科（Saururaceae） | ドクダミ、ハンゲショウ |
| | ウマノスズクサ科（Aristolochiaceae） | ウマノスズクサ、カンアオイ類 |
| **センリョウ目** | センリョウ科（Chloranthaceae） | センリョウ、フタリシズカ |

## 真正双子葉類（Eudicots）

### 真正双子葉類の特徴

| 特徴 | 内容 |
|------|------|
| 花粉 | **三溝粒（tricolpate）または派生型** |
| 花弁数 | 4または5の倍数が多い |
| 葉脈 | 網状脈 |

| | アケビ科（Lardizabalaceae） | アケビ、ムベ |
| | ツヅラフジ科（Menispermaceae） | アオツヅラフジ |
| **ヤマモガシ目** | ヤマモガシ科（Proteaceae） | マカダミア |
| | ハス科（Nelumbonaceae） | ハス |
| **ツゲ目** | ツゲ科（Buxaceae） | ツゲ、フッキソウ |

## 従来の分類体系との主な違い

### ユリ科の分割

**最重要！必ず覚える**

旧ユリ科（Liliaceae sensu lato）はAPG体系で大幅に分割されました：

| 旧分類（ユリ科） | 新分類（APG） |
|------------------|---------------|
| ユリ | ユリ科（Liliaceae s.s.） |
| ネギ、ニラ | ヒガンバナ科（Amaryllidaceae）ネギ亜科 |
| アスパラガス | キジカクシ科（Asparagaceae） |
| スズラン | キジカクシ科 スズラン亜科 |
| ギボウシ | キジカクシ科 ツルボ亜科 |
| ツルボ | キジカクシ科 ツルボ亜科 |
| アロエ | ススキノキ科（Xanthorrhoeaceae） |
| ヒガンバナ | ヒガンバナ科（Amaryllidaceae） |

### ゴマノハグサ科の再編成

旧ゴマノハグサ科は分子系統解析により大幅に再編成：

| 旧分類 | 新分類（APG） |
|--------|---------------|
| オオバコ属 | オオバコ科（Plantaginaceae） |
| キンギョソウ | オオバコ科 |
| ジギタリス | オオバコ科 |
| カワヂシャ | オオバコ科 |
| ゴマノハグサ属 | ゴマノハグサ科（狭義） |

### カエデ科・トチノキ科の統合

| 旧分類 | 新分類（APG） |
|--------|---------------|
| カエデ科（Aceraceae） | **ムクロジ科（Sapindaceae）に統合** |
| トチノキ科（Hippocastanaceae） | ムクロジ科に統合 |

### ガガイモ科の統合

| 旧分類 | 新分類（APG） |
|--------|---------------|
| ガガイモ科（Asclepiadaceae） | **キョウチクトウ科（Apocynaceae）に統合** |

## 確認問題

### 問1
APG体系において、被子植物の最も原始的（基部に位置する）とされる科は何か。

<details>
<summary>解答を見る</summary>

**アンボレラ科（Amborellaceae）**

ニューカレドニア固有の1属1種。被子植物で唯一、道管を持たず仮道管のみを持つ。

</details>

### 問2
旧ユリ科のネギ属は、APG体系では何科に分類されるか。

<details>
<summary>解答を見る</summary>

**ヒガンバナ科（Amaryllidaceae）ネギ亜科**

旧ユリ科は多くの科に分割された。

</details>

### 問3
APG体系で、カエデ科は何科に統合されたか。

<details>
<summary>解答を見る</summary>

**ムクロジ科（Sapindaceae）**

トチノキ科も同様にムクロジ科に統合された。

</details>

### 問4
被子植物で仮道管のみを持ち、道管を持たない科は何か。

<details>
<summary>解答を見る</summary>

**アンボレラ科（Amborellaceae）**

被子植物の中で唯一の例。これが原始的特徴とされる根拠の一つ。

</details>

### 問5
APG体系の特徴として最も適切なものはどれか。
1. 形態に基づく分類
2. 化石に基づく分類
3. DNA塩基配列に基づく分類
4. 生態に基づく分類

<details>
<summary>解答を見る</summary>

**3. DNA塩基配列に基づく分類**

APG体系は分子系統解析（特にDNA塩基配列の比較）に基づく分類体系。

</details>

## 暗記用まとめ表

### APG体系の系統関係

| 群 | 主な目・科 | 覚え方 |
|-----|------------|--------|
| 最基部 | アンボレラ科 | 「アンボレラが一番原始的」 |
| 基部被子植物 | スイレン科 | 水生植物 |
| モクレン類 | モクレン科、クスノキ科 | 「原始的双子葉」の代表 |
| 単子葉類 | イネ科、ラン科など | 平行脈、3数性 |
| 真正双子葉類 | 大多数の双子葉類 | 三溝粒花粉 |

### 従来との主な違い

| 変更点 | 内容 |
|--------|------|
| ユリ科 | 多数の科に分割 |
| ゴマノハグサ科 | オオバコ科等に再編成 |
| カエデ科 | ムクロジ科に統合 |
| トチノキ科 | ムクロジ科に統合 |
| ガガイモ科 | キョウチクトウ科に統合 |
