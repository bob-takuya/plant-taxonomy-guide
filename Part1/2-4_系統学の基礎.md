[← 目次に戻る](../NEW_00_目次.md) | [← 前：学名と命名規約](./2-3_学名と命名規約.md)

## はじめに

系統学は生物間の進化的関係を研究する分野です。植物分類においては、形態的特徴に基づく**形態系統学**と、DNA配列に基づく**分子系統学**の両方が重要です。

---

## 系統学の基本概念

### 系統樹（Phylogenetic Tree）

| 用語 | 説明 |
|------|------|
| **系統樹** | 生物間の進化的関係を樹形図で表したもの |
| **クレード** | 共通祖先とそのすべての子孫からなる単系統群 |
| **姉妹群** | 最も近縁な2つのクレード |
| **外群** | 比較対象となる系統群の外にある群 |

### 群の分類

| 用語 | 説明 | 例 |
|------|------|-----|
| **単系統群** | 共通祖先とすべての子孫を含む | 被子植物 |
| **側系統群** | 共通祖先を含むが、一部の子孫を除く | 旧「双子葉植物」 |
| **多系統群** | 共通祖先を含まない人為的なまとまり | 旧「藻類」 |

```
単系統群と側系統群の違い

    A ─┬─ B  ← AとBは単系統群
       │
       └─ C  ← A, B, Cも単系統群

「双子葉植物」は単子葉類を除いた側系統群
 → APG分類では「真正双子葉類」として再定義
```

## 形態系統学

### 概要

| 項目 | 内容 |
|------|------|
| 方法 | 形態的特徴の比較による系統推定 |
| 歴史 | リンネ以来の伝統的分類法 |
| 利点 | 化石にも適用可能、直感的 |
| 欠点 | 収斂進化による誤りの可能性 |

### ==形質の種類（頻出）==

| 用語 | 説明 | 例 |
|------|------|-----|
| **相同形質** | 共通祖先から受け継いだ形質 | 脊椎動物の前肢 |
| **相似形質** | 異なる起源だが似た形質（収斂） | サボテンとユーフォルビアの多肉茎 |
| **祖先形質** | 祖先から受け継いだ原始的形質 | 離弁花 |
| **派生形質** | 進化の過程で新たに獲得した形質 | 合弁花 |

---

## ==分岐分類学（Cladistics）の用語（超頻出）==

### 形質の進化的解釈

```
        祖先状態(0) → 派生状態(1)
             │
    ┌────────┼────────┐
    A        B        C
   (0)      (1)      (1)

A: 祖先形質を保持
B, C: 派生形質を共有
```

### ==主要な用語==

| 用語 | 英語 | 説明 | 例 |
|------|------|------|-----|
| ==**祖先形質**== | Plesiomorphy | 祖先的な形質状態 | 被子植物における離弁花 |
| ==**派生形質**== | Apomorphy | 進化的に新しい形質状態 | 被子植物における合弁花 |
| ==**共有派生形質**== | Synapomorphy | 共通祖先で獲得し子孫で共有される派生形質 | ==**真正双子葉類の三溝型花粉**== |
| ==**自己派生形質**== | Autapomorphy | その群固有の派生形質 | イチョウの扇形の葉 |
| **共有祖先形質** | Symplesiomorphy | 複数の群で共有されるが祖先的な形質 | 種子植物における種子形成 |

### ==共有派生形質の重要性==

| 項目 | 説明 |
|------|------|
| **定義** | 単系統群を特徴づける最も重要な形質 |
| **系統推定** | ==共有派生形質の存在が単系統群の証拠== |
| **分類への応用** | APG分類は共有派生形質に基づく |

## ==主要な植物群の共有派生形質（必須暗記）==

### 陸上植物（Embryophyta）の共有派生形質

| 形質 | 説明 |
|------|------|
| **胚の形成** | 配偶体上で胚が保護される |
| **クチクラ層** | 乾燥への適応 |
| **気孔** | ガス交換（最も原始的なコケ類の一部を除く） |
| **多細胞性の生殖器官** | 造卵器、造精器 |

### ==維管束植物（Tracheophyta）の共有派生形質==

| 形質 | 説明 |
|------|------|
| ==**維管束**== | 木部と師部による輸送系 |
| ==**リグニン**== | 細胞壁の木化 |
| **真の根** | 定根系の発達 |
| **胞子体優占** | 世代交代で胞子体が主体 |

### ==種子植物（Spermatophyta）の共有派生形質==

| 形質 | 説明 |
|------|------|
| ==**種子**== | 胚珠が発達した繁殖器官 |
| ==**花粉管**== | 精細胞の輸送 |
| **異形胞子性** | 大胞子と小胞子の分化 |
| **胚珠内での配偶体発達** | 胞子体に依存 |

### ==被子植物（Angiospermae）の共有派生形質==

| 形質 | 説明 |
|------|------|
| ==**心皮**== | 雌しべを構成、胚珠を包む |
| ==**重複受精**== | 2つの精細胞が卵細胞と中央細胞を受精 |
| ==**胚乳**== | 3倍体（2n+n）の栄養組織 |
| ==**導管**== | 裸子植物の仮道管より進化的 |
| **花** | 生殖器官が凝集した構造 |

### ==真正双子葉類（Eudicots）の共有派生形質==

| 形質 | 説明 |
|------|------|
| ==**三溝型花粉**== | ==**花粉に3本の溝（colpus）**== |
| - | これが最も重要な共有派生形質 |

```
三溝型花粉の模式図

    ───┐ 溝
       │
   ○   │   ← 花粉粒
       │
    ───┘
   3本の溝が放射状に配置
```

### ==単子葉類（Monocots）の共有派生形質==

| 形質 | 説明 |
|------|------|
| ==**平行脈**== | 葉脈が平行 |
| ==**散在維管束**== | 茎の維管束が不規則配列 |
| ==**3数性の花**== | 花の部品が3の倍数 |
| **単子葉** | 子葉が1枚 |
| **ひげ根系** | 主根が早期に退化 |

## 派生形質と祖先形質の対比

### 花の構造

| 祖先形質 | 派生形質 | 例（派生形質を持つ群） |
|----------|----------|------------------------|
| 離弁花 | ==合弁花== | ==キク科、シソ科、ナス科== |
| 放射相称 | 左右相称 | マメ科、ラン科、シソ科 |
| 花弁多数 | 花弁数固定 | 多くの真正双子葉類 |
| 雌雄異花 | 両性花 | 多くの被子植物 |

### 果実の構造

| 祖先形質 | 派生形質 | 例（派生形質を持つ群） |
|----------|----------|------------------------|
| 乾果（袋果など） | 液果 | バラ科の一部、ウリ科 |
| 裂開果 | 閉果 | イネ科、キク科 |

### ==収斂進化の例==

**植物で頻出の収斂進化例**

| 形質 | 系統的に異なる植物群 |
|------|----------------------|
| 多肉植物 | サボテン科（新大陸）とトウダイグサ科（旧大陸） |
| 食虫性 | モウセンゴケ科、ウツボカズラ科、タヌキモ科 |
| つる性 | ウリ科、マメ科、ブドウ科、ヒルガオ科 |
| 無葉緑素 | ギンリョウソウ（ツツジ科）、ナンバンギセル（ハマウツボ科） |

## 分子系統学

### 概要

| 項目 | 内容 |
|------|------|
| 方法 | DNA・RNA・タンパク質の配列比較 |
| 発展 | 1980年代以降急速に発展 |
| 利点 | 客観的、大量のデータ |
| 欠点 | 化石には適用困難 |

### 使用される遺伝子領域

| 領域 | 特徴 | 用途 |
|------|------|------|
| **rbcL** | 葉緑体遺伝子、進化速度が遅い | 科以上の高次分類 |
| **matK** | 葉緑体遺伝子、変異が多い | 属・種レベル |
| **ITS** | 核リボソームDNA | 属・種レベル |
| **trnL-F** | 葉緑体遺伝子間領域 | 種レベル |

### 分子時計

| 項目 | 内容 |
|------|------|
| 概念 | DNA変異は一定速度で蓄積する |
| 用途 | 分岐年代の推定 |
| 限界 | 速度は一定でない場合も |

## APG分類体系と分子系統学

### APG体系の成立

| 項目 | 内容 |
|------|------|
| APG | Angiosperm Phylogeny Group |
| 発表 | 1998年（APG I）〜2016年（APG IV） |
| 特徴 | ==分子系統学に基づく単系統群の分類== |

### 分子系統で明らかになったこと

| 従来の分類 | 分子系統の結果 |
|------------|----------------|
| 双子葉植物 | ==側系統群==（単子葉類を含まない） |
| ユリ科（広義） | ==多系統群==（多数の科に分割） |
| ゴマノハグサ科 | ==側系統群==（オオバコ科などに分割） |
| ガガイモ科 | ==キョウチクトウ科に統合== |
| アカザ科 | ==ヒユ科に統合== |

### ==旧「双子葉植物」の解体==

```
旧分類での「双子葉植物」

被子植物
├── 双子葉類（側系統群）
│   ├── モクレン類（基部被子植物の一部）
│   ├── 真正双子葉類
│   └── その他
└── 単子葉類

APG分類

被子植物
├── 基部被子植物（ANA grade + モクレン類）
├── 単子葉類
└── 真正双子葉類（Eudicots）
    ├── 基部真正双子葉類
    └── コア真正双子葉類
        ├── バラ類
        └── キク類
```

## 系統学で重要な概念

### 共有派生形質（Synapomorphy）

| 用語 | 説明 |
|------|------|
| 定義 | 単系統群のみが共有する派生形質 |
| 重要性 | ==単系統群を認識する根拠== |
| 例 | 被子植物の「二重受精」 |

### ==被子植物の共有派生形質==

| 形質 | 説明 |
|------|------|
| **二重受精** | 精細胞2個が卵細胞と中央細胞に受精 |
| **胚乳** | 3n（三倍体）の栄養組織 |
| **心皮** | 胚珠を包む構造 |
| **花粉管** | 花粉から伸びる管 |

### ==真正双子葉類の共有派生形質==

| 形質 | 説明 |
|------|------|
| **三溝型花粉** | 花粉に3本の溝（colpus） |
| 花弁数 | 4または5の倍数が多い |

## 系統推定の方法

### 主な手法

| 方法 | 説明 | 特徴 |
|------|------|------|
| **最節約法** | 変化が最も少ない系統樹を選ぶ | 古典的手法 |
| **最尤法** | 確率的に最も可能性の高い系統樹 | 計算量が多い |
| **ベイズ法** | 事後確率に基づく推定 | 信頼度評価が容易 |
| **近隣結合法** | 距離行列に基づく | 高速 |

### 系統樹の信頼性評価

| 方法 | 説明 |
|------|------|
| **ブートストラップ値** | 再サンプリングによる支持率（70%以上で信頼） |
| **事後確率** | ベイズ法での信頼度（0.95以上で信頼） |

## 試験で問われるポイント

### 頻出事項

1. **単系統群・側系統群・多系統群の区別**
2. **APG分類で解体された科**（旧ユリ科、旧ゴマノハグサ科）
3. **分子系統で統合された科**（ガガイモ科→キョウチクトウ科）
4. **被子植物の共有派生形質**（二重受精）
5. **真正双子葉類の特徴**（三溝型花粉）

## 確認問題

### 問1
共通祖先とそのすべての子孫を含む群を何というか。

<details>
<summary>解答</summary>

**単系統群（monophyletic group）**

側系統群は一部の子孫を除いたもの。

</details>

### 問2
APG分類体系は主にどのような手法に基づいているか。

<details>
<summary>解答</summary>

**分子系統学（DNA配列の解析）**

rbcL、matK、ITSなどの遺伝子領域が使用される。

</details>

### 問3
旧「双子葉植物」はなぜAPG分類で使われなくなったか。

<details>
<summary>解答</summary>

**側系統群であることが分子系統解析で判明したため**

単子葉類を除いた残りの群であり、単系統群ではない。

</details>

### 問4
真正双子葉類に共通する花粉の特徴は何か。

<details>
<summary>解答</summary>

**三溝型花粉（tricolpate pollen）**

花粉に3本の溝を持つことが共有派生形質。

</details>

### 問5
APG分類でガガイモ科はどの科に統合されたか。

<details>
<summary>解答</summary>

**キョウチクトウ科（Apocynaceae）**

分子系統解析により近縁であることが判明。

</details>
