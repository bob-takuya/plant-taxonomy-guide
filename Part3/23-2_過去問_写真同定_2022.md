# 23-2. 過去問：写真同定問題（2022年度）

> **出題形式**：問56（植物写真10枚同定）、問57（特徴記述）  
> **対象種数**：11種  
> 海岸植物・亜高山帯植物・帰化植物が多く出題された年度。

## 問56：写真同定 10問（2022年度）

### 56-1. アダン *Pandanus odoratissimus*

**科・目**：タコノキ科（Pandanaceae）- タコノキ目

| 項目 | 内容 |
|------|------|
| 生活型 | 常緑高木（高さ3〜5m） |
| 茎 | **気根（支柱根）が発達**してタコの足のように見える |
| 葉 | 長線形（長さ1m以上）、螺旋状につき3列配列、葉縁・中肋に鋸歯 |
| 花 | 雌雄異株、穂状花序 |
| 果実 | **パイナップル様の大型集合果**（赤〜橙色に熟す） |
| 分布 | **沖縄・小笠原、海岸沿い**（熱帯〜亜熱帯） |

!!! warning "試験頻出ポイント"
    - 気根（支柱根）+ パイナップル様の集合果 → 即アダン
    - 分布：沖縄・小笠原の海岸の代表植物
    - 単子葉類（タコノキ目）だが木本 → 「木本の単子葉」として記憶
    - 果実は食べられる（生食・加工）

---

### 56-2. ハンゲショウ / カタシログサ *Saururus chinensis*

**科・目**：ドクダミ科（Saururaceae）- コショウ目

| 項目 | 内容 |
|------|------|
| 生活型 | 多年生草本 |
| 葉 | 卵形〜広卵形、互生、**花期に上部の葉が白くなる（半夏生）** |
| 花 | 白色、花被なし、穂状花序（ドクダミ科の特徴） |
| 花期 | 6〜8月（半夏生＝夏至から11日目頃） |
| 分布 | 本州〜九州、水辺・湿地 |

!!! info "和名の由来"
    「半夏生」= 夏至から11日目（7月2日頃）に花を咲かせることから。または葉が半分白くなる（半分化粧）から。

!!! warning "試験頻出ポイント"
    - 白い葉（葉の白化）が最大の識別点 → 花期のみ白くなる
    - ドクダミ科：コショウ目に属す基部被子植物（ANA grade外）
    - ドクダミとの違い：ドクダミは葉全体が緑（白い部分は総苞）

---

### 56-3. ボタンボウフウ *Peucedanum japonicum*

**科・目**：セリ科（Apiaceae）- セリ目

| 項目 | 内容 |
|------|------|
| 生活型 | 多年生草本 |
| 葉 | 1〜2回3出複葉〜羽状複葉、小葉は卵形〜広卵形で厚い |
| 花 | 白色、複散形花序 |
| 生育環境 | **海岸崖地・岩場**（ハマボウフウとは別種） |
| 分布 | 関東以西〜九州・沖縄の海岸 |
| 利用 | **食用・薬用**（若葉を天ぷら・おひたしに） |

!!! warning "試験頻出ポイント"
    - ハマボウフウ（*Glehnia littoralis*、砂浜）との混同注意 → **ボタンボウフウは崖・岩場**
    - セリ科の識別：複散形花序（umbel）が基本
    - 海岸植物として出題される

---

### 56-4. クサトベラ *Scaevola taccada*

**科・目**：クサトベラ科（Goodeniaceae）- キク目（APG IV）

| 項目 | 内容 |
|------|------|
| 生活型 | 常緑低木 |
| 葉 | へら形〜倒卵形、厚くて光沢あり、互生 |
| 花 | **白色・5弁だが左右相称（半開き扇形）**、花被が片側にのみ開く |
| 果実 | 白色の液果 |
| 分布 | **沖縄・小笠原、海岸砂地・砂浜** |

!!! success "独特な花形"
    クサトベラの花は5弁花が半分しか開かない「扇状」の形。これはクサトベラ科に特徴的な構造（花冠の裂片がすべて片側に並ぶ）。

!!! warning "試験頻出ポイント"
    - 半開きの扇形の花 → クサトベラ
    - 南西諸島・小笠原の海岸植物
    - キク目に属す（APG IV）

---

### 56-5. オオシラビソ / アオモリトドマツ *Abies mariesii*

**科・目**：マツ科（Pinaceae）- マツ目

| 項目 | 内容 |
|------|------|
| 生活型 | 常緑針葉高木（樹高25m） |
| 葉 | 扁平線形、**裏面に2本の白色気孔帯**、先は丸い |
| 球果 | **紫色〜青紫色の直立球果**（マツ科モミ属の特徴：枝から直立） |
| 分布 | **本州（東北〜中部）の亜高山帯**、日本固有種 |
| 森林 | 亜高山帯針葉樹林の優占種（オオシラビソ林） |

!!! warning "試験頻出ポイント"
    - モミ属（*Abies*）の特徴：球果が**直立**（落下しない）、葉が扁平
    - 球果が**紫色** → オオシラビソ（他のモミ属は緑〜褐色）
    - トドマツ（北海道）との混同注意：**オオシラビソ＝本州亜高山帯**
    - 「アオモリトドマツ」は地方名（青森で多い）

---

### 56-6. ノウルシ *Euphorbia adenochlora*

**科・目**：トウダイグサ科（Euphorbiaceae）- キントラノオ目（APG IV）

| 項目 | 内容 |
|------|------|
| 生活型 | 多年生草本 |
| 葉 | 卵形〜倒披針形、無柄、輪生状に見える |
| 花 | **杯状花序（Cyathium）**：トウダイグサ属特有の偽花 |
| 汁液 | **白色乳液（有毒）** |
| 生育環境 | 河川敷・湿地 |
| 分布 | 本州・九州 |

!!! warning "試験頻出ポイント"
    - トウダイグサ属はすべて杯状花序（Cyathium）と白色乳液
    - ノウルシ vs トウダイグサ：**ノウルシは湿地、トウダイグサは畑地**
    - 乳液は皮膚炎を起こすほど有毒

---

### 56-7. コアジサイ / シバアジサイ *Hydrangea hirta*

**科・目**：アジサイ科（Hydrangeaceae）- ミズキ目（APG IV）

| 項目 | 内容 |
|------|------|
| 生活型 | 落葉低木 |
| 葉 | 卵形〜楕円形、対生、鋸歯 |
| 花 | **淡青〜淡紫色の細かい花が密に集まる（装飾花なし！）** |
| 花期 | 6〜7月 |
| 分布 | 本州〜九州、山地の林縁・林内 |

!!! warning "試験頻出ポイント"
    - 普通のアジサイ（ガクアジサイ等）との違い：**コアジサイは装飾花（大型萼片）を持たない**
    - 小さな淡紫色の花のみが密集 → コアジサイ
    - アジサイ科（旧ユキノシタ科からAPGで独立）

---

### 56-8. コウボウムギ / フデクサ *Carex kobomugi*

**科・目**：カヤツリグサ科（Cyperaceae）- イネ目

| 項目 | 内容 |
|------|------|
| 生活型 | 多年生草本 |
| 葉 | 線形、硬くて先が鋭い、断面は三稜形気味 |
| 花序 | 穂状（カヤツリグサ科の特徴）、**太くて大型の穂** |
| 生育環境 | **海岸砂浜・砂丘** |
| 分布 | 全国（海岸砂丘の優占種） |

!!! info "カヤツリグサ科の特徴"
    茎は**三稜形（三角形）**が多く、中実。葉は3列互生（イネ科は2列）。

!!! warning "試験頻出ポイント"
    - 海岸砂丘の代表的草本 → コウボウムギ
    - 「弘法麦」= 弘法大師の錫杖に似た穂から
    - カヤツリグサ科 vs イネ科：茎の断面（三稜形 vs 円形）

---

### 56-9. ヒメオドリコソウ *Lamium purpureum*

**科・目**：シソ科（Lamiaceae）- シソ目

| 項目 | 内容 |
|------|------|
| 生活型 | 越年生草本（帰化植物） |
| 葉 | 対生、円形〜卵形、**上部の葉が赤紫色を帯びる**、縮れた感じ |
| 花 | **小型の淡紅紫色唇形花** |
| 花期 | 3〜5月 |
| 原産 | ヨーロッパ（帰化植物） |

!!! warning "試験頻出ポイント"
    - オドリコソウとの違い：**ヒメオドリコソウ＝帰化植物・葉が赤紫・小型**
    - 春の路傍・畑地の代表的帰化雑草
    - 四角茎（シソ科）+ 赤紫の上葉 → ヒメオドリコソウ

---

### 56-10. イタチハギ / クロバナエンジュ *Amorpha fruticosa*

**科・目**：マメ科（Fabaceae）- マメ目

| 項目 | 内容 |
|------|------|
| 生活型 | 落葉低木（高さ1〜3m） |
| 葉 | 奇数羽状複葉、小葉11〜25枚 |
| 花 | **暗紫色〜黒紫色の旗弁1枚のみ**（翼弁・竜骨弁なし）、黄色の雄蕊が目立つ |
| 果実 | 短い鎌形豆果 |
| 原産 | 北アメリカ |
| 法規 | 要注意外来生物（侵略性が高い） |

!!! success "特異な花形"
    マメ科の蝶形花は通常「旗弁1 + 翼弁2 + 竜骨弁2」だが、イタチハギは**旗弁1枚のみ**。黄色の雄蕊が束になって突き出す独特な外観。

!!! warning "試験頻出ポイント"
    - 黒紫色の旗弁1枚 + 黄色雄蕊 → 即イタチハギ
    - 河川敷・荒地に侵入する外来低木

## 問57-5. アカメガシワ *Mallotus japonicus*

**科・目**：トウダイグサ科（Euphorbiaceae）- キントラノオ目

| 項目 | 内容 |
|------|------|
| 生活型 | 落葉高木 |
| 葉 | 卵形〜広卵形、互生、**若葉が赤色**、葉柄基部に腺点2個 |
| 花 | 雌雄異株、白色、総状花序 |
| 果実 | 星状毛密生の蒴果 |
| 分布 | 本州〜沖縄、林縁・二次林の先駆種 |

!!! warning "試験頻出ポイント"
    - 「赤め（芽）柏」= 春の新葉が鮮やかな赤色
    - トウダイグサ科だが白色乳液はなし（*Mallotus*属は乳液が少ない）
    - 林縁の先駆樹種として重要
