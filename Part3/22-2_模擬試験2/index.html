    22-2. 模擬試験 第2回（100問）¶

    試験形式：4択・100問・120分
    第1回より難易度を上げた応用問題を多く含みます。

    共通問題（問1〜20）¶

    問1 大顎がある動物群をひとつ選びなさい。 1. カニムシ類　2. サソリ類　3. ダニ類　4. フラジムシ類

    問2 性的二型の程度が比較的小さい科をひとつ選びなさい。 1. キジ科　2. カワセミ科　3. ゴクラクチョウ科　4. カモ科

    問3 マボヤと異なる動物群に属する種をひとつ選びなさい。 1. ギボシムシ　2. オオサルパ　3. ナメクジウオ　4. ミドリシャミセンガイ

    問4 魚類の時代といえる地質時代をひとつ選びなさい。 1. カンブリア紀　2. デボン紀　3. ジュラ紀　4. 古第三紀

    問5 国の特別天然記念物に指定されていないものをひとつ選びなさい。 1. タンチョウ　2. コウノトリ　3. ニホンカモシカ　4. ツシマヤマネコ

    問6 「環境省レッドリスト2020」において魚類のうち絶滅危惧ⅠA類にランクされる種が最も多く含まれる分類群をひとつ選びなさい。 1. シラウオ類　2. タナゴ類　3. トゲウオ類　4. ハゼ類

    問7 小笠原諸島の固有種のうち、現在母島列島にのみ分布する種をひとつ選びなさい。 1. オオバシマムラサキ　2. オガサワラツツジ　3. マルハチ　4. ワダンノキ

    問8 爬虫類や両生類の液浸標本の記述として誤りであるものをひとつ選びなさい。 1. 両生類は皮膚を通してホルマリンが浸透するため、腹腔内にホルマリン注入は不要 2. 爬虫類は皮膚を通してホルマリンがほとんど浸透しないため、腹腔内への注入が必要 3. 両生類は長期保存すると変色するため液浸標本の体色を生時と同じとみなしてはいけない 4. 爬虫類は長期保存しても変色しないため液浸標本の体色を生時と同じとみなしてよい

    問9 種子に翼が発達している植物をひとつ選びなさい。 1. カエデ　2. カタバミ　3. ナズナ　4. ヤマノイモ

    問10 ラン科オフリス属の一部の種は花冠がマルハナバチ類のメスに似るため、疑似交接により送粉が行われる。この解明の先駆けとなった研究を行った生物学者をひとつ選びなさい。 1. ダーウィン　2. ビュフォン　3. ヘッケル　4. ラマルク

    問11 日本での絶滅種をひとつ選びなさい。 1. オキナワオオコウモリ　2. オガサワラオオコウモリ 3. ダイトウオオコウモリ　4. エラブオオコウモリ

    問12 伊豆諸島の生物地理に関する記述として正しいものをひとつ選びなさい。 1. 伊豆大島と新島に分布するモリアオガエルは在来種である 2. 伊豆諸島に自然分布するコウモリ類以外の哺乳類はアカネズミのみである 3. オカダトカゲは伊豆諸島のほか伊豆半島にも自然分布する 4. ミクラミヤマクワガタは御蔵島の固有種である

    問13 国際動物命名規約に関する記述として誤りであるものをひとつ選びなさい。 1. すでに命名されている種に別の学名を与えた場合は先に発表された学名が優先される 2. 絶滅種も現生種と同じ命名規約によって規定される 3. 属名を変更する際には必ず種小名も変更しなければならない 4. 新種として学名を発表する際にはホロタイプを指定する

    問14 富栄養化した淡水域で発生するアオコの主体となる単細胞微細藻が属するものをひとつ選びなさい。 1. 菌界　2. 原核生物界　3. 原生生物界　4. 植物界

    問15 子嚢菌類をひとつ選びなさい。 1. オオゴムタケ　2. オニフスベ　3. ショウロ　4. シロキクラゲ

    問16 生物地理学上のブラキストン線の一方にのみ分布する種の組み合わせをひとつ選びなさい。 1. シマフクロウ－ナキウサギ－エゾリス（北海道のみ） 2. フクロウ－ウミネコ－シマヘビ（両側に分布） 3. ヤチネズミ－エゾオコジョ－エゾモモンガ（北海道のみ） 4. タヌキ－キツネ－テン（本州のみ）

    問17 特定外来生物に指定されていないものが含まれる組み合わせをひとつ選びなさい。 1. ヒアリ－カダヤシ－オオヒキガエル－オオハンゴンソウ 2. ツマアカスズメバチ－ブルーギル－ウシガエル－オオキンケイギク 3. クビアカツヤカミキリ－オオクチバス－カミツキガメ－オオフサモ 4. アルゼンチンアリ－グッピー－グリーンアノール－ナルトサワギク

    問18 コムギを実験材料として遺伝学に貢献した人物をひとつ選びなさい。 1. 木原均　2. 木村資生　3. 今西錦司　4. 八杉龍一

    問19 外来生物法で特定外来生物に指定されていない種をひとつ選びなさい。 1. アレチウリ　2. セアカゴケグモ　3. ソウシチョウ　4. モウソウチク

    問20 主に伊豆諸島・伊豆半島に分布する植物の組み合わせをひとつ選びなさい。 1. スカシユリ－ヒロハクサフジ－ハマダイコン－ソナレマツムシソウ 2. サクユリ－シマクサギ－ソナレセンブリ－シマホタルブクロ 3. ハマオモト－イワタイゲキ－ソナレムグラ－アゼトウナ 4. ハマカンゾウ－ガクアジサイ－ハマコウゾリナ－イソギク

    専門問題（問21〜100）¶

    問21 蘚類に属する種をひとつ選びなさい。 1. ゼニゴケ　2. ジャゴケ　3. スギゴケ　4. ウキゴケ

    問22 APG分類体系（APG IV）でオモダカ目（Alismatales）に含まれる科をひとつ選びなさい。 1. サトイモ科　2. ショウブ科　3. ホロムイソウ科　4. トチカガミ科のみ

    問23 イネ科植物で、小穂が無柄であるものをひとつ選びなさい。 1. ヨシ　2. カヤツリグサ　3. ススキ　4. オギ

    問24 エリカ葉をもつ種をひとつ選びなさい。 1. イワヒゲ　2. ガンコウラン　3. コケモモ　4. シラタマノキ

    問25 ガマズミ属で装飾花をもたない種をひとつ選びなさい。 1. オオカメノキ　2. カンボク　3. ハクサンボク　4. ヤブデマリ

    問26 子房と萼筒が合着し肥大した果実をつける植物をひとつ選びなさい。 1. カスミザクラ　2. オオウラジロノキ　3. ナワシロイチゴ　4. ノウゴウイチゴ

    問27 岩礁に生育する海草をひとつ選びなさい。 1. ウミショウブ　2. ウミジグサ　3. エビアマモ　4. コアマモ

    問28 ブナの記述として正しいものをひとつ選びなさい。 1. 常緑高木である 2. コナラ属に属する 3. 雌雄異花である 4. コナラ類と同じ目に属する

    問29 カツラの記述として誤りであるものをひとつ選びなさい。 1. 日本の固有種である　2. 雌雄異株である 3. 花は葉の展開に先立って開花する　4. 果実は袋果である

    問30 ハルニレの記述として誤りであるものをひとつ選びなさい。 1. 落葉高木である 2. 花は春に開花する 3. 花は両性である 4. 種子は翼果の中央にあり、翼の先端はくぼまない

    問31 モミの記述として誤りであるものをひとつ選びなさい。 1. 常緑高木である　2. 花は春に開花する 3. 球果は開花の年に成熟する　4. 若い枝に毛がない

    問32 アスナロの記述として誤りであるものをひとつ選びなさい。 1. 葉はふつう鱗片状である　2. 種鱗は3〜4対である 3. 球果の鱗片は互生である　4. 本州・四国・九州に分布する

    問33 マツ属の記述として誤りであるものをひとつ選びなさい。 1. 常緑樹である　2. 葉は2型である 3. 針形葉は2〜10本束生する　4. 雌雄同株である

    問34 落葉高木、雄花は鱗片状の1次苞の腋に3個ずつつき、球果の鱗片は宿存する。あてはまる属をひとつ選びなさい。 1. ハンノキ属　2. カバノキ属　3. ハシバミ属　4. クマシデ属

    問35 落葉高木、雄花は無柄の2出集散花序をなして苞腋につき、さらにそれらは穂状花序につく。あてはまる属をひとつ選びなさい。 1. ブナ属　2. コナラ属　3. クリ属　4. マテバシイ属

    問36 落葉性、花は穂状につき、両性で花弁がある。あてはまる属をひとつ選びなさい。 1. イスノキ属　2. トサミズキ属　3. マルバノキ属　4. マンサク属

    問37 ユズリハの記述として誤りであるものをひとつ選びなさい。 1. 雌雄異株でまれに同株 2. 常緑高木 3. 雄蕊は20本前後 4. 花は春に開花する

    問38 オサバグサ、トガクシソウ、シラネアオイ、レンゲショウマに共通ではない記述をひとつ選びなさい。 1. 日本の固有種である 2. 単型属である 3. キンポウゲ科に分類される 4. 九州・琉球には分布しない

    問39 萼片・花弁とも4枚、四強雄蕊、二心皮性雌蕊を特徴とする科をひとつ選びなさい。 1. Brassicaceae　2. Saxifragaceae　3. Lamiaceae　4. Rubiaceae

    問40 花冠は合弁の放射相称花で5裂し、雄蕊5本で、子房は4裂して分果をつくる科をひとつ選びなさい。 1. ムラサキ科　2. ナス科　3. キキョウ科　4. シソ科

    問41 葉は対生し全縁、花は五数性で花柱は2・3または5本、子房は独立中央胎座である科をひとつ選びなさい。 1. サクラソウ科　2. ナデシコ科　3. リンドウ科　4. キキョウ科

    問42 オトギリソウ属に関する記述として誤りであるものをひとつ選びなさい。 1. 葉は対生し明点や黒点がある 2. 花は一日花で花弁は黄色 3. 雄蕊は多数で三体または五体雄蕊になる 4. 果実は核果で種子は1個

    問43 木本・葉は対生・花は五数性。あてはまる属をひとつ選びなさい。 1. アオキ属　2. カキノキ属　3. ガマズミ属　4. ヤマブキ属

    問44 草本・散形花序・子房下位。あてはまる属をひとつ選びなさい。 1. バイモ属　2. ヒガンバナ属　3. ホトトギス属　4. ユリ属

    問45 落葉高木、花序の苞は殻斗をつくる。あてはまる属をひとつ選びなさい。 1. カバノキ属　2. クマシデ属　3. ハシバミ属　4. ブナ属

    問46 草本・花序は掌状（指状）。あてはまる属をひとつ選びなさい。 1. イヌビエ属　2. オヒシバ属　3. シバ属　4. ネズミガヤ属

    問47 草本・花被片は6個・果実は蒴果。あてはまる属をひとつ選びなさい。 1. ウバユリ属　2. ツバメオモト属　3. チゴユリ属　4. キキョウラン属

    問48 日本に野生または野生状態でみられる植物で、雌蕊は柄がなく、茎葉は有柄で、花は花茎に単生し、萼片は5または6枚で、痩果は卵形で翼がない種をひとつ選びなさい。 1. アズマイチゲ　2. イチリンソウ　3. ニリンソウ　4. ハクサンイチゲ

    問49 ニシキギ科に分類されない属をひとつ選びなさい。 1. ウメバチソウ属　2. クロヅル属　3. ツゲモドキ属　4. ツルウメモドキ属

    問50 アオイ科に分類されない種をひとつ選びなさい。 1. イチビ　2. オクラ　3. キササゲ　4. フヨウ

    問51 散形花序・子房上位。あてはまる属をひとつ選びなさい。 1. アヤメ属　2. セリ属　3. ネギ属　4. ホタルブクロ属

    問52 常緑高木・短枝はない。あてはまる属をひとつ選びなさい。 1. イチョウ属　2. カツラ属　3. ツガ属　4. マツ属

    問53 落葉高木、花は尾状花序につき、苞は果実期まで残るが木化しない。あてはまる属をひとつ選びなさい。 1. クマシデ属　2. シイノキ属　3. ブナ属　4. ハンノキ属

    問54 落葉性・葉は単葉。あてはまる属をひとつ選びなさい。 1. カエデ属　2. ツバキ属　3. フジ属　4. ホザキナナカマド属

    問55 ブドウ科の属への検索表を参考にして、「花弁は開花時展開、花は5数性、巻きひげの先端は吸盤にならない」に該当する属をひとつ選びなさい。 1. ブドウ属　2. ツタ属　3. エビヅル属（ヤマブドウ属）　4. ミツバカズラ属

    問56 ヤナギ科の特徴として正しいものをひとつ選びなさい。 1. ヤマナラシ属は葉が狭卵形で花序は直立する 2. ヤナギ属は葉が広卵形で花序は垂れ下がる 3. ヤナギ属の花序は多くは直立し花は無柄 4. ヤナギ科の果実は全て液果である
