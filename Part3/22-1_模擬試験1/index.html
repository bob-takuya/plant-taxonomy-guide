    22-1. 模擬試験 第1回（100問）¶

    試験形式：4択・100問・120分（本番と同じ）
    合格目標：70点以上
    注意：本番試験ではCBTのため選択肢の順番が変わることがあります。

    共通問題（問1〜20）¶

    問1 昆虫やクモ類の排出器官をひとつ選びなさい。 1. ボーマン嚢　2. 原腎管　3. マルピーギ管　4. ゴルジ体

    問2 体外受精をする種をひとつ選びなさい。 1. アカウミガメ　2. ウミウ　3. トノサマガエル　4. バンドウイルカ

    問3 海中で採餌する種を含まない動物群をひとつ選びなさい。 1. 哺乳類　2. 鳥類　3. 爬虫類　4. 両生類

    問4 光合成色素としてクロロフィルaとクロロフィルbをもつ分類群をひとつ選びなさい。 1. 褐藻類　2. 珪藻類　3. 藍藻類　4. 緑藻類

    問5 日本の冷温帯林の主要樹種をひとつ選びなさい。 1. モミ　2. クロマツ　3. スダジイ　4. ブナ

    問6 シダ植物の世代交代にあてはまるものをひとつ選びなさい。 1. 配偶体は胞子体内にある　2. 配偶体だけで胞子体はない 3. 胞子体が発達し配偶体はめだたない　4. 胞子体は配偶体上にある

    問7 最近の生物分類において、菌類が所属するスーパーグループ名をひとつ選びなさい。 1. アーケプラスチダ　2. アメーボゾア　3. オピストコンタ　4. ストラメノパイル

    問8 ハーディ・ワインベルクの法則が成立するための条件として誤りであるものをひとつ選びなさい。 1. 集団内の交配は作為的でなければいけない 2. 集団内の個体数は十分に大きくなければいけない 3. 他の集団との間に個体の流出や流入がないこと 4. 突然変異がおこらないこと

    問9 生物の進化に関して天変地異説を唱えた生物学者をひとつ選びなさい。 1. ウォーレス　2. キュヴィエ　3. ヘッケル　4. ラマルク

    問10 本州亜高山帯に発達する森林の主要な高木ではない種をひとつ選びなさい。 1. オオシラビソ　2. コメツガ　3. トウヒ　4. モミ

    問11 ブラキストン線のどちら側にも自然分布する組み合わせをひとつ選びなさい。 1. ヒグマ－ツキノワグマ　2. エゾシカ－ニホンジカ 3. フクロウ－シマヘビ　4. シマフクロウ－エゾリス

    問12 次の記述のうち正しいものをひとつ選びなさい。 1. 北海道にブナは分布しない 2. 屋久島にブナは分布しない 3. 北海道にツクツクボウシは分布しない 4. 屋久島にツクツクボウシは分布しない

    問13 特定外来生物に指定されていない種をひとつ選びなさい。 1. オオキンケイギク　2. モウソウチク　3. オオハンゴンソウ　4. アレチウリ

    問14 学名 Mus musculus Linnaeus 1758 について正しい記述をひとつ選びなさい。 1. Linnaeusは発見者を、1758は発見年を意味する 2. Linnaeusは命名者を、1758は発見年を意味する 3. Linnaeusは発見者を、1758は公表年を意味する 4. Linnaeusは命名者を、1758は公表年を意味する

    問15 原核生物に関する記述として誤りであるものをひとつ選びなさい。 1. 原核生物は大きく細菌と古細菌より構成される 2. 原核生物ではRNAが遺伝子の働きをする 3. 原核生物は細胞壁をもつ 4. 原核生物は細胞核をもたない

    問16 単細胞生物だけの組み合わせをひとつ選びなさい。 1. ウミホタル－ヤコウチュウ　2. ワラジムシ－ゾウリムシ 3. ゾウリムシ－ヤコウチュウ　4. ツリガネムシ－ハリガネムシ

    問17 種子植物とシダ植物に共通する形質をひとつ選びなさい。 1. 仮導管　2. 種子　3. 托葉　4. 二次木部

    問18 ゴカイが属する動物群をひとつ選びなさい。 1. 環形動物　2. 節足動物　3. 線形動物　4. 扁形動物

    問19 カンブリアの大爆発とはどのような現象か、ひとつ選びなさい。 1. 多様な海洋生物が一斉に分化したこと 2. 巨大隕石落下の影響で多くの生物が絶滅したこと 3. 巨大火山の噴火後の気温低下が絶滅の原因になったこと 4. 被子植物の爆発的な大繁栄が恐竜の絶滅の一因になったこと

    問20 集団遺伝学の基礎をなすハーディ・ワインベルクの法則が成立する条件で正しいのはひとつ選びなさい。 1. 集団は小規模であること 2. 突然変異が頻繁に起こること 3. ランダムな交配が行われること 4. 他集団からの個体の流入が常にあること

    問22 苔類に属する種をひとつ選びなさい。 1. アカウキクサ　2. イチョウウキゴケ　3. カブトゴケ　4. ナンジャモンジャゴケ

    問26 花序のタイプとそれを特徴とする属の組み合わせで誤りであるものをひとつ選びなさい。 1. かたつむり形花序 ― ワスレグサ属 2. さそり形花序 ― ワスレナグサ属 3. 肉穂花序 ― ショウブ属 4. 穂状花序 ― オカトラノオ属

    問27 海草を含む科をひとつ選びなさい。 1. オモダカ科　2. サトイモ科　3. トチカガミ科　4. ホロムイソウ科

    問28 最新の分類体系（APG IV）でモクレン類として分類される目をひとつ選びなさい。 1. キンポウゲ目　2. クスノキ目　3. スイレン目　4. ヤマグルマ目

    問29 単子葉類が単系統群であることを示す共有派生形質ではないものをひとつ選びなさい。 1. 子葉は1枚である 2. 茎の中心柱は不整中心柱である 3. 根系は多数の不定根からなるひげ根系である 4. 花は3数性である

    問30 果実が袋果である分類群をひとつ選びなさい。 1. ゲンゲ属（マメ科）　2. サルトリイバラ属 3. トリカブト属（キンポウゲ科）　4. ヤマノイモ属

    問31 萼片・花弁とも4枚、四強雄蕊、二心皮性雌蕊を特徴とする科をひとつ選びなさい。 1. Brassicaceae　2. Saxifragaceae　3. Lamiaceae　4. Rubiaceae

    問32 花冠は合弁の放射相称花で5裂し、雄蕊5本で、子房は4裂して分果をつくる科をひとつ選びなさい。 1. ムラサキ科　2. ナス科　3. キキョウ科　4. シソ科

    問33 アブラナ科に分類される属をひとつ選びなさい。 1. Aeschynomene　2. Barbarea　3. Convallaria　4. Mallotus

    問34 キョウチクトウ科に分類されない種をひとつ選びなさい。 1. チトセカズラ　2. テイカカズラ　3. キジョラン　4. カモメヅル

    問35 花粉に粘着糸がある種をひとつ選びなさい。 1. シロヤシオ（ツツジ科）　2. サラサドウダン（ツツジ科） 3. クロマツ（マツ科）　4. オニユリ（ユリ科）

    問36 水中で開花し受粉する種をひとつ選びなさい。 1. ミズオオバコ　2. トチカガミ　3. イバラモ　4. スブタ

    問37 托葉を起源とする棘をもつ種をひとつ選びなさい。 1. ハリエンジュ（ニセアカシア）　2. カギカズラ　3. サイカチ　4. ボケ

    問38 托葉を起源とする巻きひげをもつ属をひとつ選びなさい。 1. Vicia　2. Smilax　3. Parthenocissus　4. Clematis

    問39 真葉植物に分類される科をひとつ選びなさい。 1. ヒカゲノカズラ科　2. ミズニラ科　3. イワヒバ科　4. ゼンマイ科

    問40 イワヒバ科の記述として誤りであるものをひとつ選びなさい。 1. 葉は同形または二形である 2. 同形胞子性である 3. 胞子嚢穂をつくる 4. 葉には小舌がある

    問41 スギの記述として誤りであるものをひとつ選びなさい。 1. 雌雄異株である　2. 常緑高木である 3. 短枝はない　4. 花は春に開花する

    問42 コウヤマキの記述として誤りであるものをひとつ選びなさい。 1. 日本の固有種である　2. 雌雄異株である 3. 短枝につく葉は線状である　4. コウヤマキ属に属する

    問43 アカマツの記述として誤りであるものをひとつ選びなさい。 1. 短枝につく葉は2枚束生である　2. 花は春に開花する 3. 球果は開花の年に成熟する　4. 常緑樹である

    問44 ゴヨウマツの記述として誤りであるものをひとつ選びなさい。 1. 日本に自生する　2. 短枝につく葉は5枚束生である 3. 花は秋に開花する　4. 球果は開花の翌年に成熟する

    問45 トウヒ属の記述として誤りであるものをひとつ選びなさい。 1. 常緑樹である　2. 短枝はない　3. 葉枕はない　4. 苞鱗は微小である

    問46 ツガ属の記述として誤りであるものをひとつ選びなさい。 1. 常緑樹である　2. 葉は2型である　3. 葉は有柄である　4. 葉枕がある

    問47 ヒノキの記述として誤りであるものをひとつ選びなさい。 1. 葉は十字対生である　2. 花は春に開花する 3. 種鱗の縁は敷石状に接する　4. 球果の鱗片は互生である

    問48 葉が十字対生または三輪生し、鱗片状または針状となる属をひとつ選びなさい。 1. Chamaecyparis　2. Juniperus　3. Thuja　4. Thujopsis

    問49 落葉高木、殻斗の総苞片は瓦重ね状、果実は2年目に熟す。あてはまる分類群をひとつ選びなさい。 1. アカガシ類　2. ウバメガシ類　3. クヌギ類　4. コナラ類

    問50 スダジイの記述として誤りであるものをひとつ選びなさい。 1. 果実は開花の翌年に成熟する 2. 殻斗の鱗片状突起は環状に配列する 3. 落葉高木である 4. シイノキ属に属する

    問51 ナナカマド属の説明として正しい記述をひとつ選びなさい。 1. 常緑高木である　2. 葉は対生である 3. 花弁は白色である　4. 果実は核果である

    問52 ウワミズザクラの記述として正しいものをひとつ選びなさい。 1. 常緑高木である　2. 葉は対生である 3. 花序は総状である　4. 果実は堅果である

    問53 果実が袋果である属をひとつ選びなさい。 1. シモツケ属　2. バラ属　3. ダイコンソウ属　4. リンゴ属

    問55 次の記述にあてはまる科をひとつ選びなさい。

    1. アサ科　2. イラクサ科　3. タデ科　4. ヒユ科

    問56 次の記述にあてはまる科をひとつ選びなさい。

    1. ウリ科　2. ガガイモ科　3. サルトリイバラ科　4. マタタビ科

    問57 オトギリソウ属に関する記述として誤りであるものをひとつ選びなさい。 1. 葉は対生し、明点や黒点がある 2. 花は一日花で、花弁は黄色 3. 雄蕊は多数で三体雄蕊または五体雄蕊になる 4. 果実は核果で、種子は1個

    問58 ブナの記述として正しいものをひとつ選びなさい。 1. 日本の暖帯を代表する高木である 2. コナラ属に属する 3. 雌雄異花である 4. 花序は尾状である

    1. Abies　2. Castanea　3. Castanopsis　4. Pinus
