# 23-4. 過去問：写真同定問題（2024年度）

> **出題形式**：問58（カバノキ科4属識別）、問59（植物写真10枚同定）  
> **対象種数**：14種  
> カバノキ科4属の一括出題、希少な寄生植物など難問が揃った年度。

---

## 問58：カバノキ科4属の識別（2024年度）

2024年度は同一科の4属を写真で識別する出題形式。カバノキ科（Betulaceae）を4種まとめて覚える。

**科・目**：カバノキ科（Betulaceae）- ブナ目

| 属名 | 代表種 | 果序・果実の特徴 | その他特徴 |
|------|--------|--------------|----------|
| ハンノキ属 *Alnus* | ハンノキ | **球果状の木質果序（松かさ状）** | 湿地・水辺・窒素固定 |
| カバノキ属 *Betula* | シラカンバ等 | **細長い円柱形の果序→砕けて散る** | 白い樹皮（カンバ類） |
| ハシバミ属 *Corylus* | ハシバミ | **苞に包まれたドングリ型の堅果** | ヘーゼルナッツの仲間 |
| クマシデ属 *Carpinus* | クマシデ等 | **羽根状の苞葉に包まれた堅果が連なる** | シデ類・材が硬い |

---

### 58-1. ハンノキ *Alnus japonica*

**科・目**：カバノキ科 - ブナ目

| 項目 | 内容 |
|------|------|
| 生活型 | 落葉高木 |
| 葉 | 卵形〜楕円形、鋸歯、互生 |
| 果序 | **卵形〜楕円形の木質化した果序（松かさ状）**、1〜5個が集まる |
| 雄花序 | 尾状花序、冬芽時から垂れる |
| 根粒 | **フランキア菌（放線菌）による窒素固定** |
| 生育環境 | **湿地・水辺・低湿地の代表樹木** |

!!! warning "試験頻出ポイント"
    - 松かさ状の木質果序 → ハンノキ属
    - 湿地林（ハンノキ林）の優占種
    - 窒素固定 → 根粒菌ではなくフランキア（放線菌類）との共生

---

### 58-2. カバノキ（シラカンバ）*Betula platyphylla* var. *japonica*

**科・目**：カバノキ科 - ブナ目

| 項目 | 内容 |
|------|------|
| 生活型 | 落葉高木 |
| 樹皮 | **白色で紙状に剥がれる**（シラカバの特徴） |
| 葉 | 三角卵形〜卵形、鋭鋸歯 |
| 果序 | 細長い円柱形の果序（熟すと砕けて種子を散布） |
| 分布 | 北海道〜本州中部の亜高山帯、パイオニア種 |

!!! warning "試験頻出ポイント"
    - 白い樹皮が識別の第一歩
    - 果序が熟すと砕けて散る（松かさ状の果序が残るハンノキと異なる）
    - ダケカンバ（*B. ermanii*）との違い：シラカンバは樹皮が白・低地〜亜高山、ダケカンバは樹皮が灰褐色〜橙褐色・高山帯

---

### 58-3. ハシバミ *Corylus heterophylla* var. *thunbergii*

**科・目**：カバノキ科 - ブナ目

| 項目 | 内容 |
|------|------|
| 生活型 | 落葉低木 |
| 葉 | 倒卵形〜円形、2重鋸歯、頂部がくびれた後に尖る |
| 果実 | **坅（どんぐり）型の堅果を包む鐘形〜筒形の苞葉（総苞）** |
| 種子 | 食用（ヘーゼルナッツの近縁種） |
| 分布 | 北海道〜九州、山地の低木林 |

!!! success "苞葉が果実を包む"
    他のカバノキ科と異なり、果実全体が総苞（苞葉）に包まれている。西洋のヘーゼルナッツ（*C. avellana*）の国内近縁種。

!!! warning "試験頻出ポイント"
    - 果実を包む鐘形の苞葉 → ハシバミ属
    - ドングリに見えるが本種はブナ科ではなくカバノキ科
    - 食用の種子（ヘーゼルナッツ）

---

### 58-4. クマシデ *Carpinus japonica*

**科・目**：カバノキ科 - ブナ目

| 項目 | 内容 |
|------|------|
| 生活型 | 落葉高木 |
| 葉 | 卵形〜長楕円形、鋭鋸歯、側脈が多い（平行に並ぶ） |
| 果序 | **羽根状の大型苞葉に包まれた小堅果が連なる穂（ホップに似る）** |
| 材 | 非常に硬い（シデ材） |
| 分布 | 本州〜九州の山地 |

!!! warning "試験頻出ポイント"
    - 羽根状苞葉が連なる果序 → クマシデ属（イヌシデ・アカシデ・クマシデ）
    - 葉の側脈が多く平行 → カバノキ科全般の特徴
    - シデ類の識別：苞葉の形（クマシデ＝苞葉の切れ込みが深い・最大）

## 問59：写真同定 10問（2024年度）

### 59-1. チョウノスケソウ *Dryas octopetala*

**科・目**：バラ科（Rosaceae）- バラ目

| 項目 | 内容 |
|------|------|
| 生活型 | 常緑低木（高山の地表を這う） |
| 葉 | 小型・楕円形〜卵形、上面は光沢・下面は白毛密生 |
| 花 | **白色・8弁（通常の花より多い）**、直径2〜3cm |
| 果実 | **綿毛のついた痩果が集まるモップ状の集合果** |
| 分布 | 北海道〜本州中部、**高山帯の岩礫地・雪田周辺** |
| 環北極 | 環北極分布種 |

!!! warning "試験頻出ポイント"
    - バラ科で8弁花 → 非常に珍しい → チョウノスケソウ一択
    - 「長之助草」= 植物学者 伊藤長之助にちなむ
    - 果実のモップ状集合体も識別点
    - 高山帯・環北極分布

---

### 59-2. コウリンカ *Tephroseris flammea* subsp. *glabrifolius*

**科・目**：キク科（Asteraceae）- キク目

| 項目 | 内容 |
|------|------|
| 生活型 | 多年生草本 |
| 葉 | 根出葉は有柄、茎葉は茎を抱く、毛がある |
| 花 | **橙赤色の舌状花 + 筒状花**、舌状花は垂れ下がる（反り返る） |
| 花期 | 7〜9月 |
| 分布 | 北海道〜本州、亜高山帯の草原 |
| 保護 | 準絶滅危惧（NT） |

!!! warning "試験頻出ポイント"
    - 橙赤色で舌状花が反り返る → コウリンカ
    - 「紅輪花」= 橙赤の頭花を指す
    - 草原性の高山植物（草原の減少で個体数が減っている）

---

### 59-3. ヤナギラン *Epilobium angustifolium*（= *Chamerion angustifolium*）

**科・目**：アカバナ科（Onagraceae）- フトモモ目

| 項目 | 内容 |
|------|------|
| 生活型 | 多年生草本（高さ1〜2m） |
| 葉 | 線形〜披針形（ヤナギに似る）、互生 |
| 花 | **鮮やかな紅紫色の4弁花**、総状花序（上から順に開花） |
| 果実 | 線形の蒴果、**種子に絹毛状の白毛** |
| 花期 | 7〜9月 |
| 生育環境 | **山火事跡地・林道沿い・亜高山帯の草地** |

!!! warning "試験頻出ポイント"
    - アカバナ科の4弁花 + 細長い蒴果（絹毛種子）
    - 山火事後の裸地・撹乱地に大群落を形成（パイオニア植物）
    - 鮮やかな紅紫の穂状花序 → 写真で見やすい

---

### 59-4. キイレツチトリモチ *Balanophora tobiracola*

**科・目**：ツチトリモチ科（Balanophoraceae）- ビャクダン目（APG IV）

| 項目 | 内容 |
|------|------|
| 生活型 | **完全寄生植物（葉緑素なし）** |
| 寄生先 | **トベラ（*Pittosporum tobira*）の根** |
| 外観 | **橙赤色〜赤褐色のキノコ状塊状体** |
| 葉 | なし（鱗片葉のみ） |
| 花 | 非常に小型、雌雄同株〜異株 |
| 分布 | **九州南部〜沖縄、海岸林** |
| 保護 | 絶滅危惧 II 類（VU） |

!!! danger "完全寄生植物"
    葉緑素を持たず、光合成ができない完全寄生植物。外見はキノコのようで、トベラの根から直接栄養を取る。

!!! warning "試験頻出ポイント"
    - キノコ状・赤褐色 + 完全寄生 → ツチトリモチ科
    - 寄生先がトベラ → 「トビラ（戸扉）草」の意味
    - 絶滅危惧種 + 九州南部〜沖縄の海岸
    - ビャクダン目（APG IV）＝半寄生・全寄生植物が多い目

---

### 59-5. ズダヤクシュ *Tiarella polyphylla*

**科・目**：ユキノシタ科（Saxifragaceae）- ユキノシタ目

| 項目 | 内容 |
|------|------|
| 生活型 | 多年生草本 |
| 葉 | 心形、掌状浅裂、長柄 |
| 花 | **白色の小さな5弁花**、総状花序 |
| 花期 | 5〜7月 |
| 分布 | 北海道〜九州、山地の林床 |

!!! info "和名の由来"
    「喘息薬種」= 喘息（ズダ）の薬になる種草。薬用植物として利用された歴史がある。

!!! warning "試験頻出ポイント"
    - ユキノシタ科の小型白色5弁花
    - 心形の長柄葉 + 山地林床 → ズダヤクシュ

---

### 59-6. ヤマトグサ *Theligonum japonicum*

**科・目**：アカネ科（Rubiaceae）- リンドウ目（APG IV）

| 項目 | 内容 |
|------|------|
| 生活型 | 一年生草本 |
| 葉 | 卵形、茎下部では対生・上部では互生（ユニーク） |
| 花 | **雄花：3〜4本の長い垂れ下がる雄蕊が目立つ、花被なし（目立たない）** |
| 分布 | 本州〜九州・沖縄、山地の日陰 |

!!! info "APGによる科の変更"
    旧分類：ヤマトグサ科（Theligonaceae）→ APG IVではアカネ科（Rubiaceae）に統合。

!!! warning "試験頻出ポイント"
    - 旧ヤマトグサ科 → 現アカネ科 のAPG変更
    - 茎の同一個体で葉序が変わる（下部＝対生、上部＝互生）
    - 長く垂れ下がる雄蕊が写真に写りやすい

---

### 59-7. キヅタ *Hedera rhombea*

**科・目**：ウコギ科（Araliaceae）- セリ目（APG IV）

| 項目 | 内容 |
|------|------|
| 生活型 | 常緑つる性木本 |
| 葉 | **菱形〜3〜5浅裂（幼木葉）or 楕円形（成木葉）**、革質・光沢 |
| 付着根 | 気根（付着根）で岩・樹幹に這い登る |
| 花 | 黄緑色、散形花序 |
| 果実 | 黒色の液果（秋〜冬） |
| 分布 | 本州〜沖縄、岩・樹幹・石垣 |

!!! warning "試験頻出ポイント"
    - セイヨウキヅタ（アイビー）の国産版
    - 幼葉（3〜5浅裂）と成葉（全縁楕円形）が同一株で異なる → 異型葉性
    - ウコギ科（セリ目）→ セリ目がキク上群に属す

---

### 59-8. ヤブニンジン *Osmorhiza aristata*

**科・目**：セリ科（Apiaceae）- セリ目

| 項目 | 内容 |
|------|------|
| 生活型 | 多年生草本 |
| 葉 | 2〜3回3出複葉 |
| 花 | 白色、複散形花序（小型） |
| 果実 | **棍棒状〜線形の長い果実（分果）**、上に向けて集まる |
| 香り | 根・葉に人参様の香り |
| 分布 | 北海道〜九州、山地林床 |

!!! warning "試験頻出ポイント"
    - セリ科の林床種 → 複散形花序 + 分裂葉
    - 棍棒状の長い果実（上を向いて集合する）→ 識別点
    - 「藪人参」= 林内に生え、根が人参様の香り

---

### 59-9. ケイワタバコ / イワタバコ *Conandron ramosissimus*

**科・目**：イワタバコ科（Gesneriaceae）- シソ目（APG IV）

| 項目 | 内容 |
|------|------|
| 生活型 | 多年生草本 |
| 葉 | 大型・卵形〜楕円形、**毛があり葉柄基部が茎を抱く**、鋸歯 |
| 花 | **淡紫色〜淡紅紫色の5裂の合弁花**、雄蕊が雌蕊を囲む |
| 生育環境 | **渓谷の岩壁・湿った岩上**（垂直面に張り付く） |
| 分布 | 本州〜九州・沖縄、渓谷の日陰 |

!!! warning "試験頻出ポイント"
    - 岩壁に這う + 大型葉 + 淡紫色合弁花 → イワタバコ
    - イワタバコ科（Gesneriaceae）= アフリカスミレ科の仲間（シソ目）
    - 「岩煙草」= 岩上に生え、葉がタバコの葉に似る

---

### 59-10. クコ *Lycium chinense*

**科・目**：ナス科（Solanaceae）- ナス目

| 項目 | 内容 |
|------|------|
| 生活型 | 落葉低木（長く枝が垂れ下がる） |
| 葉 | 卵形〜披針形、柔らかい、互生 |
| 花 | **淡紫色の5裂合弁花**（ナス科型） |
| 果実 | **鮮赤色の楕円形液果**（クコの実、薬用） |
| 棘 | 短い棘あり |
| 分布 | 全国（帰化・半野生化）、海岸・土手 |
| 利用 | **漢方・薬用・健康食品**（枸杞子） |

!!! warning "試験頻出ポイント"
    - ナス科の典型的な5裂合弁花（ナス・トマト型）
    - 赤い楕円形の果実 → クコ（枸杞の実）として有名
    - 海岸〜人里の低木、枝が垂れ下がる
    - 中国原産の薬用植物
