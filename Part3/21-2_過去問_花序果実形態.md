## 🌸 花序の種類

### 無限花序（求心的花序）
主軸が伸び続け、下から上、外から内に開花する。

| 花序名 | 特徴 | 代表種 |
|--------|------|--------|
| **穂状花序** | 花柄なし・主軸に直接つく | オカトラノオ、イノコズチ |
| **総状花序** | 花柄あり・主軸に並ぶ | フジ、ナタネ、スズラン |
| **散房花序** | 下の花ほど花柄が長い・花が水平に並ぶ | コデマリ、ノリウツギ |
| **散形花序** | 全ての花柄が1点から出る | セリ科全般、ヒガンバナ属 |
| **頭状花序** | 多数の花が短縮した花軸に密集 | キク科（タンポポ等）、マツムシソウ |
| **肉穂花序** | 多肉質の花軸に花が密生 | サトイモ科（マムシグサ等） |
| **尾状花序** | 穂状で垂れ下がる・風媒花 | ブナ科、カバノキ科 |

### 有限花序（遠心的花序）
頂端の花から開花し、側枝の花へと進む。

| 花序名 | 特徴 | 代表種 |
|--------|------|--------|
| **二出集散花序** | 頂花の下から2本の側枝 | タンポポ（一部）、ナデシコ科 |
| **巻散（さそり形）花序** | コイル状に巻いた単軸 | ムラサキ科（ワスレナグサ等） |
| **かたつむり形花序** | 円形に巻く | ワスレグサ属 |

!!! warning "有限花序 vs 無限花序"
    **有限花序（例：二出集散花序）**は頂端から開花→有限  
    **無限花序（例：総状花序）**は基部から開花→無限  
    試験では「散房花序は有限か無限か」が問われます → **無限花序**

## 📝 過去問演習：花序

### 問題1（2021年度 問26 改題）
花序のタイプとそれを特徴とする属の組み合わせで、**誤り**であるものをひとつ選びなさい。

1. かたつむり形花序 ― ワスレグサ属
2. さそり形花序 ― ワスレナグサ属
3. 肉穂花序 ― ショウブ属
4. 穂状花序 ― オカトラノオ属

<details>
<summary>解答を見る</summary>

**正解：4**

オカトラノオ（サクラソウ科）の花序は**総状花序**です（穂状花序ではない）。

穂状花序は花柄がなく、総状花序は花柄があります。オカトラノオは花柄があるので総状花序です。

</details>

3. 穂状花序
4. 二出集散花序

## 🍎 果実の種類

### 乾果（成熟すると乾燥する果実）

| 果実名 | 特徴 | 代表種 |
|--------|------|--------|
| **袋果（蓑果）** | 心皮1枚が縫合線で裂開 | トリカブト属、シャクナゲ属、シモツケ属 |
| **蓋果（蒴果）** | 多数の心皮、多方向に裂開 | スミレ、チューリップ、イチョウウキクサ |
| **豆果（莢果）** | マメ科の果実・縫合線と背缝で裂開 | ダイズ、エンドウ |
| **痩果** | 1心皮・1種子・裂開しない | タンポポ（冠毛つき）、バラ科の一部 |
| **分果** | 2つ以上の分果に分裂 | セリ科（2分果）、シソ科（4小堅果） |
| **堅果** | 硬い外皮・1種子 | ドングリ（ブナ科）、クリ |
| **翼果（有翼果）** | 翼状の突起をもつ | カエデ、ニレ |

### 液果（成熟しても多肉質の果実）

| 果実名 | 特徴 | 代表種 |
|--------|------|--------|
| **液果（berry）** | 果皮全体が多肉質 | ブドウ、トマト、キウイ |
| **核果（drupe）** | 外果皮＋中果皮（多肉）＋内果皮（硬い核） | モモ、ウメ、サクラ |
| **梨果（偽果）** | 花托が発達 | リンゴ、ナシ、カリン |
| **集合果** | 複数の花の子房が合着 | クワ、イチジク、ヤマボウシ |

## 📝 過去問演習：果実

### 問題4（2021年度 問33 改題）
果実が**袋果**である分類群をひとつ選びなさい。

1. ゲンゲ属（マメ科）
2. サルトリイバラ属（シオデ科）
3. トリカブト属（キンポウゲ科）
4. ヤマノイモ属（ヤマノイモ科）

<details>
<summary>解答を見る</summary>

**正解：3**

トリカブト属（キンポウゲ科）の果実は**袋果**（蓑果）です。

- ゲンゲ属 → **豆果**（莢果）
- サルトリイバラ属 → **液果**
- ヤマノイモ属 → **蒴果**（翼果状）

</details>

### 問題5（2022年度 問46 改題）
果実が**袋果**である属をひとつ選びなさい。

1. シモツケ属
2. バラ属
3. ダイコンソウ属
4. リンゴ属

<details>
<summary>解答を見る</summary>

**正解：1**

シモツケ属（バラ科）の果実は**袋果**です。

- バラ属 → 痩果（多数の痩果が花托の中に入る偽果）
- ダイコンソウ属 → 痩果
- リンゴ属 → 梨果（偽果）

バラ科の中でシモツケ属・ホザキナナカマド属などは袋果をもつことに注意。

</details>

### 問題6（2024年度 問44 改題）
**合心皮雌蕊**による**裂開果**であるものをひとつ選びなさい。

1. 蒴果
2. 袋果
3. 痩果
4. 豆果

<details>
<summary>解答を見る</summary>

**正解：1**

蒴果は**合心皮雌蕊**（複数の心皮が合着した雌蕊）から生じる**裂開果**です。

- 袋果 → **離心皮雌蕊**（1心皮）
- 痩果 → 裂開しない閉果
- 豆果 → 離心皮雌蕊（1心皮）、縫合線と背縫で裂開

</details>

## 🌱 特殊形態：托葉・巻きひげ・棘の起源

### 問題7（2022年度 問43 改題）
**托葉を起源とする棘**をもつ種をひとつ選びなさい。

1. ハリエンジュ（ニセアカシア）
2. カギカズラ
3. サイカチ
4. ボケ

<details>
<summary>解答を見る</summary>

**正解：1**

ハリエンジュ（マメ科）の棘は**托葉が変化**したものです（托葉棘）。

- カギカズラ → 花序の軸が変化したかぎ状突起
- サイカチ → 幹から直接出る複合棘（枝が変化）
- ボケ → **枝が変化**した棘（枝棘）

</details>

**托葉を起源とする巻きひげ**をもつ属をひとつ選びなさい。

1. *Vicia*（ソラマメ属）
2. *Smilax*（サルトリイバラ属）
3. *Parthenocissus*（ツタ属）
4. *Clematis*（センニンソウ属）

<details>
<summary>解答を見る</summary>

**正解：2**

サルトリイバラ属（シオデ科）の巻きひげは**托葉が変化**したものです。

- ソラマメ属（*Vicia*）→ 葉の先端が変化した巻きひげ（小葉が変化）
- ツタ属 → 葉と対生する枝が変化した巻きひげ
- センニンソウ属 → 葉柄が巻きつく（巻きひげではなく葉柄）

</details>

## 🌿 葉・気孔・花粉の形態

### 問題9（2021年度 問25 改題）
気孔の説明として**誤り**であるものをひとつ選びなさい。

1. 気孔は通常は葉の上面より下面のほうが多い。
2. 1個の気孔には2個の孔辺細胞がある。
3. 孔辺細胞に葉緑体はない。
4. 孔辺細胞の周りに副細胞があることが多い。

<details>
<summary>解答を見る</summary>

**正解：3**

孔辺細胞には**葉緑体がある**のが特徴です（光合成によりアブシシン酸・糖の変化で開閉を調節）。

葉緑体がないのは**表皮細胞**（孔辺細胞以外）です。

</details>

### 問題10（2022年度 問35 改題）
花粉に**粘着糸**がある種をひとつ選びなさい。

1. シロヤシオ（ツツジ科）
2. サラサドウダン（ツツジ科）
3. クロマツ（マツ科）
4. オニユリ（ユリ科）

<details>
<summary>解答を見る</summary>

**正解：4**

オニユリ（ユリ科）の花粉には**粘着糸（粘着性の糸状物質）**があり、昆虫に粘着しやすくなっています。

ツツジ科の花粉も粘着糸をもちます。マツ科の花粉は乾燥した風媒花粉（粘着糸なし、気嚢あり）。

</details>

## 📊 花序・果実のまとめ表

```
花序の識別チートシート：

・花柄なし → 穂状花序
・花柄あり・主軸伸長 → 総状花序
・花柄あり・水平に揃う → 散房花序
・全花柄が1点から → 散形花序
・多数の花が密集 → 頭状花序
・肉質軸に密生 → 肉穂花序
・垂れ下がる風媒 → 尾状花序
・コイル状に巻く → さそり形花序
```
