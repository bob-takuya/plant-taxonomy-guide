# 20-2. 形態用語問題

## はじめに

この章では、植物形態学の用語に関する演習問題を収録しています。正確な用語の理解は、植物を記載・識別する上で不可欠です。

---

## 第1節 葉の形態

### 問題1：葉形の用語

**A. 以下の葉形を表す用語を答えよ。**

1. 基部から先端まで幅がほぼ等しい長い葉
2. 中央部が最も幅広く、両端に向かって狭くなる葉
3. 先端が最も幅広く、基部に向かって狭くなる葉
4. 基部が最も幅広く、先端に向かって狭くなる葉
5. ほぼ円形の葉

<details>
<summary>解答を見る</summary>

1. **線形**（りんけい）
2. **楕円形**（だえんけい）
3. **倒卵形**（とうらんけい）
4. **卵形**（らんけい）
5. **円形**（えんけい）

</details>

---

### 問題2：葉縁の用語

**B. 以下の葉縁の形態を表す用語を答えよ。**

1. 鋸の歯のような切れ込み
2. 波状の浅い切れ込み
3. 縁に切れ込みがない
4. 二重の鋸歯
5. 深く切れ込んで裂片になる

<details>
<summary>解答を見る</summary>

1. **鋸歯縁**（きょしえん）
2. **波状縁**（はじょうえん）
3. **全縁**（ぜんえん）
4. **重鋸歯縁**（じゅうきょしえん）
5. **分裂**（ぶんれつ）または**裂状**

</details>

---

### 問題3：複葉の用語

**C. 以下の複葉の型を答えよ。**

1. 小葉が3枚
2. 小葉が1本の軸の両側に羽状に並ぶ
3. 葉軸が2回分岐して小葉がつく
4. 小葉が1点から放射状に広がる
5. 小葉が2枚

<details>
<summary>解答を見る</summary>

1. **三出複葉**（さんしゅつふくよう）
2. **羽状複葉**（うじょうふくよう）
3. **2回羽状複葉**（にかいうじょうふくよう）
4. **掌状複葉**（しょうじょうふくよう）
5. **2小葉複葉**（にしょうようふくよう）

</details>

---

### 問題4：葉序の用語

**D. 以下の葉序を表す用語と、その代表的な植物を1つ挙げよ。**

1. 茎の各節に1枚ずつ葉がつく
2. 茎の各節に2枚ずつ向かい合って葉がつく
3. 茎の各節に3枚以上の葉が輪状につく
4. 茎の基部に葉が集まってつく

<details>
<summary>解答を見る</summary>

1. **互生**（ごせい）- 例：サクラ、ブナ
2. **対生**（たいせい）- 例：ツツジ、シソ
3. **輪生**（りんせい）- 例：クルマバソウ、オカトラノオ
4. **根生**（こんせい）または**ロゼット状** - 例：タンポポ

</details>

## 第2節 花の形態

### 問題5：花被の用語

**E. 以下の花被の型を説明せよ。**

1. 離弁花
2. 合弁花
3. 同花被花
4. 異花被花

<details>
<summary>解答を見る</summary>

1. **離弁花**：花弁が互いに離れている花（例：バラ、サクラ）
2. **合弁花**：花弁が合着している花（例：キク、ツツジ）
3. **同花被花**：萼と花弁の区別がない花（例：チューリップ、ユリ）
4. **異花被花**：萼と花弁が明確に区別できる花（例：サクラ、バラ）

</details>

---

### 問題6：花冠の型

**F. 以下の花冠の型を答えよ。**

1. 花弁が筒状に合着し、先端が広がる漏斗のような形
2. 上下2つの唇に分かれた花冠
3. 花弁が合着して釣鐘のような形
4. 花弁が水平に広がり、中央から裂片が放射状に出る
5. 旗弁、翼弁、竜骨弁からなる蝶のような形

<details>
<summary>解答を見る</summary>

1. **漏斗形花冠**（ろうとがたかかん）- 例：アサガオ
2. **唇形花冠**（しんけいかかん）- 例：シソ科
3. **鐘形花冠**（しょうけいかかん）- 例：キキョウ
4. **車形花冠**（しゃけいかかん）- 例：ナス
5. **蝶形花冠**（ちょうけいかかん）- 例：マメ科

</details>

---

### 問題7：子房の位置

**G. 以下の子房の位置を説明し、それぞれの例を挙げよ。**

1. 上位子房
2. 下位子房
3. 半下位子房

<details>
<summary>解答を見る</summary>

1. **上位子房**：花托の上に子房がある
   - 例：サクラ、キンポウゲ、マメ科

2. **下位子房**：花托が子房を包み込む
   - 例：キク科、ウリ科、アヤメ科

3. **半下位子房**：子房の半分が花托に埋まる
   - 例：ユキノシタ、アジサイ

</details>

---

### 問題8：花序の型

**H. 以下の花序の型を説明せよ。**

1. 総状花序
2. 穂状花序
3. 散房花序
4. 散形花序
5. 頭状花序
6. 集散花序

<details>
<summary>解答を見る</summary>

1. **総状花序**：主軸に花柄のある花が下から順につく
   - 例：フジ、ナズナ

2. **穂状花序**：主軸に無柄の花が下から順につく
   - 例：オオバコ、イネ（小穂が集まった複穂状）

3. **散房花序**：下の花柄が長く、花が同じ高さに並ぶ
   - 例：ナシ、ナナカマド

4. **散形花序**：花柄が1点から放射状に出る
   - 例：セリ科、ウコギ科

5. **頭状花序**：多数の花が頭状に密集
   - 例：キク科

6. **集散花序**：頂端の花から順に咲く
   - 例：ムラサキ科（さそり状集散花序）

</details>

## 第3節 果実の形態

### 問題9：果実の分類

**I. 以下の果実の型を説明し、例を挙げよ。**

1. 液果
2. 核果
3. 梨果
4. 蒴果
5. 痩果
6. 穎果
7. 莢果

<details>
<summary>解答を見る</summary>

1. **液果**：果皮が多肉質で多汁
   - 例：トマト、ブドウ

2. **核果**：内果皮が硬い核になる
   - 例：ウメ、モモ、サクランボ

3. **梨果**：花托が発達して果肉となる（偽果）
   - 例：リンゴ、ナシ

4. **蒴果**：乾燥して裂開する果実
   - 例：アサガオ、ツバキ

5. **痩果**：1種子を含み、裂開しない小型の果実
   - 例：キク科、キンポウゲ

6. **穎果**：果皮と種皮が癒合
   - 例：イネ、コムギ（イネ科）

7. **莢果**：縫合線に沿って裂開
   - 例：マメ科

</details>

---

### 問題10：種子散布様式

**J. 以下の種子散布様式と、それをもつ植物例を挙げよ。**

1. 風散布
2. 動物散布（付着型）
3. 動物散布（被食型）
4. 水散布
5. 自動散布

<details>
<summary>解答を見る</summary>

1. **風散布**：種子や果実に翼や冠毛がある
   - 例：タンポポ（冠毛）、カエデ（翼果）

2. **動物散布（付着型）**：鉤や粘液で動物に付着
   - 例：オナモミ、ヌスビトハギ

3. **動物散布（被食型）**：果実を動物が食べて散布
   - 例：サクランボ、柿

4. **水散布**：種子が水に浮いて運ばれる
   - 例：ハマナス、ヤシ

5. **自動散布**：自らの力で種子を弾き出す
   - 例：ホウセンカ、スミレ

</details>

## 第4節 総合問題

### 問題11：用語の組み合わせ

**K. 以下の植物について、葉形、葉縁、葉序、花序、果実の型をそれぞれ答えよ。**

1. タンポポ
2. サクラ
3. ツバキ
4. アジサイ

<details>
<summary>解答を見る</summary>

1. **タンポポ**
   - 葉形：へら形～倒披針形
   - 葉縁：羽状に深裂
   - 葉序：根生（ロゼット）
   - 花序：頭状花序
   - 果実：痩果（冠毛つき）

2. **サクラ**
   - 葉形：卵形～楕円形
   - 葉縁：鋸歯縁～重鋸歯縁
   - 葉序：互生
   - 花序：散形花序または散房花序
   - 果実：核果

3. **ツバキ**
   - 葉形：楕円形～長楕円形
   - 葉縁：鋸歯縁
   - 葉序：互生
   - 花序：単生
   - 果実：蒴果

4. **アジサイ**
   - 葉形：広卵形
   - 葉縁：鋸歯縁
   - 葉序：対生
   - 花序：集散花序（装飾花と両性花）
   - 果実：蒴果

</details>

### 問題12：用語の正誤

**L. 以下の記述の正誤を答え、誤りがあれば訂正せよ。**

1. 「互生」とは、茎の各節に2枚の葉が向かい合ってつくことである。
2. 「下位子房」とは、子房が花被の下にあることを意味する。
3. 「総状花序」では、花は先端から咲き始める。
4. 「核果」の種子は、果皮の内側にある硬い殻（核）の中にある。
5. 「蝶形花冠」は、キク科の特徴である。

<details>
<summary>解答を見る</summary>

1. **誤り** → 互生は各節に1枚ずつつく。向かい合ってつくのは「対生」

2. **正しい** → 下位子房は花托が子房を包み込み、花被が子房の上につく

3. **誤り** → 総状花序は下（基部）から咲き始める（無限花序）

4. **正しい** → 核果は内果皮が硬化して核となり、その中に種子がある

5. **誤り** → 蝶形花冠はマメ科の特徴。キク科は頭状花序で管状花・舌状花

</details>
