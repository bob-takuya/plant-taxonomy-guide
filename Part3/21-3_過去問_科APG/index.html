    21-3. 過去問演習：科の特徴・APG分類¶

    出題頻度：★★★★☆
    科の特徴と APG 分類体系の理解は、専門問題の根幹となる分野です。

    🌿 科の特徴から科名を答える問題¶

    問題1（2021年度 問38 改題）¶

    次の記述にあてはまる科をひとつ選びなさい。

    多くは草本。花は小型で単性、花被片は2〜5個で、雄蕊は花被片と同数でそれと対生し、花糸は開花の時伸びて反り返る。

    1. アサ科
    2. イラクサ科
    3. タデ科
    4. ヒユ科
    解答を見る **正解：2** **イラクサ科（Urticaceae）**の特徴： - 草本が多い - 花は小型で**単性** - 花被片2〜5個、雄蕊が花被片と**対生** - 開花時に花糸が伸びて**反り返る**（この動作で花粉を飛ばす） タデ科は花被片5個で両性花が多く、反り返らない。

    問題2（2021年度 問39 改題）¶

    次の記述にあてはまる科をひとつ選びなさい。

    つる草。らせんに巻く巻きひげがある。花は単性で放射相称。萼片、花弁とも5個。子房下位で、柱頭は3裂。果実は液果または蓋果。

    1. ウリ科
    2. ガガイモ科
    3. サルトリイバラ科（シオデ科）
    4. マタタビ科
    解答を見る **正解：1** **ウリ科（Cucurbitaceae）**の特徴： - つる草・巻きひげあり（茎が変化） - 花は単性・放射相称・5数性 - **子房下位**・柱頭3裂 - 果実は液果（ウリ、カボチャなど）または蓋果 ウリ科の覚え方：「キュウリ・カボチャ・スイカ → 子房下位・単性花」

    問題3（2022年度 問28 改題）¶

    萼片、花弁とも4枚、四強雄蕊、二心皮性雌蕊を特徴とする科をひとつ選びなさい。

    1. Brassicaceae（アブラナ科）
    2. Saxifragaceae（ユキノシタ科）
    3. Lamiaceae（シソ科）
    4. Rubiaceae（アカネ科）
    解答を見る **正解：1** **アブラナ科（Brassicaceae）**の特徴： - 花弁4枚（十字形）→ 旧名「十字花科」 - **四強雄蕊**（4長2短の6本） - 二心皮性の雌蕊 - 角果（シリキュア/シリクル） アブラナ科の覚え方：「十字架のアブラナ、4枚の花弁、6本の雄しべ（4長2短）」

    問題4（2022年度 問29 改題）¶

    花冠は合弁の放射相称花で5裂し、雄蕊5本で、子房は4裂して分果をつくる科をひとつ選びなさい。

    1. ムラサキ科
    2. ナス科
    3. キキョウ科
    4. シソ科
    解答を見る **正解：1** **ムラサキ科（Boraginaceae）**の特徴： - 合弁花・放射相称・5裂 - 雄蕊5本（花弁と互生） - 子房が**4裂**→ 熟すと**4個の小堅果（分果）**に分離 ムラサキ科の覚え方：「ワスレナグサのさそり形花序、4つの小堅果」

    問題5（2022年度 問30 改題）¶

    葉は対生し、全縁、花は五数性で花柱は2、3、または5本、子房は独立中央胎座である科をひとつ選びなさい。

    1. サクラソウ科
    2. ナデシコ科
    3. リンドウ科
    4. キキョウ科
    解答を見る **正解：2** **ナデシコ科（Caryophyllaceae）**の特徴： - 葉は対生・全縁 - 茎の節が膨らむ - **独立中央胎座**（子房の中央に胎座が柱状に立つ） - 花柱2〜5本 - 代表種：カワラナデシコ、ハコベ、マンテマ

    🌿 APG分類体系¶

    APG IVの主要クレード¶

    被子植物
├── ANA grade（基部被子植物）
│   ├── アンボレラ目（Amborellales）
│   ├── スイレン目（Nymphaeales）
│   └── アウストロバイレヤ目（Austrobaileyales）
├── マグノリアクレード（旧「双子葉類」の一部）
│   ├── モクレン目（Magnoliales）→ モクレン科・バンレイシ科
│   ├── クスノキ目（Laurales）→ クスノキ科・ロウバイ科
│   └── その他
├── 単子葉類（Monocots）
│   ├── オモダカ目（Alismatales）→ オモダカ科・サトイモ科・海草含む
│   ├── ユリ目（Liliales）→ ユリ科
│   ├── キジカクシ目（Asparagales）→ ラン科・アヤメ科・ヒガンバナ科
│   └── ツユクサ類（Commelinids）→ イネ目・ヤシ目等
└── 真正双子葉類（Eudicots）
    ├── キンポウゲ目（Ranunculales）→ キンポウゲ科・ケシ科
    ├── コア真正双子葉類（Core Eudicots）
    │   ├── ナデシコ目（Caryophyllales）
    │   ├── バラ上群（Superrosids）
    │   │   ├── バラ目・マメ目・ブナ目
    │   │   └── アブラナ目・ムクロジ目等
    │   └── キク上群（Superasterids）
    │       ├── ミズキ目・ツツジ目
    │       ├── シソ目・ナス目
    │       └── キク目・セリ目等

    問題6（2022年度 問22 改題）¶

    APG分類体系III（2009）で原始的被子植物に含まれる種をひとつ選びなさい。

    1. カナクギノキ（クスノキ科）
    2. イチイガシ（ブナ科）
    3. マンサク（マンサク科）
    4. ケショウヤナギ（ヤナギ科）
    解答を見る **正解：1** カナクギノキは**クスノキ科**に属します。クスノキ科はクスノキ目（Laurales）に属し、これはマグノリアクレードの一員として「原始的被子植物」（基部被子植物＋マグノリアクレード）に含まれます。 - イチイガシ → ブナ科（Fagales）= 真正双子葉類 - マンサク → マンサク科（Saxifragales）= 真正双子葉類 - ケショウヤナギ → ヤナギ科（Malpighiales）= 真正双子葉類

    問題7（2021年度 問36 改題）¶

    最新の分類体系（APG IV）でモクレン類（マグノリア類）として分類される目をひとつ選びなさい。

    1. キンポウゲ目
    2. クスノキ目
    3. スイレン目
    4. ヤマグルマ目
    解答を見る **正解：2** **クスノキ目（Laurales）**はモクレン類（マグノリアクレード）に含まれます。 - キンポウゲ目 → 真正双子葉類の基部 - スイレン目 → ANA grade（基部被子植物） - ヤマグルマ目（Trochodendrales）→ 真正双子葉類（コア真正双子葉類）

    問題8（2022年度 問20 改題）¶

    真葉植物に分類される科をひとつ選びなさい。

    1. ヒカゲノカズラ科
    2. ミズニラ科
    3. イワヒバ科
    4. ゼンマイ科
    解答を見る **正解：4** ゼンマイ科（Osmundaceae）は**真葉植物**（大葉植物・シダ植物）に分類されます。 - ヒカゲノカズラ科・ミズニラ科・イワヒバ科 → **小葉植物**（石松類） 小葉植物と大葉植物の違い： - **小葉植物（石松類）**：葉が小さく維管束は1本（仮葉脈）→ ヒカゲノカズラ・イワヒバ・ミズニラ - **大葉植物（真葉植物）**：葉に複雑な葉脈あり → シダ植物の大多数

    問題9（2022年度 問21 改題）¶

    イワヒバ科の記述として、誤りであるものをひとつ選びなさい。

    1. 葉は同形または二形である。
    2. 同形胞子性である。
    3. 胞子嚢穂をつくる。
    4. 葉には小舌がある。
    解答を見る **正解：2** イワヒバ科（Selaginellaceae）は**異形胞子性**です（大胞子と小胞子の2種類）。 - 同形胞子性（胞子が1種類）= ヒカゲノカズラ科・本物のシダ植物の多く - 異形胞子性（胞子が2種類）= イワヒバ科・ミズニラ科・一部の水生シダ 小舌（リグラ）はイワヒバ科・ミズニラ科に特有の器官です。

    ウリ科 ウリ目（Cucurbitales） ウリ
