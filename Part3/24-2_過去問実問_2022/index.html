    24-2. 過去問実問演習（2022年度）全問・全選択肢¶

    出題形式：共通問題（問1〜19）+ 専門問題（問20〜57）
    APG IIIに基づく（2022年度の試験は「APGIII」と明記）

    共通問題（問1〜19）¶

    問1　昆虫・クモの排出器官¶

    1. 原腎管　2. ゴルジ体　3. ボーマン嚢　4. マルピーギ管 ✅

    解説：マルピーギ管は節足動物（昆虫・クモ類）の排出器官。原腎管は扁形動物。ボーマン嚢は脊椎動物の腎臓。

    問2　もっとも類縁の遠い動物門¶

    1. 環形動物　2. 線形動物 ✅　3. 軟体動物　4. 扁形動物

    解説：環形動物・軟体動物・扁形動物は「冠輪動物」のグループ。線形動物は「脱皮動物」で別系統。

    問3　ウシにはみられない歯¶

    1. 下顎の臼歯　2. 下顎の切歯　3. 上顎の臼歯　4. 上顎の切歯 ✅

    解説：ウシは草食反芻動物。上顎前方には切歯がなく（歯板のみ）、下顎の切歯で草をちぎる。

    問4　体外受精をする種¶

    1. アカウミガメ　2. ウミウ　3. トノサマガエル ✅　4. ハンドウイルカ

    問5　本州亜高山帯の高木ではない種¶

    1. オオシラビソ　2. コメツガ　3. トウヒ　4. モミ ✅

    解説：本州亜高山帯の主要高木：オオシラビソ・コメツガ・トウヒ・ダケカンバ等。モミは暖温帯〜山地帯（太平洋側）の樹種で亜高山帯には達しない。

    問6　種子に翼が発達している植物¶

    1. カエデ（翼果だが翼は果皮）　2. カタバミ　3. ナズナ　4. ヤマノイモ ✅

    解説：ヤマノイモの果実は3翼のある翼果（蒴果）で、種子自体に翼がある。カエデは翼果だが翼は果皮（外果皮）由来。ヤマノイモの翼は種皮由来。

    問7　APGでもっとも原始的な科¶

    1. キク科　2. マツブサ科 ✅　3. マメ科　4. ユリ科

    解説：マツブサ科（Schisandraceae）はモクレン目に属し、被子植物の中でも基部被子植物に近い原始的な科。キク科・マメ科は真正双子葉類の末端に近い進化した科。

    問8　子嚢菌類¶

    1. オオゴムタケ ✅　2. オニフスベ　3. ショウロ　4. シロキクラゲ

    解説：子嚢菌類（Ascomycota）の例：トリュフ・アミガサタケ・チャワンタケ・オオゴムタケ。担子菌類：オニフスベ・ショウロ・シロキクラゲ。

    問9　原核生物に関する誤りの記述¶

    1. 細菌と古細菌の2グループからなる。　2. 原核生物ではRNAが遺伝子の働きをする。 ✅（誤り）
    2. 細胞壁をもつ。　4. 細胞核をもたない。

    解説：原核生物もDNAが遺伝情報を担う。「RNAが遺伝子」は誤り（RNAウイルスとの混同）。

    問10　単細胞生物だけの組み合わせ¶

    1. ウミホタル－ヤコウチュウ　2. ワラジムシ－ゾウリムシ　3. ゾウリムシ－ヤコウチュウ ✅　4. ツリガネムシ－ハリガネムシ

    問11-1　アオコの分類：界¶

    1. 菌界　2. 原核生物界 ✅　3. 原生生物界　4. 植物界

    問11-2　アオコの分類：種類¶

    1. 渦鞭毛藻　2. 珪藻　3. 藍藻 ✅　4. 緑藻

    解説：アオコの主体はミクロキスティス等の藍藻（シアノバクテリア）。原核生物で光合成する。富栄養化した湖沼で大量発生し、毒素（ミクロキスチン等）を産生。

    問12　ブラキストン線の両側に分布する種の組み合わせ¶

    1. アカネズミ－フクロウ－カジカガエル－ジャノメチョウ　2. アカネズミ－ウミネコ－シマヘビ－オニヤンマ ✅
    2. ヤチネズミ－フクロウ－シマヘビ－オニヤンマ　4. ヤチネズミ－ウミネコ－カジカガエル－ジャノメチョウ

    ブラキストン線

    津軽海峡を境に動物相が異なる境界線。北海道（ユーラシア区）と本州（日本区）を区分。 ブラキストン線の両側に自然分布：鳥類は多くが両側に分布（ウミネコ・フクロウ等）。 ヤチネズミ・エゾリス等は北海道のみ。アカネズミは全国分布（両側）。

    問13　伊豆諸島の生物地理（正しい記述）¶

    1. 伊豆大島・新島のモリアオガエルは在来種。　2. コウモリ以外の哺乳類はアカネズミのみ。 3. オカダトカゲは伊豆半島にも自然分布する。 ✅　4. ミクラミヤマクワガタは御蔵島固有種。

    問14　学名の命名者と年の意味（正しい記述）¶

    1. 発見者、発見年。　2. 命名者、発見年。　3. 発見者、公表年。　4. 命名者、公表年。 ✅

    問15　日本での絶滅種¶

    1. オキナワオオコウモリ ✅　2. オガサワラオオコウモリ　3. ダイトウオオコウモリ　4. エラブオオコウモリ

    問16　カンブリアの大爆発¶

    1. 多様な海洋生物が一斉に分化したこと。 ✅　2. 巨大隕石落下による絶滅。　3. 巨大火山噴火。　4. 被子植物の繁栄と恐竜絶滅。

    問17　特定外来生物に指定されていないものが含まれる組み合わせ¶

    1. ヒアリ－カダヤシ－オオヒキガエル－オオハンゴンソウ　2. ツマアカスズメバチ－ブルーギル－ウシガエル－オオキンケイギク
    2. クビアカツヤカミキリ－オオクチバス－カミツキガメ－オオフサモ　4. アルゼンチンアリ－グッピー－グリーンアノール－ナルトサワギク ✅

    解説：グッピーは特定外来生物に指定されていない（観賞魚として飼育可能）。

    問18　国際動物命名規約（誤りの記述）¶

    1. 先に発表された学名が優先される（先取権の原則）。　2. 絶滅種も同じ規約に従う。 3. 属名を変更する際には、必ず種小名も変更しなければならない。 ✅（誤り）　4. 新種記載時にホロタイプを指定する。

    解説：属名変更（組み合わせ変更）の際、種小名はそのまま保持される（国際命名規約の基本原則）。命名者名は括弧内に入れる。

    問19　ランの送粉研究の先駆者¶

    1. ダーウィン ✅　2. ビュフォン　3. ヘッケル　4. ラマルク

    解説：チャールズ・ダーウィンは『ランの受粉』（1862）を著し、ランの巧妙な送粉機構を詳細に研究した。

    専門問題（問20〜57）¶

    問20　真葉植物（euphyllophyta）に分類される科¶

    1. ヒカゲノカズラ科　2. ミズニラ科　3. イワヒバ科　4. ゼンマイ科 ✅

    真葉植物とは

    真葉植物（Euphyllophyta）：大型葉（ megaphyll）をもつ植物。シダ植物（ゼンマイ科・オシダ科等）と種子植物が含まれる。 石松類（ヒカゲノカズラ科・ミズニラ科・イワヒバ科）は小型葉（microphyll）をもち、真葉植物ではない。

    問21　イワヒバ科の誤りの記述¶

    1. 葉は同形または二形である。　2. 同形胞子性である。 ✅（誤り）
    2. 胞子嚢穂をつくる。　4. 葉には小舌がある。

    イワヒバ科は異形胞子性

    イワヒバ科（Selaginella）は異形胞子性（heterosporous）：大胞子（雌性）と小胞子（雄性）の2種類の胞子をつくる。「同形胞子性」は誤り。小舌（ligule）はイワヒバ科とミズニラ科に特有の器官。

    問22　APG III（2009）で原始的被子植物に含まれる種¶

    1. カナクギノキ ✅　2. イチイガシ　3. マンサク　4. ケショウヤナギ

    解説：カナクギノキ（クスノキ科）はモクレン類（Magnoliids）に属す。APG IIIで「原始的被子植物」に位置づけられる基部被子植物（ANA clade＋モクレン類）の一つ。

    問23　アブラナ科に分類される属¶

    1. Aeschynomene（クサネム属、マメ科）　2. Barbarea（ヤマガラシ属、アブラナ科） ✅
    2. Convallaria（スズラン属、キジカクシ科）　4. Mallotus（アカメガシワ属、トウダイグサ科）

    問24　ニシキギ科に分類されない属¶

    1. ウメバチソウ属　2. クロヅル属　3. ツゲモドキ属 ✅　4. ツルウメモドキ属

    解説：ウメバチソウ属はAPG IVでニシキギ科に統合。ツゲモドキ（Maytenus）はニシキギ科に含まれることも多いが、本問では「含まれない」とされている。要確認。

    問25　アオイ科に分類されない種¶

    1. イチビ　2. オクラ　3. キササゲ ✅　4. フヨウ

    解説：キササゲはノウゼンカズラ科。イチビ・オクラ・フヨウはアオイ科。APGでは旧アオイ科にシナノキ科・ツバキ科の一部が統合されている。

    問26　キョウチクトウ科に分類されない種¶

    1. チトセカズラ ✅　2. テイカカズラ　3. キジョラン　4. カモメヅル

    解説：チトセカズラ（Lonicera japonica）はスイカズラ科。テイカカズラ・キジョラン・カモメヅルはキョウチクトウ科（旧ガガイモ科がAPGで統合）。

    問27　4種に共通でない記述（オサバグサ・トガクシソウ・シラネアオイ・レンゲショウマ）¶

    1. 日本の固有種である。　2. 単型属である。 3. キンポウゲ科に分類される。 ✅（共通でない）　4. 九州・琉球には分布しない。

    4種の科の所属

    種名 科（APG） 備考
    オサバグサ ケシ科 単型属、高山植物
    トガクシソウ メギ科 単型属
    シラネアオイ シラネアオイ科（→キンポウゲ科近縁説も） 単型属
    レンゲショウマ キンポウゲ科 単型属
    - キンポウゲ科でないものがある（オサバグサ＝ケシ科、トガクシソウ＝メギ科）

    問28　アブラナ科の特徴（萼片・花弁4枚・四長雄蕊・二心皮）¶

    1. Brassicaceae（アブラナ科） ✅　2. Saxifragaceae（ユキノシタ科） 3. Lamiaceae（シソ科）　4. Rubiaceae（アカネ科）

    問29　合弁5裂・雄蕊5・子房4裂して分果¶

    1. ムラサキ科 ✅　2. ナス科　3. キキョウ科　4. シソ科

    ムラサキ科の識別

    • 合弁花冠・5裂・放射相称（シソ科は唇形花）
    • 雄蕊5本
    • 子房は4裂し、各室1胚珠 → 分果4個（小堅果）
    • 花序は巻散（さそり形）花序
    • 代表：ムラサキ・ワスレナグサ・キュウリグサ

    問30　対生・全縁・5数性・花柱2-3-5・独立中央胎座¶

    1. サクラソウ科（独立中央胎座だが花柱1）　2. ナデシコ科 ✅
    2. リンドウ科（側膜胎座）　4. キキョウ科（側膜胎座）

    問31　オトギリソウ属の誤りの記述¶

    1. 葉は対生し、明点や黒点がある。　2. 花は一日花で、花弁は黄色。
    2. 雄蕊は多数で三体または五体雄蕊。　4. 果実は核果で、種子は1個。 ✅（誤り）

    解説：オトギリソウ属の果実は蒴果（3心皮・多数の種子）。核果ではない。

    問32-3 クスノキ科の完全雄蕊

    問35　花粉に粘着糸がある種¶

    1. シロヤシオ（ツツジ科） ✅　2. サラサドウダン　3. クロマツ　4. オニユリ

    解説：ツツジ科の多くは花粉粒が粘着糸（viscin thread）でつながり、塊状になる。これは昆虫媒花の適応。裸子植物（クロマツ）は粘着糸がない。

    問36　唇形花冠ではない科¶

    1. タヌキモ科　2. ハマウツボ科　3. キツネノマゴ科　4. モクセイ科 ✅

    解説：モクセイ科（ライラック・キンモクセイ等）は放射相称の合弁花冠（4裂）。唇形花冠をもつのはシソ科・タヌキモ科・ハマウツボ科・キツネノマゴ科等。

    問37　弁開葯（孔開葯）をもつ科¶

    1. メギ科 ✅　2. ツツジ科　3. キク科　4. ヤッコソウ科

    解説：弁開葯（valvate dehiscence）：葯が弁状に開裂して花粉を放出。メギ科（メギ・ナンテン）の特徴。

    問38　水中で開花・受粉する種¶

    1. ミズオオバコ（水面で開花）　2. トチカガミ（水面で開花）　3. イバラモ ✅　4. スブタ

    解説：イバラモ（Najas）は完全に水中で開花・受粉する（水中受粉）。ミズオオバコ・トチカガミは水面上で開花する。

    問39-1 オカトラノオは総状花序

    2021年度問26でも出題。オカトラノオの花序は「総状花序」（各花に花柄あり）。穂状花序ではない。繰り返し出題される重要ポイント。

    問40　葉が互生する種¶

    1. コンロンソウ（アブラナ科） ✅　2. トウネズミモチ（モクセイ科、対生） 3. キバナアキギリ（シソ科、対生）　4. ルリミノキ（アカネ科、対生）

    問41　葉が十字対生または三輪生し鱗片状・針状となる属¶

    1. Chamaecyparis（ヒノキ属）　2. Juniperus（ビャクシン属） ✅
    2. Thuja（クロベ属）　4. Thujopsis（アスナロ属）

    解説：ビャクシン属（Juniperus）は葉が三輪生する針葉または鱗片葉。ヒノキ属・クロベ属・アスナロ属は十字対生の鱗片葉のみ（針葉は持たない）。

    問42　葉に蜜腺がない種¶

    1. ナンキンハゼ　2. シラキ　3. イイギリ　4. ヒメコウゾ ✅

    解説：トウダイグサ科（ナンキンハゼ・シラキ）は葉柄基部または葉身基部に蜜腺を持つ。イイギリも蜜腺あり。ヒメコウゾ（クワ科）は蜜腺なし。

    問43　托葉を起源とする棘をもつ種¶

    1. ハリエンジュ（ニセアカシア、マメ科） ✅　2. カギカズラ（茎の変形） 3. サイカチ（枝の変形）　4. ボケ（枝の変形）

    解説：ハリエンジュの棘は托葉が硬化したもの（托葉棘）。カギカズラの棘は葉柄変形、サイカチは枝変形、ボケは短枝変形。

    問44　托葉を起源とする巻きひげをもつ属¶

    1. Vicia（ソラマメ属、葉の先端変化）　2. Smilax（サルトリイバラ属） ✅
    2. Parthenocissus（ツタ属、茎変形）　4. Clematis（センニンソウ属、葉柄変形）

    解説：Smilax（サルトリイバラ属）の巻きひげは托葉変形。Viciaの巻きひげは葉の先端（小葉）の変形。

    問45　1回結実性の多年草¶

    1. イワレンゲ ✅　2. キツネノカミソリ　3. ムサシアブミ　4. クロユリ

    解説：1回結実性多年草（一回繁殖型）：数年間栄養成長した後、1回だけ開花・結実して枯死。イワレンゲ（ベンケイソウ科）はその典型。

    問46　果実が袋果の属¶

    1. シモツケ属（バラ科） ✅　2. バラ属（偽果）　3. ダイコンソウ属（痩果）　4. リンゴ属（梨状果）

    問47　アブラナ科で裂開しない果実をもつ属¶

    1. ダイコン属 ✅　2. タネツケバナ属（長角果）　3. ナズナ属（短角果）　4. ワサビ属（長角果）

    解説：ダイコンの果実は海綿状で不裂開（ロメント果の変形）。通常アブラナ科の果実は角果（長角果・短角果）だが、ダイコン属は例外。

    問48　半寄生性である種¶

    1. シコクマコナ（ハマウツボ科） ✅　2. ムジナモ（食虫植物・非寄生） 3. ツクバネソウ（非寄生）　4. アメリカネナシカズラ（全寄生）

    解説：半寄生：葉緑体をもち光合成するが、他植物の根から水・無機塩類を吸収。シコクマコナ（コシオガマ属近縁）は根半寄生。全寄生（ネナシカズラ等）は葉緑体なし。

    問49　菌従属栄養植物ではない種¶

    1. オニク ✅（全寄生植物・ハマウツボ科）　2. ギンリョウソウ（菌従属） 3. タヌキノショクダイ（菌従属）　4. タシロラン（菌従属）

    解説：菌従属栄養植物（腐生植物）はすべて菌根菌に栄養依存。オニクは被子植物の根に寄生する全寄生植物（菌類には依存しない）。

    問50　カブの食用部分¶

    1. 幼芽　2. 子葉　3. 胚軸 ✅　4. 主根

    解説：カブの白い食用部分は胚軸（hypocotyl）が肥大したものが主体（下部の主根も一部含む）。ダイコンは胚軸＋根の両方。サツマイモは根（塊根）。ジャガイモは茎（塊茎）。

    問51　本州中部から琉球に分布する種¶
