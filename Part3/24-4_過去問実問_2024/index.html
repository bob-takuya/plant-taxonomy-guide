    24-4. 過去問実問演習（2024年度）全問・全選択肢¶

    出題形式：共通問題（問1〜20）+ 専門問題（問21〜60）

    共通問題（問1〜20）¶

    問1　胎生の種が含まれていない動物群¶

    1. 鳥類 ✅　2. 爬虫類　3. 哺乳類　4. 魚類

    解説：鳥類は全て卵生。爬虫類・哺乳類・魚類には胎生の種が存在する（ウミヘビ、有袋類、サメの一部等）。

    問2　閉鎖血管系をもつ動物門¶

    1. 環形動物門 ✅　2. 棘皮動物門　3. 刺胞動物門　4. 節足動物門

    問3　一生を通じて脊索がある動物群¶

    1. ヒモムシ類　2. ナメクジウオ類 ✅　3. ホシムシ類　4. ニムシ類

    問4　完全変態をする種だけの組み合わせ¶

    1. クロオオアリ−クロゴキブリ−ウシアブ　2. ヤマトシロアリ−トビナナフシ−ウスバカゲロウ 3. ゴマダラカミキリ−モンキチョウ−キムネクマバチ ✅　4. アブラゼミ−ナナホシテントウ−シオカラトンボ

    問5　シダ植物¶

    1. クジラグサ（被子植物）　2. カニクサ（シダ植物） ✅　3. ムシクサ（被子植物）　4. ウリクサ（被子植物）

    問6　藻類¶

    1. ホシツリモ（車軸藻類・緑藻） ✅　2. ウミヒルモ（海草・被子植物） 3. エビモ（水草・被子植物）　4. バイカモ（水草・被子植物）

    問7　菌根をつくらない組み合わせ¶

    1. アカヤマドリ−コナラ（外生菌根）　2. ショウロ−クロマツ（外生菌根） 3. ツキヨタケ−ブナ ✅（腐生菌、菌根形成しない）　4. ベニテングタケ−シラカンバ（外生菌根）

    問8　被子植物の記述（誤り）¶

    1. 仮導管をもつものがある。　2. 重複受精をする。 3. 精子をつくるものがある。 ✅（誤り）　4. 胚珠は子房に包まれる。

    解説：被子植物の雄性配偶体（花粉管）は精細胞（非運動性）を送り込む。精子（べん毛で泳ぐ）をつくるのはシダ植物・コケ植物・ソテツ・イチョウ（裸子植物の古い系統）。

    問9　原核生物¶

    1. スイゼンジノリ（藍藻） ✅　2. ヒカリモ（黄緑藻）　3. ミドリムシ（ユーグレナ）　4. カワノリ（緑藻）

    問10　単細胞生物だけの組み合わせ¶

    1. ゾウリムシ−クロレラ ✅　2. ミジンコ−ヒカリモ（ミジンコは多細胞） 3. ワムシ−ミカヅキモ（ワムシは多細胞）　4. コケムシ−ユレモ（コケムシは多細胞）

    問11　カブトクラゲが属する動物門¶

    1. 棘皮動物門　2. 扁形動物門　3. 有櫛動物門 ✅　4. 刺胞動物門

    問12　棘皮動物の誤りの記述¶

    1. 基本的体制は五放射相称である。　2. 体表が殻や棘で覆われていることが多い。
    2. 中枢神経系や独立した排出系は存在しない。　4. 口も肛門も体の腹側に位置する。 ✅（誤り）

    解説：棘皮動物は口が腹側・肛門が背側に位置する（ウニ・ヒトデ等）。

    問13　トロコフォア幼生がある動物群¶

    1. 環形動物 ✅　2. 節足動物　3. 棘皮動物　4. 輪形動物

    問14　沖縄県にのみ自然分布する種の組み合わせ¶

    1. ニッパヤシ−オキナワトゲネズミ−リュウキュウウラボシシジミ−ヤンバルクイナ ✅

    問15　アロエ・ストレリチア・マツバギクが属する植物区系¶

    1. オーストラリア区　2. 旧熱帯区　3. ケープ区 ✅　4. 新熱帯区

    解説：アロエ・ストレリチア・マツバギク（Aizoaceae）はいずれも南アフリカ（ケープ植物区）を起源とする植物。

    問16　環境省レッドリスト2020の絶滅危惧種合計数¶

    1. 約2,700種　2. 約3,200種　3. 約3,700種 ✅　4. 約4,200種

    問17　シダ植物が地球上に出現した時期¶

    1. 約2億年前　2. 約4億年前 ✅　3. 約6億年前　4. 約8億年前

    問18　特定第二種国内希少野生動植物種¶

    1. ケナガネズミ　2. シジュウカラガン　3. ゲンゴロウ　4. タデスミレ ✅

    問19　特定外来生物に指定されていない種¶

    1. アレチウリ　2. セアカゴケグモ　3. ソウシチョウ　4. モウソウチク ✅

    解説：モウソウチクは生態系への影響はあるものの、現時点で特定外来生物には指定されていない（農業・林業等への利用があるため）。

    問20　コムギで遺伝学に貢献した人¶

    1. 木原均 ✅　2. 木村資生　3. 今西錦司　4. 八杉龍一

    解説：木原均はコムギの倍数性（ゲノム構造）を解明した遺伝学者。木村資生は中立進化論。

    専門問題（問21〜60）¶

    問21　シャジクモ類の誤りの記述¶

    1. クロロフィルa+bをもつ。　2. 生卵器と造精器をつくり卵生殖する。 3. 胞子体と配偶体があり世代交代する。 ✅（誤り）　4. 陸上植物にもっとも近縁な藻類。

    解説：シャジクモ類は藻類であり、配偶体のみ（有性生殖世代＝n）が生活する。明確な胞子体世代（2n）はなく、受精卵が直接発芽する。

    問22　包膜をもつ種¶

    1. イワガネソウ　2. ノキシノブ　3. ミゾシダ　4. リョウメンシダ ✅（？）

    解説：シダ植物の包膜（indusium）は胞子嚢群（ソーラス）を覆う薄い膜。種によって形状が異なる（腎形・杯形・線形等）。

    問23　オオハナワラビ・ゼンマイ・マメヅタ・コウヤワラビの共通特徴¶

    1. 栄養葉と胞子葉がある。 ✅　2. 常緑性である。　3. 根茎は長くはう。　4. 北海道には分布しない。

    問24　モミ属の種¶

    1. エゾマツ（トウヒ属）　2. ヒメバラモミ（トウヒ属）　3. ハリモミ（トウヒ属）　4. トドマツ ✅（モミ属・Abies）

    問25　クロウメモドキ科に属さない種¶

    1. ケンポナシ　2. クマヤナギ　3. ムクノキ（アサ科） ✅　4. ヨコグラノキ

    問26　ベンケイソウ科に属さない種¶

    1. アズマツメクサ　2. イワボタン（ユキノシタ科） ✅　3. ミセバヤ　4. ヒメキリンソウ

    問27　ヤマブキの学名¶

    1. Kerria japonica（バラ科ヤマブキ属） ✅　2. Angelica japonica（セリ科） 3. Persicaria japonica（タデ科）　4. Swertia japonica（リンドウ科）

    問28　植物の学名に関する誤りの記述¶

    1. 属名は1個の単語。　2. 属名の語源はギリシャ語またはラテン語に限られる。 ✅（誤り）
    2. 種の学名は属名と形容語の2語。　4. 科の学名は属名から語尾-aceaeで作る。

    解説：属名の語源はギリシャ語・ラテン語に限られない。人名・地名・民族語など様々な語源が使われる（ただしラテン語として扱われる）。

    問29　種子植物の成長に関する正しい記述¶

    1. 芽が根から発生することはない。（誤り：不定芽が根から出ることがある）
    2. 芽が葉から発生することはない。（誤り：葉から不定芽）
    3. 根が葉から発生することはない。（誤り：葉から不定根） 4. 葉が根から発生することはない。 ✅

    解説：芽・根は茎・根・葉どこからでも（不定芽・不定根として）発生しうるが、葉（葉身や葉柄）は根から直接発生しない。

    問30　水草と生育形の正しい組み合わせ¶

    1. アサザ：抽水植物（誤り→浮葉植物）　2. ハス：浮葉植物（誤り→浮葉） 3. ミズオオバコ：沈水植物 ✅　4. エビモ：浮遊植物（誤り→沈水植物）

    問31　葉は網状脈の種（単子葉類だが網状脈）¶

    1. ミヤマエンレイソウ（ユリ科・網状脈） ✅　2. クマガイソウ（ラン科） 3. コオニユリ（ユリ科・平行脈）　4. ナルコユリ（キジカクシ科・平行脈）

    解説：ミヤマエンレイソウ（Trillium tschonoskii）は単子葉類だが葉に顕著な網状脈をもつ（3枚の葉がホウの枝先に輪生）。単子葉類でも網状脈をもつ例外：エンレイソウ属・ヤマノイモ属等。

    問32　鋸歯縁の葉をもつ種（モチノキ科）¶

    1. クロガネモチ（全縁）　2. ソヨゴ（全縁）　3. ナナミノキ（鋸歯） ✅　4. モチノキ（全縁）

    問33　全縁の葉をもつ種¶

    1. カナメモチ（鋸歯縁）　2. ヤマグルマ（全縁） ✅　3. ガンピ（全縁）→正解はヤマグルマ　4. イイギリ（鋸歯縁）

    問34　葉が対生する種¶

    1. シロザ（互生）　2. シモツケソウ（互生）　3. シモバシラ（シソ科・対生） ✅　4. シデシャジン（互生）

    問35　誤りの記述（攀縁の方法）¶

    1. ツタは茎が変化した巻きひげで吸盤をもつ。　2. ジャケツイバラは短枝先端が棘になる。 2. ジャケツイバラは短枝の先が棘になり… ✅（誤り）

    正解は2：ジャケツイバラの棘は枝の変形（托葉変形ではなく枝棘）。短枝が棘に変化するのはボケ等。

    問36　ツツジ科で花粉に粘着糸をもたない種¶

    1. アカヤシオ　2. バイカツツジ　3. ウラジロヨウラク　4. ウスノキ ✅

    解説：ツツジ科の多くは粘着糸をもつが、ウスノキ（Vaccinium smallii）は粘着糸がない。ワクシン属（ブルーベリー等）は粘着糸なし。

    問37　頭状花序をつくらない属¶

    1. ネムノキ属（頭状花序）　2. スズカケノキ属（頭状花序）　3. ミツマタ属（頭状花序）　4. ガマズミ属（散房花序または円錐花序） ✅

    問 花序 正解

    問42　果実タイプとその例の誤りの組み合わせ¶

    1. 長角果：タネツケバナ属　2. 穎果：カヤツリグサ属 ✅（誤り）
    2. 節果：クサネム属　4. 核果：サクラ属

    解説：穎果（caryopsis）はイネ科の果実（果皮と種皮が融合）。カヤツリグサ属（カヤツリグサ科）の果実は痩果（nut）（果皮と種皮が分離可能）。

    問43　胎生種子をつくる種¶

    1. オオバヒルギ（ヒルギ科、マングローブ） ✅　2. ニッパヤシ　3. ハマザクロ　4. ヒルギモドキ

    解説：胎生種子（viviparous seed）：種子が親植物についたまま発芽し、ある程度まで成長してから脱落する。マングローブ植物に多い適応（干潟に落下してすぐ根付く）。オオバヒルギ等ヒルギ科が代表。

    問44　合心皮雌蕊による裂開果¶

    1. 蒴果 ✅　2. 袋果（一心皮）　3. 痩果（不裂開）　4. 豆果（一心皮）

    問45　種皮¶

    1. 玄米の褐色部分（果皮＋種皮が一体化）　2. ハルニレの翼（果皮）
    2. ウメの核（内果皮）　4. クリの渋皮（種皮） ✅

    解説：クリの「渋皮」は種皮（testa）。外側の硬い殻（鬼皮）は果皮（内果皮）。タンニンを含む渋皮が種皮。

    問46　誤りの記述¶

    1. フジカンゾウの豆果−全面に鉤毛がある。　2. フサザクラの翼果−ゆがんだ倒卵形。
    2. カツラの果実は袋果で種子に翼がある。　4. ムベの果実は蒴果で多数の種子がある。 ✅（誤り）

    解説：ムベ（アケビ科）の果実は液果（berry）（裂開しない）。種子は多数だが蒴果ではない。アケビも液果（熟すと裂開するが、分類上は液果）。

    問47　二数性・4裂・雄蕊2本の科¶

    解説：モクセイ科の特徴： - 花冠は放射相称・4裂（合弁） - 雄蕊2本（他の科は4〜5本が多い） - 子房上位・2室 - 代表：モクセイ・キンモクセイ・ライラック・オリーブ・アオダモ
