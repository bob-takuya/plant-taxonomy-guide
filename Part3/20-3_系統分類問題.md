## はじめに

この章では、APG体系を中心とした系統分類に関する演習問題を収録しています。分類体系の変更点を正確に理解することが重要です。

---

## 第1節 APG体系の基礎

### 問題1：APG体系の概要

**A. 以下の問いに答えよ。**

1. APGとは何の略称か。
2. APG体系は何に基づいて構築されているか。
3. 最新のAPG体系（2016年発表）のバージョン名は何か。
4. 被子植物の主要な系統群を4つ挙げよ。

<details>
<summary>解答を見る</summary>

1. **Angiosperm Phylogeny Group**（被子植物系統群）

2. **分子系統解析**（DNA配列に基づく解析）

3. **APG IV**

4. 以下の4つ（他の回答も可）
   - **ANA grade**（基部被子植物）
   - **モクレン類**
   - **単子葉類**
   - **真正双子葉類**

</details>

---

### 問題2：被子植物の系統樹

**B. 以下の被子植物の系統関係について、正しいものを選べ。**

1. 単子葉類の姉妹群は？
   - a) モクレン類
   - b) 真正双子葉類
   - c) マグノリア類

2. バラ類の姉妹群は？
   - a) モクレン類
   - b) キク類
   - c) 単子葉類

3. スイレン目はどの系統群に属するか？
   - a) 基部被子植物（ANA grade）
   - b) モクレン類
   - c) 真正双子葉類

<details>
<summary>解答を見る</summary>

1. **a) モクレン類**（単子葉類はモクレン類と姉妹群関係）

2. **b) キク類**（バラ類とキク類は姉妹群）

3. **a) 基部被子植物（ANA grade）**

</details>

## 第2節 科の分類変更

### 問題3：APG体系での科の変更

**C. 以下の植物群について、旧分類とAPG分類での科名を答えよ。**

1. カエデ属（*Acer*）
2. トチノキ属（*Aesculus*）
3. クスノキ属（*Cinnamomum*）
4. オモト属（*Rohdea*）
5. アジサイ属（*Hydrangea*）

<details>
<summary>解答を見る</summary>

| 植物群 | 旧分類 | APG分類 |
|--------|--------|---------|
| 1. カエデ属 | カエデ科 | **ムクロジ科** |
| 2. トチノキ属 | トチノキ科 | **ムクロジ科** |
| 3. クスノキ属 | クスノキ科 | **クスノキ科**（変更なし） |
| 4. オモト属 | ユリ科 | **キジカクシ科** |
| 5. アジサイ属 | ユキノシタ科 | **アジサイ科** |

</details>

### 問題4：旧ユリ科の分割

**D. 旧ユリ科に含まれていた以下の属は、APG体系でどの科に分類されるか。**

1. ユリ属（*Lilium*）
2. チューリップ属（*Tulipa*）
3. ギボウシ属（*Hosta*）
4. スズラン属（*Convallaria*）
5. ヤブラン属（*Liriope*）
6. アロエ属（*Aloe*）
7. ネギ属（*Allium*）

<details>
<summary>解答を見る</summary>

1. ユリ属 → **ユリ科**（そのまま）
2. チューリップ属 → **ユリ科**
3. ギボウシ属 → **キジカクシ科**
4. スズラン属 → **キジカクシ科**
5. ヤブラン属 → **キジカクシ科**
6. アロエ属 → **ツルボラン科**（または広義キジカクシ科）
7. ネギ属 → **ヒガンバナ科**

</details>

---

### 問題5：ゴマノハグサ科の再編

**E. 旧ゴマノハグサ科に含まれていた以下の属は、APG体系でどの科に分類されるか。**

1. オオバコ属（*Plantago*）
2. クワガタソウ属（*Veronica*）
3. ジギタリス属（*Digitalis*）
4. ゴマノハグサ属（*Scrophularia*）
5. ママコナ属（*Melampyrum*）

<details>
<summary>解答を見る</summary>

1. オオバコ属 → **オオバコ科**
2. クワガタソウ属 → **オオバコ科**
3. ジギタリス属 → **オオバコ科**
4. ゴマノハグサ属 → **ゴマノハグサ科**（狭義、少数の属のみ）
5. ママコナ属 → **ハマウツボ科**

</details>

---

### 問題6：キョウチクトウ科への統合

**F. 旧ガガイモ科に含まれていた以下の属は、APG体系でどの科に分類されるか。また、その理由を述べよ。**

1. ガガイモ属（*Metaplexis*）
2. イケマ属（*Cynanchum*）
3. カモメヅル属（*Vincetoxicum*）

<details>
<summary>解答を見る</summary>

すべて**キョウチクトウ科**に含まれる

**理由**：分子系統解析の結果、旧ガガイモ科は旧キョウチクトウ科の中に入れ子になっており、両者を合わせて1つの科（キョウチクトウ科）とするのが系統的に適切であるため。

</details>

**I. 以下の問いに答えよ。**

1. PPGとは何の略称か。
2. 薄嚢シダ類と厚嚢シダ類の違いを述べよ。
3. 小葉植物に含まれる科を2つ挙げよ。

<details>

   - **厚嚢シダ類**：胞子嚢壁が多層の細胞からなる

   - **イワヒバ科**
   - **ミズニラ科**

</details>

## 第5節 総合問題

### 問題11：系統分類の正誤

**K. 以下の記述の正誤を答え、誤りがあれば訂正せよ。**

1. APG体系では、単子葉類と双子葉類という分類は廃止された。
2. カエデ科は独立した科として認められている。
3. アジサイ属はバラ目に含まれる。
4. キク科はキク類（キキョウ類）に属する。
5. トクサ類は薄嚢シダ類に含まれる。

<details>
<summary>解答を見る</summary>

1. **正しい** → 「双子葉類」は側系統群であり、「真正双子葉類」と「基部被子植物＋モクレン類」に分けられた

2. **誤り** → カエデ科はムクロジ科に統合された

3. **誤り** → アジサイ属は**ミズキ目**に含まれる

4. **正しい** → キク科はキク類（Campanulids）のキク目に属する

5. **誤り** → トクサ類は薄嚢シダ類ではなく、**トクサ綱**として独立（厚嚢シダでもない）

</details>

a) APG IV
b) エングラー体系
c) クロンキスト体系
d) APG I

<details>
<summary>解答を見る</summary>

**b) → c) → d) → a)**

- d) APG I（1998年）
