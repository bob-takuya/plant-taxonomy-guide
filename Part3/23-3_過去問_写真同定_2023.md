# 23-3. 過去問：写真同定問題（2023年度）

> **出題形式**：問53（植物写真10枚同定）、問54-5（特徴記述）  
> **対象種数**：11種  
> 高山植物・湿地植物・半寄生植物など、難度の高い種が集中した年度。

## 問53：写真同定 10問（2023年度）

### 53-1. シコタンソウ *Saxifraga bronchialis*

**科・目**：ユキノシタ科（Saxifragaceae）- ユキノシタ目

| 項目 | 内容 |
|------|------|
| 生活型 | 常緑多年生草本（マット状） |
| 葉 | 小型・線形、硬くて先が刺状、密に互生 |
| 花 | **白色5弁、花弁に黄・赤のドット（斑点）模様** |
| 花期 | 7〜8月 |
| 分布 | 北海道〜本州北部・中部、**高山帯の岩礫地** |

!!! success "花弁のドット模様が決め手"
    白い花弁に黄色や橙赤色の斑点が散る。同じ高山帯でも斑点のある5弁花はシコタンソウと特定できる。

!!! warning "試験頻出ポイント"
    - 「色丹草」= 北方領土の色丹島に多い → 北方系の高山植物
    - ユキノシタ科 → ユキノシタ目（APG IV）
    - マット状 + 斑点花弁 → 高山の岩礫地

---

### 53-2. ヒゲネワチガイソウ *Pseudostellaria palibiniana*

**科・目**：ナデシコ科（Caryophyllaceae）- ナデシコ目

| 項目 | 内容 |
|------|------|
| 生活型 | 多年生草本 |
| 葉 | 対生、卵形〜広卵形 |
| 花 | 白色、花弁5枚・先が浅く2裂、**萼片に長い白毛（ヒゲ）** |
| 分布 | 北海道〜本州、山地の林床 |

!!! warning "試験頻出ポイント"
    - ナデシコ科の特徴：節がある茎、対生葉、花弁先端が2裂
    - 「髭根」= 萼片の白毛（ヒゲ）が名前の由来
    - ワチガイソウ属（*Pseudostellaria*）は日本に数種分布

---

### 53-3. ツルボ *Scilla scilloides*（= *Barnardia japonica*）

**科・目**：キジカクシ科（Asparagaceae）- キジカクシ目（APG IV）

| 項目 | 内容 |
|------|------|
| 生活型 | 鱗茎性多年生草本 |
| 葉 | 線形2枚、開花時は枯れている場合が多い |
| 花 | **淡紅紫色、総状花序、多数の花が密につく** |
| 花期 | 8〜9月 |
| 分布 | 全国、日当たりのよい草地・土手 |

!!! info "APGによる科の変更"
    旧分類：ユリ科 → APG IVでキジカクシ科（Asparagaceae）に。

!!! warning "試験頻出ポイント"
    - 夏に葉が枯れて花が咲く（ヒガンバナと同じパターン）
    - 淡紅紫色の小花が穂状に密集 → ツルボ

---

### 53-4. ミヤマアケボノソウ *Swertia perennis*

**科・目**：リンドウ科（Gentianaceae）- リンドウ目

| 項目 | 内容 |
|------|------|
| 生活型 | 多年生草本 |
| 葉 | 対生、基部葉はへら形〜卵形 |
| 花 | **白色〜淡紫色、5弁、各花弁に緑色の蜜腺窩2個（目立つ）** |
| 花期 | 8〜9月 |
| 分布 | 北海道〜本州中部、**亜高山帯〜高山帯の湿地・沢沿い** |

!!! success "蜜腺窩が決め手"
    花弁中央部にある2個の緑色のへこみ（蜜腺窩）が最大の特徴。拡大写真でこれが見えたらミヤマアケボノソウと断言できる。

!!! warning "試験頻出ポイント"
    - アケボノソウ属（*Swertia*）= センブリの仲間
    - センブリとの違い：センブリは低地〜山地・秋の1年草、ミヤマアケボノソウは高山帯・多年草
    - リンドウ目 → シソ目に近縁（APG IV）

---

### 53-5. ハマニガナ *Ixeris repens*

**科・目**：キク科（Asteraceae）- キク目

| 項目 | 内容 |
|------|------|
| 生活型 | 多年生草本（匍匐茎で砂上を這う） |
| 葉 | 多肉質、**腎形〜円形、深く切れ込む（掌状3〜5裂）** |
| 花 | **黄色の舌状花のみ**（筒状花なし）、直径2〜3cm |
| 花期 | 4〜7月 |
| 分布 | 全国、**海岸の砂浜・砂丘** |

!!! warning "試験頻出ポイント"
    - 海岸砂浜に這って広がる、黄色頭花のみ（タンポポ様）
    - 葉が多肉質で深く3〜5裂 → ハマニガナ
    - ニガナ（*Ixeris polycephala*、山野の普通種）との違い：**ハマニガナは砂浜・葉が深裂**

---

### 53-6. ツクバネ *Buckleya lanceolata*

**科・目**：ビャクダン科（Santalaceae）- ビャクダン目（APG IV）

| 項目 | 内容 |
|------|------|
| 生活型 | 落葉低木 |
| 寄生性 | **ツガ・カラマツ等の針葉樹の根に半寄生** |
| 葉 | 卵形〜披針形、対生 |
| 花 | 小型・緑色、目立たない |
| 果実 | **緑色の核果、先に4枚の葉状苞が羽根つきの羽根のように残る** |
| 分布 | 本州〜四国（太平洋側）、山地林内 |

!!! info "APGによる科の変更"
    旧分類：ツクバネ科（Buckleyaceae）→ APG IVではビャクダン科（Santalaceae）に統合。

!!! success "果実の形が超ユニーク"
    果実の先端に4枚の苞（葉状）が残り、まるで「羽根つきの羽根（シャトル）」のような形になる。この果実を見れば即答できる。

!!! warning "試験頻出ポイント"
    - 針葉樹根への半寄生 → 自分でも光合成する（全寄生ではない）
    - 羽根つき様の果実 → 写真で見たら迷わずツクバネ

---

### 53-7. ウワバミソウ *Elatostema involucratum*

**科・目**：イラクサ科（Urticaceae）- バラ目（APG IV）

| 項目 | 内容 |
|------|------|
| 生活型 | 多年生草本 |
| 葉 | 互生、楕円形〜長楕円形、鋸歯、**著しく左右非対称** |
| 花 | 小型・目立たない、雌雄同株 |
| 茎 | 半透明・多汁（食用） |
| 分布 | 全国、**渓谷の岩壁・湿った林内** |

!!! info "食用植物"
    茎が多汁でシャキシャキした食感。山菜として「ミズ」の名で知られる（ミズタデとも）。

!!! warning "試験頻出ポイント"
    - イラクサ科 → APGではバラ目（旧分類ではイラクサ目）
    - 葉の著しい左右非対称 → イラクサ科の特徴
    - 渓谷・沢沿いの湿地に密生

---

### 53-8. ムサシアブミ *Arisaema ringens*

**科・目**：サトイモ科（Araceae）- ショウブ目

| 項目 | 内容 |
|------|------|
| 生活型 | 多年生草本（球茎） |
| 葉 | 3出複葉、2枚の葉が向き合う |
| 花 | **仏炎苞（佛焔苞）に覆われた肉穂花序**、仏炎苞の口辺部が強く前に折れ曲がる |
| 仏炎苞 | 暗紫色〜緑色に白い縞模様、**縁が耳のように水平に張り出す** |
| 分布 | 本州（関東以西）〜沖縄、暖温帯林床 |

!!! success "仏炎苞の形が独特"
    仏炎苞の開口部の縁が横に張り出し、武蔵の鐙（あぶみ）の形に見える。この張り出した「耳」がムサシアブミの最大の識別点。

!!! warning "試験頻出ポイント"
    - サトイモ科の基本：肉穂花序 + 仏炎苞
    - テンナンショウ属（*Arisaema*）は日本に50種以上
    - ウラシマソウ（付属体が長い糸状）、マムシグサ（仏炎苞の縁に耳なし）との区別

---

### 53-9. ザゼンソウ *Symplocarpus foetidus* var. *latissimus*（= *S. renifolius*）

**科・目**：サトイモ科（Araceae）- ショウブ目

| 項目 | 内容 |
|------|------|
| 生活型 | 多年生草本 |
| 発熱 | **発熱植物（肉穂花序が発熱し、約22℃を保つ）**、雪を溶かして開花 |
| 仏炎苞 | 暗紫褐色〜暗赤色、丸みがある（座禅を組む姿） |
| 葉 | 花後に展開、大型・心形 |
| 臭気 | 腐臭（スカンク臭）→ 英名 Skunk Cabbage |
| 分布 | 北海道〜本州中部、湿地・湿った林床 |

!!! success "春一番の発熱植物"
    肉穂花序が発熱して雪を溶かし、2〜3月に真っ先に咲く。花序温度は外気が0℃でも22℃前後を維持する。ミトコンドリア呼吸による発熱。

!!! warning "試験頻出ポイント"
    - 発熱植物 → 試験に超頻出
    - 暗紫褐色の丸い仏炎苞 + 早春に雪の中で開花 → 即ザゼンソウ
    - 「座禅草」= 仏炎苞の形が座禅を組む達磨大師に見える

---

### 53-10. ユウゲショウ / アカバナユウゲショウ *Oenothera rosea*

**科・目**：アカバナ科（Onagraceae）- フトモモ目

| 項目 | 内容 |
|------|------|
| 生活型 | 多年生草本（帰化植物） |
| 葉 | 互生、長楕円形〜披針形、波状鋸歯 |
| 花 | **淡紅〜濃桃色の4弁花**、直径1〜1.5cm |
| 花期 | 5〜9月 |
| 原産 | 南アメリカ |

!!! info "和名の由来"
    「夕化粧」= 夕方に開花するイメージ（実際は昼間も咲く）。「アカバナ」はアカバナ科の和名から。

!!! warning "試験頻出ポイント"
    - アカバナ科の特徴：4弁花・4萼片・花柱先端が十字状
    - 4弁の桃色花 + 路傍・草地 → ユウゲショウ
    - マツヨイグサ類（黄色・夜咲き）と同科だが花色・開花時間が異なる

---

## 問54-5. アオキ / ガリア *Aucuba japonica*

**科・目**：ガリア科（Garryaceae）- ガリア目（APG IV）

| 項目 | 内容 |
|------|------|
| 生活型 | 常緑低木 |
| 葉 | 対生、**光沢のある革質の大型楕円形葉**、鋸歯あり |
| 花 | 暗紫色、花弁4枚、雌雄異株 |
| 果実 | **鮮やかな赤色の核果**（冬に熟す） |
| 分布 | 本州〜沖縄、林床（耐陰性が高い） |

!!! info "APGによる科の変更"
    旧分類：ミズキ科（Cornaceae）→ APG IVではガリア科（Garryaceae）、ガリア目に。
    これは**試験で最も出やすいAPG科変更の一つ**。

!!! warning "試験頻出ポイント"
    - 旧ミズキ科 → 現ガリア科のAPG変更を必ず覚える
    - 光沢のある常緑大型葉 + 赤い果実 → アオキ
    - 耐陰性が高く、林床の暗い場所にも生育
