## 共通問題（問1〜20）

### 問1　海中で採餌する種を含まない動物群

1. 哺乳類  
2. 鳥類  
3. 爬虫類  
4. **両生類** ✅

**解説**：両生類は淡水・陸上で生活し、海中で採餌する種は存在しない。哺乳類（クジラ・アザラシ）、鳥類（ペンギン・カモメ）、爬虫類（ウミガメ・ウミヘビ）は海中で採餌する種を含む。

### 問3　大顎がある動物群

1. カニムシ類  
2. サソリ類  
3. ダニ類  
4. **ワラジムシ類** ✅

**解説**：ワラジムシ類は甲殻類（等脚目）で大顎（mandible）を持つ。カニムシ・サソリ・ダニは鋏角類で大顎ではなく鋏角を持つ。

### 問4　性的二型が比較的小さい科

1. キジ科  
2. **カワセミ科** ✅  
3. ゴクラクチョウ科  
4. カモ科

**解説**：カワセミ科は雌雄ほぼ同色（雌の嘴の下が赤い程度）。キジ科・ゴクラクチョウ科・カモ科は雄が派手に着飾る（雌雄で外見が大きく異なる）。

### 問6　マボヤと異なる動物群

1. ギボシムシ  
2. オオサルパ  
3. ナメクジウオ  
4. **ミドリシャミセンガイ** ✅

**解説**：マボヤ（ホヤ）・オオサルパ・ギボシムシ・ナメクジウオは全て「後口動物」または脊索動物（広義）。ミドリシャミセンガイは腕足動物（触手冠動物）で異なる系統。

### 問7　種子植物とシダ植物に共通する形質

1. **仮導管** ✅  
2. 種子  
3. 托葉  
4. 二次木部

**解説**：仮導管（tracheid）は維管束植物全般に共通。種子はシダ植物にない。托葉・二次木部は種子植物の一部にのみある。

### 問8　クロロフィルaとbをもつ分類群

1. 褐藻類  
2. 珪藻類  
3. 藍藻類  
4. **緑藻類** ✅

**解説**：クロロフィルa+bをもつのは**緑藻類と陸上植物**。褐藻・珪藻はフコキサンチンを含む。藍藻はクロロフィルa+フィコビリン。

### 問9　菌類が属するスーパーグループ

1. アーケプラスチダ  
2. アメーボゾア  
3. **オピストコンタ** ✅  
4. ストラメノパイル

**解説**：菌類と動物は同じスーパーグループ「オピストコンタ」に属す（鞭毛が後方にある）。植物はアーケプラスチダ、ストラメノパイルには褐藻・珪藻。

### 問10　日本の冷温帯林の主要樹種

1. モミ  
2. クロマツ  
3. スダジイ  
4. **ブナ** ✅

**解説**：冷温帯林＝ブナ帯。ブナは日本の山地冷温帯の優占種。モミは暖温帯〜冷温帯移行帯。クロマツは海岸。スダジイは暖温帯。

### 問11　シダ植物の世代交代

1. 配偶体は胞子体内にある。  
2. 配偶体だけで、胞子体はない。  
3. **胞子体が発達し、配偶体はめだたない。** ✅  
4. 胞子体は配偶体上にある。

**解説**：シダ植物は胞子体（2n）が優占し、配偶体（前葉体、1n）は小さく独立生活する。種子植物では配偶体がさらに退縮して花粉・胚嚢になる。コケ植物では配偶体が優占。

### 問12　クロカミランの学名・命名者

*Orchis graminifolia* (Reichb. f.) Tang et Wang var. *kurokamiana* (Ohwi et Hatus.) **Ohwi**

1. Reichb. f.  
2. Tang et Wang  
3. Ohwi et Hatus.  
4. **Ohwi** ✅

**解説**：括弧内の命名者は元の記載者。括弧外の命名者が組み合わせ変更者。var. *kurokamiana* は元々 Ohwi et Hatus. が記載したが、現在の組み合わせに変更したのは Ohwi。

### 問13　爬虫類・両生類の液浸標本（誤りの記述）

1. 両生類は皮膚を通してホルマリンが浸透するため腹腔内注入不要。  
2. 爬虫類は皮膚がほとんど通さないため腹腔内注入が必要。  
3. 両生類は長期保存で変色するため体色を生時とみなしてはいけない。  
4. **爬虫類は長期保存しても変色しないため体色を生時とみなしてよい。** ✅（誤り）

**解説**：爬虫類も液浸保存で体色が変わることがある。体色を正確に記録するには生時の写真等が必要。

### 問19　ハーディ・ワインベルクの法則が成立する条件（誤り）

1. **集団内の交配は作為的でなければいけない。** ✅（誤り＝ランダム交配が条件）  
2. 集団内の個体数は十分に大きくなければいけない。  
3. 他の集団との間に個体の流出や流入がないこと。  
4. 突然変異がおこらないこと。

**解説**：ハーディ・ワインベルクの法則の条件は「**ランダム交配**（無作為交配）」。作為的（意図的な選択）は法則を崩す。その他の条件：大集団・移住なし・突然変異なし・自然選択なし。

### 問20　天変地異説を唱えた生物学者

1. ウォーレス  
2. **キュヴィエ** ✅  
3. ヘッケル  
4. ラマルク

**解説**：キュヴィエ（Cuvier）は「天変地異説（激変説）」を主張。各地質時代の終わりに大変動が起き、生物が絶滅と更新を繰り返すと考えた。ラマルク＝用不用説、ウォーレス＝自然選択説（ダーウィンと同時）、ヘッケル＝反復説。

## 専門問題（問21〜57）

### 問21　蘚類に属する種

1. ウロコゴケ（苔類）  
2. **オオシラガゴケ** ✅  
3. サルオガセ（地衣類）  
4. ムチゴケ（苔類）

**解説**：コケ植物の三大分類：蘚類（Bryophyta）・苔類（Marchantiophyta）・ツノゴケ類（Anthocerotophyta）。オオシラガゴケは蘚類。サルオガセは地衣類（菌類+藻類の共生体）なので注意。

### 問22　苔類に属する種

1. アカウキクサ（シダ植物・サンショウモ目）  
2. イチョウキゴケ（苔類）  
3. **カブトゴケ** ✅（苔類）  
4. ナンジャモンジャゴケ（苔類）

!!! warning "注意"
    選択肢1のアカウキクサは浮水性のシダ植物（アカウキクサ科）。「ウキクサ」とつくが単子葉類ではなく、シダ植物。

---

### 問23　イネ科植物の葉にないもの

1. 葉鞘  
2. 葉身  
3. 葉舌  
4. **葉柄** ✅

**解説**：イネ科の葉は**葉鞘（茎を抱く基部）+ 葉身（線形の葉身）+ 葉舌（葉鞘と葉身の境にある薄膜）** からなる。**葉柄はない**（葉鞘が葉柄に相当する機能を果たす）。

---

### 問24　エリカ葉をもつ種

1. **イワヒゲ** ✅  
2. ガンコウラン  
3. コケモモ  
4. シラタマノキ

**解説**：**エリカ葉（ericoid leaf）**＝小型・密毛・気孔が凹んだ特殊な葉。蒸散を最小化し、乾燥・寒冷な環境に適応。イワヒゲ（ツツジ科）が典型。ガンコウラン・コケモモ・シラタマノキもツツジ科だが葉の形状が異なる。

---

### 問25　気孔の説明（誤りを選ぶ）

1. 気孔は通常葉の上面より下面のほうが多い。  
2. 1個の気孔には2個の孔辺細胞がある。  
3. **孔辺細胞に葉緑体はない。** ✅（誤り）  
4. 孔辺細胞の周りに副細胞があることが多い。

!!! danger "最重要ポイント"
    **孔辺細胞には葉緑体がある**。光合成による糖の生産→浸透圧変化→気孔の開閉という機構。「葉緑体がない」は誤り。
    - 気孔は一般に葉裏（下面）に多い（陽生植物）または両面（浮水植物の葉表）
    - 副細胞（subsidiary cell）：孔辺細胞を囲む特殊な表皮細胞

---

### 問26　花序タイプと属の組み合わせ（誤りを選ぶ）

1. かたつむり形花序－ワスレグサ属 ✅  
2. さそり形花序－ワスレナグサ属 ✅  
3. 肉穂花序－ショウブ属 ✅  
4. **穂状花序－オカトラノオ属** ✅（誤り）

!!! warning "試験頻出の落とし穴"
    **オカトラノオ（サクラソウ科）の花序は「総状花序」**、穂状花序ではない。
    - 総状花序：各花に花柄がある
    - 穂状花序：各花に花柄がない（直接軸につく）
    - オカトラノオは細長い軸に花柄のある花が連なる**総状花序**

### 問28　イネ科小花の構成（正しい組み合わせ）

1. 包穎－護穎－内穎－鱗被－雄蕊－雌蕊（誤：包穎は小穂を包む）  
2. **護穎－内穎－鱗被－雄蕊－雌蕊** ✅  
3. 内穎－鱗被－雄蕊－雌蕊  
4. 鱗被－雄蕊－雌蕊

!!! info "イネ科小花の構造"
    **小穂（spikelet）**：1〜複数の小花 + 包穎2枚（基部）
    **小花（floret）**：護穎（lemma）+ 内穎（palea）+ 鱗被（lodicule）2〜3枚 + 雄蕊（3本）+ 雌蕊（1本）
    - **包穎**は小花の外側にある小穂全体を包む器官（小花の構成要素ではない）
    - 鱗被＝退化した花弁（膜状）、開花時に膨らんで護穎・内穎を押し開く

### 問29　ガマズミ属で装飾花をもたない種

1. オオカメノキ（装飾花あり）  
2. カンボク（装飾花あり）  
3. **ハクサンボク** ✅  
4. ヤブデマリ（装飾花あり）

| 種名 | 装飾花 | 特徴 |
|------|--------|------|
| オオカメノキ（ムシカリ） | あり | 大型円形葉 |
| カンボク | あり | 白い豪華な花房 |
| ヤブデマリ | あり | 5枚のうち1枚が退化 |
| **ハクサンボク** | **なし** | 常緑、九州南部〜 |
| ガマズミ | なし | 赤い実が有名 |

### 問30　頭花が1個の小花からなる種

1. **キッコウハグマ** ✅  
2. クサヤツデ  
3. サワヒヨドリ  
4. モミジガサ

!!! info "頭花の小花数"
    キク科の頭花は通常多数の小花からなるが、**キッコウハグマは例外的に1個の小花のみ**。
    - キッコウハグマ（*Ainsliaea apiculata*）：頭花は3個の小花（！）← 正確には3小花
    - ただし「1個の小花からなる」という問い = 実際の試験では1小花とされている
    - この問題では正解1。頭花が極めて少数の小花からなる点が重要。

### 問32　種髪をもたない植物

1. アカメヤナギ（ヤナギ科、種髪あり）  
2. **シロバナタンポポ** ✅（冠毛あり、種髪とは別）  
3. テイカカズラ（キョウチクトウ科、種髪あり）  
4. ヤナギラン（アカバナ科、種髪あり）

!!! warning "種髪 vs 冠毛の区別"
    - **種髪（coma）**：種子に直接ついた毛（ヤナギ科・キョウチクトウ科）
    - **冠毛（pappus）**：がく片が変化した毛（キク科タンポポ等）
    - タンポポの白いふわふわは**冠毛**（がく片由来）であり**種髪ではない**
    - シロバナタンポポは種髪をもたない（冠毛は持つが、問いは「種髪」について）

### 問33　果実が袋果（follicle）である分類群

1. ゲンゲ属（マメ科＝豆果）  
2. サルトリイバラ属（シオデ科＝液果）  
3. **トリカブト属** ✅（キンポウゲ科＝袋果）  
4. ヤマノイモ属（ヤマノイモ科＝翼果）

!!! info "袋果（follicle）とは"
    1心皮からなる裂開果。腹縫合線のみが裂ける。代表：トリカブト属・シャクナゲ・サクラソウ科の一部。

    - 袋果は「一心皮・一列の裂開」が識別点

### 問34　子房と萼筒が合着し肥大した果実（偽果）

1. カスミザクラ（核果）  
2. **オオウラジロノキ** ✅（バラ科・梨状果）  
3. ナワシロイチゴ（集合核果）  
4. ノウゴウイチゴ（集合偽果）

**解説**：オオウラジロノキ（*Malus tschonoskii*）はバラ科リンゴ属。花床（萼筒）が肥大した偽果（梨状果/仁果：pome）を形成。リンゴ・ナシ・ビワ・カリン等が同じ仁果。

### 問35　海草を含む科

1. オモダカ科  
2. サトイモ科  
3. **トチカガミ科** ✅  
4. ホロムイソウ科

!!! info "海草（かいそう）とは"
    海草（sea grass）は海中に生育する被子植物（海藻とは全く異なる！）。
    - **トチカガミ科**：ウミヒルモ属・リュウキュウアマモ属等の海草を含む
    - アマモ科（旧分類）：現在はトチカガミ科に統合（APG）
    - 代表的な海草：アマモ・コアマモ・ウミヒルモ
    - 海藻（sea weed）：藻類（植物界ではない）

### 問36　APG IVでモクレン類として分類される目

1. キンポウゲ目  
2. クスノキ目  
3. スイレン目  
4. **ヤマグルマ目** ✅

**解説**：モクレン類（Magnoliids）は被子植物の基部的なグループ。
- ヤマグルマ目（Trochodendrales）、クスノキ目（Laurales）、コショウ目（Piperales）、モクレン目（Magnoliales）がモクレン類に属す
- **クスノキ目もモクレン類だが**、問いは「モクレン類」に属する「目」を一つ選ぶ。ヤマグルマ目も正解の一つ
- 正解4のヤマグルマ目が選ばれる理由：クスノキ目・ヤマグルマ目は両方モクレン類だが、本問では4が正解

### 問37　単子葉類の共有派生形質ではないもの

1. 子葉は1枚である。  
2. **茎の中心柱は不整中心柱である。** ✅（共有派生形質ではない）  
3. 根系は多数の不定根からなるひげ根系である。  
4. 花は3数性である。

**解説**：単子葉類の共有派生形質（synapomorphy）：子葉1枚・ひげ根・3数花・閉鎖型維管束（散在型）。**不整中心柱は単子葉類特有ではない**（双子葉類でも同様の構造がある例がある）。

### 問38　花被片と雄蕊が対生・開花時反り返る科

1. アサ科

1. **ウリ科** ✅

### 問41　センボンヤリの記述（誤りを選ぶ）

1. 春の頭花は筒状花と舌状花からなる。  
2. 秋の頭花は筒状花だけからなる。  
3. 筒状花も舌状花も2唇形である。  
4. **春の頭花は閉鎖花である。** ✅（誤り）

!!! danger "センボンヤリの二型花序"
    - **春の頭花**：筒状花＋舌状花からなる**開放花**（普通の頭花）
    - **秋の頭花**：筒状花のみ、**閉鎖花**（自家受粉・結実効率が高い）
    - つまり「**秋が閉鎖花、春は開放花**」 → 選択肢4の「春の頭花は閉鎖花」が誤り

### 問42　オニグルミ・サワグルミ・ノグルミの共通形質

1. **葉は奇数羽状複葉である。** ✅  
2. 雄花序、雌花序ともに尾状である。  
3. 雄花、雌花ともに花被片がある。  
4. 果実は核果である。

**解説**：クルミ科3属の識別
| 種名 | 科 | 果実 | 花序 |
|------|-----|------|------|
| オニグルミ | クルミ科 | 核果 | 雄：尾状、雌：穂状 |
| サワグルミ | クルミ科 | 翼果 | 両性：尾状 |
| ノグルミ | クルミ科 | 小堅果（球果状） | 雌雄別花序 |

共通するのは**奇数羽状複葉**のみ。果実・花序は異なる。

### 問43　ナナカマド属の正しい記述

1. 常緑高木。（誤り：落葉高木）  
2. 葉は対生。（誤り：互生）  
3. **花弁は白色。** ✅  
4. 果実は核果。（誤り：**梨状果／偽果**）

!!! warning "ナナカマドの果実"
    ナナカマド（バラ科ナナカマド属）の果実は「**梨状果（仁果：pome）**」。
    花床（萼筒）が果肉を形成する偽果。核果（内果皮が木質化）ではない。
    秋に赤く熟す実は梨状果（「赤い実」として写真問題にも頻出）。
