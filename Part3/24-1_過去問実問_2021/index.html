    24-1. 過去問実問演習（2021年度）全問・全選択肢¶

    出題形式：共通問題（問1〜20）+ 専門問題（問21〜57）
    正解付き：各問に正解番号・解説を添付
    正解・不正解の選択肢全てが参考書として機能するよう、誤りの選択肢にも解説をつける。

    共通問題（問1〜20）¶

    問1　海中で採餌する種を含まない動物群¶

    1. 哺乳類
    2. 鳥類
    3. 爬虫類
    4. 両生類 ✅

    解説：両生類は淡水・陸上で生活し、海中で採餌する種は存在しない。哺乳類（クジラ・アザラシ）、鳥類（ペンギン・カモメ）、爬虫類（ウミガメ・ウミヘビ）は海中で採餌する種を含む。

    問3　大顎がある動物群¶

    1. カニムシ類
    2. サソリ類
    3. ダニ類
    4. ワラジムシ類 ✅

    解説：ワラジムシ類は甲殻類（等脚目）で大顎（mandible）を持つ。カニムシ・サソリ・ダニは鋏角類で大顎ではなく鋏角を持つ。

    問4　性的二型が比較的小さい科¶

    1. キジ科
    2. カワセミ科 ✅
    3. ゴクラクチョウ科
    4. カモ科

    解説：カワセミ科は雌雄ほぼ同色（雌の嘴の下が赤い程度）。キジ科・ゴクラクチョウ科・カモ科は雄が派手に着飾る（雌雄で外見が大きく異なる）。

    問5　ゴカイが属する動物群¶

    1. 環形動物 ✅
    2. 節足動物
    3. 線形動物
    4. 扁形動物

    問6　マボヤと異なる動物群¶

    1. ギボシムシ
    2. オオサルパ
    3. ナメクジウオ
    4. ミドリシャミセンガイ ✅

    解説：マボヤ（ホヤ）・オオサルパ・ギボシムシ・ナメクジウオは全て「後口動物」または脊索動物（広義）。ミドリシャミセンガイは腕足動物（触手冠動物）で異なる系統。

    問7　種子植物とシダ植物に共通する形質¶

    1. 仮導管 ✅
    2. 種子
    3. 托葉
    4. 二次木部

    解説：仮導管（tracheid）は維管束植物全般に共通。種子はシダ植物にない。托葉・二次木部は種子植物の一部にのみある。

    問8　クロロフィルaとbをもつ分類群¶

    1. 褐藻類
    2. 珪藻類
    3. 藍藻類
    4. 緑藻類 ✅

    解説：クロロフィルa+bをもつのは緑藻類と陸上植物。褐藻・珪藻はフコキサンチンを含む。藍藻はクロロフィルa+フィコビリン。

    問9　菌類が属するスーパーグループ¶

    1. アーケプラスチダ
    2. アメーボゾア
    3. オピストコンタ ✅
    4. ストラメノパイル

    解説：菌類と動物は同じスーパーグループ「オピストコンタ」に属す（鞭毛が後方にある）。植物はアーケプラスチダ、ストラメノパイルには褐藻・珪藻。

    問10　日本の冷温帯林の主要樹種¶

    1. モミ
    2. クロマツ
    3. スダジイ
    4. ブナ ✅

    解説：冷温帯林＝ブナ帯。ブナは日本の山地冷温帯の優占種。モミは暖温帯〜冷温帯移行帯。クロマツは海岸。スダジイは暖温帯。

    問11　シダ植物の世代交代¶

    1. 配偶体は胞子体内にある。
    2. 配偶体だけで、胞子体はない。
    3. 胞子体が発達し、配偶体はめだたない。 ✅
    4. 胞子体は配偶体上にある。

    解説：シダ植物は胞子体（2n）が優占し、配偶体（前葉体、1n）は小さく独立生活する。種子植物では配偶体がさらに退縮して花粉・胚嚢になる。コケ植物では配偶体が優占。

    問12　クロカミランの学名・命名者¶

    Orchis graminifolia (Reichb. f.) Tang et Wang var. kurokamiana (Ohwi et Hatus.) Ohwi

    1. Reichb. f.
    2. Tang et Wang
    3. Ohwi et Hatus.
    4. Ohwi ✅

    解説：括弧内の命名者は元の記載者。括弧外の命名者が組み合わせ変更者。var. kurokamiana は元々 Ohwi et Hatus. が記載したが、現在の組み合わせに変更したのは Ohwi。

    問13　爬虫類・両生類の液浸標本（誤りの記述）¶

    1. 両生類は皮膚を通してホルマリンが浸透するため腹腔内注入不要。
    2. 爬虫類は皮膚がほとんど通さないため腹腔内注入が必要。
    3. 両生類は長期保存で変色するため体色を生時とみなしてはいけない。
    4. 爬虫類は長期保存しても変色しないため体色を生時とみなしてよい。 ✅（誤り）

    解説：爬虫類も液浸保存で体色が変わることがある。体色を正確に記録するには生時の写真等が必要。

    問14　小笠原諸島・母島列島のみ分布する固有種¶

    1. オオバシマムラサキ
    2. オガサワラツツジ
    3. マルハチ ✅
    4. ワダンノキ

    解説：マルハチ（ヘゴ科）は母島列島にのみ分布する日本最大のシダ植物。

    問15　分布の正しい記述¶

    1. 北海道にブナは分布しない。
    2. 屋久島にブナは分布しない。
    3. 北海道にツクツクボウシは分布しない。 ✅
    4. 屋久島にツクツクボウシは分布しない。

    解説：ブナは北海道渡島半島にも分布する。ツクツクボウシは本州〜九州・屋久島に分布し、北海道には分布しない。

    問16　魚類の絶滅危惧IA類が最多の分類群¶

    1. シラウオ類
    2. タナゴ類 ✅
    3. トゲウオ類
    4. ハゼ類

    問17　国の特別天然記念物に指定されていないもの¶

    1. タンチョウ
    2. コウノトリ
    3. ニホンカモシカ
    4. ツシマヤマネコ ✅（特別天然記念物ではなく天然記念物）

    問18　「魚類の時代」の地質時代¶

    1. カンブリア紀
    2. デボン紀 ✅
    3. ジュラ紀
    4. 古第三紀

    解説：デボン紀（約4.2〜3.6億年前）は魚類が爆発的に多様化した時代。カンブリア紀＝無脊椎動物。ジュラ紀＝爬虫類（恐竜）。

    問19　ハーディ・ワインベルクの法則が成立する条件（誤り）¶

    1. 集団内の交配は作為的でなければいけない。 ✅（誤り＝ランダム交配が条件）
    2. 集団内の個体数は十分に大きくなければいけない。
    3. 他の集団との間に個体の流出や流入がないこと。
    4. 突然変異がおこらないこと。

    解説：ハーディ・ワインベルクの法則の条件は「ランダム交配（無作為交配）」。作為的（意図的な選択）は法則を崩す。その他の条件：大集団・移住なし・突然変異なし・自然選択なし。

    問20　天変地異説を唱えた生物学者¶

    1. ウォーレス
    2. キュヴィエ ✅
    3. ヘッケル
    4. ラマルク

    解説：キュヴィエ（Cuvier）は「天変地異説（激変説）」を主張。各地質時代の終わりに大変動が起き、生物が絶滅と更新を繰り返すと考えた。ラマルク＝用不用説、ウォーレス＝自然選択説（ダーウィンと同時）、ヘッケル＝反復説。

    専門問題（問21〜57）¶

    問21　蘚類に属する種¶

    1. ウロコゴケ（苔類）
    2. オオシラガゴケ ✅
    3. サルオガセ（地衣類）
    4. ムチゴケ（苔類）

    解説：コケ植物の三大分類：蘚類（Bryophyta）・苔類（Marchantiophyta）・ツノゴケ類（Anthocerotophyta）。オオシラガゴケは蘚類。サルオガセは地衣類（菌類+藻類の共生体）なので注意。

    問22　苔類に属する種¶

    1. アカウキクサ（シダ植物・サンショウモ目）
    2. イチョウキゴケ（苔類）
    3. カブトゴケ ✅（苔類）
    4. ナンジャモンジャゴケ（苔類）

    注意

    選択肢1のアカウキクサは浮水性のシダ植物（アカウキクサ科）。「ウキクサ」とつくが単子葉類ではなく、シダ植物。

    問23　イネ科植物の葉にないもの¶

    1. 葉鞘
    2. 葉身
    3. 葉舌
    4. 葉柄 ✅

    解説：イネ科の葉は葉鞘（茎を抱く基部）+ 葉身（線形の葉身）+ 葉舌（葉鞘と葉身の境にある薄膜） からなる。葉柄はない（葉鞘が葉柄に相当する機能を果たす）。

    問24　エリカ葉をもつ種¶

    1. イワヒゲ ✅
    2. ガンコウラン
    3. コケモモ
    4. シラタマノキ

    解説：エリカ葉（ericoid leaf）＝小型・密毛・気孔が凹んだ特殊な葉。蒸散を最小化し、乾燥・寒冷な環境に適応。イワヒゲ（ツツジ科）が典型。ガンコウラン・コケモモ・シラタマノキもツツジ科だが葉の形状が異なる。

    問25　気孔の説明（誤りを選ぶ）¶

    1. 気孔は通常葉の上面より下面のほうが多い。
    2. 1個の気孔には2個の孔辺細胞がある。
    3. 孔辺細胞に葉緑体はない。 ✅（誤り）

    最重要ポイント

    • ツガ属（Tsuga）：葉枕なし → 葉が脱落するとほぼ平滑
    • モミ属（Abies）：葉枕あり（葉痕が丸い跡）

    4. 葉枕がある。 ✅（誤り：ツガ属は葉枕がない）

    問51-1　クスノキ科：3行脈をもたない常緑高木¶

    1. シロダモ（3行脈あり）
    2. イヌガシ ✅
    3. ハマビワ（3行脈あり）
    4. シロモジ（落葉）

    クスノキ科の3行脈

    • 3行脈あり：クスノキ・タブノキ・シロダモ・ハマビワ・ヤブニッケイ等
    • 3行脈なし：イヌガシ・カナクギノキ・ヤマコウバシ等

    問51-2　クスノキ科：常緑木本で雄蕊9本¶

    1. タブノキ（完全雄蕊9本+退化雄蕊3本=合計12）
    2. カナクギノキ ✅
    3. ヤマコウバシ
    4. ヤマグルマ（ヤマグルマ科・別科）

    クスノキ科の雄蕊

    クスノキ科の完全雄蕊は9本（3輪×3本）が基本。外側2輪の葯は内向き（内開）、内側1輪の葯は外向き（外開）で腺体を持つ。退化雄蕊3本を加えると12本。
