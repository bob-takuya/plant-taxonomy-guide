    23-1. 過去問：写真同定問題（2021年度）¶

    出題形式：問54（果実写真）、問55（樹木写真）、問56（植物写真10枚同定）、問57（特徴記述）
    対象種数：15種
    写真から学名・科・形態的特徴を即座に答えられるようにする。

    問54：ブドウ科の識別（2021年度）¶

    54-1. ノブドウ Ampelopsis glandulosa var. heterophylla¶

    科・目：ブドウ科（Vitaceae）- ブドウ目

    項目 内容
    生活型 落葉つる性木本
    葉 掌状3〜5裂、互生、葉柄長い
    花 小型・黄緑色、集散花序
    果実 光沢ある青〜紫〜白色の液果（虫こぶ由来で色が変わる）
    分布 北海道〜九州、林縁・荒地

    試験頻出ポイント

    • ヤブカラシとの違い：ノブドウ＝果実を食べない（有毒・無味）、ヤブカラシ＝葉がより単純
    • 果実の虹色・青緑・紫の混在が最大の識別点
    • 巻きひげ：葉と対生

    54-2. ヤブカラシ Cayratia japonica¶

    科・目：ブドウ科（Vitaceae）- ブドウ目

    項目 内容
    生活型 多年生つる性草本（根茎あり）
    葉 鳥足状複葉（5小葉）、互生
    花 淡紅色〜白、4弁、花盤が橙赤色で目立つ
    果実 黒紫色の液果、小粒
    分布 本州〜沖縄、人里・荒地（旺盛に繁茂）

    試験頻出ポイント

    • ノブドウ（単葉3〜5裂）とヤブカラシ（鳥足状5小葉の複葉）を混同しない
    • 花盤（オレンジ色の環状構造）が特徴的 → 写真で見えたら即ヤブカラシ
    • 「藪を枯らす」ほどの繁殖力

    問55：ヤナギ科の識別（2021年度）¶

    55-1. ヤナギ類 Salix spp.¶

    科・目：ヤナギ科（Salicaceae）- キントラノオ目（APG IV）

    項目 内容
    生活型 落葉高木〜低木
    葉 単葉・互生・托葉あり、多くは線形〜披針形
    花序 尾状花序（catkin）、風媒花、花被なし
    果実 蒴果、種子に綿毛あり（風散布）
    花期 早春（葉より先に開花する種が多い）

    APGによる科の変更

    旧分類ではヤナギ科はヤナギ目（Salicales）だったが、APG IVではキントラノオ目（Malpighiales）に含まれる。試験では「キントラノオ目」が正解。

    試験頻出ポイント

    • 雌雄異株
    • 葉の形と托葉の有無が属・種同定の鍵
    • イイギリとの比較：ヤナギ＝葉が細い・尾状花序、イイギリ＝葉が広い・総状花序

    55-2. イイギリ Idesia polycarpa¶

    科・目：ヤナギ科（Salicaceae）- キントラノオ目（APG IV）

    項目 内容
    生活型 落葉高木（樹高15〜20m）
    葉 大型心形、長柄、葉柄に腺体2個あり
    花 黄緑色、花被片5、総状花序、芳香あり
    果実 赤色の液果が垂れ下がるブドウ状の集合（秋〜冬に美しい）
    分布 本州〜九州〜沖縄、山地

    試験頻出ポイント

    • 旧分類：イイギリ科（Flacourtiaceae）→ APGでヤナギ科に編入（試験頻出！）
    • 冬の赤い果実の写真 → イイギリと即答できるようにする
    • 葉柄の腺体（2個）が形態的識別点

    問56：写真同定 10問（2021年度）¶

    56-1. ハマギク Nipponanthemum nipponicum¶

    科・目：キク科（Asteraceae）- キク目

    項目 内容
    生活型 多年生草本〜半木本
    葉 倒披針形〜ヘラ形、厚くて光沢あり、互生
    花 頭花、白色の舌状花＋黄色の筒状花、直径4〜6cm
    花期 9〜11月
    分布 日本固有種、太平洋側の海岸崖地・砂地（茨城県〜静岡県）

    一属一種（日本固有属）

    Nipponanthemum は単型属（1種のみ）。日本海岸の固有種として試験頻出。

    試験頻出ポイント

    • ノジギクやシオギクとの違い：ハマギク＝葉に鋸歯なし・海岸崖地・日本固有種
    • 白い大型頭花 + 厚くて光沢のある葉 → ハマギク

    56-2. ハマゼリ Cnidium japonicum¶

    科・目：セリ科（Apiaceae）- セリ目

    項目 内容
    生活型 多年生草本
    葉 1〜2回羽状複葉
    花 白色、複散形花序
    生育環境 海岸の砂地・砂丘（ハマで始まる名前通り）
    分布 北海道〜九州

    試験頻出ポイント

    • セリ科の識別：複散形花序（umbel）+ 分裂葉・複葉
    • 「ハマ（浜）」系植物の棲み分け：砂浜ならハマゼリ、崖ならハマギク

    56-3. イワブクロ / タルマイソウ Pennellianthus frutescens（= Penstemon frutescens）¶

    科・目：オオバコ科（Plantaginaceae）- シソ目（APG IV）

    項目 内容
    生活型 半低木（木本性の基部）
    葉 対生、倒卵形〜卵形
    花 淡紫〜白色の筒形花冠、5裂、仮雄蕊あり
    花期 7〜8月
    分布 北海道・大雪山系固有種（高山帯〜亜高山帯）

    APGによる科の変更

    旧分類：ゴマノハグサ科（Scrophulariaceae）→ APG IVではオオバコ科（Plantaginaceae）に統合。この科変更は試験頻出！

    試験頻出ポイント

    • 大雪山系の固有種 → 北海道の高山固有種として記憶
    • 仮雄蕊（4本の稔性雄蕊 + 1本の不稔性雄蕊）が特徴

    56-4. オドリコソウ Lamium album var. barbatum¶

    科・目：シソ科（Lamiaceae）- シソ目

    項目 内容
    生活型 多年生草本
    葉 対生、卵形〜三角卵形、粗い鋸歯
    茎 四角形（シソ科の特徴）
    花 淡紅〜白色の唇形花、輪散花序、上唇が兜型
    花期 4〜6月
    分布 北海道〜九州、人里・林縁

    試験頻出ポイント

    • 花の形が踊り子の衣装に見える → 「踊り子草」
    • ヒメオドリコソウ（帰化植物・葉が赤紫）との混同注意
    • シソ科の基本特徴：四角茎・対生葉・唇形花

    56-5. ミゾソバ Persicaria thunbergii¶

    科・目：タデ科（Polygonaceae）- ナデシコ目

    項目 内容
    生活型 一年生草本
    葉 戟形（ほこ形）：基部が矢じり状に張り出す、互生
    托葉鞘 筒状（タデ科の特徴）、縁に長毛あり
    花 淡紅〜白色、頭状に密集
    花期 8〜10月
    分布 全国、水田・溝・湿地

    試験頻出ポイント

    • タデ科識別：托葉鞘（ocrea）の存在が最重要
    • 葉の形：戟形（矢じり形）が最大の特徴 → 溝沿いに群生
    • 「溝蕎麦」= 溝（水辺）に生えるソバ似の草

    56-6. キバナアキギリ / コトジソウ Salvia nipponica¶

    科・目：シソ科（Lamiaceae）- シソ目

    項目 内容
    生活型 多年生草本
    葉 対生、長三角形〜矛状形、基部に1〜2対の小裂片
    花 黄色の唇形花（アキギリ類で黄色は珍しい）
    花期 8〜10月
    分布 本州〜九州、林縁・山地の日陰

    試験頻出ポイント

    • アキギリ属（Salvia）= セージ属、雄蕊が特殊構造（槓桿状）
    • 黄色の花 → アキギリ類の中でも即答できる
    • コトジソウは同種の別名（異名）

    56-7. ハッカ Mentha canadensis（= M. arvensis var. piperascens）¶

    科・目：シソ科（Lamiaceae）- シソ目

    項目 内容
    生活型 多年生草本
    葉 対生、卵形〜長楕円形、鋸歯あり
    特徴 メントール（薄荷脳）含有、揉むと強い香り
    花 淡紫〜白色、輪散花序
    花期 8〜10月
    分布 全国（北海道で大規模栽培された歴史あり）

    試験頻出ポイント

    • ハッカはミント類の原種（日本在来種）
    • 北海道北見地方で明治〜昭和にかけて世界的なハッカ産地だった歴史
    • 揉むと香るシソ科 → 香りが強いものはハッカ・シソ・バジルなど

    56-8. イワウメ Diapensia lapponica var. obovata¶

    科・目：イワウメ科（Diapensiaceae）- ツツジ目

    項目 内容
    生活型 常緑低木（マット状に地表を覆う）
    葉 小型・倒卵形〜ヘラ形、密に互生、常緑・革質
    花 白色・5弁・単生、短い花柄
    花期 6〜7月
    分布 北海道〜本州中部、高山帯〜亜高山帯の岩礫地・雪田周辺
    環北極 環北極分布種

    一科一属一種（日本では）

    イワウメ科は日本では本種1種のみ。APGではツツジ目に含まれる。

    試験頻出ポイント

    • 2021年度出題。白い5弁花 + マット状の常緑小型低木 → 高山帯
    • ツツジ目だが花弁が合着しない（離弁花）点がツツジ科と異なる

    56-9. ホテイアオイ Eichhornia crassipes¶

    科・目：ミズアオイ科（Pontederiaceae）- ツユクサ目

    項目 内容
    生活型 浮水性多年生草本
    葉柄 膨らんだ浮き袋状の葉柄（布袋様）が最大の特徴
    花 淡紫色、花被6枚、上方の1枚に黄斑
    原産 南アメリカ（ブラジル）
    法規 特定外来生物（外来生物法）

    特定外来生物！

    ホテイアオイは特定外来生物。水面を覆い尽くし、在来の水生植物・水生動物を脅かす。 「観賞用として持ち込まれ → 野外逸出 → 爆発的繁殖」のパターン。

    試験頻出ポイント

    • 膨らんだ葉柄 → 即ホテイアオイ
    • 特定外来生物の扱い（栽培・販売・移動の禁止）も試験範囲

    56-10. オオハンゴンソウ Rudbeckia laciniata¶

    科・目：キク科（Asteraceae）- キク目

    項目 内容
    生活型 多年生草本（高さ1〜3m）
    葉 深く羽状分裂、互生
    花 頭花、黄色の舌状花（折れ下がる）+ 緑色〜黄緑色の筒状花（半球形に盛り上がる）
    原産 北アメリカ
    法規 特定外来生物

    特定外来生物！

    オオハンゴンソウは特定外来生物。河川敷・湿地・山地に旺盛に侵入。 在来のキク科植物（ヒマワリ属近縁）と競合。

    試験頻出ポイント

    • 筒状花が緑色〜黄緑色・半球形に盛り上がる → 他のキク科と異なる
    • ヒマワリ・キク属との区別：花托が大きく突出

    問57-5. オオカメノキ / ムシカリ Viburnum furcatum¶

    科・目：ガマズミ科（Adoxaceae）- マツムシソウ目（APG IV）

    項目 内容
    生活型 落葉低木〜小高木
    葉 円形〜広卵形、長さ10〜20cm、葉脈が目立つ、葉柄に翼状突起
    花 白色、散房状集散花序、周辺に大型の装飾花（中性花）
    果実 赤色→黒色に熟す核果
    分布 北海道〜九州、山地〜亜高山帯

    APGによる科の変更

    旧分類：スイカズラ科（Caprifoliaceae）→ APG IVではガマズミ科（Adoxaceae）に。 この科変更は頻出。

    試験頻出ポイント

    • 「ムシカリ（虫食われ）」= 葉が虫に食われやすいことから
    • 装飾花（周辺の大型不稔花）+ 大きな円形葉 → 識別容易
    • ガクアジサイに似るが全くの別科
